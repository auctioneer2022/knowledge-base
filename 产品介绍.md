# 众益会计专业知识库 — 产品介绍

## 产品概述

众益会计专业知识库是四川众益会计师事务所自主研发的专业法规与准则知识管理平台，覆盖审计、税务、咨询三大核心业务线。产品以结构化 Markdown 知识库为底座，通过 MkDocs Material 主题构建现代化 Web 站点，为事务所执业人员提供权威、高效、一站式的法规查阅服务，同时为 AI 智能化审计工具提供标准化的底层知识供给。

当前知识库已收录法律法规、会计准则、审计准则、税法法规、行业知识五大核心领域共 182 项知识条目，整体完成率 92.4%。

## 核心技术与功能

### 知识内容体系

产品围绕会计师事务所执业需求，构建了九大知识模块：

**法律法规模块（16部，100%）**

涵盖会计法律（会计法、注册会计师法、企业财务会计报告条例、会计基础工作规范、会计档案管理办法）、公司商事法律（公司法、破产法、合伙企业法、外商投资法）、合同与民事法律（民法典合同编/物权编、合同编通则司法解释、担保制度司法解释）、劳动法律（劳动法、劳动合同法、社会保险法）、其他相关法律（反洗钱法、数据安全法、个人信息保护法）。

**会计准则模块（91/104项，87.5%）**

收录企业会计准则基本准则 1 项、具体准则 CAS 1~42 共 41 项、应用指南 40 项、准则解释第 1~20 号共 20 项、小企业会计准则 1 项、制度与规范 8 项（含企业内部控制基本规范及配套指引、政府会计准则、民间非营利组织会计制度等）。

**审计准则模块（47/50项，94.0%）**

收录中国注册会计师审计准则 CSA 1101~1631 系列 37 项（含验资、商业银行审计等专项准则）、审阅准则第 2101 号（财务报表审阅）、其他鉴证业务准则第 3101 号和第 3111 号（预测性财务信息审核）、职业道德守则第 1~3 号和第 5 号及术语表、质量管理准则第 5102 号（项目质量复核）、鉴证业务基本准则，以及审计准则应用指南索引（36 项）。

**税法法规模块（7部，100%）**

按税种分类收录增值税暂行条例及实施细则、企业所得税法及实施条例、个人所得税法、印花税法、税收征收管理法。

**行业知识模块（17项，100%）**

覆盖制造业、房地产、高新技术、批发零售、建筑、餐饮住宿、农业、行政事业单位、民间非营利组织等 9 个行业领域共 17 个主题文件，涵盖各行业特殊会计处理、审计要点、实务操作及常见问题。

### 技术架构

- **静态站点生成**：基于 MkDocs Material 主题，支持中文本地化、深浅色模式切换、响应式布局
- **导航体系**：顶部标签页导航（sticky）、章节侧边栏、目录展开、返回顶部按钮
- **全文搜索**：内置中文分词搜索引擎，支持搜索建议与关键词高亮
- **Markdown 增强**：支持提示框（admonition）、可折叠内容、代码块高亮、标签页切换、表格、永久目录链接
- **CI/CD**：通过 GitHub Actions 实现推送至 main 分支即自动构建部署至 GitHub Pages

### 知识编码体系

每条知识条目采用 `领域编码-模块编号-条目编号` 的统一编码规则，覆盖 LAW（法律法规）、CAS（会计准则）、CSA（审计准则）、TAX（税法法规）、IND（行业知识）、TPL（实务模板）、CASE（案例库）、MTH（工具方法）、TRN（培训资源）九大领域，确保知识条目的唯一标识与可追溯性。

## 创新点

### 四层知识架构设计

产品采用从底层到顶层的四层递进式知识架构：法规原文层（权威、时效、完整的法律与准则原文）→ 结构化知识层（按主题组织的准则解读与法规速查）→ 场景应用层（可直接执行的实务指引与检查清单）→ AI 应用层（智能工具与推理引擎）。当前已完成法规原文层的大量收录工作，为后续层级的知识提炼与 AI 产品化奠定数据基础。

### AI 智能化审计知识底座

知识库定位于 AI 智能化审计工具的底层知识供给层，通过标准化编码、结构化 Markdown 格式和层级化目录组织，使法规与准则数据具备机器可读性，可直接对接大语言模型进行智能问答、准则比对、审计程序推荐等高阶应用。

### 规模化法规收录与质量管控

从 CICPA 官方 PDF 等权威渠道批量提取审计准则原文，经过系统性的编号修复、格式清理（删除页码残留、恢复条文编号空格、清理多余空格）等质量管控流程，确保法规文本的准确性与可读性。

## 应用场景

审计、税务、咨询业务人员在项目执行过程中，可快速定位并查阅相关法律法规、会计准则、审计准则及税法法规的完整原文，替代传统纸质查阅或分散的网站搜索。

## 行业痛点解决

### 法规分散、检索低效

会计师事务所执业涉及的法律法规、会计准则、审计准则、税法法规分散于财政部、中注协、人大网等多个权威网站，检索效率低且版本管理困难。本产品将核心法规集中收录至统一平台，通过中文全文搜索与层级化导航，实现一次检索、精准定位。

### 准则体系复杂、关联性难以把握

企业会计准则体系包含基本准则、42 项具体准则、20 项准则解释、40 项应用指南及多项制度规范，条文间存在大量交叉引用。本产品通过统一的编码体系、完整的索引页和层级化目录组织，清晰呈现准则间的从属与关联关系。

### 知识更新滞后

法规与准则持续修订，传统文档管理方式难以保证时效性。本产品基于 Git 版本控制，配合结构化的更新记录机制（首页记录每次更新的具体内容），确保知识库的时效性与可追溯性。

## 部署方案

### 在线部署（当前方案）

- **平台**：GitHub Pages
- **触发方式**：推送至 `main` 分支自动触发 GitHub Actions 工作流
- **CI 环境**：ubuntu-latest + Python 3.12
- **构建工具**：mkdocs-material
- **访问地址**：https://auctioneer2022.github.io/knowledge-base/
- **部署流程**：代码检出 → Python 环境配置 → 安装 mkdocs-material → mkdocs build → 上传构建制品 → 部署至 GitHub Pages
