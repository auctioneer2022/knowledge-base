# 数据分析

本模块收录审计数据分析方法与异常检测技术，为注册会计师在审计过程中运用数据分析工具提供实务指引。内容依据中注协《注册会计师审计数据分析指引》及相关执业准则编制。

## 子模块

- [审计数据分析模型](MTH-04-001-审计数据模型.md) — 数据采集、清洗、转换及七大核心分析模型（账龄分析、毛利率分析、费用分析、往来款分析、收入确认分析、存货分析、固定资产分析）
- [审计异常检测方法](MTH-04-002-异常检测.md) — 阈值法、趋势分析法、聚类分析法、本福特定律、关联规则分析、机器学习方法及异常跟进程序

## 适用范围

本模块适用于财务报表审计中的数据分析环节，涵盖从数据采集到分析结果呈现的全流程。审计人员可根据被审计单位的信息化水平和数据条件，选择适当的数据分析方法和工具。
