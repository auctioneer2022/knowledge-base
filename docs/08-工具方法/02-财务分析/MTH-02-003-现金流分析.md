# 现金流分析

现金流分析是以现金流量表为核心，对企业在一定期间内现金和现金等价物的流入和流出进行系统分析的方法。依据《企业会计准则第31号——现金流量表》（CAS 31），现金流量分析能够揭示企业利润质量、偿债能力和财务健康状况的真实状况，是财务分析和审计风险评估中不可或缺的重要工具。

## 一、现金流量表的基本结构

### 1.1 三大活动分类

根据CAS 31准则，现金流量表将企业的现金流量分为三大类：

**（1）经营活动现金流**

经营活动是指企业投资活动和筹资活动以外的所有交易和事项。经营活动现金流主要包括：

- 销售商品、提供劳务收到的现金
- 收到的税费返还
- 收到其他与经营活动有关的现金
- 购买商品、接受劳务支付的现金
- 支付给职工以及为职工支付的现金
- 支付的各项税费
- 支付其他与经营活动有关的现金

**（2）投资活动现金流**

投资活动是指企业长期资产的购建和不包括在现金等价物范围内的投资及其处置活动。投资活动现金流主要包括：

- 收回投资收到的现金
- 取得投资收益收到的现金
- 处置固定资产、无形资产和其他长期资产收回的现金
- 购建固定资产、无形资产和其他长期资产支付的现金
- 投资支付的现金

**（3）筹资活动现金流**

筹资活动是指导致企业资本及债务规模和构成发生变化的活动。筹资活动现金流主要包括：

- 吸收投资收到的现金
- 取得借款收到的现金
- 偿还债务支付的现金
- 分配股利、利润或偿付利息支付的现金

### 1.2 编制方法

- **经营活动现金流**：可采用直接法或间接法编制。中国企业会计准则要求采用直接法编制经营活动现金流，同时在附注中按间接法将净利润调节为经营活动现金流。
- **投资活动现金流**：采用直接法编制。
- **筹资活动现金流**：采用直接法编制。

### 1.3 间接法调节

间接法以净利润为起点，通过调整非现金项目和非经营项目，将其调节为经营活动现金流净额：

```
经营活动现金流净额 = 净利润
  + 资产减值准备
  + 固定资产折旧
  + 无形资产摊销
  + 长期待摊费用摊销
  + 处置固定资产、无形资产和其他长期资产的损失（减：收益）
  + 固定资产报废损失
  + 公允价值变动损失（减：收益）
  + 财务费用（属于投资或筹资活动的部分）
  + 投资损失（减：收益）
  + 递延所得税资产减少（减：增加）
  + 递延所得税负债增加（减：减少）
  + 存货的减少（减：增加）
  + 经营性应收项目的减少（减：增加）
  + 经营性应付项目的增加（减：减少）
```

## 二、经营活动现金流分析

### 2.1 经营活动现金流与净利润的对比

经营活动现金流净额与净利润的对比是评估盈余质量的核心方法：

**盈余现金比率：**

```
盈余现金比率 = 经营活动现金流净额 / 净利润
```

**评价标准：**

| 比率范围 | 评价等级 | 含义 |
|----------|----------|------|
| ≥ 1.0 | 优秀 | 利润有充足的现金流支撑 |
| 0.7 ~ 1.0 | 良好 | 利润基本有现金流保障 |
| 0.3 ~ 0.7 | 一般 | 部分利润缺乏现金流支撑 |
| < 0.3 | 较差 | 利润质量堪忧，存在较大风险 |
| < 0 且净利润为正 | 异常 | 利润完全缺乏现金流支撑，高度警惕 |

**分析要点：**
- 盈余现金比率长期大于1的企业，通常具有较强的盈利质量和财务健康度。
- 该比率低于1不一定意味着利润造假，可能由以下合理原因导致：应收账款大幅增加、存货大量增加、预付款项增加等。
- 但如果该比率持续低于1且无合理商业解释，则应高度关注利润质量问题。

### 2.2 销售收现比率

**计算公式：**

```
销售收现比率 = 销售商品、提供劳务收到的现金 / 营业收入
```

**经济含义：** 销售收现比率反映企业营业收入的现金回收程度。该比率越接近或超过1（考虑增值税因素，理论上应约为1.13或1.06），说明收入的现金回收情况越好。

**分析要点：**
- 销售收现比率持续低于1，说明企业存在大量赊销，收入质量较低。
- 该比率大于1可能是预收账款增加或以前期间应收账款在本期收回。
- 审计中应将销售收现比率与应收账款变动趋势结合分析。

### 2.3 经营活动现金流的质量评估标准

高质量的经营活动现金流应具备以下特征：

1. **持续性**：经营活动现金流应持续为正，且与净利润保持合理比例关系。
2. **充足性**：经营活动现金流应能覆盖日常经营支出、偿债支出和投资支出。
3. **稳定性**：经营活动现金流不应出现剧烈波动，季节性波动应在合理范围内。
4. **匹配性**：经营活动现金流应与营业收入、净利润的变动趋势基本一致。
5. **成长性**：经营活动现金流应随企业规模扩大而稳步增长。

## 三、投资活动现金流分析

### 3.1 投资活动的类型与现金流特征

企业的投资活动可分为以下几类：

**（1）扩张性投资**

- 特征：购建固定资产、无形资产支付的现金大幅增加。
- 含义：企业处于扩张期，未来产能将增加。
- 关注点：投资方向是否符合企业战略，投资回报率预期如何。

**（2）维持性投资**

- 特征：购建固定资产支付的现金与折旧额基本相当。
- 含义：企业主要进行设备更新和产能维护。
- 关注点：维持性投资是否足以保持现有竞争力。

**（3）收缩性投资**

- 特征：处置固定资产、无形资产收回的现金增加，投资活动现金流为正。
- 含义：企业可能在收缩业务或处置资产。
- 关注点：是否为企业主动战略调整，还是被动应对财务困难。

**（4）投资性活动**

- 特征：对外投资支付的现金增加或收回投资收到的现金增加。
- 含义：企业进行金融资产配置或股权投资。
- 关注点：投资规模是否合理，投资回报如何。

### 3.2 资本支出与折旧的对比

```
资本支出/折旧比率 = 购建固定资产、无形资产和其他长期资产支付的现金 / 折旧与摊销总额
```

| 比率范围 | 含义 |
|----------|------|
| > 1.5 | 积极扩张，产能大幅增长 |
| 1.0 ~ 1.5 | 适度扩张 |
| 0.7 ~ 1.0 | 维持性投资，产能基本不变 |
| < 0.7 | 投资不足，产能可能萎缩 |

### 3.3 投资回报分析

```
投资现金回报率 = 经营活动现金流净额 / 平均总资产
```

该指标衡量企业每1元资产投资所产生的经营现金流，比ROA更难被会计政策操纵，是评估资产盈利质量的有力工具。

## 四、筹资活动现金流分析

### 4.1 筹资结构分析

筹资活动现金流分析应关注企业的融资来源结构：

**股权融资 vs 债权融资：**

```
股权融资比率 = 吸收投资收到的现金 / 筹资活动现金流入总额
债权融资比率 = 取得借款收到的现金 / 筹资活动现金流入总额
```

**分析要点：**
- 过度依赖债权融资的企业面临较大的偿债压力和财务风险。
- 频繁进行股权融资的企业可能存在经营现金流不足的问题。
- 应关注融资结构与企业生命周期阶段的匹配性。

### 4.2 偿债能力分析

通过筹资活动现金流分析企业的偿债行为：

```
偿债保障比率 = 经营活动现金流净额 / (偿还债务支付的现金 + 利息支出)
```

该比率反映企业以经营现金流偿还到期债务的能力。比率大于1说明企业经营活动产生的现金流足以覆盖偿债支出。

### 4.3 股利支付与现金流匹配

```
股利支付率（现金基础）= 分配股利支付的现金 / 经营活动现金流净额
```

**分析要点：**
- 股利支付应建立在充足的经营现金流基础之上。
- 如果企业在经营现金流不足的情况下仍然大额分红，可能损害企业的财务健康。
- 审计中应关注大股东通过高额分红套现的行为。

## 五、自由现金流计算

### 5.1 经营性自由现金流

```
经营性自由现金流 = 经营活动现金流净额 - 资本支出
其中：资本支出 = 购建固定资产、无形资产和其他长期资产支付的现金
```

**经济含义：** 经营性自由现金流反映企业在维持和扩大生产经营规模后，剩余的可自由支配的现金。这是企业可用于偿还债务、分配股利、回购股票或进行新投资的现金来源。

### 5.2 企业自由现金流（FCFF）

```
FCFF = 息税前利润(EBIT) × (1 - 所得税税率)
       + 折旧与摊销
       - 资本支出
       - 营运资本增加额

或等价地：

FCFF = 经营活动现金流净额
       + 税后利息费用
       - 资本支出
```

**经济含义：** FCFF反映企业在满足经营需要后，可以向所有资本提供者（包括债权人和股东）分配的现金流总额。FCFF是企业估值（DCF模型）中最核心的现金流概念。

### 5.3 股权自由现金流（FCFE）

```
FCFE = FCFF - 税后利息费用 + 净借款增加额
     = 经营活动现金流净额 - 资本支出 + 净借款增加额
```

**经济含义：** FCFE反映企业在满足经营需要和偿债义务后，可以向股东分配的现金流总额。FCFE是股权估值的基础。

### 5.4 自由现金流的分析应用

| 自由现金流状况 | 含义 | 企业阶段特征 |
|----------------|------|--------------|
| FCFE > 0 且 FCFF > 0 | 现金充裕，可同时满足债权人和股东 | 成熟期企业 |
| FCFE > 0 但 FCFF < 0 | 经营现金流不足，靠借债维持分红 | 需关注财务风险 |
| FCFE < 0 且 FCFF > 0 | 经营现金流充足，但偿债压力大 | 高杠杆企业 |
| FCFE < 0 且 FCFF < 0 | 现金流全面紧张 | 初创期或困难期企业 |

## 六、现金流异常识别

### 6.1 利润与现金流长期背离

**预警信号：**
- 净利润持续增长但经营活动现金流持续下降或为负。
- 盈余现金比率连续3年以上低于0.5。
- 营业收入增长但销售收现比率持续下降。

**可能原因：**
- 收入确认政策激进，大量赊销形成应收账款。
- 存货积压，资金被大量占用。
- 关联方资金占用，以应收款项形式流出。
- 存在收入虚增的可能。

**审计应对：**
- 重点审计应收账款的真实性和可回收性。
- 检查收入确认政策是否符合准则要求。
- 关注关联方交易的商业实质。

### 6.2 经营活动现金流持续为负

**预警信号：**
- 连续3年以上经营活动现金流净额为负。
- 经营活动现金流净额与净利润的差额持续扩大。

**可能原因：**
- 企业处于初创期或快速扩张期，经营现金流出大于流入。
- 行业周期性下行，经营环境恶化。
- 企业竞争力下降，经营陷入困境。
- 存在大额关联方资金占用。

**审计应对：**
- 评估企业持续经营能力。
- 关注管理层对持续经营假设的评估和披露。
- 检查是否存在影响持续经营的重大不确定性事项。

### 6.3 大额异常现金流项目

**关注事项：**
- 收到或支付的大额"其他与经营活动有关的现金"——可能隐藏关联方资金往来。
- 投资活动中的大额异常支出——可能为资金体外循环。
- 筹资活动中突然出现的非正常融资渠道。

**审计应对：**
- 追查大额异常现金流的具体去向和来源。
- 核实相关交易的商业实质。
- 关注是否存在资金占用或利益输送。

### 6.4 现金流季节性异常

**关注事项：**
- 经营活动现金流呈现与行业特征不符的季节性模式。
- 年末或季末出现异常的大额现金流入——可能存在截止性问题。
- 期初出现大额现金流出——可能存在期后退回的情况。

**审计应对：**
- 分析月度或季度现金流分布的合理性。
- 重点关注期末前后的大额现金交易。
- 执行截止测试，验证现金流归属期间的正确性。

## 七、现金流分析在审计风险评估中的应用

### 7.1 舞弊风险评估

现金流分析是识别财务舞弊的重要手段：

| 异常模式 | 舞弊风险指向 | 审计程序建议 |
|----------|--------------|--------------|
| 净利润高增长但经营现金流低迷 | 收入虚增 | 函证应收账款、检查期后回款 |
| 经营现金流正常但投资现金流异常 | 资金体外循环 | 追踪投资资金去向、核实投资项目 |
| 筹资现金流异常增长 | 隐性负债或关联方融资 | 检查借款合同、担保协议 |
| 期末现金余额与银行对账单差异 | 货币资金舞弊 | 全面银行函证、核对银行对账单 |

### 7.2 持续经营评估

现金流分析在评估企业持续经营能力方面具有不可替代的作用：

- **短期持续经营**：关注经营活动现金流是否足以覆盖未来12个月的到期债务和必要支出。
- **中期持续经营**：分析自由现金流的趋势，判断企业是否能维持正常运营。
- **长期持续经营**：评估企业现金流创造能力的可持续性。

### 7.3 分析性程序的应用

在审计中，现金流分析可作为实质性分析程序使用：

- 将经营活动现金流与预期值进行比较，识别重大差异。
- 分析现金流各项目之间的勾稽关系，验证数据的合理性。
- 利用现金流信息验证其他审计证据的可靠性。

### 7.4 三表勾稽关系验证

现金流量表与资产负债表、利润表之间存在严格的勾稽关系，审计人员应验证这些关系是否成立：

```
期末现金余额 - 期初现金余额 = 经营活动现金流净额 + 投资活动现金流净额 + 筹资活动现金流净额
```

```
净利润与经营活动现金流净额的差异 ≈ 非现金费用 + 营运资本变动 + 非经营项目
```

勾稽关系的异常可能暗示财务报表存在错报，应作为审计重点进行追查。

## 八、综合分析要点总结

现金流分析的核心价值在于"以现金为镜，照见企业真实面貌"。在实务应用中，应把握以下要点：

1. **利润与现金流并重**：不能仅看利润表，必须结合现金流验证利润质量。
2. **结构分析与趋势分析结合**：既关注现金流的构成结构，也关注变动趋势。
3. **行业特征考量**：不同行业的现金流特征差异较大，分析时应充分考虑行业因素。
4. **多期数据对比**：至少分析3年以上的现金流数据，才能识别趋势性变化。
5. **与比率分析、杜邦分析联动**：现金流分析应与比率分析和杜邦分析相互印证，形成完整的分析体系。
