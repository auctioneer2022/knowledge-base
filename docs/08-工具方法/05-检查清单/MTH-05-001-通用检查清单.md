# 通用审计检查清单

## 一、清单使用说明

### 1.1 适用范围

本检查清单适用于会计师事务所执行的各类财务报表审计业务，包括但不限于年度财务报表审计、中期财务报表审计、专项审计等。本清单依据《中国注册会计师审计准则》体系编制，覆盖审计业务从承接到报告的全流程。

### 1.2 编制依据

- 《中国注册会计师审计准则第1101号——注册会计师的总体目标和审计工作的基本要求》
- 《中国注册会计师审计准则第1111号——就审计业务约定条款达成一致意见》
- 《中国注册会计师审计准则第1141号——财务报表审计中与舞弊相关的责任》
- 《中国注册会计师审计准则第1201号——计划审计工作》
- 《中国注册会计师审计准则第1211号——通过了解被审计单位及其环境识别和评估重大错报风险》
- 《中国注册会计师审计准则第1231号——针对评估的重大错报风险采取的应对措施》
- 《中国注册会计师审计准则第1301号——审计证据》
- 《中国注册会计师审计准则第1501号——对财务报表形成审计意见和出具审计报告》

### 1.3 使用指引

1. 审计项目组应在计划阶段根据项目具体情况对本清单进行适当调整
2. 每个检查项应逐一执行并记录，标注完成状态（已完成/不适用/待完成）
3. 检查结果应与工作底稿索引号关联，确保审计轨迹可追溯
4. 清单执行过程中发现的异常事项应记录在"备注"栏，并跟踪处理结果
5. 清单完成后应由项目合伙人或指定复核人进行复核确认
