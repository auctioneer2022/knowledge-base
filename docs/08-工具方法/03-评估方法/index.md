## 适用场景

- 企业并购重组中的价值评估
- 股权转让、增资扩股中的定价参考
- 财务报告中的资产减值测试
- 税务筹划中的资产估值
- 破产清算中的资产评估
- 投资决策中的价值分析

## 评估方法选择原则

1. **匹配性原则**: 评估方法应与评估目的、价值类型相匹配
2. **适用性原则**: 根据评估对象特点、数据可获得性选择方法
3. **交叉验证原则**: 采用两种以上方法进行评估并相互验证
4. **谨慎性原则**: 关键参数选取应谨慎合理，避免高估或低估
