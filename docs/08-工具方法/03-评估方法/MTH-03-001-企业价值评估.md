# 企业价值评估方法

## 一、基本概念与目的

### 1.1 企业价值评估的定义

企业价值评估是指资产评估机构及其资产评估专业人员在遵守法律、行政法规和资产评估准则的基础上，根据委托，对评估基准日特定目的下的企业整体价值、股东全部权益价值或部分权益价值进行分析、估算并发表专业意见的行为和过程。

### 1.2 评估目的

企业价值评估的主要目的包括：
- **并购重组定价**：为企业合并、分立、资产置换等交易提供价值参考
- **股权转让**：为股东转让股权提供公允价值参考
- **增资扩股**：为新股东入股、老股东增资提供定价依据
- **财务报告**：满足商誉减值测试、合并对价分摊等财务报告需求
- **税务筹划**：为关联交易定价、股权转让税务处理提供依据
- **破产清算**：为破产企业的价值评估提供专业意见

### 1.3 价值类型

企业价值评估中常见的价值类型包括：
- **市场价值**：自愿买方和自愿卖方在各自理性、不受强迫的情况下，评估对象在评估基准日进行正常公平交易的价值估计数额
- **投资价值**：评估对象对于具有特定投资目标的投资者所具有的价值
- **内在价值**：基于企业未来收益能力所体现的价值
- **清算价值**：企业停止经营、出售资产并偿还债务后的剩余价值

## 二、三大评估方法概述

### 2.1 方法比较

| 评估方法 | 核心思路 | 适用条件 | 优点 | 局限性 |
|---------|---------|---------|------|--------|
| **收益法** | 将企业未来预期收益折现 | 持续经营、收益可预测 | 反映企业未来盈利能力 | 依赖主观预测 |
| **市场法** | 参考可比公司或交易价格 | 存在活跃市场、可比对象 | 反映市场供求关系 | 可比性难以保证 |
| **成本法** | 资产重置成本扣除贬值 | 资产重置可行 | 客观性强 | 忽视企业整体盈利能力 |

### 2.2 方法选择原则

1. **评估目的匹配**：评估方法应与评估目的和价值类型相匹配
2. **数据可获得性**：根据可获取的数据和信息选择适当方法
3. **评估对象特点**：考虑企业所处行业、发展阶段、资产结构等特点
4. **交叉验证**：通常应采用两种以上方法进行评估，相互验证

## 三、收益法

### 3.1 基本原理

收益法是指将评估对象的未来预期收益以适宜的折现率折算至评估基准日的现值，借以确定评估对象价值的一种评估方法。收益法的理论基础是货币的时间价值和风险价值。

### 3.2 现金流量折现法（DCF）

#### 3.2.1 企业自由现金流模型（FCFF）

企业自由现金流是指企业经营活动产生的、在满足再投资需要后可供所有资本提供者（股东和债权人）分配的现金流量。

**计算公式**：
```
企业价值 = Σ[FCFFt / (1+WACC)^t] + 终值 / (1+WACC)^n

FCFF = EBIT × (1-税率) + 折旧摊销 - 资本性支出 - 营运资金增加
     = 税后经营净利润 + 折旧摊销 - 资本性支出 - 营运资金增加
```

**股权价值** = 企业价值 - 付息债务价值

#### 3.2.2 股权自由现金流模型（FCFE）

股权自由现金流是指企业经营活动产生的、在满足再投资需要和偿还债务后可供股东分配的现金流量。

**计算公式**：
```
股权价值 = Σ[FCFEt / (1+Ke)^t] + 终值 / (1+Ke)^n

FCFE = 净利润 + 折旧摊销 - 资本性支出 - 营运资金增加 + 净债务增加
```

#### 3.2.3 预测期确定

- **详细预测期**：通常为5-10年，应覆盖企业从当前状态到稳定增长状态的过渡期
- **预测期后**：假设企业进入稳定增长阶段，采用永续增长模型或退出倍数法计算终值

### 3.3 折现率的确定

#### 3.3.1 加权平均资本成本（WACC）

WACC是企业各种资本来源成本的加权平均值，用于折现企业自由现金流。

**计算公式**：
```
WACC = Ke × E/(D+E) + Kd × (1-T) × D/(D+E)

其中：
Ke = 股权资本成本
Kd = 债务资本成本
E = 股权市场价值
D = 债务市场价值
T = 企业所得税税率
```

#### 3.3.2 资本资产定价模型（CAPM）

CAPM是计算股权资本成本最常用的模型。

**计算公式**：
```
Ke = Rf + β × (Rm - Rf) + Rs

其中：
Rf = 无风险收益率（通常采用长期国债收益率）
β = 贝塔系数，反映系统性风险
Rm = 市场组合预期收益率
Rm - Rf = 市场风险溢价（ERP）
Rs = 特定风险调整系数（规模溢价、特定风险溢价等）
```

#### 3.3.3 贝塔系数的确定

**贝塔系数**反映个股相对于整个市场的系统性风险：
- **有杠杆贝塔（βL）**：反映包含财务杠杆的风险
- **无杠杆贝塔（βU）**：剔除财务杠杆影响后的经营风险

**计算方法**：
```
βU = βL / [1 + (1-T) × D/E]

重新加载杠杆：
βL = βU × [1 + (1-T) × D/E]
```

**数据来源**：
- 可比上市公司的贝塔系数
- 行业平均贝塔系数
- 历史数据回归分析

#### 3.3.4 市场风险溢价（ERP）

市场风险溢价是市场组合预期收益率与无风险收益率之差，反映市场整体的风险补偿。

**确定方法**：
- **历史风险溢价法**：基于历史市场数据计算长期平均风险溢价
- **隐含风险溢价法**：基于当前市场估值水平反推未来预期风险溢价
- **国家风险溢价调整**：对于新兴市场，需要在成熟市场风险溢价基础上加国家风险溢价

**中国市场的参考值**：
- 历史风险溢价：约6%-8%
- 国家风险溢价：约1%-2%

### 3.4 终值计算

#### 3.4.1 永续增长法（Gordon模型）

假设企业在预测期后以固定增长率永续增长。

**计算公式**：
```
终值 = FCFFn+1 / (WACC - g) = FCFFn × (1+g) / (WACC - g)

其中：
g = 永续增长率（通常不超过长期经济增长率）
FCFFn+1 = 预测期后第一年的企业自由现金流
```

**永续增长率的确定**：
- 不应超过宏观经济长期增长率（通常2%-3%）
- 应考虑行业成熟度、市场竞争格局
- 应与通货膨胀率相协调

#### 3.4.2 退出倍数法

假设企业在预测期末按市场可比倍数出售。

**计算公式**：
```
终值 = 预测期最后一年的指标 × 退出倍数

常用指标：EBITDA、净利润、营业收入等
退出倍数：参考当前可比公司倍数或历史交易倍数
```

### 3.5 收益法的关键假设和参数选取

#### 3.5.1 收入增长假设
- 基于历史增长趋势、行业发展前景、市场竞争格局
- 应考虑产品生命周期、市场容量、价格趋势
- 区分详细预测期和稳定增长期的增长率

#### 3.5.2 利润率假设
- 基于历史利润率水平、规模效应、成本控制能力
- 考虑行业竞争程度、原材料价格趋势
- 预测期利润率应逐步趋向行业平均水平

#### 3.5.3 资本性支出假设
- 区分维持性资本支出和扩张性资本支出
- 与收入增长相匹配
- 考虑资产更新周期和技术进步

#### 3.5.4 营运资金假设
- 基于历史营运资金周转效率
- 考虑业务模式、信用政策、存货管理
- 通常以营运资金占收入比例表示

### 3.6 适用条件

收益法适用于以下情况：
- **持续经营假设成立**：企业具有持续经营的能力和意愿
- **未来收益可预测**：企业未来收益能够合理预测并量化
- **风险可衡量**：与未来收益相关的风险能够合理衡量
- **收益与资产存在对应关系**：企业收益与投入资产之间存在相对稳定的对应关系

**不适用情况**：
- 初创期企业（难以预测收益）
- 周期性行业（收益波动大）
- 亏损企业（收益为负）
- 面临重大不确定性的企业

## 四、市场法

### 4.1 基本原理

市场法是指将评估对象与可比上市公司或可比交易进行比较，通过分析可比公司的价值乘数或交易价格，确定评估对象价值的一种评估方法。市场法的理论基础是替代原则。

### 4.2 可比公司分析法

#### 4.2.1 选取可比公司的标准

**行业可比性**：
- 主营业务相同或相似
- 所处行业、细分市场相同
- 面临的监管环境相似

**经营可比性**：
- 商业模式相似
- 规模相当（收入、资产、员工数量）
- 发展阶段相近（成长期、成熟期）
- 地域分布相似

**财务可比性**：
- 盈利能力相近
- 资本结构相似
- 增长模式类似

**选取数量**：通常选取3-8家可比公司

#### 4.2.2 价值乘数分析

**市盈率（P/E）**
```
P/E = 股权市值 / 净利润
或 = 每股价格 / 每股收益
```
- 适用于盈利稳定的企业
- 受资本结构影响较大
- 不适用于亏损企业

**市净率（P/B）**
```
P/B = 股权市值 / 净资产
或 = 每股价格 / 每股净资产
```
- 适用于资产密集型行业（银行、保险、房地产）
- 受会计政策影响较大
- 不适用于轻资产企业

**企业价值/息税折旧摊销前利润（EV/EBITDA）**
```
EV/EBITDA = 企业价值 / EBITDA
企业价值 = 股权市值 + 付息债务 - 现金及等价物
```
- 不受资本结构影响
- 不受折旧政策影响
- 适用于资本密集型行业
- 最常用的价值乘数之一

**市销率（P/S）**
```
P/S = 股权市值 / 营业收入
或 = 每股价格 / 每股销售收入
```
- 适用于亏损但有收入的企业
- 适用于高增长的互联网企业
- 不受会计政策影响

#### 4.2.3 评估值计算

**步骤**：
1. 计算可比公司的价值乘数（中位数或平均值）
2. 将可比公司乘数应用于评估对象的相应指标
3. 进行必要的调整（规模、流动性、控制权等）
4. 确定评估对象的评估值

**示例**：
```
可比公司平均EV/EBITDA = 10倍
评估对象EBITDA = 1亿元
初步评估值 = 10 × 1 = 10亿元
调整因素：
- 规模折扣：-10%
- 流动性折扣：-15%
调整后评估值 = 10 × (1-10%-15%) = 7.5亿元
```

### 4.3 可比交易分析法

#### 4.3.1 基本原理

通过分析近期发生的可比并购交易的价格和条件，确定评估对象价值。

#### 4.3.2 交易选取标准

- **交易时间**：通常为近3-5年内的交易
- **交易类型**：并购、收购、私有化等
- **交易规模**：与评估对象相当
- **交易条件**：正常公平交易，非关联交易

#### 4.3.3 并购交易乘数

**交易价值乘数**：
```
交易EV/EBITDA = 交易企业价值 / 目标公司EBITDA
交易P/E = 交易股权价格 / 目标公司净利润
```

**控制权溢价**：
并购交易通常包含控制权溢价，需要与少数股权交易进行比较分析。

### 4.4 市场法的调整因素

#### 4.4.1 规模调整

小规模企业通常面临更高的经营风险和融资成本，需要进行规模折扣调整。

**调整方法**：
- 参考规模溢价研究（如Duff & Phelps数据）
- 基于规模与价值乘数的回归分析
- 通常调整幅度为10%-30%

#### 4.4.2 流动性折扣

非上市公司的股权缺乏流动性，需要进行流动性折扣调整。

**折扣幅度**：
- 限制性股票研究：15%-35%
- IPO前交易研究：20%-40%
- 通常采用25%-35%

#### 4.4.3 控制权溢价

控股股权通常具有控制权溢价，少数股权则需要折价。

**溢价幅度**：
- 基于控制权交易与少数股权交易的比较
- 通常为20%-40%

### 4.5 适用条件

市场法适用于以下情况：
- **存在活跃市场**：有充分的市场交易数据
- **可比对象可获取**：能够找到足够的可比公司或交易
- **信息透明**：可比对象的信息能够可靠获取
- **市场有效**：市场价格能够反映公允价值

**不适用情况**：
- 缺乏可比公司的特殊行业企业
- 市场交易不活跃的情况
- 企业具有独特性，难以找到可比对象
- 市场处于非理性状态（泡沫或恐慌）

## 五、成本法（资产基础法）

### 5.1 基本原理

成本法（资产基础法）是指以评估基准日企业资产负债表为基础，合理评估企业表内各项资产和负债的价值，同时识别和评估表外资产和负债，从而确定企业整体价值的一种评估方法。

**计算公式**：
```
企业价值 = 各项资产评估值之和 - 负债评估值之和 + 表外净资产价值
```

### 5.2 各项资产的评估方法选择

#### 5.2.1 流动资产评估

- **货币资金**：按账面价值确认
- **应收账款**：账面价值扣除坏账准备，或采用账龄分析法
- **存货**：采用市场法或成本法，考虑变现能力
- **预付账款**：按实际可收回金额评估

#### 5.2.2 非流动资产评估

- **长期股权投资**：采用收益法或市场法评估被投资企业价值
- **固定资产**：采用成本法、市场法或收益法
- **无形资产**：采用收益法、市场法或成本法
- **投资性房地产**：采用市场法或收益法

#### 5.2.3 负债评估

- **流动负债**：按实际需偿还金额评估
- **非流动负债**：按折现值或实际需偿还金额评估
- **或有负债**：根据可能性评估是否确认

### 5.3 表外资产和无形资产的识别与评估

#### 5.3.1 表外资产识别

**常见表外资产**：
- 自主研发的专利技术
- 品牌价值和市场声誉
- 客户关系和客户网络
- 人力资源和团队价值
- 特许经营权、许可证
- 长期合同和订单

#### 5.3.2 无形资产评估

**可辨认无形资产**：
- **专利权**：采用收益法（许可费节省法、超额收益法）
- **商标权**：采用收益法（许可费节省法、超额收益法）
- **客户关系**：采用多期超额收益法
- **著作权**：采用收益法

**不可辨认无形资产（商誉）**：
```
商誉 = 企业整体价值 - 各项可辨认净资产公允价值之和
```

### 5.4 适用条件

成本法适用于以下情况：
- **资产重置可行**：各项资产能够重新购置或建造
- **资产基础型企业**：资产密集型企业的价值主要体现在资产上
- **持续经营假设不成立**：企业面临清算或停止经营
- **控股公司**：主要价值来源于投资资产

**不适用情况**：
- **轻资产企业**：价值主要来源于人力资本、客户关系等表外资产
- **高成长企业**：未来收益远高于资产重置成本
- **服务型企业**：主要价值来源于人力资源和品牌价值

## 六、评估方法的选择原则和交叉验证

### 6.1 方法选择原则

1. **评估目的导向**：根据评估目的选择最能反映价值类型的方法
2. **数据可获得性**：优先选择数据充分、可靠性高的方法
3. **评估对象特点**：根据企业所处行业、发展阶段、资产结构选择方法
4. **市场条件**：考虑市场发育程度、交易活跃度

### 6.2 交叉验证

**验证目的**：
- 检验评估结果的合理性
- 识别评估方法中的偏差
- 提高评估结论的可信度

**验证方法**：
1. **多方法比较**：采用两种以上方法评估，分析差异原因
2. **敏感性分析**：分析关键参数变化对评估结果的影响
3. **合理性测试**：将评估结果与市场价格、交易案例进行比较

**差异处理**：
- 差异在合理范围内（通常10%-20%）：可取加权平均值
- 差异较大：分析原因，判断哪种方法更适用
- 某种方法明显不适用：排除该方法，采用其他方法结果

## 七、评估报告的基本要素

### 7.1 报告内容

企业价值评估报告应当包括下列基本内容：

**声明部分**：
- 资产评估机构及评估专业人员声明
- 评估报告使用限制说明

**摘要部分**：
- 评估目的、评估对象和评估范围
- 价值类型、评估基准日
- 评估方法、评估结论

**正文部分**：
1. **委托人及其他报告使用人**
2. **评估目的**
3. **评估对象和评估范围**
4. **价值类型及其定义**
5. **评估基准日**
6. **评估依据**
7. **评估方法**
8. **评估程序实施过程和情况**
9. **评估假设**
10. **评估结论**
11. **特别事项说明**
12. **评估报告使用限制说明**
13. **评估报告日**

**附件部分**：
- 评估汇总表及明细表
- 评估方法说明
- 重要参数选取依据
- 其他相关资料

### 7.2 评估结论的披露

评估结论应当清晰、准确地表述，包括：
- 评估值及其构成
- 评估增减值情况及原因
- 评估结论的使用有效期
- 评估结论的使用限制条件

### 7.3 特别事项说明

应当披露可能对评估结论产生影响的特别事项：
- 权属资料不完整或存在瑕疵
- 重大期后事项
- 评估资料不完整或受限
- 评估假设的特殊性
- 其他需要说明的事项

## 八、实务要点

### 8.1 评估基准日的选择

- 通常选择会计期末（月末、季末、年末）
- 与评估目的实现时间相近
- 考虑数据的可获得性和时效性

### 8.2 评估假设的合理性

- **持续经营假设**：企业是否具有持续经营能力
- **公开市场假设**：是否存在公开活跃的市场
- **最佳用途假设**：资产是否按最佳用途使用

### 8.3 评估参数的验证

- 关键参数应有充分的依据和支持
- 参考行业数据、市场数据、历史数据
- 进行敏感性分析，测试参数的合理性

### 8.4 评估结论的有效期

- 评估结论自评估基准日起一年内有效
- 超过有效期需要重新评估
- 期后发生重大事项需要调整评估结论

## 九、常见问题与风险提示

### 9.1 收益法常见问题

- **预测过于乐观**：收入增长率、利润率假设缺乏依据
- **折现率选取不当**：未充分考虑风险因素
- **终值占比过大**：终值占总价值比例超过70%需要重点关注

### 9.2 市场法常见问题

- **可比性不足**：选取的可比公司与评估对象差异较大
- **调整因素主观性强**：规模调整、流动性折扣缺乏客观依据
- **数据时效性**：市场数据过时，不能反映当前市场状况

### 9.3 成本法常见问题

- **表外资产遗漏**：未充分识别和评估表外无形资产
- **资产高估或低估**：资产状况调查不充分
- **负债遗漏**：未充分识别或有负债

## 十、总结

企业价值评估是一项复杂的专业工作，需要评估专业人员具备扎实的理论基础、丰富的实践经验和良好的职业判断能力。在实际工作中，应当：

1. **充分理解评估目的**：不同评估目的可能影响价值类型和方法选择
2. **深入分析企业**：了解企业的业务模式、竞争优势、风险因素
3. **合理选择评估方法**：根据评估对象特点选择最适用的方法
4. **谨慎选取参数**：关键参数应有充分依据，避免主观臆断
5. **充分披露**：在评估报告中充分披露评估假设、参数选取依据和特别事项

通过科学、规范的评估工作，为企业价值评估提供可靠的专业意见。
