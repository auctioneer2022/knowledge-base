# 审计抽样方法

> 本文件依据《中国注册会计师审计准则第1314号——审计抽样》编制，为注册会计师在审计过程中运用抽样技术提供规范指引。

## 一、审计抽样的定义与基本概念

### 1.1 审计抽样的定义

审计抽样是指注册会计师对某类交易或账户余额中低于百分之百的项目实施审计程序，使所有抽样单元都有被选取的机会，为注册会计师针对整个总体得出结论提供合理基础。

审计抽样的核心特征包括：

- **部分测试**：仅对总体中低于百分之百的项目实施审计程序；
- **随机选取**：所有抽样单元都有被选取的机会，避免人为偏见；
- **推断总体**：根据样本结果推断总体特征，形成审计结论。

### 1.2 抽样风险与非抽样风险

#### 抽样风险

抽样风险是指注册会计师根据样本得出的结论，与对总体全部项目实施与样本同样的审计程序得出的结论存在差异的可能性。抽样风险与样本规模呈反比——样本规模越大，抽样风险越低。

在**控制测试**中，抽样风险表现为：
- **信赖过度风险**：推断的控制有效性高于其实际有效性的风险。信赖过度风险影响审计效果，可能导致注册会计师不适当地减少实质性程序，从而发表不恰当的审计意见。
- **信赖不足风险**：推断的控制有效性低于其实际有效性的风险。信赖不足风险影响审计效率，通常会导致注册会计师执行额外的审计程序。

在**细节测试**中，抽样风险表现为：
- **误受风险**：推断某一账户余额不存在重大错报，而实际上该余额存在重大错报的风险。误受风险影响审计效果，可能导致发表不恰当的无保留意见。
- **误拒风险**：推断某一账户余额存在重大错报，而实际上该余额不存在重大错报的风险。误拒风险影响审计效率，通常会导致注册会计师扩大审计范围。

#### 非抽样风险

非抽样风险是指注册会计师由于任何与抽样风险无关的原因而得出错误结论的风险。非抽样风险包括但不仅限于：

- 采用了不恰当的审计程序；
- 对审计证据的误解或遗漏；
- 审计程序执行不当；
- 对审计结论的错误判断。

非抽样风险无法量化，但可以通过适当的审计计划、督导和质量控制程序加以降低。

## 二、统计抽样与非统计抽样

### 2.1 统计抽样

统计抽样是指同时具备以下两个特征的抽样方法：

1. **随机选取样本项目**：每个抽样单元被选取的概率是已知的；
2. **运用概率论评价样本结果**：包括计量抽样风险。

常用的统计抽样方法包括属性抽样和变量抽样。

**优点**：
- 能够客观地确定样本规模；
- 能够精确地计量抽样风险；
- 抽样结果具有可重复性和可验证性；
- 便于审计质量控制和复核。

**适用场景**：
- 总体规模较大、项目数量较多的审计测试；
- 需要精确计量抽样风险的场合；
- 监管机构或质量控制要求使用统计方法的场合。

### 2.2 非统计抽样

非统计抽样（也称判断抽样）是指不同时具备统计抽样两个特征的抽样方法。在非统计抽样中，注册会计师运用职业判断确定样本规模和评价样本结果。

**优点**：
- 灵活性较高，可根据具体情况调整；
- 不需要复杂的统计知识；
- 适用于总体特征不均匀或存在特殊项目的情况。

**适用场景**：
- 总体规模较小；
- 存在金额特别大或风险特别高的个别项目需要单独测试；
- 审计人员具有丰富的经验且能够做出合理判断。

### 2.3 两者的选择

无论使用统计抽样还是非统计抽样，只要设计得当，均能提供充分、适当的审计证据。注册会计师应当根据具体情况选择适当的方法，关键考虑因素包括：审计目标、总体的特征、可用的审计资源、成本效益原则等。

## 三、属性抽样（用于控制测试）

### 3.1 定义

属性抽样是一种用于估计总体中某一控制特征偏离率（即偏差率）的抽样方法，主要应用于控制测试。属性抽样的结论通常以百分比表示，例如"在95%的可信赖程度下，总体偏差率不超过3%"。

### 3.2 适用条件

- 审计目标是测试内部控制运行的有效性；
- 控制的运行可以留下审计证据轨迹；
- 总体中的每个项目可以被明确地判定为"偏差"或"无偏差"（即二值判定）。

### 3.3 样本量确定

属性抽样中样本量的确定受以下因素影响：

| 影响因素 | 与样本量的关系 | 说明 |
|---------|--------------|------|
| 可接受的信赖过度风险 | 反向变动 | 风险越低，所需样本量越大 |
| 可容忍偏差率 | 反向变动 | 可容忍偏差率越低，所需样本量越大 |
| 预计总体偏差率 | 同向变动 | 预计偏差率越高，所需样本量越大 |
| 总体规模 | 影响很小 | 当总体规模足够大时，对样本量影响几乎为零 |

**基本公式**：

```
样本量(n) = 可信赖程度系数(R) / 可容忍误差率(P)
```

其中：
- **可信赖程度系数(R)**：根据所要求的置信水平查表确定。例如，95%的可信赖程度对应的系数约为3.0。
- **可容忍误差率(P)**：注册会计师能够接受的最大偏差率，通常根据控制的重要程度和计划评估的控制风险水平确定。

**示例**：若要求95%的可信赖程度（R=3.0），可容忍偏差率为3%（P=0.03），则：
n = 3.0 / 0.03 = 100（样本量）

### 3.4 属性抽样的步骤

1. **确定审计目标**：明确要测试的控制及其相关的认定；
2. **定义偏差**：明确构成控制偏差的条件；
3. **定义总体**：确定抽样所针对的总体及其完整性；
4. **确定样本量**：综合考虑上述影响因素，确定适当的样本规模；
5. **选取样本**：采用适当的选样方法从总体中选取样本；
6. **实施审计程序**：对选取的样本项目实施审计程序；
7. **推断总体**：根据样本偏差率推断总体偏差率；
8. **评价结果**：将推断的总体偏差率与可容忍偏差率进行比较，得出结论。

## 四、变量抽样（用于细节测试）

### 4.1 概述

变量抽样是一种用于估计总体金额（如账户余额或交易总额）的抽样方法，主要应用于细节测试。变量抽样的目的是就账户余额或交易总额是否不存在重大错报获取审计证据。

### 4.2 均值估计抽样

**原理**：通过计算样本项目的平均值，乘以总体项目数，来估计总体金额。

**计算公式**：
```
估计的总体金额 = 样本均值 × 总体项目数
即：Ŷ = ȳ × N
```

其中：
- ȳ = 样本项目的平均值
- N = 总体项目数

**适用场景**：总体中各项目的金额分布相对均匀，且未分层时较为适用。

### 4.3 差额估计抽样

**原理**：通过计算样本项目的审定金额与账面金额之差的平均值，来估计总体错报金额。

**计算公式**：
```
估计的总体错报 = 样本平均差额 × 总体项目数
即：Δ = d̄ × N

其中 d̄ = Σ(审定金额 - 账面金额) / n
```

**适用场景**：当错报金额与项目账面金额之间不存在比例关系时较为适用。

### 4.4 比率估计抽样

**原理**：通过计算样本审定金额与账面金额的比率，乘以总体账面金额，来估计总体实际金额。

**计算公式**：
```
估计的总体金额 = 总体账面金额 × (样本审定金额合计 / 样本账面金额合计)
即：Ŷ = B × (Σy / Σb)
```

**适用场景**：当错报金额与项目账面金额之间存在比例关系时较为适用。

### 4.5 PPS抽样（概率比例规模抽样）

**原理**：PPS抽样（Probability-Proportional-to-Size Sampling）以货币单元作为抽样单元，总体中每个货币单元被选取的概率相等，金额越大的项目被选取的概率越高。PPS抽样实质上是一种属性抽样在金额上的应用，用于测试总体是否存在高估。

**核心特点**：
- 自动分层：金额大的项目被选取的概率高，实质上实现了自动分层；
- 适用于测试高估：对于检测账户余额的高估特别有效；
- 样本量相对较小：通常比传统变量抽样所需的样本量更小；
- 不适用于低估：如果预期存在低估，PPS抽样可能不适用。

**PPS抽样的基本步骤**：
1. 确定抽样间隔（选样间距）= 总体账面金额 / 样本量；
2. 确定随机起点，在第一个抽样间隔内随机选取一个起点；
3. 以后每隔一个抽样间隔选取一个样本；
4. 对选取的样本项目实施审计程序；
5. 评价样本结果，推断总体错报。

## 五、样本量确定方法

### 5.1 影响样本量的因素

在确定样本量时，注册会计师应当综合考虑以下因素：

| 因素 | 与样本量的关系 | 说明 |
|------|--------------|------|
| **可接受的抽样风险** | 反向 | 可接受的风险越低，所需样本量越大 |
| **可容忍误差** | 反向 | 可容忍的误差越小，所需样本量越大 |
| **预计总体误差** | 同向 | 预计的误差越大，所需样本量越大 |
| **总体规模** | 影响很小 | 对大规模总体而言，总体规模对样本量的影响几乎为零 |
| **总体的变异性** | 同向 | 总体项目的变异性越大，所需样本量越大 |

### 5.2 实务中的样本量确定

在实务中，注册会计师通常使用以下方法确定样本量：

- **统计公式法**：根据统计公式计算，适用于统计抽样；
- **查表法**：根据审计抽样表，结合可容忍误差和预计总体误差确定样本量；
- **审计软件法**：使用专业审计软件（如IDEA、ACL等）自动计算样本量；
- **职业判断法**：在非统计抽样中，根据经验和判断确定样本量。

## 六、抽样风险的控制

### 6.1 控制信赖过度风险和误受风险

信赖过度风险和误受风险直接影响审计效果，是注册会计师最为关注的抽样风险。控制措施包括：

- 合理确定可容忍误差，保持适当的谨慎性；
- 适当提高可信赖程度（置信水平）的要求；
- 当样本结果接近可容忍误差时，扩大样本量或实施替代程序；
- 结合其他审计程序的结果综合评价。

### 6.2 控制信赖不足风险和误拒风险

信赖不足风险和误拒风险主要影响审计效率。当出现这两种情况时，注册会计师可以：

- 扩大样本规模以进一步验证；
- 实施替代审计程序；
- 获取额外的审计证据以支持结论。

## 七、样本选取方法

### 7.1 随机选样

随机选样是指总体中的每个抽样单元都有同等机会被选取的选样方法。实现方式包括：

- **随机数表法**：使用随机数表，将抽样单元与随机数对应选取；
- **计算机辅助法**：使用计算机程序生成随机数进行选样。

随机选样是统计抽样的基本要求，能够避免人为偏见。

### 7.2 系统选样

系统选样（也称等距选样）是指在总体中按照固定的间隔选取样本的方法。具体步骤为：

1. 计算选样间距 = 总体规模 / 样本量；
2. 在第一个间距内随机确定起点；
3. 从起点开始，每隔一个间距选取一个样本。

**注意**：系统选样要求总体的排列不具有周期性特征，否则可能导致样本不具有代表性。

### 7.3 随意选样

随意选样是指注册会计师不带任何偏见地随意选取样本的方法。随意选样不同于随意选取，审计人员应当避免有意识地偏向于选取特定类型的项目（如金额较大、容易获取的项目等）。

随意选样不适用于统计抽样，但在非统计抽样中可以使用。

## 八、抽样结果评价

### 8.1 推断总体误差

根据样本结果推断总体误差是抽样结果评价的核心步骤：

- **属性抽样**：将样本偏差率作为总体偏差率的最佳估计，考虑抽样风险后计算总体偏差率上限；
- **变量抽样**：根据样本错报金额推断总体错报金额，考虑抽样风险计算总体错报的置信区间。

### 8.2 形成审计结论

注册会计师应当将推断的总体误差与可容忍误差进行比较：

- **推断的总体误差远低于可容忍误差**：可以得出结论，总体不存在重大错报；
- **推断的总体误差接近或超过可容忍误差**：不能得出结论，总体可能存在重大错报，应当扩大样本量、实施替代程序或要求管理层调整；
- **推断的总体误差超过可容忍误差**：有充分理由认为总体存在重大错报。

## 九、实务操作要点和注意事项

### 9.1 实务操作要点

1. **明确审计目标**：在开始抽样之前，明确抽样是用于控制测试还是细节测试，据此选择适当的抽样方法；
2. **确保总体完整性**：验证总体的完整性，确保总体包含了所有应当纳入的项目，避免遗漏或重复；
3. **合理定义误差**：在控制测试中明确定义什么构成偏差，在细节测试中明确错报的认定标准；
4. **分层考虑**：对于金额差异较大的总体，考虑将总体分层，对金额较大的层选取更大的样本或进行百分之百检查；
5. **关注异常项目**：对于样本中发现的异常项目，应当调查原因，评价其对总体的影响；
6. **记录工作底稿**：详细记录抽样方法的选择依据、样本量的确定过程、选样方法、样本结果和结论。

### 9.2 注意事项

1. **抽样不能替代其他审计程序**：抽样只是审计程序的一种手段，不能替代对重要项目的百分之百检查或其他必要的审计程序；
2. **非抽样风险同样重要**：即使抽样风险得到控制，非抽样风险仍可能导致错误结论，应当通过适当的督导和质量控制加以防范；
3. **发现舞弊迹象时的处理**：如果在抽样过程中发现舞弊迹象，不论样本结果如何，都应当重新评估舞弊风险并扩大审计范围；
4. **总体定义的一致性**：确保抽样所针对的总体与审计目标所针对的总体一致；
5. **偏差或错报的性质考虑**：在评价抽样结果时，不仅要看偏差或错报的数量，还要考虑其性质。某些性质严重的偏差或错报（如舞弊导致的错报），即使金额很小，也可能影响审计结论。
