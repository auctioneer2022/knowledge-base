# 审计方法

本模块收录审计实务中常用的核心方法与技术，依据中国注册会计师审计准则编制，为审计人员提供规范化的操作指引。

## 子模块导航

| 编号 | 名称 | 说明 | 主要依据 |
|------|------|------|----------|
| MTH-01-001 | [审计抽样方法](MTH-01-001-抽样方法.md) | 审计抽样的基本概念、统计抽样与非统计抽样、属性抽样与变量抽样、样本量确定、抽样风险控制和结果评价 | CSA 1314 审计抽样 |
| MTH-01-002 | [分析性程序](MTH-01-002-分析性程序.md) | 分析性程序的定义与目的、趋势分析法、比率分析法、合理性测试、实施步骤及实务案例 | CSA 1313 分析程序 |
| MTH-01-003 | [风险评估方法](MTH-01-003-风险评估方法.md) | 审计风险模型、重大错报风险评估、检查风险确定、风险评估程序、舞弊风险评估及风险应对策略 | CSA 1211 风险评估 |

## 使用说明

1. **审计抽样方法**：适用于控制测试和细节测试中需要从总体中选取样本进行推断的场景，帮助审计人员合理确定样本量、选取样本并评价抽样结果。
2. **分析性程序**：适用于审计的风险评估阶段、实质性程序阶段和总体复核阶段，通过对财务数据与非财务数据之间关系的研究，识别异常波动和潜在错报。
3. **风险评估方法**：适用于审计计划阶段和审计全过程，帮助审计人员系统识别和评估财务报表层次及认定层次的重大错报风险，并据此设计应对措施。
