# 众益会计专业知识库

四川众益会计师事务所专业知识库，服务于审计、税务、咨询三大业务主线，为AI智能化审计工具提供底层知识支撑。

## 知识库概览

本知识库采用四层架构设计，从底层法规原文到顶层AI应用逐层提炼：

| 层级 | 内容 | 说明 |
|------|------|------|
| 法规原文层 | 法律法规、准则原文 | 权威、时效、完整 |
| 结构化知识层 | 准则解读、法规速查 | 按主题组织的知识模块 |
| 场景应用层 | 实务指引、检查清单 | 可直接执行的操作指引 |
| AI应用层 | 智能工具、推理引擎 | 未来产品化方向 |
