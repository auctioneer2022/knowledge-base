# 建筑

建筑行业具有项目周期长、合同金额大、成本构成复杂、收入确认依赖履约进度判断等特点。该行业的会计核算核心在于按照履约进度准确确认合同收入与成本，合理计量合同资产与合同负债，规范处理工程结算、变更签证及质保金等事项。

## 主题导航

### [完工百分比法](./完工百分比法.md)

涵盖完工进度确认的三大方法（成本比例法、产出测量法、技术测量法）、合同收入与成本的确认原则、工程施工与工程结算科目的核算规范，以及合同预计损失的会计处理。

### [工程结算](./工程结算.md)

涵盖工程结算全流程（预付款、进度款、结算款、质保金）、工程量确认依据、工程变更与索赔处理、质保金核算规范以及竣工结算审计要点。
