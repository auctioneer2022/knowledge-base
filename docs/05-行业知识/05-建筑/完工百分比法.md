# 建筑行业完工百分比法

## 一、完工进度确认方法

### 1.1 成本比例法

**核心概念**：成本比例法（投入法）是根据累计实际发生的合同成本占合同预计总成本的比例确定完工进度。这是最常用、最客观的完工进度计量方法。

**计算公式**：
- 完工进度 = 累计实际发生的合同成本 / 合同预计总成本 × 100%
- 当期合同收入 = 合同总收入 × 完工进度 - 以前期间累计已确认收入
- 当期合同成本 = 合同预计总成本 × 完工进度 - 以前期间累计已确认成本
- 当期合同毛利 = 当期合同收入 - 当期合同成本

**成本计入规则**：
- **应计入累计实际发生成本**：直接人工、直接材料、机械使用费、分包成本、其他直接费用
- **不应计入累计实际发生成本**：
  - 预付给分包单位的款项（未形成工程实体）
  - 已采购但尚未领用的材料物资（未形成工程实体）
  - 尚未安装或使用的设备（未形成工程实体）
  - 预付款项及已结算但未支付的分包工程款（按权责发生制，已发生才计入）

**实务操作要点**：
- 合同预计总成本应在项目开工前编制详细预算，并随工程进展动态调整
- 每季度末重新评估合同预计总成本，若发生变更（如设计变更、材料价格大幅波动），应及时修订
- 成本归集需按项目设置明细账，区分直接成本与间接成本，间接成本按合理标准分摊
- 累计实际成本与工程结算额（甲方确认产值）通常存在差异，前者用于计算完工进度，后者用于确认应收账款

**常见问题及解决方法**：
- **问题**：项目初期大量采购材料堆放在工地，是否计入累计实际成本？
  - **解决方法**：未领用或未形成工程实体的材料不应计入累计实际成本，应记入"工程物资"或"原材料"科目。待实际领用并用于工程后，方可转入"工程施工——合同成本"。
- **问题**：合同预计总成本频繁调整，导致前期已确认收入需要修正？
  - **解决方法**：成本比例法下，完工进度按最新预计总成本计算，前期已确认收入不再追溯调整，差额计入当期。但需在财务报表附注中披露估计变更的性质、金额及原因。

### 1.2 产出测量法

**核心概念**：产出测量法是根据已完成的工作量或工程产出占合同总工作量的比例确定完工进度。适用于工程量可以明确计量、各阶段产出价值相对均衡的项目。

**具体形式**：
- **已完工工程量比例**：适用于土方、砌筑、混凝土等可按物理量计量的分部分项工程
  - 完工进度 = 已完成工程量 / 合同总工程量 × 100%
- **已完成产值比例**：适用于甲方按月审核确认产值的工程项目
  - 完工进度 = 甲方累计确认产值 / 合同总价 × 100%
- **里程碑法**：按合同约定的关键节点划分阶段，每完成一个里程碑确认相应比例收入

**适用场景**：
- 工程量清单清晰、计量规则明确的工程合同
- 甲方审核确认产值及时、准确的项目
- 各阶段投入与产出相对匹配的项目

**实务操作要点**：
- 产出测量法要求工程量的计量具有客观性和可验证性
- 采用甲方确认产值作为产出指标时，需关注甲方审核周期及确认产值与实际的差异
- 若产出与投入严重不匹配（如前期投入大但产出小），应优先采用成本比例法或调整产出计量方式

**常见问题及解决方法**：
- **问题**：甲方拖延审核产值，导致产出数据滞后，收入确认延迟？
  - **解决方法**：产出测量法依赖甲方确认数据时存在此弊端。建议与甲方约定审核时限，超期未审视为同意申报产值；或采用成本比例法作为替代，在甲方确认后调整差异。

### 1.3 技术测量法

**核心概念**：技术测量法是由专业工程师根据工程实际完成情况，通过现场测量、检测、评估确定完工进度。适用于产出难以用简单物理量计量、成本与产出关联度不高的特殊项目。

**适用场景**：
- 定制化程度高的设备安装工程
- 技术复杂的系统集成项目
- 设计施工一体化（EPC）项目中设计工作占比较大的阶段
- 海洋工程、地下工程等难以按常规方法计量的项目

**操作要求**：
- 技术测量应由具备专业资质的工程师或技术团队执行
- 测量标准应在合同中明确约定或经甲乙双方共同认可
- 测量结果需形成书面报告，经甲乙双方签认
- 测量频率应与财务报告期间相匹配，至少每季度一次

**实务操作要点**：
- 技术测量法主观性较强，需建立完善的内控程序，防止人为操纵完工进度
- 财务部门应参与技术测量过程，了解测量依据和方法
- 保留完整的技术测量报告、现场照片、检测记录等证据资料
- 若技术测量结果与成本比例法差异重大，应分析原因并在附注中说明

**常见问题及解决方法**：
- **问题**：技术测量与成本比例法结果差异较大，以哪个为准？
  - **解决方法**：根据《企业会计准则第14号——收入》，应选择最能如实反映履约进度的方法。若产出法或投入法均无法单独提供可靠结果，可考虑综合多种方法。但需在会计政策中明确优先顺序，并保持一贯应用。

## 二、合同收入与成本的确认原则

### 2.1 收入确认基本公式

**核心概念**：建造合同收入应按履约进度在资产负债表日确认。当期确认的合同收入为合同总收入乘以完工进度扣除以前期间累计已确认收入后的金额。

**基本公式**：
- 当期合同收入 = 合同总收入 × 完工进度 - 以前期间累计已确认收入
- 当期合同成本 = 合同预计总成本 × 完工进度 - 以前期间累计已确认成本
- 当期合同毛利 = 当期合同收入 - 当期合同成本

**合同总收入的构成**：
- 合同规定的初始收入（合同标价）
- 合同变更收入（满足收入确认条件的变更）
- 索赔收入（预计很可能流入且金额能可靠计量）
- 奖励收入（满足确认条件时）

**实务操作要点**：
- 合同变更、索赔、奖励收入应在满足收入确认条件时计入合同总收入，不得提前确认
- 若合同结果不能可靠估计（如预计总成本无法合理确定），应按已发生且预计可收回的成本金额确认收入，成本于发生时计入当期损益
- 合同完成后，应及时进行合同清算，将"工程施工"与"工程结算"科目对冲

### 2.2 合同变更收入确认

**核心概念**：合同变更是指客户对原合同范围或价格作出的调整。建造合同变更需满足特定条件方可确认为收入。

**确认条件**：
- 客户能够认可因变更而增加的收入（书面确认或口头认可但有客观证据）
- 收入金额能够可靠计量（有明确的变更报价单或签证单）

**核算方法**：
- 变更收入经甲方书面确认后，计入合同总收入
- 若变更收入在资产负债表日尚未取得甲方确认，但预计很可能获得认可且金额能可靠估计，可按最佳估计数计入合同总收入
- 对应变更成本同时计入合同预计总成本

**实务操作要点**：
- 变更签证应在发生后及时办理，不得超过合同约定的办理时限
- 未取得甲方确认的变更，不得在财务报表中确认收入，但应在附注中披露或有事项
- 重大变更可能导致原合同履约进度计算基础发生根本性变化，需重新评估整个合同

**常见问题及解决方法**：
- **问题**：甲方口头同意变更但未出具书面签证，能否确认变更收入？
  - **解决方法**：原则上需取得书面确认。若有充分客观证据（如会议纪要、往来邮件、监理签字、已实际施工且甲方未提出异议），可在附注充分披露的前提下按最佳估计数确认，但需评估审计风险。

### 2.3 索赔与奖励收入确认

**核心概念**：索赔款是指因客户或第三方原因造成的、向客户或第三方收取的用于补偿不包括在合同造价中的款项。奖励款是指工程达到或超过规定标准时，客户同意支付的额外款项。

**确认条件**：
- **索赔收入**：根据谈判情况预计对方同意这项索赔，且同意金额能可靠计量
- **奖励收入**：工程进度大幅提前或质量显著优于标准，有足够证据证明很可能获得奖励，且金额能可靠计量

**核算方法**：
- 满足确认条件的索赔款、奖励款计入合同总收入
- 不满足确认条件的，在附注中作为或有资产披露

**实务操作要点**：
- 索赔收入确认应极为审慎，通常需在索赔事项基本谈妥、金额基本确定后方可确认
- 奖励款需有合同明确约定或甲方书面承诺，不得仅凭可能性确认
- 索赔和奖励收入通常税务处理与会计处理一致，但需注意发票开具时点

## 三、工程施工与工程结算科目核算规范

### 3.1 工程施工——合同成本

**核心概念**："工程施工——合同成本"科目归集建造合同发生的直接费用和间接费用，是计算完工进度的分子（累计实际发生成本）。

**核算内容**：
- **直接费用**：直接人工、直接材料、机械使用费、分包成本、其他直接费用
- **间接费用**：施工单位下属施工单位（项目部）为组织和管理施工生产活动所发生的费用，包括管理人员薪酬、折旧费、办公费、水电费、差旅费等

**会计分录示例**：
- 发生人工费：借记"工程施工——合同成本（人工费）"，贷记"应付职工薪酬"
- 领用材料：借记"工程施工——合同成本（材料费）"，贷记"原材料"
- 支付分包款：借记"工程施工——合同成本（分包成本）"，贷记"银行存款/应付账款"
- 分摊间接费用：借记"工程施工——合同成本（间接费用）"，贷记"工程施工——间接费用"

**实务操作要点**：
- 间接费用应按系统合理的方法分摊至各工程项目，通常按直接费用比例或人工费比例分摊
- 项目部管理费用与总部管理费用需严格区分，总部管理费用计入期间费用，不得资本化
- 分包成本应按权责发生制确认，已完工未结算分包款应暂估入账

### 3.2 工程施工——合同毛利

**核心概念**："工程施工——合同毛利"是"工程施工"科目的明细科目，反映已确认的合同收入与合同成本之间的差额。该科目余额在借方表示累计已确认毛利，在贷方表示累计已确认亏损。

**核算逻辑**：
- 资产负债表日确认收入时：
  - 借记"主营业务成本"（当期合同成本）
  - 借记"工程施工——合同毛利"（当期合同毛利，亏损时在贷方）
  - 贷记"主营业务收入"（当期合同收入）

**实务操作要点**：
- "工程施工——合同成本"与"工程施工——合同毛利"之和反映该项目累计已确认收入
- 该余额应与"工程结算"科目对比，判断在资产负债表中列示为合同资产还是合同负债

### 3.3 工程结算

**核心概念**："工程结算"科目核算根据合同约定向客户办理结算的累计金额。该科目是"工程施工"的备抵科目，反映甲方已确认并同意支付的工程价款。

**核算时点**：
- 按合同约定向甲方报送结算资料并经甲方确认后
- 资产负债表日，若已完工未结算，应按完工进度确认合同资产，不通过"工程结算"科目

**会计分录**：
- 办理结算时：借记"应收账款"，贷记"工程结算""应交税费——应交增值税（销项税额）"
- 收到工程款时：借记"银行存款"，贷记"应收账款"

**实务操作要点**：
- "工程结算"与甲方确认产值口径应保持一致
- 已结算未收款部分计入"应收账款"，已完工未结算部分计入"合同资产"
- 工程竣工决算后，"工程施工"与"工程结算"科目对冲，结平余额

### 3.4 合同资产与合同负债

**核心概念**：根据新收入准则，建造合同不再使用"已完工未结算资产"和"已结算未完工款"概念，改为使用"合同资产"和"合同负债"。

**判断标准**：
- **合同资产**：企业已向客户转让商品而有权收取对价的权利，且该权利取决于时间流逝之外的其他条件。在建造合同中，"工程施工"科目余额大于"工程结算"科目余额的差额，表明企业已完工但尚未获得无条件收款权，应列示为"合同资产"
- **合同负债**：企业已收或应收客户对价而应向客户转让商品的义务。在建造合同中，"工程结算"科目余额大于"工程施工"科目余额的差额，表明企业已结算但尚未完成相应履约义务，应列示为"合同负债"

**会计处理**：
- 合同资产减值：按预期信用损失模型计提减值准备，借记"信用减值损失"，贷记"合同资产减值准备"
- 合同资产重分类：当合同资产获得无条件收款权时（如结算完成、质保期满），转入"应收账款"

**实务操作要点**：
- 资产负债表日应按项目逐项比较"工程施工"与"工程结算"余额
- 合同资产与应收账款的本质区别在于收款权是否附条件
- 质保金通常先计入合同资产，质保期满且无质量问题时转入应收账款

**常见问题及解决方法**：
- **问题**：老项目使用"工程施工"和"工程结算"科目，新收入准则下如何衔接？
  - **解决方法**：首次执行新收入准则时，比较各项目"工程施工"与"工程结算"余额。若"工程施工"大于"工程结算"，差额重分类至"合同资产"；若"工程结算"大于"工程施工"，差额重分类至"合同负债"。首次执行日调整计入期初留存收益。

## 四、合同预计损失的会计处理

### 4.1 损失确认条件

**核心概念**：当合同预计总成本超过合同预计总收入时，形成合同预计损失（亏损合同）。企业应在资产负债表日对亏损合同确认预计损失。

**确认时点**：
- 资产负债表日，若预计总成本 > 预计总收入，应立即确认全部预计损失
- 预计损失 = 预计总成本 - 预计总收入

**会计处理原则**：
- 当期确认的合同费用 = 当期合同收入 + 预计损失总额 - 以前期间累计已确认费用
- 或者：当期确认的合同毛利为负数，且累计确认的合同毛利等于预计损失总额

### 4.2 核算方法

**会计分录**：
- 确认当期收入与成本时：
  - 借记"主营业务成本"（按上述公式计算的金额）
  - 贷记"主营业务收入"（按完工进度计算的收入）
  - 贷记"工程施工——合同毛利"（差额，即当期应补提的亏损）
- 计提存货跌价准备（备选方法）：
  - 借记"资产减值损失"
  - 贷记"存货跌价准备——合同预计损失准备"

**备查登记**：
- 建立亏损合同台账，记录每个亏损合同的预计损失金额、已确认金额、剩余待确认金额
- 每季度跟踪预计总成本与总收入的变化，及时调整预计损失金额

**实务操作要点**：
- 预计损失应在亏损发生当期全额确认，不得递延至以后期间
- 若后续期间预计总成本下降或合同总收入增加，导致亏损减少或转为盈利，应在后续期间冲减原确认的预计损失，但冲减金额不得超过原确认金额
- 合同完成后，将"存货跌价准备——合同预计损失准备"余额冲销

**常见问题及解决方法**：
- **问题**：合同预计损失与履约进度计算的关系？
  - **解决方法**：即使合同整体亏损，仍应按履约进度确认收入。亏损金额通过当期成本大于收入的差额体现。例如：完工进度50%，预计总收入1000万元，预计总成本1200万元。当期确认收入500万元，当期确认成本600万元（含补提的亏损100万元），当期毛利-100万元，累计毛利-200万元（即预计亏损总额）。
- **问题**：亏损合同的税务处理？
  - **解决方法**：会计上确认的预计损失，在税务上需按实际发生原则扣除。未实际发生的预计损失不得在企业所得税前扣除，应做纳税调增。待损失实际发生时（如合同完工结算），再做纳税调减。
