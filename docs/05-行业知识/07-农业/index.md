# 农业行业会计知识

农业行业会计核算具有鲜明的行业特点，核心在于生物资产的确认、计量与披露，以及政府补助的会计处理。农业企业需遵循《企业会计准则第5号——生物资产》及相关政府补助准则，结合行业实务进行准确核算。

## 主题导航

### [生物资产](生物资产.md)

涵盖消耗性生物资产、生产性生物资产、公益性生物资产的分类与核算，包括初始计量、后续计量、收获与处置等完整会计处理流程，以及减值与跌价准备的计提方法。

### [政府补助](政府补助.md)

系统梳理农业企业常见的政府补助类型，包括与资产相关、与收益相关及综合性项目补助的会计处理，重点讲解总额法与净额法的适用场景，以及补助资金的监管合规要求。
