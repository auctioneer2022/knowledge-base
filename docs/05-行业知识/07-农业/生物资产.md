# 生物资产

## 一、核心概念

生物资产是指有生命的动物和植物，是农业企业最核心的资产形态。根据《企业会计准则第5号——生物资产》，生物资产按照其用途和持有目的分为三类：

- **消耗性生物资产**：持有目的为出售，如存栏待售的牲畜、生长中的大田作物、蔬菜、用材林等。该类资产属于流动资产范畴，在资产负债表中列示于存货项目。
- **生产性生物资产**：持有目的为产出农产品、提供劳务或出租，如果树、产畜、役畜、经济林、薪炭林等。该类资产属于非流动资产，具备固定资产或无形资产的属性，需计提折旧。
- **公益性生物资产**：以防护、环境保护为主要目的，如防风固沙林、水土保持林、水源涵养林等。该类资产不计提折旧，也不计提减值准备。

**准则适用边界**：本准则适用于与农业生产相关的生物资产，企业从事农产品收获后的加工、流通环节不适用本准则，应适用存货、固定资产等其他准则。天然起源的生物资产，企业仅有轻微付出或几乎无付出的，通常不予确认。

## 二、生物资产分类

### 2.1 消耗性生物资产

| 类别 | 典型示例 | 持有目的 |
|------|----------|----------|
| 大田作物 | 小麦、水稻、玉米、棉花 | 收获后出售 |
| 蔬菜 | 叶菜类、果菜类、根茎类 | 收获后出售 |
| 林木 | 用材林、竹林 | 成材后出售 |
| 畜禽 | 存栏待售的猪、牛、羊、家禽 | 直接出售 |
| 水产 | 养殖鱼、虾、蟹、贝类 | 育成后出售 |

### 2.2 生产性生物资产

| 类别 | 典型示例 | 产出形式 |
|------|----------|----------|
| 经济林 | 苹果园、茶园、橡胶园 | 水果、茶叶、橡胶 |
| 产畜 | 奶牛、蛋鸡、种猪 | 牛奶、鸡蛋、仔猪 |
| 役畜 | 耕牛、马、骡 | 耕作、运输劳务 |
| 薪炭林 | 速生薪炭林 | 燃料 |

生产性生物资产可进一步划分为**成熟生产性生物资产**与**未成熟生产性生物资产**。达到预定生产经营目的（如果树开始挂果、奶牛开始产奶）前为未成熟阶段，达到后为成熟阶段，成熟后方可计提折旧。

### 2.3 公益性生物资产

公益性生物资产的核心特征在于其非经营性和公共服务属性。企业持有该类资产不以获取经济利益为主要目的，因此会计处理上相对简化，不计提折旧和减值。

## 三、初始计量方法

生物资产初始计量按成本进行，成本构成因取得方式不同而存在差异。

### 3.1 外购取得

外购生物资产的成本包括购买价款、相关税费、运输费、保险费以及可直接归属于购买该资产的其他支出。

**实务操作要点**：
- 购入种畜、鱼苗等需隔离检疫的，检疫期间的相关饲养费用计入资产成本。
- 一笔款项购入多项没有单独标价的生物资产，应按各项生物资产的公允价值比例对总成本进行分配。
- 外购过程中发生的合理损耗（如运输途中正常死亡），不扣减总成本，但需调整单位成本。

**常见问题**：外购生物资产时，购买价款中包含已到期但尚未领取的政府补助款项，应如何处理？
**解决方法**：应将应收政府补助款单独确认为其他应收款，不计入生物资产成本。实际收到补助时，按政府补助准则进行会计处理。

### 3.2 自行营造或繁殖

自行营造或繁殖的生物资产，其成本由达到预定生产经营目的前发生的必要支出构成。

**消耗性生物资产成本构成**：
- 直接材料：种苗、饲料、肥料、农药等
- 直接人工：种植、养殖人员薪酬
- 其他直接费用：机械作业费、灌溉费、防疫费
- 应分摊的间接费用：制造费用按合理标准分配

**生产性生物资产成本构成**：
- 未成熟阶段：与消耗性生物资产成本构成相同，归集至"生产性生物资产——未成熟"科目
- 达到预定生产经营目的时：将累计成本从"未成熟"转入"成熟"明细科目

**实务操作要点**：
- 自行繁殖的幼畜成龄转为产畜或役畜时，按账面价值结转。
- 自行营造的林木在郁闭或达到预定生产经营目的前发生的管护费用计入资产成本，之后发生的计入当期费用。
- 农业生产过程中发生的共同费用，应按种植面积、饲养头数、工时等合理标准在各成本核算对象之间分配。

**常见问题**：果树的成本何时停止资本化，转为计提折旧？
**解决方法**：果树开始挂果并产生稳定经济产出时，视为达到预定生产经营目的。实务中通常以首次稳定结果期为判断标准，此时将累计培育成本从"未成熟生产性生物资产"转入"成熟生产性生物资产"。

### 3.3 投资者投入

投资者投入的生物资产，按投资合同或协议约定的价值确定成本，但合同或协议约定价值不公允的除外。不公允时，应以公允价值作为入账成本。

### 3.4 非货币性资产交换取得

按《企业会计准则第7号——非货币性资产交换》处理。具有商业实质且公允价值能够可靠计量的，以换出资产公允价值（或换入资产公允价值）加相关税费作为成本；不满足条件的，以换出资产账面价值加相关税费作为成本。

### 3.5 债务重组取得

按《企业会计准则第12号——债务重组》处理。以放弃债权的公允价值加上可直接归属于该资产的相关税费作为成本。

## 四、后续计量方法

### 4.1 历史成本模式

历史成本模式是生物资产后续计量的基础方法，适用于绝大多数农业企业。

**消耗性生物资产**：按成本与可变现净值孰低计量。资产负债表日，当可变现净值低于成本时，计提消耗性生物资产跌价准备，计入当期损益（资产减值损失）。

**可变现净值确定方法**：
- 持有以备出售的，按估计售价减去估计销售费用和相关税费确定
- 持有以待收获的，按估计收获后产成品的估计售价减去至收获时估计将要发生的成本、销售费用和相关税费确定

**生产性生物资产**：按成本减去累计折旧和累计减值准备后的金额计量。成熟生产性生物资产应当按期计提折旧，折旧方法包括年限平均法、工作量法、产量法等。

**折旧实务要点**：
- 折旧年限：果树通常10-20年，产畜3-5年，役畜5-8年，经济林15-30年
- 预计净残值：产畜、役畜通常按产畜或役畜淘汰时预计的变现价值确定
- 折旧计提时点：当月增加的成熟生产性生物资产，当月不计提折旧，从下月起计提；当月减少的，当月仍计提折旧

**公益性生物资产**：不计提折旧，也不计提减值准备。后续发生的管护费用计入当期损益。

### 4.2 公允价值模式

公允价值模式的使用受到严格限制。仅当同时满足以下条件时，生物资产才可采用公允价值计量：
1. 生物资产存在活跃的交易市场
2. 能够从交易市场上取得同类或类似生物资产的市场价格及其他相关信息，从而对生物资产的公允价值作出合理估计

**适用情形**：我国目前主要用于上市农业企业的部分畜禽、大宗农作物等具有公开报价的品种。绝大多数农业企业仍采用历史成本模式。

**公允价值确定层次**：
- 第一层次：活跃市场中的报价
- 第二层次：同类或类似资产在活跃市场中的报价，经适当调整后确定
- 第三层次：采用估值技术，如预期现金流量折现法

**会计处理差异**：采用公允价值模式的，不计提折旧和减值，公允价值变动计入当期损益。

### 4.3 折旧与摊销

**生产性生物资产折旧政策**：

| 类别 | 折旧方法建议 | 预计使用年限 | 预计净残值率 |
|------|-------------|-------------|-------------|
| 果树 | 年限平均法或产量法 | 10-20年 | 5%-10% |
| 经济林 | 年限平均法 | 15-30年 | 5%-10% |
| 产畜 | 工作量法或年限平均法 | 3-5年 | 淘汰变现价值 |
| 役畜 | 工作量法 | 5-8年 | 淘汰变现价值 |

**产量法折旧公式**：
单位产量折旧额 = （资产成本 - 预计净残值）/ 预计总产量
当期折旧额 = 单位产量折旧额 × 当期实际产量

**常见问题**：生产性生物资产更新改造期间是否停止计提折旧？
**解决方法**：生产性生物资产进行更新改造（如果树品种改良、畜舍改扩建）且预期能延长使用寿命或提高产出效率的，应将账面价值转入在建工程，改造期间停止计提折旧；改造完工后按新的账面价值和尚可使用寿命重新确定折旧政策。日常维修维护期间不停止计提折旧。

## 五、收获与处置核算

### 5.1 农产品收获成本结转

农产品收获时，应将相应的生物资产账面价值结转为农产品的成本。

**消耗性生物资产收获**：
```
借：农产品
    消耗性生物资产跌价准备（如有）
贷：消耗性生物资产
```

**生产性生物资产产出**：生产性生物资产产出的农产品成本，包括该资产折旧费、管护费用等间接分摊费用和直接材料、直接人工。产出时按合理方法（如面积比例、产量比例）将累计费用在农产品与在产品之间分配。

**实务操作要点**：
- 大田作物收获：将"消耗性生物资产"账面价值全额转入"农产品"
- 畜禽出栏：将出栏畜禽对应的账面价值从"消耗性生物资产"转入"主营业务成本"
- 果树产果：采摘成本计入当期农产品成本，果树本身的折旧计入制造费用后分配

### 5.2 生物资产出售

**消耗性生物资产出售**：
```
借：银行存款
贷：主营业务收入
借：主营业务成本
    消耗性生物资产跌价准备（如有）
贷：消耗性生物资产
```

**生产性生物资产出售**：比照固定资产清理处理，通过"生产性生物资产清理"科目核算，处置净损益计入资产处置损益。

### 5.3 生物资产盘亏、死亡毁损

生物资产发生盘亏、死亡或毁损时，应将账面价值扣除责任人赔偿、保险赔款后的余额，根据原因计入不同科目：
- 因管理不善造成的：计入管理费用
- 因自然灾害等非常原因造成的：计入营业外支出

**实务操作要点**：
- 发生死亡毁损时，应及时取得兽医检疫证明、保险公司查勘报告等外部证据。
-  herdsman或饲养人员责任造成的死亡，应追究赔偿责任，赔偿款冲减损失。
- 重大疫情导致的群体性死亡，应按政府扑杀补贴文件处理，扑杀补贴与账面价值的差额计入当期损益。

**常见问题**：奶牛场发生口蹄疫，政府组织扑杀并给予补偿，会计如何处理？
**解决方法**：
1. 将扑杀奶牛账面价值转入待处理财产损溢
2. 实际收到政府扑杀补偿时，确认其他收益（与日常活动相关）或营业外收入
3. 补偿款不足覆盖账面价值的差额，计入营业外支出
4. 如政府补偿明确为对价组成部分的，按收入准则处理

### 5.4 跌价准备与减值准备

**消耗性生物资产跌价准备**：
资产负债表日，按成本与可变现净值孰低计量。可变现净值低于成本的差额计提跌价准备，计入资产减值损失。以后期间影响因素消失的，可在原已计提金额内转回。

**生产性生物资产减值准备**：
资产负债表日判断是否存在减值迹象。存在减值迹象的，估计可收回金额（公允价值减处置费用后的净额与资产预计未来现金流量的现值孰高）。可收回金额低于账面价值的，计提减值准备，计入资产减值损失。

**减值迹象判断**：
- 遭受重大自然灾害（旱灾、水灾、疫情）导致生产能力大幅下降
- 市场价格持续下跌且短期内无回升希望
- 因技术原因导致品种被淘汰
- 长期闲置且预计未来不再使用
- 其他表明资产已发生减值的客观证据

**重要原则**：生产性生物资产减值准备一经计提，在以后会计期间不得转回。消耗性生物资产跌价准备可以转回。

**常见问题**：如何对生产性生物资产进行减值测试？
**解决方法**：生产性生物资产通常难以单独产生现金流量，应结合所属资产组进行减值测试。如果树应与果园整体作为资产组，估计其未来现金流量现值。预计未来现金流量时，应考虑农产品价格走势、产量预测、生产成本等因素，采用适当的折现率进行折现。

## 六、列报与披露

资产负债表日，企业应在资产负债表附注中披露与生物资产相关的下列信息：
- 生物资产的类别及各类别的账面价值与公允价值（如适用）
- 与生物资产相关的风险情况说明（病害、疫情、自然灾害等）
- 采用公允价值计量时，公允价值的确定方法
- 生产性生物资产的折旧政策、使用寿命、预计净残值
- 生物资产增减变动情况
- 生物资产减值或跌价准备的计提情况
