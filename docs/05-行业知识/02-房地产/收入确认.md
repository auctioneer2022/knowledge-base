# 房地产收入确认

## 一、不同销售模式下的收入确认原则

房地产企业销售模式多样，不同模式下的收入确认原则和时点存在显著差异。根据《企业会计准则第14号——收入》（2017修订），房地产企业应当在履行了合同中的履约义务，即在客户取得相关商品控制权时确认收入。

### 1.1 现售模式

**业务特征**：商品房已竣工验收，具备交付条件，购房者一次性或短期内支付全部房款。

**收入确认原则**：
- 在商品房控制权转移给购房者时确认收入
- 通常以商品房实际交付（交钥匙）作为控制权转移的标志
- 交付前收到的款项确认为合同负债（或预收账款）

**确认条件**：
1. 商品房已竣工验收合格，取得竣工验收备案表
2. 已签订正式商品房买卖合同
3. 已按合同约定收取房款（或取得收款权利）
4. 商品房已实际交付给购房者，购房者取得控制权
5. 相关的经济利益很可能流入企业

**会计处理**：
- 交付前：借：银行存款，贷：合同负债（预收账款）
- 交付时：借：合同负债（预收账款），贷：主营业务收入；同时结转成本：借：主营业务成本，贷：开发产品

### 1.2 预售模式

**业务特征**：商品房尚未竣工或虽已竣工但未达到交付条件，房地产开发企业预先向购房者收取房款。

**收入确认时点**：
- 预售阶段不得确认销售收入
- 预售房款应作为合同负债（或预收账款）核算
- 待商品房竣工验收并实际交付后，方可确认收入

**会计处理**：
- 收到预售款：借：银行存款，贷：合同负债（预收账款）
- 预缴增值税：借：应交税费——预交增值税，贷：银行存款
- 交付确认收入：借：合同负债，贷：主营业务收入；借：主营业务成本，贷：开发产品

**重要提示**：
- 预售款不得提前确认收入，不得虚构交付提前确认收入
- 需关注是否存在"假交付"（形式上交付但实质上未转移控制权）以调节收入
- 预售资金监管账户的款项使用应符合当地监管规定

### 1.3 按揭销售模式

**业务特征**：购房者支付首付款后，以所购商品房抵押向银行申请按揭贷款，银行将贷款直接划入开发商账户。

**收入确认原则**：
- 收入确认时点仍以商品房实际交付、控制权转移为准
- 按揭贷款的审批和发放不构成收入确认的条件
- 首付款和按揭贷款到账时均作为预收款核算

**风险考量**：
- **按揭担保责任**：开发商通常需为购房者的按揭贷款提供阶段性担保（至办妥抵押登记为止）。在担保期内，如购房者断供，开发商可能承担回购责任。需评估预计负债。
- **银行按揭未获批**：如购房者按揭贷款未获批准，需按合同约定处理（退房、变更付款方式等），已收款项可能需退还。

**会计处理要点**：
- 收到首付款：借：银行存款，贷：合同负债
- 银行发放按揭贷款：借：银行存款，贷：合同负债
- 交付确认收入时：借：合同负债，贷：主营业务收入
- 计提担保预计负债（如适用）：借：信用减值损失/营业外支出，贷：预计负债

### 1.4 分期收款销售模式

**业务特征**：购房者将房款分多次支付，付款期限通常超过正常信用期（如超过1年）。

**收入确认原则**：
- 按照《企业会计准则第14号——收入》的规定，合同中存在重大融资成分的，应当按照假定客户在取得商品控制权时即以现金支付的应付金额（即现销价格）确定交易价格
- 合同对价与现销价格之间的差额，应当在合同期间内采用实际利率法摊销

**会计处理**：
- 交付时按现销价格确认收入：借：长期应收款（合同对价），贷：主营业务收入（现销价格），贷：未实现融资收益（差额）
- 按期收款：借：银行存款，贷：长期应收款
- 摊销融资收益：借：未实现融资收益，贷：财务费用

**税务处理差异**：
- 企业所得税：根据《企业所得税法实施条例》，以分期收款方式销售商品的，按照合同约定的收款日期确认收入的实现
- 增值税：发生应税行为并收讫销售款项或取得索取销售款项凭据的当天为纳税义务发生时间
- 需关注会计与税务处理的暂时性差异

### 1.5 售后回租模式

**业务特征**：房地产开发企业将商品房出售给购房者后，又与购房者签订租赁协议，回租该商品房。

**业务实质判断**：
- 需根据《企业会计准则第14号——收入》和《企业会计准则第21号——租赁》判断交易实质
- 如售后回租构成融资租赁或经营租赁，且售价不公允，可能需要将售价与公允价值的差额递延
- 重点关注是否存在融资安排、回购条款、固定收益承诺等表明交易实质上不是销售的情形

**会计处理**：
- **构成真实销售**：按销售确认收入，同时按租赁准则处理回租业务
- **售后租回形成融资租赁**：卖方（承租人）应将售价与账面价值的差额递延，并按租赁资产的折旧进度分摊
- **售后租回形成经营租赁**：
  - 售价按公允价值达成：确认销售收入，回租按经营租赁处理
  - 售价低于公允价值：损失递延并按租金支付比例分摊；未来租赁付款额低于市场租金，损失与未来低于市场租金的付款额相抵，按租金支付比例分摊
  - 售价高于公允价值：超额部分递延并按租金支付比例分摊

**风险警示**：
- 如存在固定回购条款、回购价格与原价挂钩、购房者不承担资产主要风险和报酬等情形，交易实质上可能是融资安排，不应确认销售收入
- 售后返租模式在商业地产销售中较为常见，需审慎判断收入确认的适当性

## 二、收入确认时点判断标准

### 2.1 交付时点

在房地产销售中，商品房的实际交付是判断控制权转移的最重要标志之一。

**交付的认定标准**：
- 开发商将商品房的钥匙交给购房者，购房者实际占有该商品房
- 双方签署房屋交接单或交付确认书
- 商品房已竣工验收合格，符合合同约定的交付条件
- 相关配套设施（水、电、气等）已可正常使用或符合合同约定条件

**审计关注**：
- 检查交付记录的真实性，防止为提前确认收入而虚构交付
- 对比交付记录与物业费起算时间、水电燃气开通时间等第三方证据
- 关注期末集中交付的情况，检查是否存在突击交付以完成业绩目标
- 检查延期交付的合同约定及实际处理，是否存在违约赔偿未计提

### 2.2 控制权转移

根据新收入准则，企业应当在客户取得相关商品控制权时确认收入。控制权的转移需要综合判断。

**控制权转移的迹象**：
1. **现时收款权**：企业就该商品享有现时收款权利，即客户就该商品负有现时付款义务
2. **法定所有权转移**：企业已将该商品的法定所有权转移给客户
3. **实物转移**：企业已将该商品实物转移给客户
4. **主要风险和报酬转移**：企业已将该商品所有权上的主要风险和报酬转移给客户
5. **客户接受**：客户已接受该商品

**房地产销售的特殊考量**：
- 商品房交付后，如存在重大质量保证义务（如结构性质量问题保修期较长），需判断质量保证是否构成单项履约义务
- 如存在物业管理服务承诺、配套设施建设承诺等，需判断这些承诺是否构成合同中的单项履约义务，是否需要将交易价格分摊至各项履约义务
- 部分地区的"交房即交证"政策下，产权过户是否作为收入确认的前提条件需结合合同约定判断

### 2.3 风险报酬转移

虽然新收入准则以控制权转移作为收入确认的核心标准，但风险报酬的转移仍是判断控制权是否转移的重要依据。

**房地产企业主要风险**：
- **价格波动风险**：商品房交付前，房价波动的风险由开发商承担；交付后由购房者承担
- **毁损灭失风险**：交付前商品房毁损灭失的风险由开发商承担；交付后通常由购房者承担（除非另有约定）
- **功能性风险**：商品房不符合居住功能要求的风险在交付前由开发商承担

**审计程序**：
- 检查商品房买卖合同中关于风险转移的条款
- 检查交付验收记录，确认双方对交付状态的确认
- 检查期后是否存在大规模退房、质量纠纷等情况，判断收入确认的适当性
- 关注是否存在附回购条款、保底收益等未完全转移风险的交易安排

## 三、面积差异处理

### 3.1 面积补差款的会计处理

商品房销售通常按建筑面积或套内建筑面积计价，合同约定面积与产权登记面积可能存在差异，由此产生面积补差款。

**处理原则**：
- 面积补差款是对原交易价格的调整，应当调整已确认的收入金额
- 如面积差异在交付时确定，面积补差款与交付时点的收入一并确认或调整
- 如面积差异在交付后确定（如产权登记面积实测后），在实际确定时调整收入

**会计处理**：
- **面积补差款（补收）**：借：应收账款/银行存款，贷：主营业务收入
- **面积补差款（退还）**：借：主营业务收入（红字），贷：银行存款/应收账款；或借：主营业务收入，贷：银行存款（如金额较小可直接冲减当期收入，如金额重大需追溯调整）

**实务要点**：
- 面积补差款通常金额不大，且与主要交易密切相关，一般在确定时计入当期损益
- 如面积差异涉及金额重大，且是对交付时即已存在情况的调整，需考虑是否作为前期差错更正处理
- 通常在交付时按合同面积确认收入，后续面积实测差异直接调整当期收入

### 3.2 退补流程

**面积差异产生原因**：
- 预售阶段按预测面积计价，交付后按实测面积结算
- 建筑设计变更导致面积变化
- 测量误差或标准理解差异

**标准处理流程**：
1. 取得具有资质的测绘机构出具的实测面积报告
2. 对比合同约定面积与实测面积，计算差异比例和金额
3. 按照合同约定的面积差异处理条款（如"多退少补"、"超过3%部分双倍返还"等）计算应退补金额
4. 通知购房者确认，办理退补手续
5. 进行相应的会计处理

**合同条款常见约定**：
- 面积误差比绝对值在3%以内（含3%）：据实结算房价款
- 面积误差比绝对值超出3%：购房者有权退房；如不退房，超出3%部分的房价款由开发商承担，产权归购房者；面积小于合同约定且误差超出3%的，超出3%部分的房价款双倍返还购房者

**审计关注**：
- 检查面积差异的计算是否准确，是否按合同约定执行
- 检查面积补差款的确认和收付是否及时、完整
- 关注是否存在大量面积差异纠纷，评估对财务报表的影响
- 检查面积补差款的会计处理是否符合会计准则

## 四、预售款核算规范

### 4.1 预收账款科目使用

房地产企业预售阶段收到的款项，在未达到收入确认条件前，应作为负债核算。

**科目选择**：
- 执行新收入准则的企业：使用"合同负债"科目核算已收或应收客户对价而应向客户转让商品的义务
- 未执行新收入准则的企业（已较少）：使用"预收账款"科目
- 实务中很多企业仍沿用"预收账款"科目，或同时设置"合同负债"和"预收账款"

**核算内容**：
- 购房者按合同约定支付的定金、首付款、按揭贷款、分期付款等
- 在商品房交付前，上述款项均不得转入收入
- 预售款应按客户或项目设置明细账，详细记录每笔款项的性质、金额、时间

### 4.2 合同负债与预收账款区分

根据新收入准则，"合同负债"与"预收账款"存在概念差异。

**合同负债**：
- 核算企业已收或应收客户对价而应向客户转让商品的义务
- 强调基于合同产生的履约义务
- 企业向客户转让相关商品时，将该负债转为收入

**预收账款**：
- 核算企业按照合同规定向购货单位预收的款项
- 不强调合同是否存在，范围更广
- 实务中常用于核算不适用收入准则的预收款项

**房地产企业的应用**：
- 商品房预售款基于商品房买卖合同产生，符合合同负债的定义，理论上应使用"合同负债"科目
- 由于实务习惯和系统设置原因，很多企业仍在使用"预收账款"科目，或两者并用
- 在财务报表列报时，如使用"预收账款"科目核算基于合同的预售款，应在附注中充分披露

### 4.3 增值税预缴

房地产企业采取预收款方式销售自行开发的房地产项目，应在收到预收款时按照3%的预征率预缴增值税。

**预缴计算**：
- **一般计税方法**：应预缴税款 = 预收款 /（1 + 9%）x 3%
- **简易计税方法**：应预缴税款 = 预收款 /（1 + 5%）x 3%

**会计处理**：
- 预缴增值税：借：应交税费——预交增值税，贷：银行存款
- 纳税义务发生时（交付时）：借：应交税费——应交增值税（销项税额），贷：应交税费——预交增值税（冲减预缴）

**预缴附加税费**：
- 城市维护建设税、教育费附加、地方教育附加通常随同增值税预缴
- 会计处理：借：应交税费——应交城市维护建设税等，贷：银行存款

**土地增值税预征**：
- 房地产开发项目在未达到清算条件前，按照当地税务机关规定的预征率预征土地增值税
- 会计处理：借：应交税费——应交土地增值税，贷：银行存款（或先计入税金及附加，再结转）

**企业所得税预缴**：
- 房地产开发企业销售未完工开发产品取得的收入，应先按预计计税毛利率分季（或月）计算出预计毛利额，计入当期应纳税所得额
- 预计毛利额 = 预售收入 x 预计计税毛利率（由各省税务机关确定，一般不低于15%）
- 开发产品完工后，企业应及时结算其计税成本并计算此前销售收入的实际毛利额，同时将其实际毛利额与其对应的预计毛利额之间的差额，计入当年度企业本项目与其他项目合并计算的应纳税所得额

**预售款核算的完整流程**：
1. 收到预售款：借：银行存款，贷：合同负债
2. 预缴增值税：借：应交税费——预交增值税，贷：银行存款
3. 预缴附加税费：借：应交税费——城建税/教育费附加等，贷：银行存款
4. 预缴土地增值税：借：税金及附加（或应交税费），贷：银行存款
5. 季度预缴企业所得税（按预计毛利）：按税法规定计算并申报
6. 商品房交付确认收入：借：合同负债，贷：主营业务收入；同时结转销项税额和预缴税款

**审计重点**：
- 检查预售款是否完整入账，是否存在账外收款
- 检查增值税预缴是否及时、足额，计算是否正确
