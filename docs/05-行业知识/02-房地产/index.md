# 房地产

房地产行业具有开发周期长、资金投入大、收入确认时点复杂、涉税事项繁多等特点。本模块围绕房地产行业的核心会计与税务问题，聚焦收入确认和土地增值税清算两大主题。

## 主题目录

### [收入确认](./收入确认.md)

系统梳理房地产开发销售不同模式下的收入确认原则与实务操作，包括：
- 不同销售模式下的收入确认原则（现售、预售、按揭销售、分期收款、售后回租）
- 收入确认时点判断标准（交付时点、控制权转移、风险报酬转移）
- 面积差异处理（面积补差款的会计处理、退补流程）
- 预售款核算规范（预收账款科目使用、合同负债与预收账款区分、增值税预缴）

### [土地增值税清算](./土地增值税清算.md)

全面解析土地增值税清算的条件、流程与扣除项目审核要点，包括：
- 清算条件与流程（应清算情形、可要求清算情形、清算程序）
- 扣除项目审核要点（取得土地使用权支付金额、房地产开发成本、开发费用、税金、加计扣除）
- 增值额计算方法（收入确认、扣除项目归集、增值额与增值率计算）
- 税收优惠政策应用（普通标准住宅优惠、国家征用收回优惠、旧房转让优惠）
- 清算申报要求（申报期限、资料清单、鉴证报告要求）
