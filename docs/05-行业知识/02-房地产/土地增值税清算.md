# 土地增值税清算

## 一、清算条件与流程

### 1.1 应清算情形

根据《土地增值税清算管理规程》（国税发〔2009〕91号）及相关规定，纳税人符合下列条件之一的，应进行土地增值税的清算：

1. **房地产开发项目全部竣工、完成销售的**：项目已竣工验收，且开发产品已全部售出
2. **整体转让未竣工决算房地产开发项目的**：将未完工的项目整体转让给其他单位或个人
3. **直接转让土地使用权的**：取得土地使用权后未进行开发即转让，或虽进行开发但尚未竣工即转让土地使用权

**实务判断要点**：
- "全部竣工"以取得竣工验收备案表为准
- "完成销售"指已签订销售合同并收讫价款或取得索取价款凭据
- 对于尾盘销售，如主体项目已符合清算条件，应一并清算

### 1.2 可要求清算情形

符合下列情形之一的，主管税务机关可要求纳税人进行土地增值税清算：

1. **已竣工验收的房地产开发项目，已转让的房地产建筑面积占整个项目可售建筑面积的比例在85%以上，或该比例虽未超过85%，但剩余的可售建筑面积已经出租或自用的**
2. **取得销售（预售）许可证满三年仍未销售完毕的**
3. **纳税人申请注销税务登记但未办理土地增值税清算手续的**
4. **省（自治区、直辖市、计划单列市）税务机关规定的其他情况**

**实务应对**：
- 关注当地税务机关的具体执行口径，各地对"可要求清算"的把握尺度不一
- 对于已符合可清算条件的项目，应主动评估清算义务，避免因税务机关要求清算而产生滞纳金风险
- 如项目存在大量自持或出租物业，需提前规划清算策略

### 1.3 清算程序

**自行清算程序**：
1. **清算准备**：整理项目资料，核算收入、成本、费用，计算增值额和应纳税额
2. **清算申报**：在符合应清算条件之日起90日内，到主管税务机关办理清算申报
3. **资料报送**：提交土地增值税清算申报表及其附表、项目竣工决算报表、取得土地使用权支付金额凭证、房地产开发成本凭证、与转让房地产有关的税金凭证等
4. **税款缴纳**：根据清算结果补缴或申请退税
5. **税务审核**：税务机关对清算申报进行审核，必要时进行纳税评估或税务稽查

**税务机关要求清算程序**：
1. 主管税务机关向纳税人下达《土地增值税清算通知书》
2. 纳税人在收到通知书之日起90日内办理清算手续
3. 后续程序同自行清算

**清算单位确定**：
- 土地增值税以国家有关部门审批的房地产开发项目为单位进行清算
- 对于分期开发的项目，以分期项目为单位清算
- 实务中，清算单位的确定对税负影响重大，需结合当地税务口径和项目规划许可证等资料确定

## 二、扣除项目审核要点

### 2.1 取得土地使用权支付金额

**扣除范围**：
- 纳税人为取得土地使用权所支付的地价款
- 按国家统一规定缴纳的有关费用（登记、过户手续费等）

**地价款的具体内容**：
- 以出让方式取得土地使用权的，为支付的土地出让金
- 以行政划拨方式取得土地使用权的，为按规定补缴的土地出让金
- 以转让方式取得土地使用权的，为支付的地价款
- 以其他方式取得土地使用权的，为支付的地价款和相关费用

**审核要点**：
- 检查土地出让合同、付款凭证、财政票据，确认地价款的真实性和准确性
- 检查契税完税凭证，契税计入取得土地使用权支付金额
- 关注土地价款的支付时间，逾期付款产生的滞纳金和利息不得扣除
- 如存在土地返还款、奖励款等，需关注是否冲减土地成本
- 成片受让土地使用权后分期开发的，应按合理方法分摊地价款
- 关注土地闲置费的税务处理（土地闲置费不得扣除）

### 2.2 房地产开发成本

房地产开发成本是指纳税人房地产开发项目实际发生的成本，包括：

**1. 土地征用及拆迁补偿费**：
- 土地征用费、耕地占用税、劳动力安置费
- 地上地下附着物拆迁补偿净支出、安置动迁用房支出、回迁差价支出
- 货币安置的拆迁补偿费
- **审核要点**：检查拆迁补偿协议、支付凭证、签收记录；关注回迁房视同销售的税务处理；检查是否存在虚增拆迁补偿费的情况

**2. 前期工程费**：
- 规划、设计、项目可行性研究和水文、地质、勘察、测绘、"三通一平"等支出
- **审核要点**：检查相关合同、发票、付款记录；关注费用发生的合理性和与项目的相关性；检查是否存在将期间费用混入前期工程费的情况

**3. 建筑安装工程费**：
- 以出包方式支付给承包单位的建筑安装工程费
- 以自营方式发生的建筑安装工程费
- **审核要点**：
  - 检查工程合同、结算书、发票、付款凭证
  - 关注甲供材的税务处理（甲供材金额计入开发成本，但施工方按差额开票）
  - 检查工程结算是否经过审核，有无虚增工程量或单价的情况
  - 关注关联方承包的工程定价是否公允
  - 检查质量保修金的处理（实际支付后方可扣除）

**4. 基础设施费**：
- 开发小区内道路、供水、供电、供气、排污、排洪、通讯、照明、环卫、绿化等工程发生的支出
- **审核要点**：检查合同、发票、工程结算资料；关注与建筑安装工程费的划分是否清晰

**5. 公共配套设施费**：
- 不能有偿转让的开发小区内公共配套设施发生的支出
- 如居委会、派出所、幼儿园、会所、停车场（库）、物业管理场所等
- **审核要点**：
  - 检查配套设施的性质，区分可售与不可售、有偿转让与不能有偿转让
  - 建成后产权属于全体业主所有的，成本可扣除
  - 建成后无偿移交给政府、公用事业单位用于非营利性社会公共事业的，成本可扣除
  - 建成后有偿转让的，应计算收入并扣除成本
  - 关注配套设施的成本分摊方法是否合理

**6. 开发间接费用**：
- 直接组织、管理开发项目发生的费用
- 包括工资、职工福利费、折旧费、修理费、办公费、水电费、劳动保护费、周转房摊销等
- **审核要点**：
  - 区分开发间接费用与期间费用（管理费用、销售费用、财务费用）
  - 开发间接费用仅指项目部层面发生的费用，公司总部费用不得计入
  - 检查费用发生的真实性和合理性

### 2.3 房地产开发费用

房地产开发费用是指与房地产开发项目有关的销售费用、管理费用、财务费用。

**扣除规则**：
- **能按转让房地产项目计算分摊并提供金融机构证明的**：利息支出据实扣除，但最高不能超过按商业银行同类同期贷款利率计算的金额；其他房地产开发费用按取得土地使用权支付金额与房地产开发成本金额之和的5%以内计算扣除
- **不能按转让房地产项目计算分摊利息支出或不能提供金融机构证明的**：房地产开发费用按取得土地使用权支付金额与房地产开发成本金额之和的10%以内计算扣除

**审核要点**：
- 检查利息支出的真实性和合规性，关注是否存在高利贷、关联方借款利率过高的情况
- 利息支出据实扣除的，需按转让项目计算分摊，检查分摊方法的合理性
- 超过贷款期限的利息和加罚的利息不得扣除
- 已计入房地产开发成本的利息支出，在计算房地产开发费用时不得重复扣除
- 关注企业选择扣除方式的合理性，通常选择对企业有利的方式，但需符合税法规定

### 2.4 与转让房地产有关的税金

**可扣除税金**：
- 城市维护建设税
- 教育费附加（含地方教育附加）
- 印花税（房地产开发企业缴纳的印花税列入管理费用，已相应扣除，不再单独扣除；非房地产开发企业可扣除）

**审核要点**：
- 检查税金计算是否正确，计税依据是否准确
- 关注增值税作为价外税，不作为计税依据，但附加税费以增值税为计税依据
- 检查税金的申报和缴纳情况

### 2.5 加计扣除

**扣除标准**：对从事房地产开发的纳税人，可按取得土地使用权支付金额与房地产开发成本金额之和加计20%扣除。

**适用条件**：
- 仅适用于房地产开发企业
- 适用于新建房转让
- 不适用：非房地产开发企业、转让旧房及建筑物、单纯转让土地使用权

**审核要点**：
- 确认纳税人身份是否为房地产开发企业
- 检查加计扣除的计算基数（仅包括地价款和开发成本，不包括开发费用和税金）
- 检查是否存在不得加计扣除的项目混入基数

## 三、增值额计算方法

### 3.1 收入确认

**收入总额**：纳税人转让房地产取得的全部价款及有关的经济收益，包括货币收入、实物收入和其他收入。

**不含税收入换算**：
- 适用一般计税方法的：转让房地产收入为不含增值税收入，即含税收入 /（1 + 9%）
- 适用简易计税方法的：转让房地产收入为不含增值税收入，即含税收入 /（1 + 5%）
- 免征增值税的：转让房地产收入为不含增值税收入，不得扣减增值税额

**特殊情形收入确认**：
- **视同销售**：企业将开发产品用于职工福利、奖励、对外投资、分配给股东或投资人、抵偿债务、换取其他单位和个人的非货币性资产等，发生所有权转移时应视同销售房地产
- **拆迁安置**：回迁房视同销售，按同类房地产平均价格或主管税务机关确定的价格确认收入；同时将此确认为房地产开发项目的拆迁补偿费
- **面积补差款**：计入转让收入
- **代收款项**：如计入房价向购买方一并收取的，应作为转让收入计税；如未计入房价单独收取的，可不作为转让收入

### 3.2 扣除项目归集

**归集原则**：
- 扣除项目金额必须是实际发生的、有合法有效凭证的支出
- 扣除项目应按清算项目归集，多个项目共同发生的支出应按合理方法分摊
- 未取得合法有效凭证的支出不得扣除

**分摊方法**：
- **占地面积法**：按各清算项目占地面积占总面积的比例分摊（适用于地价款）
- **建筑面积法**：按各清算项目建筑面积占总建筑面积的比例分摊（适用于公共配套设施费、开发间接费用等）
- **直接成本法**：能直接归属于某一清算项目的成本直接计入
- **税务机关确认的其他合理方法**

**不可扣除项目**：
- 取得土地使用权未支付地价款或不能提供已支付地价款凭据的
- 超过国家有关规定上浮幅度的利息支出
- 超过贷款期限的利息和加罚的利息
- 土地闲置费
- 开发企业预提的费用（除另有规定外）
- 与转让房地产无关的支出

### 3.3 增值额与增值率计算

**计算公式**：
- 增值额 = 转让房地产收入 - 扣除项目金额
- 增值率 = 增值额 / 扣除项目金额 x 100%

**税率适用**：

| 级数 | 增值率 | 税率 | 速算扣除系数 |
|------|--------|------|--------------|
| 1 | 不超过50%的部分 | 30% | 0 |
| 2 | 超过50%至100%的部分 | 40% | 5% |
| 3 | 超过100%至200%的部分 | 50% | 15% |
| 4 | 超过200%的部分 | 60% | 35% |

**应纳税额计算**：
- 应纳土地增值税 = 增值额 x 适用税率 - 扣除项目金额 x 速算扣除系数

**清算后转让**：
- 在土地增值税清算时未转让的房地产，清算后销售或有偿转让的，纳税人应按规定进行土地增值税纳税申报
- 扣除项目金额按清算时的单位建筑面积成本费用乘以销售或转让面积计算
- 单位建筑面积成本费用 = 清算时的扣除项目总金额 / 清算的总建筑面积

## 四、税收优惠政策应用

### 4.1 普通标准住宅优惠

**优惠政策**：纳税人建造普通标准住宅出售，增值额未超过扣除项目金额20%的，免征土地增值税。

**普通标准住宅认定**：
- 住宅小区建筑容积率在1.0以上
- 单套建筑面积在120平方米以下
- 实际成交价格低于同级别土地上住房平均交易价格1.2倍以下
- 具体标准由各省、自治区、直辖市根据实际情况制定

**实务要点**：
- 纳税人既建造普通住宅又建造其他类型房地产的，应分别核算增值额；不分别核算增值额或不能准确核算增值额的，其建造的普通标准住宅不适用免税规定
- 增值额超过扣除项目金额20%的，应就其全部增值额按规定计税（不是仅就超过部分计税）
- 企业在项目规划阶段应充分考虑普通标准住宅的认定标准，合理进行产品定位和成本分摊

**筹划风险提示**：
- 人为拆分项目、调节成本分摊以套用免税优惠存在税务风险
- 应确保分别核算的真实性和合理性，留存充分的证据资料

### 4.2 国家征用收回优惠

**优惠政策**：因国家建设需要依法征用、收回的房地产，免征土地增值税。

**适用情形**：
- 因城市实施规划、国家建设的需要而被政府批准征用的房产或收回的土地使用权
- 因城市实施规划、国家建设的需要而搬迁，由纳税人自行转让原房地产的

**审核要点**：
- 检查政府征用、收回的批准文件
- 检查补偿协议和补偿款支付凭证
- 自行转让原房地产的，需符合"因城市实施规划、国家建设需要而搬迁"的前提

### 4.3 旧房转让优惠

**旧房及建筑物认定**：
- 已使用一定年限的房产
- 达到固定资产折旧年限的房产
- 新建房交付使用后即视为旧房（各地口径不一）

**扣除方式**：
- 按房屋及建筑物的评估价格计算扣除（评估价格 = 重置成本价 x 成新度折扣率）
- 取得土地使用权所支付的地价款和按国家统一规定交纳的有关费用
- 转让环节缴纳的税金

**评估要求**：
- 需经当地税务机关确认的房地产评估机构评估
- 评估费用允许扣除

**特殊政策**：
- 个人销售住房：根据财税〔2008〕137号，对个人销售住房暂免征收土地增值税
- 企业改制重组：符合条件的改制重组过程中涉及房地产转移的，可暂不征土地增值税（需符合财税〔2021〕21号等文件规定）

## 五、清算申报要求

### 5.1 申报期限

**自行清算**：
- 应在满足应清算条件之日起90日内到主管税务机关办理清算手续
- 90日的起算时间：
  - 全部竣工完成销售的：以销售合同全部履行完毕之日或最后一份销售合同签订之日（各地执行口径不同）
  - 整体转让未竣工决算项目的：以转让合同签订之日
  - 直接转让土地使用权的：以转让合同签订之日

**要求清算**：
- 应在收到主管税务机关下发的清算通知之日起90日内办理清算手续

**逾期责任**：
- 未按规定期限办理清算手续的，主管税务机关可责令限期改正；逾期仍不改正的，按照《税收征收管理法》处理
- 可能涉及滞纳金、罚款等

### 5.2 资料清单

**纳税人办理土地增值税清算应报送的主要资料**：

1. **土地增值税清算表及其附表**
2. **项目基本情况说明**：项目立项、用地、开发、销售等情况
3. **项目竣工决算报表**：竣工验收报告、工程结算资料等
4. **取得土地使用权相关资料**：
   - 土地使用权出让（转让）合同
   - 土地出让金及相关费用支付凭证
   - 土地使用权证
   - 契税完税凭证
5. **房地产开发成本相关资料**：
   - 拆迁补偿协议及支付凭证
   - 前期工程合同及结算资料
   - 建筑安装工程合同、结算书、发票
   - 基础设施和公共配套设施建设资料
   - 开发间接费用归集资料
6. **与转让房地产有关的收入、税金资料**：
   - 销售合同台账
   - 销售收入明细表
   - 已缴纳税金凭证
7. **利息支出资料**：借款合同、利息支付凭证、金融机构证明等
8. **项目概算、预算资料**
9. **主管税务机关要求报送的其他资料**

**资料管理建议**：
- 房地产项目周期长，应建立完善的档案管理制度
- 从项目立项开始即系统收集、整理相关资料
- 合同、发票、付款凭证等应分类归档，便于清算时调阅
- 电子资料应备份保存，防止丢失

### 5.3 鉴证报告要求

**税务师事务所鉴证**：
- 部分地区要求纳税人在办理土地增值税清算时，附送税务师事务所出具的《土地增值税清算税款鉴证报告》
- 鉴证报告应对纳税人申报的土地增值税清算资料的真实性、准确性和完整性进行审核

**鉴证报告主要内容**：
- 项目基本情况
- 收入审核情况（销售收入、视同销售收入、其他收入）
- 扣除项目审核情况（地价款、开发成本、开发费用、税金、加计扣除）
- 增值额和应纳税额计算
- 税收优惠适用情况
- 鉴证结论

**选择鉴证机构的注意事项**：
- 选择具有资质、经验丰富的税务师事务所
- 提前与税务机关沟通，了解其对鉴证报告的具体要求
- 鉴证工作应尽早启动，避免临近申报期限才发现问题
- 对鉴证报告中的调整事项，应与鉴证机构充分沟通，了解调整原因和依据

**清算后续管理**：
- 清算审核结束后，如存在多缴税款的，可申请退税
- 清算后转让房地产的，按清算时的单位成本费用计算扣除
- 税务机关可能在清算后进行抽查或专项稽查，应保留资料备查
