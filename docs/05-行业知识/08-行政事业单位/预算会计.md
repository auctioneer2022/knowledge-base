# 预算会计

## 一、核心概念

预算会计是行政事业单位会计体系的重要组成部分，以收付实现制为核算基础，以预算执行为核心内容，反映单位预算收支的执行情况和结果。预算会计的核心目标是为预算管理服务，确保预算资金的合规、有效使用。

**预算管理一体化**：近年来，我国推行预算管理一体化改革，将预算编制、预算执行、会计核算、决算报告等环节纳入统一信息系统管理，实现预算数据全流程贯通。行政事业单位应适应一体化系统要求，规范预算会计操作。

**预算会计与财务会计的区别**：
| 对比维度 | 预算会计 | 财务会计 |
|----------|----------|----------|
| 核算基础 | 收付实现制 | 权责发生制 |
| 核算范围 | 纳入预算管理的现金收支 | 全部经济业务 |
| 会计要素 | 预算收入、预算支出、预算结余 | 资产、负债、净资产、收入、费用 |
| 报告目标 | 反映预算执行情况 | 反映财务状况和运行情况 |
| 计量属性 | 实际发生额 | 历史成本、公允价值等 |

## 二、预算编制流程

### 2.1 一上一下与二上二下程序

我国部门预算编制实行"二上二下"的基本程序，部分简单项目可采用"一上一下"程序。

**一上（单位申报）**：
- 单位根据下一年度工作任务和支出需求，编制预算建议数
- 包括基本支出预算和项目支出预算
- 基本支出按定员定额标准编制
- 项目支出需报送项目申报文本，包括立项依据、可行性论证、绩效目标等
- 报送时间：通常在上一年度的6-7月份

**一下（财政下达控制数）**：
- 财政部门根据财力状况、政策要求，对单位上报的预算建议数进行审核
- 下达预算支出控制数（基本支出控制数和项目支出控制数）
- 单位根据控制数调整预算建议数
- 下达时间：通常在上一年度的8-9月份

**二上（单位修改上报）**：
- 单位根据财政下达的控制数，修改完善预算草案
- 将修改后的预算草案上报主管部门和财政部门
- 项目支出预算需细化到具体项目和金额
- 报送时间：通常在上一年度的10-11月份

**二下（人大批准后批复）**：
- 财政部门汇总编制本级预算草案，报本级政府审定后提请人大审议
- 本级人大批准后，财政部门在法定时间内向各单位批复预算
- 单位收到批复预算后，分解下达内部预算
- 批复时间：通常在当年1-3月份

### 2.2 部门预算编制内容

部门预算编制实行"一个部门一本预算"，涵盖部门及所属各单位的所有收支。

**收入预算编制**：
- 财政拨款收入：根据财政部门下达的控制数和预算编制要求填列
- 事业收入：根据收费标准和预计业务量测算
- 上级补助收入：根据上级部门补助计划填列
- 经营收入：根据经营业务计划和预计收益填列
- 其他收入：根据实际情况合理预计

**支出预算编制**：
- 基本支出：包括人员经费和公用经费
  - 人员经费：按编制内实有人数、工资福利标准计算
  - 公用经费：按定员定额标准或分类分档定额标准计算
- 项目支出：包括专项业务费、大型修缮、购置、基础设施建设等
  - 需编制项目申报书，明确项目目标、实施计划、资金需求、绩效指标
  - 重大项目需进行事前绩效评估

**实务操作要点**：
- 预算编制应坚持"量入为出、收支平衡"原则，不得编制赤字预算
- 收入预算应积极稳妥，支出预算应统筹兼顾、保证重点
- 项目支出预算应细化到具体经济分类科目（如商品和服务支出、资本性支出等）
- 政府采购预算应与部门预算同步编制，凡使用财政性资金采购纳入集中采购目录或限额标准以上的货物、工程和服务，都应编制政府采购预算

**常见问题**：基本支出和项目支出如何区分？
**解决方法**：基本支出是保障单位机构正常运转和完成日常工作任务的支出，包括人员经费和公用经费，具有常规性、持续性的特点。项目支出是为完成特定工作任务或事业发展目标，在基本支出之外安排的支出，具有专项性、一次性的特点。判断标准主要看支出是否具有特定的工作任务目标，以及是否具有持续性。同一项支出不能同时列入基本支出和项目支出。

### 2.3 预算批复与分解

单位收到财政部门批复的预算后，应在规定时间内完成预算分解下达工作。

**预算批复流程**：
1. 财政部门向一级预算单位批复预算
2. 一级预算单位向所属二级单位批复预算
3. 各单位向内部科室或项目执行单位分解预算

**预算分解要求**：
- 按预算科目、项目、时间进度进行分解
- 明确各项目预算执行的责任主体
- 建立预算执行台账，跟踪预算执行进度

## 三、预算执行控制

### 3.1 国库集中支付

国库集中支付制度是预算执行的核心机制，包括财政直接支付和财政授权支付两种方式。

**财政直接支付**：
- 适用范围：工资支出、工程采购支出、物品和服务采购支出、转移支出等大额支出
- 支付流程：单位提出支付申请 → 财政国库支付机构审核 → 向代理银行发出支付指令 → 代理银行付款 → 与国库单一账户清算
- 会计核算：单位收到财政直接支付入账通知书时，同时进行财务会计和预算会计核算

**财政授权支付**：
- 适用范围：未纳入财政直接支付的零星支出和特殊专项支出
- 支付流程：财政部门批复授权支付额度 → 单位在额度内自行开具支付令 → 代理银行付款 → 与国库单一账户清算
- 额度管理：单位在核定的用款额度内，通过零余额账户办理支付
- 会计核算：额度到账时记"零余额账户用款额度"（财务会计）和"资金结存——零余额账户用款额度"（预算会计）；支付时冲减上述科目

**实务操作要点**：
- 单位应严格按预算批复的科目、项目、金额使用资金，不得擅自改变用途
- 财政直接支付申请应附完整、合规的原始凭证和支付依据
- 财政授权支付应在规定额度内使用，不得超额度支付
- 零余额账户日终余额必须为零，不得留存资金

**常见问题**：财政授权支付额度年内未使用完怎么办？
**解决方法**：年末，单位尚未使用的财政授权支付额度，由代理银行自动注销。单位在财务会计中借记"财政应返还额度——财政授权支付"，贷记"零余额账户用款额度"；在预算会计中借记"资金结存——财政应返还额度"，贷记"资金结存——零余额账户用款额度"。下年初，收到财政部门批复的上年未下达零余额账户用款额度时，做相反分录恢复额度。如财政部门不再下达上年结余额度，则财政应返还额度不再恢复，待财政批复后做相应处理。

### 3.2 政府采购

政府采购是预算执行的重要环节，单位使用财政性资金采购纳入集中采购目录或达到限额标准的货物、工程和服务，必须依法实行政府采购。

**采购方式**：
- 公开招标：主要采购方式，达到公开招标数额标准的必须采用
- 邀请招标：具有特殊性，只能从有限范围供应商采购的
- 竞争性谈判：招标后没有供应商投标或合格标的的
- 单一来源采购：只能从唯一供应商处采购的
- 询价：货物规格标准统一、现货货源充足、价格变化幅度小的
- 竞争性磋商：政府购买服务项目、技术复杂不能确定详细规格的

**采购程序**：
1. 编制政府采购预算（与部门预算同步编制）
2. 编报政府采购计划（明确采购项目、方式、时间等）
3. 组织实施采购（按法定程序开展采购活动）
4. 签订采购合同（在中标通知书发出后法定期限内签订）
5. 履约验收（按合同约定进行验收）
6. 资金支付（凭验收合格证明等资料申请支付）

**实务操作要点**：
- 未编报政府采购预算和计划的，不得组织采购活动
- 达到公开招标数额标准的项目，原则上不得采用非招标方式，确需改变的应报财政部门批准
- 采购合同应在采购文件发出后法定期限内签订，不得擅自变更合同实质性条款
- 采购资金支付应严格按合同约定和履约进度进行

### 3.3 预算执行进度管理

预算执行进度是衡量预算管理绩效的重要指标，单位应加强预算执行进度管理，提高财政资金使用效益。

**进度管理要求**：
- 均衡执行：避免年底突击花钱或大量结余
- 序时进度：通常要求月度执行进度不低于时间进度（如6月底执行进度不低于50%）
- 项目进度：项目支出应按项目实施计划及时支付

**影响执行进度的常见问题**：
- 预算编制不精准，项目前期准备不充分
- 政府采购程序周期长，影响资金支付
- 项目实施方案变更，需重新报批
- 项目验收滞后，尾款无法支付

**加快执行进度的措施**：
- 提前谋划项目，做好前期准备工作
- 合理选择采购方式，压缩采购周期
- 加强项目跟踪管理，及时解决执行中的问题
- 建立预算执行通报和约谈机制

**实务操作要点**：
- 单位应建立预算执行分析制度，定期分析预算执行进度和存在的问题
- 对于执行进度偏慢的项目，应分析原因并采取针对性措施
- 年底结余较多的项目，应向财政部门说明情况，按规定办理结转或收回
- 严禁为追求执行进度而违规支付、虚列支出

## 四、预算调整程序

### 4.1 调整条件

预算一经批准，原则上不得随意调整。确需调整的，应符合法定条件。

**可以调整预算的情形**：
- 上级增加或减少财政拨款
- 上级下达新增专项资金
- 因重大政策调整需要增减支出
- 因自然灾害等突发事件需要增加支出
- 其他经财政部门批准的预算调整事项

**不得调整预算的情形**：
- 单位自行改变预算资金用途
- 未经批准将项目支出调整为基本支出
- 未经批准在不同项目之间调剂使用资金
- 为规避政府采购或审批程序而拆分、调整预算

### 4.2 审批权限

预算调整实行分级审批管理。

**需报本级人大或其常委会审批的调整**：
- 预算总支出超过预算总收入需要增加举债数额的
- 需要调入预算稳定调节基金的
- 需要减少预算安排的重点支出数额的
- 其他涉及预算收支总规模变动的重大调整

**需报财政部门审批的调整**：
- 部门预算总额不变，项目之间调剂的
- 预算科目之间调整的
- 财政拨款结转结余资金的使用调整
- 新增或调整政府采购预算的

**单位内部审批**：
- 在已批复的预算总额和项目范围内，不同经济分类科目之间的调剂
- 具体支出事项的内部审批

### 4.3 调整流程

**预算调整的一般流程**：
1. **申请**：单位提出预算调整申请，说明调整原因、调整金额、调整方案
2. **审核**：主管部门审核后，报送财政部门
3. **审批**：财政部门审核并提出意见，按审批权限报批
4. **批复**：审批机关批准后，下达预算调整批复
5. **执行**：单位按调整后的预算执行

**实务操作要点**：
- 预算调整申请应提供充分的理由和依据
- 调整涉及项目变更的，应附项目变更说明和相关证明材料
- 预算调整批复下达前，不得擅自按调整后的方案执行
- 预算调整应在当年预算执行期内完成，跨年度调整需按结转结余规定办理

**常见问题**：项目支出经济分类科目之间调剂是否需要报批？
**解决方法**：根据《中央部门预算调剂管理办法》及地方相关规定，部门预算批复后，确需调剂的，应区分情况处理：
1. 基本支出人员经费与公用经费之间不得调剂
2. 基本支出与项目支出之间不得调剂
3. 不同项目之间调剂，需报财政部门审批
4. 同一项目内经济分类科目之间调剂，如金额较大的，需报财政部门备案或审批；金额较小的，可由单位按规定程序自行调剂
具体金额标准和审批程序应遵循本级财政部门的规定。

## 五、决算报告编制

### 5.1 年终清理结算

决算编制前，单位应进行全面清理结算，确保账实、账证、账账、账表相符。

**清理结算内容**：
- **核对年度预算**：核对财政批复的预算数与单位账面记录是否一致
- **清理核对各项收支**：核实各项收入是否全部入账，各项支出是否合规、准确
- **清理往来款项**：对暂收暂付款项进行全面清理，该结算的结算，该收回的收回，该退还的退还
- **清查货币资金**：核对库存现金、银行存款、零余额账户用款额度等
- **清查财产物资**：盘点固定资产、库存物品等，核实账实是否相符
- **核对财政拨款额度**：核对财政直接支付和授权支付额度使用及余额情况

**年终结账程序**：
1. 将本期所有经济业务登记入账
2. 进行期末对账，确保账证、账账相符
3. 按权责发生制原则，调整有关账项（如计提折旧、摊销，确认应收应付等）
4. 编制结账前资产负债表，试算平衡
5. 进行年终转账：将收入、支出科目余额转入结余科目
6. 编制结账后资产负债表，验证平衡

**实务操作要点**：
- 年终清理应在12月份集中进行，确保当年收支全部入账
- 跨年度的收入和支出，应按归属期确认，不得提前或延后
- 财政拨款结转结余应单独清理，按规定办理结转或上缴
- 清理中发现的问题，应及时处理并记录

### 5.2 决算报表编制

单位应根据预算会计账簿记录和相关资料，编制年度决算报表。

**主要决算报表**：
- **预算收入支出表**：反映单位年度预算收入、支出和结转结余总体情况
- **预算结转结余变动表**：反映各类预算结转结余的变动明细
- **财政拨款预算收入支出表**：反映财政拨款预算收支及结转结余情况
- **基本支出决算明细表**：按经济分类科目列示基本支出明细
- **项目支出决算明细表**：按项目列示项目支出明细

**编制要求**：
- 决算数据应与预算会计账簿数据一致
- 决算报表应与财政部门对账单核对一致
- 财政拨款数据应与国库集中支付系统数据一致
- 上下级单位之间的拨款和缴款应相互核对抵销

**实务操作要点**：
- 决算编制应做到数字真实、计算准确、内容完整、报送及时
- 决算报表中的预算数应填列年初批复预算数加上年中调整预算数
- 决算数应填列预算会计账簿中实际发生数
- 对预算执行差异较大的项目，应在决算说明中分析原因

### 5.3 决算分析报告

决算分析报告是对年度预算执行情况的全面总结和分析，是决算工作的重要组成部分。

**分析报告主要内容**：
- **预算执行总体情况**：收入完成率、支出完成率、结转结余情况
- **收入执行情况分析**：各项收入的完成情况及增减变动原因
- **支出执行情况分析**：基本支出和项目支出的完成情况，重点支出执行情况
- **结转结余情况分析**：结转结余资金的规模、构成及原因
- **预算绩效管理情况**：绩效目标完成情况、绩效评价结果及应用
- **存在的问题及改进措施**

**实务操作要点**：
- 决算分析应结合预算编制情况，对比分析预算执行差异
- 对结转结余资金规模较大的，应重点分析原因并提出改进措施
- 决算分析报告应与决算报表一并报送

## 六、预算绩效管理

### 6.1 绩效目标设定

预算绩效管理贯穿预算管理全过程，绩效目标是预算绩效管理的基础和起点。

**绩效目标分类**：
- 按照预算范围：基本支出绩效目标、项目支出绩效目标、部门整体支出绩效目标
- 按照时效性：中长期绩效目标、年度绩效目标

**绩效目标内容**：
- **产出指标**：反映预算资金预期提供的公共产品和服务情况，包括数量指标、质量指标、时效指标、成本指标
- **效益指标**：反映预算资金预期达到的产出效果，包括经济效益、社会效益、生态效益、可持续影响指标
- **满意度指标**：反映服务对象或受益人对预算支出效果的满意程度

**绩效目标设定要求**：
- 指向明确：绩效目标要符合法律法规规定、国民经济和社会发展规划、部门职能及事业发展规划
- 细化量化：绩效目标应细化分解为具体指标，尽量进行定量表述
- 合理可行：绩效目标应经过调查研究和科学论证，符合客观实际
- 相应匹配：绩效目标要与计划期内的任务数或计划数相对应，与预算确定的投资额或资金量相匹配

**实务操作要点**：
- 项目支出绩效目标应在项目申报时同步设定，作为项目入库和预算安排的前置条件
- 绩效目标应录入预算管理一体化系统，实现绩效目标与预算同步申报、同步审核、同步批复
- 绩效指标值应具有可衡量性，避免使用"明显改善""大幅提高"等模糊表述

**常见问题**：如何设定科学合理的绩效指标？
**解决方法**：设定绩效指标应遵循SMART原则：
- Specific（具体）：指标应明确具体，避免笼统
- Measurable（可衡量）：指标应可量化或可分等级评定
- Achievable（可实现）：指标值应切合实际，经过努力可以实现
- Relevant（相关性）：指标应与项目目标和预算支出密切相关
- Time-bound（时限性）：指标应有明确的时间要求
例如，"培训项目"的绩效指标可设定为：培训人数（≥100人）、培训合格率（≥95%）、培训完成时间（202X年X月前）、人均培训成本（≤500元/人）、参训人员满意度（≥90%）。

### 6.2 绩效监控

绩效监控是预算绩效管理的重要环节，通过对绩效目标实现程度和预算执行进度进行"双监控"，及时发现和纠正问题。

**监控内容**：
- 绩效目标实现情况：各项绩效指标的实际完成值与目标值的对比
- 预算执行进度：资金支付进度与项目实施进度的匹配情况
- 项目实施情况：项目是否按计划实施，是否存在偏离
- 财务管理情况：资金使用是否合规，会计核算是否规范

**监控方式**：
- 日常监控：单位自行跟踪监控，定期填报绩效监控信息
- 重点监控：财政部门或主管部门对重大项目、进度滞后项目开展重点监控
- 现场核查：对疑点项目进行现场核查，核实实际情况

**监控结果应用**：
- 对执行进度正常、绩效目标实现程度良好的项目，继续按原计划执行
- 对执行进度滞后、绩效目标实现存在偏差的项目，发出预警并督促整改
- 对问题严重的项目，暂停或收回预算资金

**实务操作要点**：
- 单位应建立绩效监控机制，明确监控责任主体和监控频次
- 绩效监控信息应及时录入预算管理一体化系统
- 对监控发现的问题，应分析原因并制定整改措施，及时反馈整改情况

### 6.3 绩效评价

绩效评价是对预算支出的经济性、效率性、效益性和公平性进行客观、公正的测量、分析和评判。

**评价类型**：
- **单位自评**：预算执行结束后，单位对预算支出的绩效目标完成情况进行自我评价
- **部门评价**：主管部门对所属单位或重点项目开展的评价
- **财政评价**：财政部门对重大政策、重点项目开展的评价

**评价程序**：
1. 确定评价对象和评价方式
2. 下达评价通知
3. 收集评价资料（预算申报资料、预算执行数据、财务数据、项目成果资料等）
4. 开展现场评价或非现场评价
5. 编制评价报告
6. 反馈评价结果

**评价方法**：
- 成本效益分析法：比较预算支出的成本和效益
- 比较法：通过对绩效目标与实施效果、历史情况与当期情况的比较，综合分析绩效目标实现程度
- 因素分析法：通过分析影响绩效目标实现的各种因素，评价绩效目标实现程度
- 最低成本法：对效益确定但不易计量的支出，比较提供同等服务或产品的不同方案的成本
- 公众评判法：通过专家评估、公众问卷、抽样调查等方式进行评判

**评价结果等级**：
通常分为四个等级：优（90分以上）、良（80-89分）、中（60-79分）、差（60分以下）。

### 6.4 评价结果应用

绩效评价结果应与预算安排、政策调整、改进管理相结合，形成"预算编制有目标、预算执行有监控、预算完成有评价、评价结果有应用"的闭环管理。

**结果应用方式**：
- **与预算安排挂钩**：对绩效好的项目，优先保障预算；对绩效差的项目，压减预算或取消项目
- **与政策调整挂钩**：对绩效评价发现的政策设计问题，及时调整完善相关政策
- **与改进管理挂钩**：针对评价发现的管理问题，制定整改措施，改进内部管理
- **与信息公开挂钩**：按规定公开绩效评价结果，接受社会监督
- **与考核问责挂钩**：将预算绩效管理情况纳入单位考核，对严重违规的追究责任

**实务操作要点**：
- 单位应在绩效评价工作完成后，及时整理归档评价资料
- 对评价发现的问题，应建立整改台账，明确整改责任、整改措施和整改时限
- 绩效评价结果应在单位内部通报，作为改进预算管理和财务管理的重要依据

## 七、常见问题及解决方法

**问题1：预算会计的支出与财务会计的费用为什么不一致？**
**解决方法**：预算会计支出与财务会计费用之间存在差异是正常现象，主要原因包括：
1. 核算基础不同：预算会计按收付实现制，财务会计按权责发生制
2. 核算范围不同：预算会计仅核算纳入预算管理的现金收支，财务会计核算全部费用
3. 确认时点不同：如预付账款在预算会计中支付时即确认为支出，在财务会计中按受益期分期确认费用；应付账款在财务会计中发生时即确认费用，在预算会计中实际支付时才确认支出
4. 非现金费用：如折旧、摊销等，财务会计确认为费用，预算会计不确认为支出
两者之间的差异可通过编制"本期预算结余与本期盈余差异调节表"进行分析和勾稽。

**问题2：项目支出预算执行中，可以自行在不同项目之间调剂资金吗？**
**解决方法**：不可以。项目支出预算一经批复，不得擅自调剂。确需调剂的，应按规定程序报批：
1. 同一部门内部不同项目之间调剂，需报财政部门审批
2. 跨部门调剂，需报本级政府或人大审批
3. 项目内部经济分类科目之间的调剂，按财政部门规定的金额标准和程序办理
未经批准擅自调剂项目资金的，属于违反预算管理的行为，应追究相关责任。

**问题3：年末财政拨款结转资金如何处理？**
**解决方法**：年末财政拨款结转资金的处理，应根据资金性质和财政部门规定办理：
1. 项目未执行完毕、下年需继续执行的，留在"财政拨款结转"科目，下年继续使用
2. 项目已执行完毕或因故终止的剩余资金，如属于项目支出结余，应按规定上缴财政或转入"财政拨款结余"
3. 基本支出结余，一般应收回财政，不再结转下年
4. 具体处理方式应遵循财政部门关于结转结余资金管理的有关规定

**问题4：政府采购合同签订后，可以变更合同金额吗？**
**解决方法**：政府采购合同签订后，原则上不得擅自变更、中止或终止合同。确需变更的，应符合以下条件：
1. 合同继续履行将损害国家利益和社会公共利益
2. 因不可抗力或情势变更导致合同无法按原约定履行
3. 采购量变更幅度在法定范围内（通常不超过10%）
4. 经采购人和供应商协商一致
变更合同应签订补充合同，并按原采购方式的审批程序报批。不得通过变更合同规避政府采购或追加与合同标的无关的内容。

**问题5：预算执行中发生不可预见的支出，如何列支？**
**解决方法**：预算执行中发生不可预见的支出，应按以下程序处理：
1. 首先在本单位预算内调剂解决，调剂不了的，申请追加预算
2. 追加预算需提供充分的理由和依据，包括支出事项的必要性、金额测算依据等
3. 按规定程序报批：一般支出追加报财政部门审批；重大支出追加报本级政府或人大审批
4. 在追加预算批复前，如情况紧急确需支付的，应按规定办理借款或垫付款手续，待预算批复后办理冲转
严禁无预算、超预算支出，严禁擅自改变预算资金用途。
