# 政府会计制度

## 一、核心概念

政府会计制度是规范行政事业单位会计核算的基本制度，自2019年1月1日起在全国行政事业单位全面施行。该制度重构了传统行政事业单位会计体系，建立了全新的"双体系、双基础、双报告"核算模式。

**双体系**：财务会计与预算会计适度分离又相互衔接的核算体系
**双基础**：财务会计采用权责发生制，预算会计采用收付实现制
**双报告**：编制财务报告和决算报告两套报告

**适用范围**：适用于各级各类行政单位和事业单位。纳入企业财务管理体系执行企业会计准则的事业单位，以及参照公务员法管理的事业单位除外（另有规定从其规定）。

## 二、双体系核算

### 2.1 财务会计与预算会计并行

政府会计制度要求单位对纳入部门预算管理的现金收支业务，同时进行财务会计和预算会计核算；对其他业务，仅进行财务会计核算。

**财务会计**：
- 核算基础：权责发生制
- 核算内容：单位全部经济业务或事项
- 计量属性：历史成本为主，公允价值为辅
- 会计要素：资产、负债、净资产、收入、费用

**预算会计**：
- 核算基础：收付实现制（国务院另有规定的，依照其规定）
- 核算内容：纳入部门预算管理的现金收支
- 会计要素：预算收入、预算支出、预算结余

**适度分离**：
- 两套体系各自独立核算、自成体系
- 财务会计反映单位的财务状况和运行情况，预算会计反映预算执行情况
- 会计要素、会计科目、核算基础、报告体系均相互独立

**相互衔接**：
- 同一笔业务在财务会计和预算会计中同时记录时，保持金额一致
- 通过"资金结存"科目建立预算会计与财务会计的对应关系
- 决算报告与财务报告之间存在勾稽关系

### 2.2 核算范围界定

**需要平行记账的业务**（同时进行财务会计和预算会计核算）：
- 财政直接支付方式下的各项支出
- 财政授权支付方式下的各项支出
- 从零余额账户提现、存现业务
- 收到财政拨款收入
- 取得纳入预算管理的非同级财政拨款
- 发生纳入预算管理的经营支出、事业支出等

**仅需财务会计核算的业务**：
- 不涉及现金收支的业务（如计提折旧、摊销，确认应收应付款项等）
- 受托代理的现金收支（如代管党费、团费）
- 应上缴财政的现金收支（如处置资产应上缴财政的收入）
- 现金形式往来款项（如暂收暂付款项，但年末需分析调整）

**实务操作要点**：
- 判断是否平行记账的核心标准是：是否纳入部门预算管理 + 是否发生现金收支
- 年末预算结转结余调整事项（如从非财政拨款结余提取专用基金），预算会计需单独记账
- 现金存入银行、从银行提取现金等货币资金内部形态转换，预算会计不做账务处理

**常见问题**：收到不涉及预算管理的往来款项（如押金、保证金），是否需要预算会计记账？
**解决方法**：不需要。此类业务虽发生现金收付，但不纳入部门预算管理，仅需进行财务会计核算。借记"银行存款"，贷记"其他应付款"等。预算会计不做账务处理。

## 三、平行记账规则

### 3.1 现金收支业务平行记账

对于纳入部门预算管理的现金收支业务，单位应当同时进行财务会计和预算会计核算，确保"一笔业务、两套分录"。

**典型业务的平行记账示例**：

**收到财政直接支付入账通知书时**：
```
财务会计：
借：库存物品 / 固定资产 / 业务活动费用等
贷：财政拨款收入

预算会计：
借：行政支出 / 事业支出——财政拨款支出
贷：财政拨款预算收入
```

**收到财政授权支付额度到账通知书时**：
```
财务会计：
借：零余额账户用款额度
贷：财政拨款收入

预算会计：
借：资金结存——零余额账户用款额度
贷：财政拨款预算收入
```

**使用零余额账户用款额度购买固定资产时**：
```
财务会计：
借：固定资产
贷：零余额账户用款额度

预算会计：
借：行政支出 / 事业支出——财政拨款支出
贷：资金结存——零余额账户用款额度
```

**收到上级主管部门拨付的非同级财政拨款时**：
```
财务会计：
借：银行存款
贷：非同级财政拨款收入

预算会计：
借：资金结存——货币资金
贷：非同级财政拨款预算收入
```

### 3.2 非现金业务仅财务会计记账

不涉及现金收支的业务，仅需进行财务会计核算，预算会计不做账务处理。

**典型业务示例**：

**计提固定资产折旧**：
```
财务会计：
借：业务活动费用 / 单位管理费用 / 经营费用等
贷：固定资产累计折旧

预算会计：不做账务处理
```

**确认应收账款**：
```
财务会计：
借：应收账款
贷：事业收入 / 经营收入等

预算会计：不做账务处理（待实际收到款项时再记账）
```

**取得无形资产**：
```
财务会计：
借：无形资产
贷：财政拨款收入 / 零余额账户用款额度 / 银行存款 / 应付账款等

预算会计：仅在支付款项时（发生现金收付）做账务处理
```

**实务操作要点**：
- 财务会计确认收入与预算会计确认预算收入的时点往往不一致
- 财务会计按权责发生制确认事业收入时，预算会计可能尚未收到款项，不做预算会计处理
- 收到以前年度已确认收入的款项时，财务会计冲减应收账款，预算会计确认预算收入

### 3.3 年末预算结转结余

年末，单位应当对预算会计的收支进行结转，形成各类预算结转结余。

**财政拨款结转结余**：
```
借：财政拨款预算收入
贷：财政拨款结转——本年收支结转

借：财政拨款结转——本年收支结转
贷：行政支出 / 事业支出（财政拨款支出部分）
```

完成上述结转后，将符合财政拨款结余性质的项目余额转入财政拨款结余：
```
借：财政拨款结转——累计结转
贷：财政拨款结余——结转转入
```

**非财政拨款结转**：
```
借：事业预算收入 / 上级补助预算收入 / 附属单位上缴预算收入 / 非同级财政拨款预算收入 / 债务预算收入 / 其他预算收入（专项资金收入）
贷：非财政拨款结转——本年收支结转

借：非财政拨款结转——本年收支结转
贷：行政支出 / 事业支出 / 其他支出（非财政拨款专项资金支出）
```

**非财政拨款结余**：
完成非财政拨款结转后，将留归本单位使用的非财政拨款专项资金剩余资金转入非财政拨款结余，以及将非专项资金收支转入非财政拨款结余：
```
借：事业预算收入 / 上级补助预算收入等（非专项资金收入）
贷：非财政拨款结余——本年收支结转

借：非财政拨款结余——本年收支结转
贷：行政支出 / 事业支出（非财政拨款非专项资金支出）
```

**实务操作要点**：
- 年末结转前，应核对财务会计的财政拨款收入与预算会计的财政拨款预算收入是否一致
- 如存在以前年度已入账但本年退回的财政拨款，应通过"财政拨款结转——归集调入/归集上缴"等明细科目调整
- 财政拨款结转结余资金应按财政部门规定进行管理，不得擅自使用结余资金

**常见问题**：年末财政拨款结转与财务会计的累计盈余为什么不一致？
**解决方法**：财政拨款结转是预算会计概念，反映财政拨款预算收支的累计差额；累计盈余是财务会计概念，反映单位全部经济业务形成的净资产累计变动。两者差异来源于：
1. 非现金业务（如折旧、摊销）影响累计盈余但不影响财政拨款结转
2. 非财政拨款业务影响累计盈余但不影响财政拨款结转
3. 财务会计与预算会计确认时点不同导致的暂时性差异
4. 应上缴财政的现金收支影响现金但不影响预算结转结余

## 四、会计科目设置

### 4.1 财务会计5大类科目

财务会计科目按照会计要素分为资产类、负债类、净资产类、收入类、费用类五大类。

**资产类科目**（主要科目）：
| 科目名称 | 核算内容 |
|----------|----------|
| 库存现金 | 单位的库存现金 |
| 银行存款 | 存入银行或其他金融机构的各种存款 |
| 零余额账户用款额度 | 财政授权支付方式下的用款额度 |
| 财政应返还额度 | 年终注销的财政直接支付额度与授权支付额度 |
| 应收账款 | 开展业务活动应收的款项 |
| 预付账款 | 按合同预付的款项 |
| 库存物品 | 库存材料、产品、包装物、低值易耗品等 |
| 固定资产 | 使用年限超过1年、单位价值在规定标准以上的资产 |
| 固定资产累计折旧 | 固定资产的累计折旧 |
| 无形资产 | 专利权、商标权、著作权、土地使用权等 |
| 无形资产累计摊销 | 无形资产的累计摊销 |
| 公共基础设施 | 政府会计主体控制的公共基础设施 |
| 政府储备物资 | 政府会计主体控制的战略及应急物资 |
| 保障性住房 | 单位控制的保障性住房原值 |
| 受托代理资产 | 受托管理的各项资产 |

**负债类科目**（主要科目）：
| 科目名称 | 核算内容 |
|----------|----------|
| 短期借款 | 期限在1年内的借款 |
| 应交增值税 | 应交纳的增值税 |
| 其他应交税费 | 除增值税外的其他税费 |
| 应缴财政款 | 应上缴财政的款项 |
| 应付职工薪酬 | 应付给职工及为职工支付的各项薪酬 |
| 应付账款 | 应付未付的款项 |
| 预收账款 | 按合同预收的款项 |
| 长期借款 | 期限超过1年的借款 |
| 长期应付款 | 偿还期限超过1年的应付款项 |
| 预计负债 | 确认的或有事项产生的义务 |
| 受托代理负债 | 受托管理的各项负债 |

**净资产类科目**：
| 科目名称 | 核算内容 |
|----------|----------|
| 累计盈余 | 历年实现的盈余扣除盈余分配后滚存的金额 |
| 专用基金 | 按规定提取或设置的具有专门用途的净资产 |
| 权益法调整 | 长期股权投资采用权益法核算时的调整项目 |
| 本期盈余 | 本期各项收入、费用相抵后的余额 |
| 本年盈余分配 | 本年盈余的分配（或弥补亏损）情况 |
| 无偿调拨净资产 | 无偿调入或调出非现金资产形成的净资产变动 |
| 以前年度盈余调整 | 以前年度盈余的调整事项 |

**收入类科目**：
财政拨款收入、事业收入、上级补助收入、附属单位上缴收入、经营收入、非同级财政拨款收入、投资收益、捐赠收入、利息收入、租金收入、其他收入。

**费用类科目**：
业务活动费用、单位管理费用、经营费用、资产处置费用、上缴上级费用、对附属单位补助费用、所得税费用、其他费用。

### 4.2 预算会计3大类科目

预算会计科目按照会计要素分为预算收入类、预算支出类、预算结余类三大类。

**预算收入类科目**：
财政拨款预算收入、事业预算收入、上级补助预算收入、附属单位上缴预算收入、经营预算收入、债务预算收入、非同级财政拨款预算收入、投资预算收益、其他预算收入。

**预算支出类科目**：
行政支出、事业支出、经营支出、上缴上级支出、对附属单位补助支出、投资支出、债务还本支出、其他支出。

**预算结余类科目**：
| 科目名称 | 核算内容 |
|----------|----------|
| 资金结存 | 纳入部门预算管理的资金的流入、流出、调整和滚存情况 |
| 财政拨款结转 | 财政拨款收支相抵后的滚存余额 |
| 财政拨款结余 | 同级财政拨款项目支出结余资金 |
| 非财政拨款结转 | 非财政拨款专项资金收支相抵后的滚存余额 |
| 非财政拨款结余 | 非财政拨款非专项资金收支相抵后的滚存余额 |
| 专用结余 | 从非财政拨款结余中提取的专用基金 |
| 经营结余 | 经营活动收支相抵后的余额 |
| 其他结余 | 除财政拨款收支、非财政拨款专项资金收支、经营收支外的其他各项收支相抵后的余额 |
| 非财政拨款结余分配 | 非财政拨款结余的分配情况 |

### 4.3 科目对照关系

预算会计与财务会计之间存在明确的科目对应关系，主要通过"资金结存"科目实现衔接。

**资金结存明细科目与财务会计科目对应**：
| 资金结存明细 | 对应财务会计科目 |
|-------------|-----------------|
| 资金结存——零余额账户用款额度 | 零余额账户用款额度 |
| 资金结存——货币资金 | 库存现金、银行存款、其他货币资金 |
| 资金结存——财政应返还额度 | 财政应返还额度 |

**预算收入与财务会计收入的对应关系**：
| 预算会计科目 | 财务会计对应科目 |
|-------------|-----------------|
| 财政拨款预算收入 | 财政拨款收入 |
| 事业预算收入 | 事业收入（专项资金部分） |
| 非同级财政拨款预算收入 | 非同级财政拨款收入 |
| 其他预算收入 | 捐赠收入、利息收入、租金收入等 |

**预算支出与财务会计费用的对应关系**：
| 预算会计科目 | 财务会计对应科目 |
|-------------|-----------------|
| 行政支出 / 事业支出 | 业务活动费用、单位管理费用等 |
| 经营支出 | 经营费用 |
| 其他支出 | 其他费用、资产处置费用等 |

**实务操作要点**：
- 日常核算中，应熟练掌握平行记账的科目对应关系
- 月末、年末对账时，应核对"资金结存"各明细科目余额与对应财务会计科目余额是否一致
- 财政应返还额度在财务会计和预算会计中均设置科目，年末余额应一致

## 五、财务报告编制要求

### 5.1 财务会计报告

财务会计报告是反映单位某一特定日期财务状况和某一会计期间运行情况的报告，包括会计报表和附注。

**资产负债表**：
- 反映单位在某一特定日期全部资产、负债和净资产的情况
- 编制基础：权责发生制
- 平衡公式：资产 = 负债 + 净资产
- 编制方法：根据各资产、负债、净资产类科目的期末余额直接或分析填列

**收入费用表**：
- 反映单位在某一会计期间内发生的收入、费用及当期盈余情况
- 编制基础：权责发生制
- 计算路径：本期收入 - 本期费用 = 本期盈余
- 编制方法：根据各收入、费用类科目的本期发生额直接或分析填列

**净资产变动表**：
- 反映单位在某一会计期间内净资产各项目变动的情况
- 展示累计盈余、专用基金、权益法调整、本期盈余、本年盈余分配、无偿调拨净资产、以前年度盈余调整等项目的年初余额、本年变动和年末余额

**现金流量表**（仅事业单位编制，行政单位可选编）：
- 反映单位在某一会计年度内现金流入和流出的情况
- 编制基础：收付实现制（但属于财务会计报告体系）
- 分三类活动列示：日常活动、投资活动、筹资活动的现金流量

**实务操作要点**：
- 资产负债表中的"货币资金"项目，应包括库存现金、银行存款、零余额账户用款额度、其他货币资金
- 固定资产应扣除累计折旧后按账面价值列示
- 收入费用表中的"业务活动费用"应按经济分类或功能分类在附注中披露明细

### 5.2 预算会计报告

预算会计报告是反映单位某一会计年度预算收支执行结果的报告，包括预算收入支出表、预算结转结余变动表和财政拨款预算收入支出表。

**预算收入支出表**：
- 反映单位在某一会计年度内预算收入、预算支出和预算结转结余情况
- 编制基础：收付实现制
- 计算路径：年初预算结转结余 + 预算收入 - 预算支出 = 年末预算结转结余

**预算结转结余变动表**：
- 反映单位在某一会计年度内预算结转结余的变动情况
- 详细展示财政拨款结转、财政拨款结余、非财政拨款结转、非财政拨款结余、专用结余、经营结余、其他结余各项目的变动明细

**财政拨款预算收入支出表**：
- 反映单位在某一会计年度内财政拨款预算收入、支出和结转结余情况
- 按基本支出和项目支出分别列示
- 基本支出再按人员经费和公用经费细分

**实务操作要点**：
- 预算收入支出表中的预算收入合计应与预算会计账簿中预算收入类科目本年发生额合计一致
- 财政拨款预算收入支出表中的数据应与财政决算数据核对一致
- 年末预算结转结余各项目余额应与预算会计账簿余额一致

### 5.3 报告编制时间要求

- 月度、季度报表：月度终了后5个工作日内、季度终了后10个工作日内编制
- 年度报表：年度终了后按照财政部门规定的时间编制，通常需在次年1-2个月内完成
- 部门决算：按财政部统一部署的时间节点编报

**常见问题**：财务会计的累计盈余与预算会计的预算结转结余之间是什么关系？
**解决方法**：累计盈余与预算结转结余之间不存在直接的相等关系，但可以通过编制"本期预算结余与本期盈余差异调节表"进行勾稽。主要差异调节项包括：
1. 当期确认为收入但没有确认为预算收入的金额（如应收款项确认的收入）
2. 当期确认为预算收入但没有确认为收入的金额（如收到预收账款）
3. 当期确认为费用但没有确认为预算支出的金额（如计提折旧）
4. 当期确认为预算支出但没有确认为费用的金额（如预付账款）
5. 应收应付、预收预付等往来款项变动
6. 应上缴财政的现金收支
7. 受托代理的现金收支

## 六、常见问题及解决方法

**问题1：平行记账时，财务会计和预算会计的记账金额为什么不一致？**
解决方法：平行记账的业务，财务会计和预算会计的记账金额应当一致。如果出现不一致，常见原因包括：
1. 记账时点不一致，如财务会计暂估入账但预算会计尚未付款
2. 错误地只对一套体系记账，漏记另一套
3. 对业务性质判断错误，不应平行记账的业务进行了平行记账
4. 科目使用错误，导致金额拆分不一致
对账发现不一致时，应逐笔核对平行记账凭证，找出差异原因并更正。

**问题2：固定资产计提折旧时预算会计不做账，年末预算结余是否虚增？**
解决方法**：不会虚增。预算会计采用收付实现制，固定资产折旧不涉及现金收付，因此不做预算会计处理。预算结余反映的是预算资金的实际收支差额，与固定资产的实物消耗无关。固定资产的价值补偿通过未来购置新资产时的预算支出来实现。

**问题3：单位收到押金、保证金等暂收款项，预算会计是否记账？**
解决方法**：押金、保证金等暂收暂付款项，虽发生现金收付，但不纳入部门预算管理，因此预算会计不做账务处理。仅在财务会计中核算：收到时借记"银行存款"，贷记"其他应付款"；退回时做相反分录。

**问题4：如何处理财务会计已确认收入但预算会计尚未收到款项的情况？**
解决方法**：财务会计按权责发生制确认收入，借记"应收账款"，贷记"事业收入"等；预算会计因尚未收到款项，不做账务处理。待实际收到款项时，财务会计借记"银行存款"，贷记"应收账款"；预算会计借记"资金结存——货币资金"，贷记"事业预算收入"等。

**问题5：年末零余额账户用款额度注销后，下年初如何恢复？**
解决方法**：年末，财政授权支付额度余额由代理银行自动注销。财务会计借记"财政应返还额度——财政授权支付"，贷记"零余额账户用款额度"；预算会计借记"资金结存——财政应返还额度"，贷记"资金结存——零余额账户用款额度"。下年初，收到财政部门批复的上年未下达零余额账户用款额度时，做相反分录恢复额度。
