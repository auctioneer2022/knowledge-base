# 行政事业单位会计知识

行政事业单位会计核算实行政府会计制度，其核心特征是"双体系、双基础、双报告"模式。行政单位和事业单位需遵循《政府会计制度——行政事业单位会计科目和报表》（财会〔2017〕25号）及相关补充规定，同时开展财务会计与预算会计的平行记账。

## 主题导航

### [政府会计制度](政府会计制度.md)

系统阐述行政事业单位双体系核算框架，涵盖平行记账规则、会计科目设置逻辑以及财务会计报告与预算会计报告的编制要求，帮助理解行政事业单位会计核算的基础架构。

### [预算会计](预算会计.md)

围绕部门预算管理全周期展开，包括预算编制流程、预算执行控制机制、预算调整程序、决算报告编制规范以及预算绩效管理的实务操作要点。
