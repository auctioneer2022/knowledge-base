# 批发零售

批发零售行业是连接生产与消费的核心环节，具有交易频繁、存货量大、销售方式多样、现金流动快等特点。该行业的会计核算需重点关注收入确认的时点与方式、各类折扣返利的税务处理、存货的精细化管理以及资金的高效周转。

## 主题导航

### [收入确认](./收入确认.md)

涵盖不同销售方式下的收入确认原则，包括直销、代销、联营、特许经营及电商销售；退换货处理的会计与税务操作；商业折扣、现金折扣、销售返利及平销返利的核算方法与增值税处理要点。

### [存货管理](./存货管理.md)

涵盖存货分类管理方法（ABC分类法、品类管理、批次管理）、库存周转率分析、存货计价方法选择、库存积压预警机制构建以及供应链协同管理模式（VMI、JIT、安全库存计算）。
