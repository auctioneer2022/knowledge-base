# 高新技术

高新技术企业以技术创新为核心驱动力，其财务核算和税务管理具有研发费用占比高、知识产权密集、政策优惠性强等显著特征。本模块围绕高新技术企业的核心财务与认定问题，聚焦研发费用归集和高新认定两大主题。

## 主题目录

### [研发费用归集](./研发费用归集.md)

系统规范高新技术企业研发费用的核算口径与归集方法，包括：
- 研发活动界定标准（创造性运用科学技术新知识、实质性改进）
- 研发费用归集范围与口径（人员人工、直接投入、折旧摊销、设计费、装备调试费、无形资产摊销、其他费用）
- 资本化与费用化划分原则（研究阶段vs开发阶段、资本化五条件）
- 研发费用辅助账设置规范（辅助账格式、科目设置、凭证要求）

### [高新认定](./高新认定.md)

全面解读高新技术企业认定的条件、材料准备与核心指标计算，包括：
- 认定条件详解（成立时间、知识产权、技术领域、科技人员占比、研发投入占比、高新技术产品收入占比、创新能力评价、无重大安全/质量/环境违法）
- 核心知识产权要求（类型、获取方式、有效性、关联性）
- 科技人员占比计算（统计口径、兼职人员处理、累计工作时间）
- 研发投入占比要求（近三年研发费用总额占销售收入总额比例，按收入规模分档）
- 申报材料准备指南（材料清单、编写要点、常见被否决原因）
