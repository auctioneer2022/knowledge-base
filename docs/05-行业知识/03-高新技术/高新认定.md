# 高新认定

## 一、认定条件详解

根据《高新技术企业认定管理办法》（国科发火〔2016〕32号）及《高新技术企业认定管理工作指引》（国科发火〔2016〕195号），高新技术企业认定需同时满足以下八项条件。

### 1.1 成立时间

**认定条件**：企业申请认定时须注册成立一年以上。

**具体要求**：
- 企业须在中国境内（不含港、澳、台地区）注册
- 申请认定时成立时间需满365个日历天数
- 注册资本和实缴资本应符合公司章程规定

**实务要点**：
- 新成立企业需提前规划，确保在申请时满足成立时间要求
- 外商投资企业、港澳台投资企业符合条件也可申请
- 企业重组、分立的，承继原企业主要经营活动且符合条件的，可以承继原企业的认定资格（需重新认定）

### 1.2 知识产权

**认定条件**：企业通过自主研发、受让、受赠、并购等方式，获得对其主要产品（服务）在技术上发挥核心支持作用的知识产权的所有权。

**具体要求**：
- 知识产权须在中国境内授权或审批审定
- 知识产权权属人应为申请企业
- 知识产权在申请高新技术企业及高新技术企业资格存续期内须有效
- 按II类评价的知识产权在申请高新技术企业时，仅限使用一次
- 在申请高新技术企业及高新技术企业资格存续期内，知识产权有多个权属人时，只能由一个权属人在申请时使用

**知识产权分类**：
- **I类知识产权**：发明专利（含国防专利、植物新品种、国家级农作物品种、国家新药、国家一级中药保护品种、集成电路布图设计专有权）
- **II类知识产权**：实用新型专利、外观设计专利、软件著作权（不含商标）

**评分标准（知识产权部分，满分30分）**：
- 技术的先进程度（8分）
- 对主要产品（服务）在技术上发挥核心支持作用（8分）
- 知识产权数量（8分）：1项及以上I类知识产权可得7-8分；5项及以上II类知识产权可得5-6分
- 知识产权获得方式（6分）：自主研发可得5-6分，受让/受赠/并购可得1-3分

**实务要点**：
- I类知识产权有效期长、含金量高，建议企业重视发明专利布局
- 知识产权应与主要产品（服务）的核心技术密切相关
- 受让取得的知识产权需关注转让手续的完整性和有效性
- 知识产权须在有效期内，临近期满的应及时续展或重新申请

### 1.3 技术领域

**认定条件**：对企业主要产品（服务）发挥核心支持作用的技术属于《国家重点支持的高新技术领域》规定的范围。

**国家重点支持的高新技术领域（八大领域）**：
1. 电子信息
2. 生物与新医药
3. 航空航天
4. 新材料
5. 高技术服务
6. 新能源与节能
7. 资源与环境
8. 先进制造与自动化

**实务要点**：
- 企业应准确判断自身技术所属的领域和子领域
- 技术领域的选择应基于核心技术的实质，而非产品的表面形态
- 一个企业的技术可能涉及多个领域，应选择发挥核心支持作用的主要领域
- 技术领域选择不当是认定被否的常见原因之一

### 1.4 科技人员占比

**认定条件**：企业从事研发和相关技术创新活动的科技人员占企业当年职工总数的比例不低于10%。

**具体解释**：
- **科技人员**：直接从事研发和相关技术创新活动，以及专门从事上述活动的管理和提供直接技术服务的人员
- **职工总数**：包括在职、兼职和临时聘用人员。在职人员通过企业是否签订了劳动合同或缴纳社会保险费来鉴别；兼职、临时聘用人员全年须在企业累计工作183天以上
- **当年**：指企业申报前1个会计年度

**计算口径**：
- 科技人员占比 = 科技人员数 / 职工总数 x 100%
- 科技人员数：按当年累计实际工作时间在183天以上的人员计算
- 职工总数：按全年月平均数计算，月平均数 =（月初数 + 月末数）/ 2

**实务要点**：
- 科技人员需有明确的岗位职责和工作内容，与研发活动相关
- 兼职和临时聘用人员需有劳动合同或劳务协议，且累计工作时间须满183天
- 科技人员学历、专业背景应与从事的研发工作相匹配
- 需留存科技人员名单、劳动合同、社保缴纳记录、工时记录等资料

### 1.5 研发投入占比

**认定条件**：企业近三个会计年度（实际经营期不满三年的按实际经营时间计算）的研究开发费用总额占同期销售收入总额的比例符合如下要求：

| 最近一年销售收入 | 研发投入占比要求 |
|------------------|------------------|
| 小于5,000万元（含） | 不低于5% |
| 5,000万元至2亿元（含） | 不低于4% |
| 2亿元以上 | 不低于3% |

**具体要求**：
- 企业在中国境内发生的研究开发费用总额占全部研究开发费用总额的比例不低于60%
- 委托外部研究开发费用按照实际发生额的80%计入委托方研发费用总额
- 研发费用归集范围按《工作指引》规定执行

**销售收入口径**：
- 销售收入为主营业务收入与其他业务收入之和
- 主营业务收入与其他业务收入按照企业所得税年度纳税申报表的口径计算
- 销售收入以企业所得税年度纳税申报表中的数据为准

**研发费用口径**：
- 人员人工费用
- 直接投入费用
- 折旧费用与长期待摊费用
- 无形资产摊销
- 设计费用
- 装备调试费用与试验费用
- 委托外部研究开发费用（按80%计入）
- 其他费用（不超过研发费用总额的20%）

**实务要点**：
- 研发费用应按项目分别核算，设置辅助账
- 委托境外研发的，按照费用实际发生额的80%计入，且不超过境内符合条件的研发费用的2/3
- 研发费用占比按近三年合计数计算，不是每年都必须达到比例要求
- 但建议每年均保持达标，以确保持续符合认定条件

### 1.6 高新技术产品（服务）收入占比

**认定条件**：近一年高新技术产品（服务）收入占企业同期总收入的比例不低于60%。

**高新技术产品（服务）收入**：
- 企业通过研发和相关技术创新活动，取得的产品（服务）收入与技术性收入的总和
- **产品收入**：企业自行研制、开发、生产的高新技术产品形成的收入
- **技术性收入**：技术转让收入、技术服务收入、接受委托研究开发收入

**总收入口径**：
- 总收入 = 收入总额 - 不征税收入
- 收入总额按照企业所得税法第六条规定计算，包括销售货物收入、提供劳务收入、转让财产收入、股息红利等权益性投资收益、利息收入、租金收入、特许权使用费收入、接受捐赠收入、其他收入
- 不征税收入按照企业所得税法第七条规定计算

**实务要点**：
- 高新技术产品收入应有知识产权支撑，且知识产权与产品收入存在对应关系
- 需编制高新技术产品收入明细表，列示产品名称、收入金额、对应知识产权
- 产品收入与技术性收入应有合同、发票等凭证支持
- 注意区分高新技术产品收入与普通产品收入

### 1.7 创新能力评价

**认定条件**：企业创新能力评价应达到相应要求。

**创新能力评价指标体系（满分100分，综合得分须达到70分以上）**：

| 一级指标 | 分值 | 二级指标 | 分值 |
|----------|------|----------|------|
| 知识产权 | 30 | 技术的先进程度 | 8 |
| | | 对主要产品（服务）在技术上发挥核心支持作用 | 8 |
| | | 知识产权数量 | 8 |
| | | 知识产权获得方式 | 6 |
| 科技成果转化能力 | 30 | 科技成果转化能力 | 30 |
| 研究开发组织管理水平 | 20 | 制定了企业研究开发的组织管理制度，建立了研发投入核算体系，编制了研发费用辅助账 | 6 |
| | | 设立了内部科学技术研究开发机构并具备相应的科研条件，与国内外研究开发机构开展多种形式产学研合作 | 6 |
| | | 建立了科技成果转化的组织实施与激励奖励制度，建立开放式的创新创业平台 | 4 |
| | | 建立了科技人员的培养进修、职工技能培训、优秀人才引进，以及人才绩效评价奖励制度 | 4 |
| 企业成长性 | 20 | 净资产增长率 | 10 |
| | | 销售收入增长率 | 10 |

**科技成果转化能力评分**：
- 科技成果转化是指为提高生产力水平而对科技成果进行的后续试验、开发、应用、推广直至形成新产品、新工艺、新材料，发展新产业等活动
- 同一科技成果分别在国内外转化的，或转化为多个产品、服务、工艺、样品、样机等的，只计为一项
- 评分标准：
  - 转化能力强，5项以上（25-30分）
  - 转化能力较强，4项（19-24分）
  - 转化能力一般，3项（13-18分）
  - 转化能力较弱，2项（7-12分）
  - 转化能力弱，1项（1-6分）
  - 转化能力无，0项（0分）

**企业成长性评分**：
- 净资产增长率 = 1/2 x（第二年末净资产 / 第一年末净资产 + 第三年末净资产 / 第二年末净资产）- 1
- 销售收入增长率 = 1/2 x（第二年销售收入 / 第一年销售收入 + 第三年销售收入 / 第二年销售收入）- 1
- 增长率大于35%得9-10分，增长率在25%-35%得7-8分，增长率在15%-25%得5-6分，增长率在5%-15%得3-4分，增长率在0-5%得1-2分，增长率为负得0分

### 1.8 无重大安全、质量、环境违法

**认定条件**：企业申请认定前一年内未发生重大安全、重大质量事故或严重环境违法行为。

**具体要求**：
- "申请认定前一年内"是指申请前的365天之内（含申报年）
- 重大安全事故、重大质量事故、严重环境违法行为的认定以相关部门的行政处罚决定为依据
- 受到警告、通报批评等轻微处罚的，一般不影响认定

**实务要点**：
- 企业应建立健全安全生产、质量管理、环境保护制度
- 定期自查，及时整改隐患
- 如发生过相关事故，应在事故处理完毕、整改到位后再申请认定

## 二、核心知识产权要求

### 2.1 知识产权类型

**I类知识产权（优先）**：
- 发明专利（含国防专利）
- 植物新品种
- 国家级农作物品种
- 国家新药
- 国家一级中药保护品种
- 集成电路布图设计专有权

**II类知识产权**：
- 实用新型专利
- 外观设计专利
- 软件著作权

### 2.2 获取方式

**获取方式与评分关系**：
- **自主研发**：得分最高（5-6分），体现企业持续创新能力
- **受让、受赠、并购**：得分较低（1-3分），仅证明企业拥有知识产权

**实务建议**：
- 企业应注重自主研发，建立知识产权战略规划
- 对于受让取得的知识产权，应确保转让手续合法有效，权属清晰
- 并购取得的知识产权，应在并购协议中明确知识产权的转让条款

### 2.3 有效性要求

**有效性标准**：
- 知识产权在申请认定时处于有效状态
- 知识产权在高新技术企业资格存续期内持续有效
- 知识产权须按规定缴纳年费，及时办理续展手续

**失效风险**：
- 未按时缴纳专利年费导致专利权终止
- 专利被宣告无效
- 软件著作权保护期满后进入公有领域

### 2.4 关联性要求

**核心支持作用**：
- 知识产权应对企业主要产品（服务）的核心技术发挥支持作用
- 知识产权与高新技术产品（服务）收入之间应存在对应关系
- 企业应能够说明知识产权如何应用于产品（服务）的生产或服务提供

**证明材料**：
- 知识产权证书复印件
- 知识产权与产品的对应关系说明
- 知识产权应用情况的证明材料（如技术报告、产品说明书）

## 三、科技人员占比计算

### 3.1 统计口径

**科技人员定义**：
- 直接从事研发活动的人员
- 专门从事研发活动管理和提供直接技术服务的人员
- 包括研究人员、技术人员、辅助人员

**非科技人员**：
- 企业行政管理人员（除非专职负责研发管理）
- 市场营销人员
- 生产一线操作人员（除非参与试制、试验等研发活动）
- 财务人员、人力资源人员（除非专职为研发部门服务）

### 3.2 兼职人员处理

**兼职人员计入条件**：
- 与企业签订正式劳动合同或劳务协议
- 全年在企业累计实际工作时间达到183天以上
- 从事的工作内容属于研发或相关技术创新活动

**工时统计方法**：
- 打卡记录
- 工作日志
- 项目工时记录表
- 其他能够证明实际工作时间的记录

### 3.3 累计工作时间

**183天计算规则**：
- 按自然年度计算（1月1日至12月31日）
- 累计实际工作时间，不要求连续
- 包括在企业工作的工作日、加班日以及因研发活动出差的时间

**注意事项**：
- 兼职和临时聘用人员须全年累计达到183天方可计入科技人员和职工总数
- 职工总数按全年月平均数计算，与科技人员的183天口径不同
- 企业应建立完善的考勤和工时记录制度，以备核查

## 四、研发投入占比要求

### 4.1 分档标准

| 档次 | 最近一年销售收入 | 近三年研发费用占销售收入比例 |
|------|------------------|------------------------------|
| 第一档 | S ≤ 5,000万元 | ≥ 5% |
| 第二档 | 5,000万元 < S ≤ 2亿元 | ≥ 4% |
| 第三档 | S > 2亿元 | ≥ 3% |

**销售收入确定**：
- 以企业申报认定前最近一个会计年度的销售收入为准
- 销售收入 = 主营业务收入 + 其他业务收入
- 以企业所得税年度纳税申报表中的数据为准

### 4.2 研发费用归集口径

**高新认定研发费用归集范围**：

1. **人员人工费用**：企业科技人员的工资薪金、基本养老保险费、基本医疗保险费、失业保险费、工伤保险费、生育保险费和住房公积金，以及外聘科技人员的劳务费用
2. **直接投入费用**：直接消耗的材料、燃料和动力费用；用于中间试验和产品试制的模具、工艺装备开发及制造费；不构成固定资产的样品、样机及一般测试手段购置费；试制产品的检验费；用于研发活动的仪器、设备的运行维护、调整、检验、维修等费用；通过经营租赁方式租入的用于研发活动的仪器、设备租赁费
3. **折旧费用与长期待摊费用**：用于研发活动的仪器、设备的折旧费；用于研发活动的仪器、设备的租赁费（经营租赁）；研发设施的改建、改装、装修和修理过程中发生的长期待摊费用
4. **无形资产摊销**：用于研发活动的软件、专利权、非专利技术的摊销费用
5. **设计费用**：为新产品和新工艺进行构思、开发和制造，进行工序、技术规范、操作特性方面的设计等发生的费用；包括创意设计费、造型设计费、工艺流程设计费等
6. **装备调试费用与试验费用**：工装准备过程中研究开发活动所发生的费用；新药研制的临床试验费、勘探开发技术的现场试验费、田间试验费等
7. **委托外部研究开发费用**：委托境内外其他机构或个人进行研发活动所发生的费用。委托外部研发费用按照实际发生额的80%计入，委托境内研发不超过总额的80%；委托境外研发不超过境内符合条件的研发费用总额的2/3，且不超过总额的80%
8. **其他费用**：与研发活动直接相关的其他费用，包括技术图书资料费、资料翻译费、专家咨询费、高新科技研发保险费、研发成果的检索、论证、评审、鉴定、验收费用、知识产权的申请费、注册费、代理费、会议费、差旅费、通讯费等。此项费用一般不得超过研究开发总费用的20%

### 4.3 境内研发费用占比

**要求**：企业在中国境内发生的研究开发费用总额占全部研究开发费用总额的比例不低于60%。

**计算口径**：
- 境内研发费用 / 全部研发费用 ≥ 60%
- 全部研发费用包括境内研发费用和委托境外研发费用（按实际发生额计算，不按80%折算）

**实务要点**：
- 委托境外个人研发的，不得加计扣除，但如符合条件可计入高新认定研发费用
- 企业应准确区分境内研发和境外研发，留存相关合同和付款凭证

## 五、申报材料准备指南

### 5.1 材料清单

**高新技术企业认定申请材料主要包括**：

1. **高新技术企业认定申请书**：在线填写并打印，加盖企业公章
2. **企业依法成立证明材料**：营业执照副本复印件
3. **知识产权相关材料**：
   - 知识产权证书复印件
   - 知识产权摘要或权利要求书
   - 知识产权与高新技术产品关联性说明
4. **科研项目立项证明**：
   - 科研项目立项决议、立项报告、立项审批文件
   - 项目结题报告或阶段性报告
5. **科技成果转化证明材料**：
   - 科技成果转化总体情况与转化形式说明
   - 科技成果证明材料（专利证书、软件著作权、查新报告、鉴定证书等）
   - 转化结果证明材料（销售合同、发票、用户报告、检测报告等）
6. **研究开发组织管理证明材料**：
   - 研发组织管理制度文件
   - 研发投入核算体系文件
   - 研发费用辅助账示例
   - 研发机构设立证明
   - 产学研合作协议
   - 科技成果转化激励制度
   - 科技人员培养、引进、绩效评价制度
7. **高新技术产品（服务）证明材料**：
   - 高新技术产品（服务）的关键技术和技术指标说明
   - 相关的生产批文、认证认可和资质证书
   - 产品质量检验报告
   - 产品销售合同和发票
8. **企业职工和科技人员情况说明材料**：
   - 企业职工和科技人员情况说明
   - 科技人员名单及其工作岗位
   - 劳动合同或社保缴纳证明
9. **经具有资质的中介机构出具的企业近三个会计年度研发费用专项审计或鉴证报告**：
   - 研发费用结构明细表
   - 研发费用辅助账审核说明
10. **经具有资质的中介机构出具的企业近一个会计年度高新技术产品（服务）收入专项审计或鉴证报告**
11. **经具有资质的中介机构鉴证的企业近三个会计年度的财务会计报告**：
    - 会计报表
    - 会计报表附注
    - 财务情况说明书
12. **近三个会计年度企业所得税年度纳税申报表**：
    - 主表
    - 附表A101010（一般企业收入明细表）
    - 附表A104000（期间费用明细表）
    - 附表A107012（研发费用加计扣除优惠明细表，如有）
    - 其他相关附表

### 5.2 编写要点

**申请书填写要点**：
- 企业基本信息应准确完整，与营业执照一致
- 技术领域选择应准确，与企业核心技术相符
- 人员结构数据应与科技人员说明材料一致
- 研发费用和高新产品收入数据应与专项审计报告一致

**知识产权材料编写要点**：
- 列出所有知识产权清单，注明类型、授权号、授权日期、有效期
- 重点说明I类知识产权及其先进性
- 说明知识产权与主要产品（服务）核心技术的对应关系

**科技成果转化材料编写要点**：
- 编制科技成果转化汇总表，列明成果名称、成果形式、转化时间、转化结果
- 每项成果附证明材料（专利证书+销售合同/发票）
- 成果转化时间应在近三年内
- 同一成果转化为多个产品的，按一项成果计算

**研发组织管理材料编写要点**：
- 制度文件应完整、规范，有签发日期和签发人
- 辅助账示例应真实反映企业的核算方式
- 产学研合作协议应在有效期内，有实质性合作内容

**专项审计报告要点**：
- 选择具有资质、经验丰富的事务所
- 审计报告应符合高新技术企业认定管理工作指引的格式要求
- 研发费用审计报告应包括近三年每年的研发费用明细
- 高新产品收入审计报告应列明高新技术产品收入明细及占比计算

### 5.3 常见被否决原因

**知识产权方面**：
- 知识产权与主要产品核心技术关联性不强
- 知识产权全部为受让取得，且无自主研发能力证明
- 知识产权已失效或在认定期间失效
- 知识产权权属不清晰，存在纠纷

**技术领域方面**：
- 技术领域选择错误，不属于国家重点支持的高新技术领域
- 技术描述过于笼统，无法体现技术先进性

**人员方面**：
- 科技人员占比未达到10%
- 科技人员学历、专业与研发工作不匹配
- 科技人员工时记录不完整，无法证明实际工作时间
- 将非研发人员计入科技人员

**研发费用方面**：
- 研发费用占比未达到要求
- 研发费用归集不准确，将非研发费用计入研发费用
- 境内研发费用占比低于60%
- 研发费用辅助账设置不规范
- 研发费用专项审计报告存在问题（如口径错误、计算错误）

**高新产品收入方面**：
- 高新产品收入占比未达到60%
- 高新产品收入认定不准确，将普通产品收入计入
- 高新产品收入与知识产权对应关系不清晰
- 缺乏充分的收入证明材料

**成长性方面**：
- 净资产或销售收入增长率为负，导致成长性得分为0
- 财务数据波动异常，引起审核人员质疑

**申报材料方面**：
- 申报材料不完整，缺少必要附件
- 申报数据前后不一致（如申请书、审计报告、纳税申报表之间数据不符）
- 申报材料逻辑混乱，无法自圆其说

**其他原因**：
- 申请认定前一年内发生重大安全、质量事故或严重环境违法行为
- 存在科研不端行为或知识产权侵权纠纷
- 企业被列入经营异常名录或严重违法失信企业名单

**续期提醒**：
- 高新技术企业资格有效期为三年
- 期满前三个月内应提出重新认定申请
- 资格有效期内每年应在"高新技术企业认定管理工作网"填报年度发展情况报表
- 未按期报送年度报表的，可能被取消高新技术企业资格
