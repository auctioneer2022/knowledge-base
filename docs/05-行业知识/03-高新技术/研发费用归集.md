# 研发费用归集

## 一、研发活动界定标准

### 1.1 研发活动的定义

根据《财政部 国家税务总局 科技部关于完善研究开发费用税前加计扣除政策的通知》（财税〔2015〕119号）及相关规定，研发活动是指企业为获得科学与技术新知识，创造性运用科学技术新知识，或实质性改进技术、产品（服务）、工艺而持续进行的具有明确目标的系统性活动。

**核心要素**：
- **创新性**：获得新知识、创造性运用知识或实质性改进
- **系统性**：有计划、有目标、有组织地进行
- **持续性**：不是偶发性的活动
- **明确目标**：有具体的技术、产品或工艺改进目标

### 1.2 适用加计扣除的研发活动范围

**允许加计扣除的活动**：
1. 企业自主、委托、合作研发项目
2. 新技术、新产品、新工艺的研发
3. 实质性改进技术、产品（工艺）的研发
4. 对引进技术的消化吸收再创新

**不适用税前加计扣除政策的活动**：
1. 企业产品（服务）的常规性升级
2. 对某项科研成果的直接应用（如直接采用公开的新工艺、材料、装置、产品、服务或知识等）
3. 企业在商品化后为顾客提供的技术支持活动
4. 对现存产品、服务、技术、材料或工艺流程进行的重复或简单改变
5. 市场调查研究、效率调查或管理研究
6. 作为工业（服务）流程环节或常规的质量控制、测试分析、维修维护
7. 社会科学、艺术或人文学方面的研究

**特殊行业限制**：
- 烟草制造业、住宿和餐饮业、批发和零售业、房地产业、租赁和商务服务业、娱乐业不得享受研发费用加计扣除政策

### 1.3 研发活动判断要点

**实质性改进的判断标准**：
- 是否提升了产品性能、质量或降低了成本
- 是否形成了新的技术秘密或专利技术
- 是否突破了现有技术瓶颈
- 是否产生了新的工艺方法或生产流程

**研发项目立项要求**：
- 企业应建立研发项目管理制度，对研发项目进行立项管理
- 研发项目应有立项决议、研发计划、预算安排
- 研发项目应有明确的目标、预期成果和技术路线
- 多个研发项目应分别核算，确保费用归集的准确性

**会计与税务口径差异**：
- 会计核算的研发活动范围通常较宽，企业管理需要即可归集
- 税务加计扣除有明确的负面清单，部分会计上的研发活动不得享受加计扣除
- 高新技术企业认定中的研发费用口径与加计扣除口径存在差异，需分别核算

## 二、研发费用归集范围与口径

高新技术企业认定和研发费用加计扣除对研发费用的归集范围既有共性也有差异。以下分别列示主要归集项目。

### 2.1 人员人工费用

**归集范围**：
- 直接从事研发活动人员的工资薪金、基本养老保险费、基本医疗保险费、失业保险费、工伤保险费、生育保险费和住房公积金
- 外聘研发人员的劳务费用

**高新认定口径**：
- 包括企业科技人员的工资薪金、五险一金及外聘科技人员的劳务费用
- 科技人员累计实际工作时间在183天以上

**加计扣除口径**：
- 包括直接从事研发活动人员的工资薪金、五险一金
- 外聘研发人员的劳务费用（需签订劳务协议或合同）

**核算要点**：
- 准确划分研发人员与生产、销售、管理人员
- 对同时参与多个项目的人员，按实际工时比例合理分配其费用
- 研发人员参与非研发活动的，相关费用不得计入研发费用
- 留存研发人员花名册、工时记录、工资分配表等资料

**常见问题**：
- 将非研发人员（如高管、行政人员）费用计入研发费用
- 未按实际工时比例分配兼岗人员费用
- 外聘人员未签订正式合同，无法提供有效凭证

### 2.2 直接投入费用

**归集范围**：
- 研发活动直接消耗的材料、燃料和动力费用
- 用于中间试验和产品试制的模具、工艺装备开发及制造费
- 不构成固定资产的样品、样机及一般测试手段购置费
- 试制产品的检验费
- 用于研发活动的仪器、设备的运行维护、调整、检验、维修等费用
- 通过经营租赁方式租入的用于研发活动的仪器、设备租赁费

**高新认定与加计扣除差异**：
- 高新认定口径较宽，包括直接消耗的材料、燃料动力、试制检验费等
- 加计扣除对直接投入有更严格的要求，强调与研发活动的直接相关性

**核算要点**：
- 建立研发领料制度，研发领料单应注明项目名称、用途
- 研发材料与生产材料应严格区分，不得混用
- 试制产品对外销售的，其材料费用不得重复计入研发费用（销售时冲减研发费用或计入成本）
- 动力费用按合理标准（如工时、设备功率）分摊至各研发项目

**常见问题**：
- 生产用料与研发用料划分不清
- 试制产品销售未作相应处理，导致研发费用虚增
- 动力费用分摊标准不合理

### 2.3 折旧费用与长期待摊费用

**归集范围**：
- 用于研发活动的仪器、设备的折旧费
- 用于研发活动的仪器、设备的租赁费（经营租赁）
- 用于研发活动的固定资产的改建、改装、装修和修理过程中发生的长期待摊费用

**核算要点**：
- 研发用固定资产应明确标识，建立台账管理
- 固定资产同时用于研发和非研发活动的，按实际使用工时或面积等合理标准分摊折旧
- 研发用固定资产的认定应有充分的证据支持（如设备清单、使用记录）
- 加速折旧的固定资产，按税前扣除的折旧金额计算加计扣除

**常见问题**：
- 将非研发用固定资产折旧计入研发费用
- 分摊标准不合理或缺乏依据
- 研发设备闲置期间折旧仍全额计入研发费用

### 2.4 设计费用

**归集范围**：
- 为新产品和新工艺进行构思、开发和制造，进行工序、技术规范、操作特性方面的设计等发生的费用
- 包括创意设计费、造型设计费、工艺流程设计费等

**高新认定口径**：
- 为获得创新性产品而进行的设计费用
- 包括工业设计、结构设计、电路设计等

**核算要点**：
- 设计费用应有明确的设计任务书、设计合同
- 常规的产品包装设计、广告设计不得计入研发费用
- 委托外部设计的，应签订技术开发合同或设计合同

### 2.5 装备调试费用与试验费用

**归集范围**：
- 工装准备过程中研究开发活动所发生的费用，包括研制特殊、专用的生产机器，改变生产和质量控制程序，或制定新方法及标准等活动所发生的费用
- 新药研制的临床试验费、勘探开发技术的现场试验费、田间试验费等

**核算要点**：
- 装备调试费应限于研发阶段的调试，量产阶段的调试费不得计入
- 临床试验费应有完整的试验方案、伦理审查批件、试验记录
- 试验费用应与研发项目直接相关，有明确的试验目的和结果

### 2.6 无形资产摊销

**归集范围**：
- 用于研发活动的软件、专利权、非专利技术（包括许可证、专有技术、设计和计算方法等）的摊销费用

**核算要点**：
- 无形资产应专门用于或部分用于研发活动
- 部分用于研发活动的无形资产，按合理标准分摊摊销额
- 外购的无形资产用于研发的，其购置成本及后续摊销可计入研发费用
- 自主研发形成的无形资产，其资本化金额在摊销时计入研发费用

**常见问题**：
- 将生产经营用软件摊销计入研发费用
- 无形资产用途划分不清，分摊依据不足

### 2.7 其他费用

**归集范围**：
- 技术图书资料费、资料翻译费、专家咨询费
- 高新科技研发保险费
- 研发成果的检索、分析、评议、论证、鉴定、评审、评估、验收费用
- 知识产权的申请费、注册费、代理费
- 差旅费、会议费
- 职工福利费、补充养老保险费、补充医疗保险费（加计扣除口径）

**高新认定口径限制**：
- 其他费用一般不得超过研究开发总费用的20%
- 单一项目或整体研发费用的其他费用占比均需关注

**加计扣除口径限制**：
- 其他相关费用总额不得超过可加计扣除研发费用总额的10%
- 计算基数不包括其他相关费用本身

**计算公式**：
- 其他相关费用限额 =（人员人工 + 直接投入 + 折旧摊销 + 设计费 + 装备调试费 + 无形资产摊销）/（1 - 10%）x 10%

**核算要点**：
- 会议费、差旅费应有完整的审批手续和相关凭证
- 专家咨询费应签订咨询协议，留存专家资质证明
- 知识产权费用应与研发项目直接相关
- 严格控制其他费用的比例，避免超标

## 三、资本化与费用化划分原则

### 3.1 研究阶段与开发阶段的划分

根据《企业会计准则第6号——无形资产》，企业内部研究开发项目的支出应区分研究阶段支出与开发阶段支出。

**研究阶段**：
- 定义：为获取新的科学或技术知识并理解它们而进行的独创性的有计划调查
- 特点：计划性、探索性、不确定性较大
- 会计处理：全部费用化，计入当期损益（管理费用——研发费用）

**开发阶段**：
- 定义：在进行商业性生产或使用前，将研究成果或其他知识应用于某项计划或设计，以生产出新的或具有实质性改进的材料、装置、产品等
- 特点：针对性、形成成果的可能性较大
- 会计处理：符合资本化条件的，确认为无形资产；不符合条件的，费用化

**阶段划分的实务判断**：
- 研究阶段通常包括：资料收集、文献调研、理论分析、实验室基础研究
- 开发阶段通常包括：小试、中试、工业化试验、产品原型开发
- 企业应建立研发阶段划分制度，明确各项目的阶段划分标准
- 同一项目在不同年份可能处于不同阶段，应动态调整

### 3.2 资本化五条件

开发阶段支出同时满足下列条件的，才能确认为无形资产：

**1. 完成该无形资产以使其能够使用或出售在技术上具有可行性**
- 有充分的技术资料和支持证明技术方案可行
- 已完成必要的实验室测试或小试
- 关键技术难点已解决或已有明确解决方案

**2. 具有完成该无形资产并使用或出售的意图**
- 企业管理层有明确的商业化计划
- 有具体的市场推广或内部使用计划
- 不是纯粹的基础研究或探索性项目

**3. 无形资产产生经济利益的方式**
- 自用：能够提高生产效率、降低成本、提升产品质量
- 对外出售：有明确的市场需求和客户群体
- 使用：能够用于生产产品或提供服务并获得收益

**4. 有足够的技术、财务资源和其他资源支持**
- 有专业的研发团队和技术储备
- 有充足的资金支持项目完成
- 有必要的设备和场地条件

**5. 归属于该无形资产开发阶段的支出能够可靠地计量**
- 建立了完善的研发费用核算体系
- 能够准确归集和分配各项目的直接支出
- 有完整的原始凭证和账簿记录

### 3.3 资本化时点的确定

**资本化开始时点**：
- 同时满足上述五个条件时，开发阶段支出开始资本化
- 通常以项目通过中期评审、进入中试阶段或取得关键技术突破作为资本化开始时点
- 资本化开始时点应有书面决议或评审记录作为依据

**资本化终止时点**：
- 无形资产达到预定用途时停止资本化
- 通常以取得专利证书、软件著作权登记证书、产品定型、技术验收通过等为标志
- 研发失败的项目，已资本化的金额应转入当期损益

**会计处理**：
- 费用化支出：借：研发支出——费用化支出，贷：原材料/应付职工薪酬/银行存款等；期末结转：借：管理费用——研发费用，贷：研发支出——费用化支出
- 资本化支出：借：研发支出——资本化支出，贷：原材料/应付职工薪酬/银行存款等；达到预定用途时：借：无形资产，贷：研发支出——资本化支出

### 3.4 税务与会计差异处理

**会计处理与税务处理的差异**：
- 会计上：研究阶段全部费用化；开发阶段符合条件的资本化，不符合条件的费用化
- 税务上：符合条件的研发费用，无论会计上资本化还是费用化，均可享受加计扣除

**加计扣除口径**：
- 费用化的研发费用：按实际发生额的100%（或120%，根据最新政策）在税前扣除，并在此基础上加计扣除
- 资本化的研发费用：按无形资产成本的200%（或根据最新政策）在税前摊销

**高新技术企业认定口径**：
- 高新认定中的研发费用口径通常比加计扣除口径宽
- 企业应分别建立会计核算、加计扣除、高新认定三套研发费用归集体系，或在一套体系中设置不同口径的取数规则

## 四、研发费用辅助账设置规范

### 4.1 辅助账格式

根据国家税务总局公告2015年第97号、2021年第28号等文件要求，企业应设置研发费用辅助账，准确归集研发费用。

**辅助账类型**：
- 自主研发"研发支出"辅助账
- 委托研发"研发支出"辅助账
- 合作研发"研发支出"辅助账
- 集中研发"研发支出"辅助账

**辅助账基本格式要素**：
1. 项目基本信息：项目名称、项目编号、研发形式、资本化/费用化时点、项目实施状态
2. 费用明细：凭证信息、费用类别（人员人工、直接投入、折旧摊销等）、金额
3. 费用小计和合计
4. 委托研发的项目，还应包括受托方信息、关联关系、委托金额等

### 4.2 科目设置

**会计科目体系**：

一级科目：研发支出
- 二级科目：费用化支出 / 资本化支出
  - 三级科目：按研发项目设置（如项目A、项目B）
    - 四级科目：人员人工、直接投入、折旧费用、设计费用、装备调试费、无形资产摊销、其他费用

**辅助核算设置**：
- 按项目辅助核算：确保每个研发项目的费用独立归集
- 按部门辅助核算：便于统计研发部门整体费用
- 按费用类别辅助核算：便于分类统计和取数

**建议账簿设置**：
1. **总账**："研发支出"总账科目
2. **明细账**：按费用化/资本化、按项目、按费用类别设置明细账
3. **辅助账簿**：按税务机关要求的格式设置辅助账（可采用台账或电子表格形式）
4. **项目台账**：记录每个项目的立项、进展、验收、成果等信息

### 4.3 凭证要求

**原始凭证要求**：
- 人员人工：工资表、社保缴纳凭证、个税申报表、工时记录表
- 直接投入：领料单（注明项目名称、用途）、采购发票、入库单
- 折旧摊销：固定资产卡片、折旧计算表、无形资产摊销表
- 委托研发：技术开发（委托）合同、付款凭证、受托方研发费用明细、受托方税务登记信息
- 其他费用：发票、合同、审批单、会议通知/纪要、差旅审批单等

**记账凭证要求**：
- 摘要应注明研发项目名称和费用类别
- 附件应完整，能够支撑费用的真实性和相关性
- 涉及多个项目的共同费用，应附费用分配表
- 资本化与费用化的划分应有明确的依据和审批

**备查资料要求**：
- 研发项目立项决议、立项报告
- 研发项目预算
- 研发人员名单及资质
- 研发设备清单
- 研发项目阶段性报告或中期检查报告
- 研发成果证明（专利证书、软件著作权、验收报告等）
- 委托研发的技术合同登记证明

**电子化管理建议**：
- 建立研发费用管理信息系统，实现费用的实时归集和监控
- 设置预警功能，对费用超标、比例异常等情况及时提醒
- 与财务系统、项目管理系统、人力资源系统对接，实现数据自动抓取
- 定期备份数据，确保资料的完整性和安全性

**常见违规情形**：
- 未设置辅助账或辅助账设置不规范
- 原始凭证不齐全，无法证明费用的真实性和相关性
- 费用归集不准确，将非研发费用混入研发费用
- 资本化与费用化划分依据不充分，随意调节利润
- 委托研发未留存受托方明细和登记证明
