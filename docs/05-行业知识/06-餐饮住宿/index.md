# 餐饮住宿

餐饮住宿行业具有交易频繁单笔金额小、现金交易占比高、存货保质期短、人工成本占比大、涉税风险点多等特点。该行业的会计核算与财务管理需重点关注现金收付的规范性、收入确认的完整性、成本费用的真实性以及税务合规性。
