# 制造业成本核算

## 一、成本构成分析

制造业产品成本由直接材料、直接人工和制造费用三大要素构成，三者共同构成产品的制造成本（生产成本）。

### 1.1 直接材料

直接材料是指构成产品实体的原材料以及有助于产品形成的主要材料和辅助材料。核算要点：

- **领用计量**：采用实际成本法或计划成本法核算，领用时按批次或加权平均法结转成本
- **退料处理**：生产剩余材料应及时办理退库手续，冲减生产成本
- **废料回收**：可回收废料应按估计价值冲减材料成本
- **共同耗用材料**：多种产品共同耗用的材料，按定额消耗量、产量比例等合理标准分配

### 1.2 直接人工

直接人工是指直接从事产品生产的工人职工薪酬。核算要点：

- **计件工资**：按产量记录和计件单价直接计入产品成本
- **计时工资**：按工时比例或定额工时比例在不同产品间分配
- **福利费归集**：按工资总额计提的职工福利费、社会保险费、住房公积金等随同工资一并分配
- **加班工资**：直接生产工人的加班工资应计入直接人工，但需关注合理性

### 1.3 制造费用

制造费用是指企业为生产产品和提供劳务而发生的各项间接费用。核算要点：

- **费用范围**：包括车间管理人员薪酬、折旧费、修理费、办公费、水电费、机物料消耗、劳动保护费、季节性和修理期间的停工损失等
- **归集方式**：按车间或生产单位设置明细账，归集当期发生的间接费用
- **分配方法**：常用分配标准包括生产工人工资比例法、生产工人工时比例法、机器工时比例法、年度计划分配率法等
- **余额处理**：采用年度计划分配率法时，年末应将制造费用余额调整计入12月份产品成本

## 二、成本计算方法

### 2.1 品种法

品种法是以产品品种作为成本计算对象，归集和分配生产成本的方法。

**适用范围**：大量大批单步骤生产，或管理上不要求分步骤计算成本的多步骤生产（如发电、采掘、水泥等）。

**计算程序**：
1. 按产品品种设置生产成本明细账
2. 归集直接材料、直接人工
3. 归集并分配制造费用
4. 月末如有在产品，将生产费用在完工产品与在产品之间分配
5. 计算完工产品总成本和单位成本

**在制品分配方法**：约当产量法、定额比例法、在产品按定额成本计价法、在产品按所耗直接材料计价法等。

### 2.2 分批法

分批法是按照产品批别归集生产费用、计算产品成本的方法，又称订单法。

**适用范围**：单件小批生产，如船舶制造、重型机械、精密仪器、专用设备、新产品试制等。

**计算特点**：
- 成本计算对象：产品批号或订单号
- 成本计算期：与生产周期一致，与会计报告期不一致
- 一般不存在完工产品与在产品之间的费用分配问题（但跨月陆续完工时需采用适当方法分配）

**跨月完工处理**：对于批内产品跨月陆续完工的情况，可采用计划成本、定额成本或最近一期实际成本对完工产品进行暂估结转，待该批产品全部完工后再计算实际总成本和单位成本，但不调整已结转成本。

### 2.3 分步法

分步法是按照产品的生产步骤归集和分配生产费用、计算各步骤半成品和最后产成品成本的方法。

**适用范围**：大量大批多步骤生产，且管理上要求分步骤核算成本的企业，如纺织、冶金、机械制造等。

#### 2.3.1 逐步结转分步法

逐步结转分步法按照产品加工顺序，逐步计算并结转半成品成本，直到最后步骤计算出产成品成本。

**综合结转法**：将上步骤半成品成本以"直接材料"或"半成品"项目综合转入下步骤成本计算单。优点是简化核算，缺点是不能反映原始成本构成。需进行成本还原。

**分项结转法**：将上步骤半成品成本按原始成本项目分别转入下步骤相应成本项目。优点是不需要成本还原，缺点是核算工作量大。

**成本还原**：从最后一个步骤起，将所耗用上步骤半成品的综合成本逐步分解为原始成本项目，直至还原为按原始成本项目反映的产成品成本。还原方法通常按本月所产该种半成品的成本结构进行还原。

#### 2.3.2 平行结转分步法

平行结转分步法各步骤只归集本步骤发生的费用，不计算也不结转半成品成本，只计算本步骤发生的费用中应计入产成品成本的份额，月末将各步骤份额平行汇总计算产成品成本。

**特点**：
- 各步骤成本计算单只登记本步骤发生的费用
- 半成品实物转移但成本不结转
- 月末将各步骤费用在产成品与广义在产品之间分配
- 产成品成本 = 各步骤应计入产成品的份额之和

**广义在产品**：包括本步骤尚未加工完成的在产品和本步骤已完工但尚未最终完工的半成品。

**份额计算**：通常采用约当产量法或定额比例法，将各步骤生产费用在最终产成品与广义在产品之间分配。

**方法选择**：逐步结转法适用于半成品对外销售或需比较考核半成品成本的情况；平行结转法适用于不对外销售半成品、管理上不要求计算半成品成本的情况。

## 三、成本控制策略

### 3.1 标准成本法

标准成本法是通过制定标准成本，将标准成本与实际成本进行比较，揭示成本差异并进行分析的成本控制方法。

**标准成本类型**：
- **理想标准成本**：以现有技术、设备在最佳状态下应达到的水平制定，要求过高，一般不宜采用
- **正常标准成本**：以正常生产经营条件为基础制定，考虑了正常的损耗和间歇，是较为合理的标准
- **现实标准成本**：根据下期最可能发生的生产要素消耗量、价格和经营能力制定，最接近实际，最切实可行

**标准成本制定**：
- 直接材料标准成本 = 标准用量 x 标准价格
- 直接人工标准成本 = 标准工时 x 标准工资率
- 制造费用标准成本 = 标准工时 x 标准制造费用分配率

**成本差异分析**：见第四节。

**会计处理**：
- 设置"原材料"、"生产成本"、"库存商品"等科目按标准成本记账
- 设置各成本差异科目（材料价格差异、材料用量差异、人工效率差异、人工工资率差异、制造费用差异等）
- 期末将成本差异按比例分配给已销产品成本和存货成本，或直接转入当期损益

### 3.2 作业成本法（ABC）

作业成本法是以作业为基础，通过对作业成本的确认和计量，对所有作业活动进行追踪和动态反映，从而更准确地计算产品成本的方法。

**核心概念**：
- **资源**：作业消耗的经济要素（人工、材料、能源、折旧等）
- **作业**：企业为提供一定量的产品或劳务所消耗的人力、技术、原材料、方法和环境的集合体
- **成本动因**：引发成本发生的因素，分为资源动因和作业动因
- **作业中心**：具有同质作业动因的作业集合

**计算程序**：
1. 确认主要作业和作业中心
2. 将资源成本归集到各作业中心
3. 选择作业成本动因，计算作业成本分配率
4. 将作业成本分配到产品（产品耗用作业量 x 作业成本分配率）
5. 汇总产品成本（直接成本 + 分配的作业成本）

**适用场景**：
- 制造费用占产品成本比重较高
- 产品品种结构复杂，多样性程度高
- 企业采用高度自动化生产技术
- 传统成本法导致成本信息严重失真

**制造业典型作业划分**：
-  unit-level作业（单位级）：与单个产品产量直接相关，如直接材料加工、直接人工操作
-  batch-level作业（批次级）：与产品批次相关，如机器调试、材料搬运、订单处理
-  product-level作业（产品级）：与特定产品线相关，如产品设计、工艺制定、产品测试
-  facility-level作业（设施级）：维持整个工厂生产能力的作业，如厂房折旧、车间管理、安保

### 3.3 目标成本法

目标成本法是以市场为导向，以目标售价和目标利润为基础确定目标成本，从产品设计阶段开始全流程成本控制的方法。

**计算公式**：目标成本 = 目标售价 - 目标利润

**实施步骤**：
1. 市场调研确定目标售价
2. 根据企业战略确定目标利润
3. 计算目标成本
4. 将目标成本分解到产品各组成部分、各生产环节
5. 跨部门团队协同实现目标成本（价值工程VE、价值分析VA）
6. 持续监控和改进

**与标准成本法的区别**：
- 目标成本法从市场价格倒推成本，标准成本法从成本计算价格
- 目标成本法覆盖产品全生命周期，标准成本法聚焦制造过程
- 目标成本法强调跨职能团队协作，标准成本法主要由财务部门和生产部门负责

## 四、成本差异分析

### 4.1 直接材料差异

**材料价格差异** = （实际价格 - 标准价格）x 实际用量

- 差异责任：一般由采购部门负责，但可能受市场价格波动、供应商选择、采购批量、运输方式等因素影响
- 分析要点：分析采购价格变动原因，区分可控因素与不可控因素

**材料用量差异** = （实际用量 - 标准用量）x 标准价格

- 差异责任：一般由生产部门负责
- 分析要点：检查生产工艺是否合理、工人操作是否规范、材料质量是否符合标准、是否存在浪费或盗窃

### 4.2 直接人工差异

**人工工资率差异** = （实际工资率 - 标准工资率）x 实际工时

- 差异原因：工资标准调整、工人等级结构变化、加班工资、季节性用工等
- 分析要点：分析用工结构与工资政策执行情况

**人工效率差异** = （实际工时 - 标准工时）x 标准工资率

- 差异原因：工人熟练程度、设备状况、材料质量、生产组织、停工待料等
- 分析要点：结合生产记录分析工时利用效率

### 4.3 制造费用差异

制造费用差异分析通常采用两差异分析法或三差异分析法。

**两差异分析法**：
- **耗费差异** = 实际制造费用 - 预算制造费用（按实际工时调整）
- **能量差异** = 预算制造费用（按实际工时调整）- 标准制造费用（按标准工时调整）

**三差异分析法**（将能量差异进一步分解）：
- **耗费差异**：实际费用与预算费用之差
- **效率差异**：（实际工时 - 标准工时）x 变动制造费用标准分配率
- **闲置能量差异**：（生产能量 - 实际工时）x 固定制造费用标准分配率

**差异处理**：
- 重大差异应查明原因，追究责任
- 期末差异可按比例分配给销售成本与存货，或直接计入当期损益
- 在标准成本制度下，差异处理需符合会计准则对存货成本的要求

## 五、制造业成本核算特殊考量

### 5.1 在制品计价

在制品是指尚未完成全部生产过程、不能作为商品销售的产品，包括正在车间加工中的在产品和已经完成一个或几个生产步骤但还需继续加工的半成品。

**在制品成本计算方法**：
- **不计算在制品成本法**：在制品数量很少，不计算在制品成本，全部费用由完工产品承担
- **在制品按年初数固定计算法**：在制品数量变动不大，各月在制品成本按年初数固定计算
- **在制品按直接材料计价法**：在制品只负担直接材料费用，人工和制造费用全部由完工产品承担（适用于材料成本占比高的情况）
- **约当产量法**：将在制品数量按其完工程度折算为相当于完工产品的产量，然后与完工产品共同分配费用
  - 直接材料约当产量：根据投料方式确定（一次投入按100%计算，陆续投入按完工程度计算）
  - 直接人工和制造费用约当产量：按完工程度计算
- **定额比例法**：按在制品与完工产品的定额消耗量或定额成本比例分配费用
- **在制品按定额成本计价法**：月末在制品成本按定额成本计算，实际成本与定额成本的差异由完工产品承担

### 5.2 联产品/副产品成本分摊

**联产品**：使用同种原料，经过同一生产过程同时生产出来的两种或两种以上的主要产品。

**成本分摊方法**：
- **实物量分配法**：按各联产品的实际产量或标准产量比例分配联合成本
- **系数分配法**：按各联产品的系数比例分配，系数根据售价、定额成本、技术参数等确定
- **售价分配法**：按各联产品的销售价格比例分配联合成本
- **可实现净值分配法**：按各联产品的可实现净值（预计售价减去可分成本）比例分配

**会计处理**：联合成本在分离点前归集为联合成本，分离后按上述方法分配；分离点后发生的可归属成本直接计入各产品成本。

**副产品**：在同一生产过程中，使用同种原料，附带生产出来的非主要产品。

**会计处理**：
- 价值较低的副产品：不计算成本，按估计售价减去销售费用和税金后的余额，从联合成本中扣除（通常冲减直接材料成本）
- 价值较高的副产品：按实际成本或计划成本计价，从联合成本中扣除
- 需进一步加工的副产品：计算分离前联合成本和分离后可归属成本

### 5.3 废品损失

废品是指不符合规定的技术标准，不能按原定用途使用，或者需要加工修复后才能使用的在产品、半成品或产成品。

**废品分类**：
- **可修复废品**：技术上可以修复，且修复费用在经济上合算
- **不可修复废品**：技术上不可修复，或修复费用在经济上不合算

**废品损失核算内容**：
- 不可修复废品的生产成本（扣除残料价值和过失人赔偿）
- 可修复废品的修复费用（材料、人工、制造费用，扣除残料价值和过失人赔偿）

**会计处理**：
- 单独核算废品损失的企业，设置"废品损失"科目，月末转入"生产成本"科目由合格品成本负担
- 不单独核算废品损失的企业，相关损失体现在"生产成本"或"制造费用"中

**停工损失**：生产车间或班组因停电、待料、机器故障、计划减产等原因停工期间发生的各项费用。季节性停工和修理期间的停工损失计入制造费用；非季节性停工损失可单独设置"停工损失"科目核算，按原因分别处理（自然灾害计入营业外支出，其他原因计入产品成本或管理费用）。
