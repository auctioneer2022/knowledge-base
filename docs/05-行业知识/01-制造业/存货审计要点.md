# 制造业存货审计要点

## 一、存货监盘程序

存货监盘是注册会计师现场观察被审计单位存货盘点，并对已盘点存货进行适当检查的程序。根据《中国注册会计师审计准则第1311号》，如果存货对财务报表是重要的，注册会计师应当实施监盘程序，除非不可行。

### 1.1 制定监盘计划

**审计前准备**：
- 了解被审计单位存货的性质、内容、存放场所、保管制度、盘点制度
- 获取被审计单位存货盘点计划，评价其合理性
- 根据存货特点和存放情况，制定具体的监盘计划
- 确定监盘范围、时间、人员分工和重点关注的存货项目

**监盘计划要点**：
- 盘点时间：尽量安排在资产负债表日或接近资产负债表日进行；如提前盘点，需实施适当的穿行测试和倒推程序
- 盘点范围：确保所有存放地点的存货均纳入盘点范围，包括第三方保管的存货、在途存货、寄售存货
- 人员安排：配备足够数量的审计人员，明确分工和职责
- 特殊存货：识别需要特殊专业技能或设备的存货（如化学品、贵金属、危险品），必要时利用专家工作

**与被审计单位沟通**：
- 要求被审计单位制定详细的盘点计划并提前提供
- 协调盘点时间和人员配合
- 要求存货在盘点期间停止流动或有效控制收发

### 1.2 实地监盘

**观察程序**：
- 观察被审计单位盘点人员是否遵守盘点计划，是否准确记录存货数量和状况
- 观察盘点现场存货的移动控制情况，防止重复盘点或漏盘
- 观察盘点人员的盘点方法和操作是否恰当
- 关注存货的存在状况，观察是否存在毁损、陈旧、过时、残次等迹象

**检查程序**：
- 对已盘点存货进行适当抽点，将抽点结果与盘点记录核对
- 从存货实物中选取项目追查至盘点记录（测试完整性）
- 从盘点记录中选取项目追查至存货实物（测试存在性）
- 检查存货的计量是否准确，计量工具是否经过校准
- 关注存货的标识和标签，确认其准确性和完整性

**关注点**：
- **盘点表单控制**：确保盘点表编号连续、完整，防止篡改和替换
- **截止测试**：关注盘点日前后的存货收发记录，检查是否计入正确的会计期间
- **所有权确认**：区分自有存货与代管存货、寄售存货、来料加工物资
- **状态确认**：关注呆滞、积压、毁损、过保质期存货的识别和记录

### 1.3 特殊类型存货处理

**第三方保管的存货**：
- 向第三方函证存货的数量和状况
- 实施检查或其他适合具体情况的审计程序
- 考虑安排审计人员到第三方仓库现场监盘
- 关注代管协议条款，确认存货所有权归属

**在途存货**：
- 检查采购合同、销售合同中的交货条款（FOB、CIF等）
- 检查运输单据、入库单、出库单的日期和签收情况
- 关注所有权转移时点，判断在途存货是否应纳入盘点范围

**寄售存货（被审计单位委托他人代销）**：
- 检查寄售协议，确认所有权未转移
- 向受托方函证寄售存货数量
- 如金额重大，可前往受托方仓库实地查看

**来料加工/受托加工物资**：
- 区分自有存货与委托方提供的原材料
- 检查来料加工协议，确认原材料所有权归属
- 关注加工费收入的确认和成本核算

**危险品/特殊材料**：
- 必要时利用具备专业资质的第三方专家参与盘点
- 采取必要的安全防护措施
- 借助专业检测设备确认数量和质量

### 1.4 监盘结果记录

**审计工作底稿要求**：
- 监盘计划及评价记录
- 盘点现场照片或视频记录（如可行）
- 抽点记录及差异处理说明
- 盘点表复印件或扫描件
- 监盘总结，包括监盘范围、监盘比例、发现的异常情况及处理
- 对盘点差异的追查记录

**差异处理**：
- 如抽点发现差异，应要求被审计单位重新盘点或扩大抽点范围
- 追查差异原因（记录错误、计量误差、收发未及时入账、盘盈盘亏等）
- 评估差异的普遍性，判断是否需要调整审计策略

## 二、存货计价测试

### 2.1 计价方法一致性

**审计程序**：
- 检查被审计单位采用的存货计价方法（先进先出法、加权平均法、移动加权平均法、个别计价法等）
- 检查计价方法是否符合会计准则规定，是否适合被审计单位的生产经营特点
- 检查本期计价方法是否与上期一致，如发生变更，检查是否属于会计政策变更，是否已恰当处理和披露
- 关注是否存在通过随意变更计价方法调节利润的情况

**会计准则要求**：企业应当采用先进先出法、加权平均法或者个别计价法确定发出存货的实际成本。对于性质和用途相似的存货，应当采用相同的成本计算方法确定发出存货的成本。

### 2.2 期末计价测试

**成本核实**：
- 检查期末存货成本的计算是否正确，计价方法是否一贯执行
- 抽查部分存货项目的入库单、领料单、成本计算单，核实成本归集和分配的准确性
- 对于采用计划成本法核算的，检查材料成本差异的计算和分配是否正确

**发出计价测试**：
- 选取样本，重新计算发出存货成本，与被审计单位账面记录核对
- 关注期末大额发出存货的计价，防止为调节利润而人为改变计价
- 检查销售成本结转的准确性，与营业收入进行配比分析

### 2.3 成本与可变现净值孰低

根据《企业会计准则第1号——存货》，资产负债表日，存货应当按照成本与可变现净值孰低计量。存货成本高于其可变现净值的，应当计提存货跌价准备。

**审计程序**：
- 检查被审计单位是否定期评估存货的可变现净值
- 获取被审计单位存货跌价准备计算表，复核计算的正确性
- 检查可变现净值的确定依据是否合理（以确凿证据为基础，考虑持有目的）
- 对于用于生产而持有的材料等，检查是否以其所生产的产成品的可变现净值与成本的比较为基础确定
- 关注以前减记存货价值的影响因素消失时，是否在原已计提的存货跌价准备金额内转回

## 三、存货周转分析

### 3.1 周转率计算

**计算公式**：
- 存货周转率 = 营业成本 / 平均存货余额
- 平均存货余额 = （期初存货余额 + 期末存货余额）/ 2
- 存货周转天数 = 365 / 存货周转率

**审计应用**：
- 计算被审计单位各期存货周转率，分析变动趋势
- 与同行业可比公司进行比较，分析是否存在异常
- 结合生产周期和销售周期，判断周转率的合理性

### 3.2 异常波动分析

**关注情形**：
- 存货周转率大幅下降：可能存在销售不畅、产品积压、生产计划不合理、虚构存货等问题
- 存货周转率大幅上升：可能存在少结转成本、存货计量不准确、销售模式变化等情况
- 各期间波动异常：可能存在成本调节、盈余管理行为

**分析维度**：
- 按存货类别分析周转情况（原材料、在产品、产成品）
- 按产品类别分析周转情况
- 按仓库/存放地点分析周转情况
- 结合销售毛利率、应收账款周转率进行综合分析

### 3.3 与同行业对比

**对比要点**：
- 选取同行业可比上市公司或行业平均数据作为参照
- 分析周转率差异的合理性（业务模式、产品结构、供应链策略等）
- 如显著低于行业水平，需重点排查滞销、呆滞、减值风险
- 如显著高于行业水平，需关注成本结转完整性和存货真实性

## 四、存货跌价准备计提依据

### 4.1 可变现净值确定

**定义**：可变现净值是指在日常活动中，存货的估计售价减去至完工时估计将要发生的成本、估计的销售费用以及相关税费后的金额。

**确定方法**：
- **直接用于出售的存货**：可变现净值 = 估计售价 - 估计销售费用和相关税费
- **需要加工的存货**：可变现净值 = 估计售价 - 至完工时估计将要发生的成本 - 估计销售费用和相关税费

**估计售价确定**：
- 有销售合同约定的，以合同价格为基础确定
- 没有销售合同约定的，以一般销售价格（市场价格）为基础确定
- 持有数量多于销售合同订购数量的，超出部分以一般销售价格为基础确定

**审计关注**：
- 检查估计售价的合理性，与期后实际售价进行对比
- 检查至完工成本估计的依据是否充分
- 检查销售费用和相关税费估计的合理性

### 4.2 迹象判断

**表明存货可能发生减值的迹象**：
- 市场价格持续下跌，并且在可预见的未来无回升的希望
- 企业使用该项原材料生产的产品的成本大于产品的销售价格
- 企业因产品更新换代，原有库存原材料已不适应新产品的需要，而该原材料的市场价格又低于其账面成本
- 因企业所提供的商品或劳务过时或消费者偏好改变而使市场的需求发生变化，导致市场价格逐渐下跌
- 其他足以证明该项存货实质上已经发生减值的情形

**审计程序**：
- 访谈销售和生产部门，了解产品销售情况和市场需求变化
- 检查存货库龄分析表，识别长库龄存货
- 检查期后销售情况，判断存货的可变现性
- 关注技术更新迭代较快行业的存货（如电子元器件、时尚产品）

### 4.3 转回处理

**转回条件**：以前减记存货价值的影响因素已经消失的，减记的金额应当予以恢复，并在原已计提的存货跌价准备金额内转回。

**审计关注**：
- 检查转回的原因是否真实合理，是否存在利用转回调节利润的情况
- 检查转回的金额是否以原已计提的存货跌价准备金额为限
- 关注转回的时点是否恰当，是否有充分的证据支持
- 结合市场价格走势和存货实际销售情况进行判断

## 五、制造业存货审计重点风险点

### 5.1 呆滞存货

**风险表现**：
- 存货长期积压，库龄过长，但未见计提跌价准备或计提不足
- 呆滞存货未及时处置，虚增资产和利润
- 将呆滞存货转移至其他科目或第三方以掩盖真实情况

**审计应对**：
- 获取存货库龄分析表，重点关注库龄超过一年的存货
- 分析存货周转情况，识别周转异常的存货项目
- 检查期后销售情况，判断长库龄存货是否仍有市场需求
- 实地查看存货状态，观察是否存在包装破损、锈蚀、过期等情况
- 评价跌价准备计提的充分性

### 5.2 委托加工物资

**风险表现**：
- 虚构委托加工业务，将资金转移至体外
- 委托加工物资长期挂账，实际已被挪用或不存在
- 委托加工物资成本归集不准确，影响产品成本计算

**审计应对**：
- 检查委托加工合同，关注加工方资质、加工内容、交货期限、价格条款
- 向主要加工方函证委托加工物资的数量和状况
- 检查委托加工物资的收发记录，关注长期未收回的物资
- 对金额重大的委托加工物资，可前往加工方现场查看
- 检查加工费的计价和支付是否公允，是否存在关联交易

### 5.3 寄售存货

**风险表现**：
- 将寄售存货确认为自有存货，虚增资产
- 寄售存货已销售但未及时确认收入，调节利润
- 寄售存货保管不善，发生毁损、灭失

**审计应对**：
- 检查寄售协议，明确所有权归属和风险转移条款
- 向受托方函证寄售存货数量和销售情况
- 检查寄售存货的发出和结算记录
- 关注期末大额寄售存货的合理性
- 必要时前往受托方仓库实地查看

### 5.4 虚构存货

**风险表现**：
- 通过虚假采购入库、虚假生产入库虚增存货
- 盘点时以其他存货顶替、重复盘点
- 利用在途存货、委托加工物资等难以核实的项目虚增资产

**审计应对**：
- 实施严格的监盘程序，保持职业怀疑
- 从存货记录追查到实物（存在测试），从实物追查到记录（完整性测试）
- 结合采购审计和销售审计，检查收发记录的合理性
- 分析存货变动与生产、销售、采购数据的匹配性
- 关注异常的大额存货增加，检查相关原始凭证
- 对存放在偏远地点或海外的存货保持高度警惕
- 检查存货的保险情况，异常低的保险金额可能暗示存货不实

**其他风险点**：
- **成本调节**：通过操纵在制品完工程度、制造费用分配、成本结转时点调节产品成本和利润
- **截止错误**：将下期收入提前确认或本期收入推迟确认，同时伴随存货截止错误
- **关联交易**：与关联方进行不公允的存货交易，转移利润或资金
