# 制造业

制造业是国民经济的基础产业，其会计核算具有成本构成复杂、核算周期长、在制品管理难度大等特点。本模块聚焦制造业的核心会计与审计问题，涵盖成本核算方法与存货审计要点两大主题。

## 主题目录

### [成本核算](./成本核算.md)

系统阐述制造业成本核算的完整框架，包括：
- 成本构成分析（直接材料、直接人工、制造费用）
- 成本计算方法：品种法、分批法、分步法（逐步结转、平行结转）
- 成本控制策略（标准成本法、作业成本法ABC、目标成本法）
- 成本差异分析（材料价差/量差、人工效率差异/工资率差异、制造费用差异）
- 制造业成本核算特殊考量（在制品计价、联产品/副产品成本分摊、废品损失）

### [存货审计要点](./存货审计要点.md)

聚焦制造业存货审计的核心程序与风险应对，包括：
- 存货监盘程序（制定监盘计划、实地监盘、特殊类型存货处理、监盘结果记录）
- 存货计价测试（计价方法一致性、期末计价测试、成本与可变现净值孰低）
- 存货周转分析（周转率计算、异常波动分析、与同行业对比）
- 存货跌价准备计提依据（可变现净值确定、迹象判断、转回处理）
- 制造业存货审计重点风险点（呆滞存货、委托加工物资、寄售存货、虚构存货）
