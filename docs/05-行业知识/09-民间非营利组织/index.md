# 民间非营利组织会计知识

民间非营利组织是指依照国家有关法律法规设立，面向公众开展慈善活动或者其他公益性活动的非营利性社会组织，包括社会团体、基金会、社会服务机构（民办非企业单位）等。民间非营利组织会计核算遵循《民间非营利组织会计制度》（财会〔2004〕7号），该制度以权责发生制为核算基础，以各项业务活动持续正常进行为前提，突出反映组织资源的来源、限制和使用情况。
