# 非营利组织会计制度

## 一、核心概念

《民间非营利组织会计制度》适用于在中华人民共和国境内依法设立的各类民间非营利组织，包括依照国家有关法律法规设立的社会团体、基金会、民办非企业单位（社会服务机构）、寺院、宫观、清真寺、教堂等。该制度自2005年1月1日起施行，是我国非营利组织会计规范的基石。

民间非营利组织应当同时具备以下特征：
- 该组织不以营利为宗旨和目的
- 资源提供者向该组织投入资源不取得经济回报
- 资源提供者不享有该组织的所有权
- 该组织清算后的剩余财产应按规定继续用于社会公益事业

**制度核心特点**：
- 会计核算基础为权责发生制
- 以各项业务活动持续正常进行为前提
- 不设"所有者权益"科目，以"净资产"反映资源提供者的剩余权益
- 净资产按是否存在限定分为非限定性净资产和限定性净资产
- 突出反映捐赠收入的来源和限制条件
- 强调业务活动成本与组织管理成本的区分

## 二、会计核算基础

### 2.1 权责发生制

民间非营利组织会计核算以权责发生制为基础。凡是当期已经实现的收入和已经发生或应当负担的费用，不论款项是否收付，都应当作为当期的收入和费用；凡是不属于当期的收入和费用，即使款项已在当期收付，也不应当作为当期的收入和费用。

**与企业会计的区别**：
- 民间非营利组织不存在所有者权益，以净资产替代
- 民间非营利组织不存在利润分配，清算剩余财产不得归属私人
- 民间非营利组织的收入以捐赠收入、会费收入、提供服务收入、政府补助收入、商品销售收入、投资收益和其他收入为主要构成，不设置"营业外收入"科目

**实务操作要点**：
- 收到捐赠款项但尚未满足收入确认条件的，应先计入"预收账款"或"受托代理负债"
- 已经发生但尚未支付的费用，应计提应付账款、应付工资等
- 预付的费用应在受益期内分期摊销，不得一次性计入当期费用

**常见问题**：民间非营利组织是否可以采用收付实现制？
**解决方法**：不可以。《民间非营利组织会计制度》明确规定会计核算采用权责发生制。部分规模较小的组织在实际操作中可能存在收付实现制的习惯，但对外报送的财务报表必须按权责发生制编制。如内部管理需要，可在权责发生制核算的基础上，另行编制收付实现制的内部管理报表。

### 2.2 持续经营前提

会计核算应当以各项业务活动持续正常进行为前提。如果组织出现终止清算情形，应按清算会计处理。

**影响持续经营判断的因素**：
- 资金链断裂，无法正常支付员工薪酬和项目费用
- 重大捐赠人撤资或停止捐赠，导致主要收入来源中断
- 重大项目失败，造成重大资产损失
- 被登记管理机关吊销登记证书
- 章程规定的终止情形出现

**实务操作要点**：
- 如果存在可能导致组织终止清算的重大不确定因素，应在财务报表附注中披露
- 进入清算程序后，应停止按正常会计制度核算，改按清算会计处理
- 清算剩余财产应按章程规定或业务主管单位、登记管理机关的要求处理，不得向出资人、设立人或会员分配

### 2.3 会计要素

民间非营利组织的会计要素包括资产、负债、净资产、收入和费用五大类。

**资产**：过去的交易或事项形成并由民间非营利组织拥有或者控制的资源，该资源预期会给民间非营利组织带来经济利益或服务潜力。包括流动资产、长期投资、固定资产、无形资产和受托代理资产等。

**负债**：过去的交易或事项形成的现时义务，履行该义务预期会导致含有经济利益或者服务潜力的资源流出民间非营利组织。包括流动负债、长期负债和受托代理负债等。

**净资产**：资产减去负债后的余额。按是否存在限定，分为限定性净资产和非限定性净资产。

**收入**：民间非营利组织开展业务活动取得的、导致本期净资产增加的经济利益或者服务潜力的流入。包括捐赠收入、会费收入、提供服务收入、政府补助收入、商品销售收入、投资收益和其他收入。

**费用**：民间非营利组织为开展业务活动所发生的、导致本期净资产减少的经济利益或者服务潜力的流出。包括业务活动成本、管理费用、筹资费用和其他费用。

## 三、捐赠收入确认原则

### 3.1 无条件捐赠

无条件捐赠是指捐赠人对捐赠资产没有设定任何时间或用途上的限制，组织可以自主决定资产的使用时间和方式。

**确认条件**：
- 捐赠交易已经实际发生，组织已收到捐赠资产或已取得收取捐赠资产的权利
- 捐赠资产能够可靠计量
- 经济利益或服务潜力很可能流入组织

**会计处理**：
```
收到货币资金捐赠时：
借：银行存款 / 现金
贷：捐赠收入——非限定性收入

收到非货币性资产捐赠时：
借：固定资产 / 库存物品等（按公允价值入账）
贷：捐赠收入——非限定性收入
```

**实务操作要点**：
- 捐赠收入的确认时点为实际收到捐赠资产或取得收取权利时
- 如捐赠人承诺捐赠但尚未实际交付，不得确认捐赠收入，但应在报表附注中披露
- 非货币性资产捐赠应按公允价值计量；公允价值不能可靠取得的，应采用估值技术或名义金额（1元）计量

### 3.2 附条件捐赠

附条件捐赠是指捐赠人对捐赠资产的使用设定了时间限制或用途限制。时间限制要求资产在特定时间之后或特定期间内使用；用途限制要求资产用于特定目的或特定项目。

**确认原则**：
- 附条件捐赠在取得捐赠资产时，即确认捐赠收入
- 同时，将捐赠收入对应的净资产计入"限定性净资产"
- 当限定条件解除时（如时间到期、项目完成），将"限定性净资产"转为"非限定性净资产"

**会计处理**：
```
收到附条件捐赠时：
借：银行存款
贷：捐赠收入——限定性收入

期末结转：
借：捐赠收入——限定性收入
贷：限定性净资产

限定条件解除时：
借：限定性净资产
贷：非限定性净资产
```

**实务操作要点**：
- 附条件捐赠的"条件"必须是捐赠人明确设定的、可识别的限制，而非组织自行设定的使用计划
- 限定条件解除的判断应依据客观事实，如项目验收完成、特定日期到达等
- 限定性净资产的重分类应在条件解除的当期及时进行

**常见问题**：捐赠协议中约定"捐赠款用于资助贫困学生"，但具体资助对象和金额由组织自行确定，这是否属于限定性捐赠？
**解决方法**：属于限定性捐赠。虽然组织有权确定具体的资助对象和金额，但捐赠资产的用途已被限定在"资助贫困学生"范围内，不得挪用于其他用途（如组织日常运营、员工薪酬等）。因此，该捐赠应确认为限定性捐赠收入，并在"资助贫困学生"项目执行完毕后或按约定期间，将相应限定性净资产转为非限定性净资产。

### 3.3 劳务捐赠

劳务捐赠是指捐赠人自愿向民间非营利组织提供志愿服务，如专业咨询、技术支持、教学辅导等。

**确认原则**：
- 劳务捐赠在满足以下条件时，应确认捐赠收入和费用：
  1. 劳务捐赠具有明确的市场价值或可以可靠计量
  2. 劳务捐赠对组织的业务活动具有实质性贡献
  3. 劳务捐赠的性质和金额能够可靠确定
- 不能满足上述条件的劳务捐赠，不予确认，但应在报表附注中作相关披露

**会计处理**：
```
借：业务活动成本 / 管理费用
贷：捐赠收入——非限定性收入（或限定性收入）
```

**实务操作要点**：
- 劳务捐赠的计量应按公允价值（即同类服务的市场价格）确定
- 对于志愿者提供的非专业性、零星服务，通常不满足确认条件，不确认捐赠收入和费用
- 对于专业人士（如律师、会计师、医生）提供的专业志愿服务，如价值重大且可计量，应确认劳务捐赠

**常见问题**：志愿者在救灾活动中提供物资搬运服务，是否应确认劳务捐赠？
**解决方法**：一般情况下，普通志愿服务的价值难以可靠计量，且多为非专业性劳动，不满足劳务捐赠的确认条件，不确认捐赠收入和费用。但应在财务报表附注中披露志愿者服务的性质、提供服务的志愿者人数、服务时间等相关信息，以反映组织获得的社会支持。如果是由专业物流公司无偿提供的运输服务，且能够取得同类运输服务的市场价格，则可以确认劳务捐赠。

### 3.4 受托代理业务

受托代理业务是指民间非营利组织从委托方收到受托资产，并按照委托人的意愿将资产转赠给指定的其他组织或者个人的活动。组织在受托代理业务中仅充当代理人角色，不享有受托资产的收益权或处置权。

**特征识别**：
- 组织不是受托资产的最终受益人
- 组织无权改变受托资产的用途、受益人、受益方式
- 组织仅按委托人指定转赠资产，通常不从中获取报酬或仅收取少量手续费

**会计处理**：
```
收到受托代理资产时：
借：受托代理资产（现金/非现金资产）
贷：受托代理负债

转赠受托代理资产时：
借：受托代理负债
贷：受托代理资产
```

**实务操作要点**：
- 受托代理资产不计入组织的自有资产，仅在资产负债表中单列项目反映
- 受托代理业务的现金流量在现金流量表中单独列示
- 组织如从受托代理业务中收取手续费，应在转赠时确认其他收入
- 应建立受托代理业务台账，详细记录每笔受托代理资产的来源、金额、指定用途、受益人、转赠情况等

**常见问题**：捐赠人指定将捐款用于资助某特定学校的学生，组织负责审核学生资格并发放资助款，这是捐赠收入还是受托代理业务？
**解决方法**：判断是否属于受托代理业务，关键在于组织是否拥有对资产的支配权。如果组织仅负责按捐赠人指定的标准和对象转交款项，无权改变受益人或用途，属于受托代理业务。如果组织在审核资格后有权决定具体资助哪些学生、资助金额多少，且捐赠人仅设定大致方向（如"资助贫困学生"），则属于限定性捐赠收入，而非受托代理业务。实务中应仔细分析捐赠协议的条款，判断组织的角色是代理人还是受益人管理人。

## 四、限定性与非限定性净资产划分

### 4.1 净资产分类

民间非营利组织的净资产按照其是否受到限制，分为限定性净资产和非限定性净资产。

**限定性净资产**：
- 资产或其经济利益的使用受到时间限制的净资产
- 资产或其经济利益的使用受到用途限制的净资产
- 资产的使用受到时间限制和用途限制双重限制的净资产

**非限定性净资产**：
- 未受到任何限制的净资产
- 限定性净资产的限制条件已经解除，转为非限定性的部分
- 其他组织和个人无偿捐赠的未设定限制的资产形成的净资产

### 4.2 时间限定

时间限定是指资源提供者对资产的使用设定了时间上的限制，要求资产在特定时间之后才能使用，或者在特定期间内使用。

**典型情形**：
- 捐赠协议约定"该笔捐款于202X年1月1日后方可使用"
- 捐赠协议约定"该笔捐款应在三年内用于指定项目"
- 政府补助文件规定补助资金应在特定会计年度内使用完毕

**会计处理**：
- 收到时：捐赠收入计入"限定性收入"，结转后形成"限定性净资产"
- 限制期间：限定性净资产不得随意动用
- 时间到期：将"限定性净资产"转入"非限定性净资产"

### 4.3 用途限定

用途限定是指资源提供者对资产的使用设定了用途上的限制，要求资产只能用于特定目的或特定项目。

**典型情形**：
- "该笔捐款专门用于贫困地区的学校建设"
- "该笔捐款用于资助患有特定疾病的患者"
- "该笔捐款用于某特定科研项目"

**会计处理**：
- 收到时：捐赠收入计入"限定性收入"，结转后形成"限定性净资产"
- 项目执行期间：按项目实际支出，将"限定性净资产"逐步转为"非限定性净资产"
- 项目完成：将剩余限定性净资产（如有）按捐赠协议约定处理，协议未约定的，经决策程序后转为非限定性净资产

**实务操作要点**：
- 用途限定净资产的释放应与项目实际支出进度相匹配
- 项目支出发生时，应按实际支出金额将限定性净资产转为非限定性净资产
- 项目完成后，如有结余资金，应按捐赠协议约定处理；协议约定退回捐赠人的，应退还；协议未约定的，一般应继续用于相同或类似公益目的，经组织决策程序后可转为非限定性净资产

### 4.4 暂时限定与永久限定

**暂时限定**：限制条件是暂时的，随着时间推移或条件满足，限制会自动解除。大多数限定性净资产属于暂时限定。

**永久限定**：限制条件是永久性的，要求本金永久保留，仅可使用投资收益或利息。常见于基金会的永续基金。

**会计处理差异**：
- 暂时限定净资产在限制解除时，全额转为非限定性净资产
- 永久限定净资产的本金部分永久保留在限定性净资产中，不得转为非限定性净资产；投资收益部分如未限定用途，可转为非限定性净资产

**实务操作要点**：
- 永久限定基金应单独核算，建立独立的基金台账
- 永久限定基金的投资收益应单独核算，明确区分本金和投资收益
- 如投资收益也被限定用途，则投资收益部分仍作为限定性净资产

**常见问题**：限定性净资产可以转为非限定性净资产的情形有哪些？
**解决方法**：限定性净资产转为非限定性净资产的情形包括：
1. 时间限制到期：捐赠协议约定的使用时间已到
2. 用途限制解除：指定项目已完成，且捐赠协议约定剩余资金可由组织自主支配
3. 捐赠人同意解除限制：经捐赠人书面同意，解除对资产使用的限制
4. 法律法规规定解除：因法律法规变化或登记管理机关要求，限制条件不再适用
5. 捐赠协议未明确结余资金用途：项目完成后，经组织理事会或类似决策机构决议，将结余资金转为非限定性净资产（需符合章程规定）
限定性净资产的重分类必须有充分的依据，不得擅自改变限定性净资产的性质。

## 五、费用分类核算

### 5.1 业务活动成本

业务活动成本是指民间非营利组织为了实现其业务活动目标、开展其项目活动或者提供服务所发生的费用。

**核算内容**：
- 项目直接费用：项目实施过程中发生的直接材料、直接人工、项目活动费用等
- 项目间接费用：为多个项目共同发生的、需要按合理标准分摊的费用
- 服务提供成本：组织提供服务过程中发生的直接成本

**典型示例**：
- 教育类组织：教学材料费、教师课酬、教学场地租赁费、学生资助款
- 医疗类组织：药品费、医疗设备使用费、医护人员薪酬、患者救助款
- 救灾类组织：救灾物资采购费、运输费、灾区重建支出
- 环保类组织：环保项目调研费、宣传费、环保设施购置费

**会计处理**：
```
借：业务活动成本——XX项目
贷：银行存款 / 应付账款 / 库存物品等
```

**实务操作要点**：
- 应按项目或业务活动类型设置明细科目，清晰反映各项目的成本支出
- 多个项目共同发生的间接费用，应按合理的分摊标准（如项目预算比例、人员投入比例、时间比例等）分摊至各项目
- 业务活动成本应与当期实现的捐赠收入、提供服务收入等相配比

### 5.2 管理费用

管理费用是指民间非营利组织为组织和管理其业务活动所发生的各项费用。

**核算内容**：
- 行政管理人员薪酬、福利费
- 办公费、水电费、物业管理费、通讯费
- 差旅费、会议费、培训费
- 折旧费、修理费、租赁费（办公场所）
- 聘请中介机构费、咨询费、审计费、诉讼费
- 房产税、车船税、土地使用税等税费
- 坏账损失、存货跌价损失
- 因预计负债所确认的损失

**典型示例**：
- 秘书长、财务负责人等行政管理人员工资
- 办公室租金、水电费
- 年度审计费用、法律咨询费用
- 办公设备折旧

**会计处理**：
```
借：管理费用——XX明细科目
贷：银行存款 / 应付职工薪酬 / 累计折旧等
```

**实务操作要点**：
- 管理费用是组织的间接费用，不与特定项目直接挂钩
- 应严格控制管理费用占总支出的比例，提高资金使用效率
- 对于规模较小的组织，如管理费用与业务活动成本难以明确区分的，可统一计入业务活动成本，但应在报表附注中说明

### 5.3 筹资费用

筹资费用是指民间非营利组织为筹集业务活动所需资金而发生的费用。

**核算内容**：
- 筹资活动产生的印刷费、宣传费
- 为筹集资金举办活动的费用
- 募捐活动产生的邮寄费、通讯费、志愿者餐饮费
- 筹资活动相关的差旅费
- 捐赠设备处置费用（用于筹资目的的资产处置）

**不属于筹资费用的支出**：
- 为捐赠人提供的感谢品、纪念品支出（一般计入管理费用或业务活动成本）
- 捐赠收入的手续费（冲减捐赠收入）

**会计处理**：
```
借：筹资费用
贷：银行存款等
```

**实务操作要点**：
- 筹资费用应单独核算，以反映组织为获取资源所付出的成本
- 应分析筹资费用与筹集资金的比率，评估筹资活动的效率
- 筹资活动如同时具有宣传组织使命、扩大社会影响的附带效果，相关费用应合理区分，难以区分的可计入筹资费用

### 5.4 其他费用

其他费用是指民间非营利组织发生的、无法归属到上述业务活动成本、管理费用或筹资费用中的费用。

**核算内容**：
- 固定资产处置净损失
- 无形资产处置净损失
- 资产减值损失
- 所得税费用（如存在应税收入）
- 无法归类的其他支出

**会计处理**：
```
借：其他费用
贷：固定资产清理 / 无形资产 / 累计减值准备等
```

### 5.5 费用归集与分配

**费用归集原则**：
- 直接费用直接计入相应的项目或类别
- 间接费用按合理标准分配至各项目或类别
- 费用归集应与组织的项目管理体系相匹配

**常见的分配标准**：
| 费用类型 | 建议分配标准 |
|----------|-------------|
| 行政管理人员薪酬 | 按各项目人员投入工时比例分配 |
| 办公场所租金 | 按各项目占用面积比例分配 |
| 办公设备折旧 | 按各项目使用频率或时间比例分配 |
| 水电费 | 按各项目占用面积或人员比例分配 |
| 通讯费 | 按各项目业务量比例分配 |

**实务操作要点**：
- 应制定明确的费用分配政策，并保持一贯性
- 费用分配标准应合理、可验证，不得随意变更
- 如变更分配政策，应在财务报表附注中披露变更原因及影响
- 对于无法合理分配的共同费用，可统一计入管理费用

**常见问题**：如何区分业务活动成本和管理费用？
**解决方法**：区分的关键在于费用是否与特定项目或业务活动直接相关：
- 业务活动成本：为开展特定项目或提供特定服务而发生的、可直接归属于该项目或服务的费用。如项目人员薪酬、项目材料费、项目差旅费、直接向受益人提供的资助款等。
- 管理费用：为组织整体运营和管理而发生的、不直接归属于特定项目的费用。如行政管理人员薪酬、办公场所租金、办公费、审计费、理事会会议费等。
实务中，部分费用可能同时服务于多个项目或既有管理功能又有业务功能，应按合理的分配标准进行分摊。对于金额不大、分配困难的支出，为简化核算可统一计入管理费用，但应在附注中说明。

## 六、财务报告披露要求

### 6.1 资产负债表

资产负债表是反映民间非营利组织在某一特定日期财务状况的报表。

**报表结构**：
- **资产**：按流动性分为流动资产和非流动资产
  - 流动资产：货币资金、短期投资、应收款项、预付账款、存货、待摊费用、一年内到期的长期债权投资等
  - 非流动资产：长期投资、固定资产、无形资产、受托代理资产等
- **负债**：按到期日分为流动负债和长期负债
  - 流动负债：短期借款、应付款项、应付工资、应交税金、预收账款、预提费用、预计负债、一年内到期的长期负债等
  - 长期负债：长期借款、长期应付款、受托代理负债等
- **净资产**：限定性净资产、非限定性净资产

**编制要点**：
- 资产和负债应按流动性分类列示
- 受托代理资产和受托代理负债应单独列示
- 限定性净资产和非限定性净资产应分别列示
- 资产负债表应满足平衡公式：资产 = 负债 + 净资产

### 6.2 业务活动表

业务活动表是反映民间非营利组织在某一会计期间内业务活动情况和结果及其净资产变动情况的报表。

**报表结构**：
- **收入部分**：捐赠收入、会费收入、提供服务收入、商品销售收入、政府补助收入、投资收益、其他收入；各项收入再按限定性/非限定性细分
- **费用部分**：业务活动成本、管理费用、筹资费用、其他费用
- **净资产变动额**：收入合计 - 费用合计 = 净资产变动额
- **净资产部分**：期初净资产 + 净资产变动额 = 期末净资产；分别列示限定性净资产和非限定性净资产的变动

**编制要点**：
- 收入应按来源分类，同时区分限定性和非限定性
- 费用应按性质分类（业务活动成本、管理费用、筹资费用、其他费用）
- 应反映限定性净资产和非限定性净资产各自的变动情况
- 业务活动表与资产负债表的净资产变动应相互勾稽

### 6.3 现金流量表

现金流量表是反映民间非营利组织在某一会计期间内现金和现金等价物流入和流出情况的报表。

**报表结构**：
- **业务活动产生的现金流量**：接受捐赠收到的现金、收取会费收到的现金、提供服务收到的现金、销售商品收到的现金、政府补助收到的现金、收到的其他与业务活动有关的现金；支付的现金包括购买商品接受劳务支付的现金、支付给员工以及为员工支付的现金、支付的其他与业务活动有关的现金
- **投资活动产生的现金流量**：收回投资收到的现金、取得投资收益收到的现金、处置固定资产和无形资产收回的现金、收到的其他与投资活动有关的现金；购建固定资产和无形资产支付的现金、对外投资支付的现金、支付的其他与投资活动有关的现金
- **筹资活动产生的现金流量**：借款收到的现金、收到的其他与筹资活动有关的现金；偿还借款支付的现金、偿付利息支付的现金、支付的其他与筹资活动有关的现金
- **汇率变动对现金的影响额**
- **现金及现金等价物净增加额**

**编制要点**：
- 现金流量表采用收付实现制编制
- 受托代理业务的现金流入流出应单独列示或在附注中披露
- 现金流量表中的"现金"应与资产负债表中的"货币资金"相勾稽

### 6.4 会计报表附注

会计报表附注是为便于会计报表使用者理解会计报表的内容而对会计报表的编制基础、编制依据、编制原则和方法及主要项目等所作的解释。

**应披露的主要内容**：
- 会计报表的编制基础，即权责发生制
- 重要会计政策和会计估计的说明
- 重要会计政策变更的说明
- 或有事项的说明
- 资产负债表日后事项的说明
- 关联方关系及其交易的说明
- 重要资产转让及其出售的说明
- 会计报表重要项目的明细资料
- 收入、成本、费用的明细资料
- 限定性净资产和非限定性净资产的变动情况
- 受托代理业务的说明
- 有助于理解和分析会计报表需要说明的其他事项

**实务操作要点**：
- 附注应详细披露限定性净资产的来源、限制条件、变动情况及释放情况
- 应披露大额捐赠人的信息（如不违反保密协议）和捐赠金额
- 应披露重大项目的收支情况和执行进度
- 应披露管理费用、筹资费用占总费用的比例，反映组织的运营效率

### 6.5 财务情况说明书

财务情况说明书是对民间非营利组织一定会计期间内业务活动、财务收支和净资产变动等情况进行分析说明的书面报告。

**主要内容**：
- 组织的业务活动基本情况
- 收入、费用、结余的实现和分配情况
- 资产、负债和净资产的构成及变动情况
- 重要资产的变动情况
- 重大项目的执行情况
- 对本期或下期财务状况发生重大影响的事项
- 其他需要说明的事项

## 七、常见问题及解决方法

**问题1：民间非营利组织收到的政府补助适用哪个会计科目？**
**解决方法**：民间非营利组织收到的政府补助，应通过"政府补助收入"科目核算，而非企业会计准则中的"递延收益"或"其他收益"。政府补助收入是指民间非营利组织接受政府拨款或者政府机构给予的补助而取得的收入。如政府补助设定了用途限制，应确认为"政府补助收入——限定性收入"；如未设定限制，确认为"政府补助收入——非限定性收入"。期末分别结转至"限定性净资产"或"非限定性净资产"。

**问题2：民间非营利组织是否需要缴纳企业所得税？哪些收入免税？**
**解决方法**：根据《企业所得税法》及实施条例，符合条件的非营利组织的收入为免税收入。免税收入包括：
1. 接受其他单位或者个人捐赠的收入
2. 除《企业所得税法》第七条规定的财政拨款以外的其他政府补助收入（但不包括因政府购买服务取得的收入）
3. 按照省级以上民政、财政部门规定收取的会费
4. 不征税收入和免税收入孳生的银行存款利息收入
5. 财政部、国家税务总局规定的其他收入
但免税资格需经财政、税务部门联合认定，未取得免税资格的非营利组织应按规定缴纳企业所得税。与取得收入有关的、合理的支出，包括成本、费用、税金、损失和其他支出，准予在计算应纳税所得额时扣除。

**问题3：限定性净资产转为非限定性净资产时，是否需要捐赠人同意？**
**解决方法**：是否需要捐赠人同意，取决于捐赠协议的约定和限制条件的性质：
1. 时间限制到期：无需捐赠人同意，时间届满自动解除
2. 用途限制（项目已完成）：如捐赠协议明确约定项目完成后结余资金可由组织自主支配，无需捐赠人同意；如协议约定需捐赠人同意或约定其他处理方式，应按协议执行
3. 捐赠人主动解除限制：需捐赠人出具书面同意文件
4. 协议未明确约定：项目完成后，结余资金的处理应经组织理事会或类似决策机构决议，并符合组织章程规定。如章程规定结余资金应继续用于相同或类似公益目的，则不能直接转为非限定性净资产，而应继续作为限定性净资产（但可变更限定用途）
擅自将限定性净资产转为非限定性净资产，属于违反捐赠协议和会计制度的行为。

**问题4：业务活动成本与管理费用如何合理分摊共同费用？**
**解决方法**：共同费用的分摊应遵循受益原则和可验证原则：
1. 确定分摊对象：明确哪些费用属于多个项目或业务活动共同发生的费用
2. 选择分摊标准：选择与费用发生原因高度相关的分摊标准
   - 人员薪酬类：按各项目投入工时比例分摊
   - 场地租金类：按各项目占用面积比例分摊
   - 设备使用类：按各项目使用时间或频率比例分摊
   - 通讯、水电类：按各项目业务量或人员比例分摊
3. 保持一贯性：分摊标准一经确定，应保持相对稳定，不得随意变更
4. 记录分摊依据：留存分摊计算表和相关依据，以备审计检查
5. 披露分摊政策：在财务报表附注中披露费用分摊政策和分摊方法

**问题5：民间非营利组织清算时，剩余财产如何分配？**
**解决方法**：根据《民间非营利组织会计制度》和相关法律法规，民间非营利组织清算后的剩余财产不得向出资人、设立人或会员分配，应当按照章程的规定转给宗旨相同或者相近的以公益为目的的其他民间非营利组织；章程未规定的，由登记管理机关组织捐赠给宗旨相同或者相近的以公益为目的的组织，并向社会公告。清算过程中，应成立清算组，清理财产、编制资产负债表和财产清单、处理未了结业务、清缴税款、清理债权债务、处理剩余财产。清算报告应经注册会计师审计，并报业务主管单位和登记管理机关备案。
