# 企业所得税税收优惠

## 一、免税收入

### 1.1 国债利息收入

| 项目 | 内容 |
|------|------|
| 适用对象 | 所有企业 |
| 优惠内容 | 国债利息收入免征企业所得税 |
| 政策依据 | 《中华人民共和国企业所得税法》第二十六条第（一）项 |
| 有效期限 | 长期有效 |

### 1.2 符合条件的居民企业之间的股息、红利等权益性投资收益

| 项目 | 内容 |
|------|------|
| 适用对象 | 居民企业 |
| 优惠内容 | 符合条件的居民企业之间的股息、红利等权益性投资收益免征企业所得税 |
| 政策依据 | 《中华人民共和国企业所得税法》第二十六条第（二）项 |
| 有效期限 | 长期有效 |
| 享受条件 | 1. 投资方为居民企业；2. 被投资方为居民企业；3. 不包括连续持有居民企业公开发行并上市流通的股票不足12个月取得的投资收益 |

### 1.3 非营利组织收入

| 项目 | 内容 |
|------|------|
| 适用对象 | 符合条件的非营利组织 |
| 优惠内容 | 非营利组织的收入免征企业所得税（不包括非营利组织从事营利性活动取得的收入） |
| 政策依据 | 《中华人民共和国企业所得税法》第二十六条第（四）项；《财政部 国家税务总局关于非营利组织企业所得税免税收入问题的通知》（财税〔2009〕122号） |
| 有效期限 | 长期有效 |

## 二、减计收入

### 2.1 综合利用资源生产产品减计收入

| 项目 | 内容 |
|------|------|
| 适用对象 | 以《资源综合利用企业所得税优惠目录》规定的资源作为主要原材料，生产非国家限制和禁止并符合国家和行业相关标准的产品 |
| 优惠内容 | 取得的收入减按90%计入收入总额 |
| 政策依据 | 《中华人民共和国企业所得税法》第三十三条；《中华人民共和国企业所得税法实施条例》第九十九条 |
| 有效期限 | 长期有效 |

## 三、加计扣除

### 3.1 研发费用加计扣除

| 项目 | 内容 |
|------|------|
| 适用对象 | 会计核算健全、能够准确归集研发费用的居民企业 |
| 优惠内容 | 未形成无形资产计入当期损益的，在按规定据实扣除的基础上，按照研究开发费用的100%加计扣除；形成无形资产的，按照无形资产成本的200%在税前摊销 |
| 政策依据 | 《中华人民共和国企业所得税法》第三十条第（一）项；《财政部 税务总局 科技部关于完善研究开发费用税前加计扣除政策的通知》（财税〔2015〕119号）；《财政部 税务总局关于进一步完善研发费用税前加计扣除政策的公告》（财政部 税务总局公告2023年第7号） |
| 有效期限 | 长期有效（具体比例以最新政策为准） |
| 享受条件 | 1. 研发活动符合《研发费用加计扣除政策适用指引》的范围；2. 研发费用归集符合规定；3. 不属于烟草制造业、住宿和餐饮业、批发和零售业、房地产业、租赁和商务服务业、娱乐业等负面清单行业 |
| 注意事项 | 企业应设置研发费用辅助账，准确归集核算研发费用 |

### 3.2 安置残疾人员工资加计扣除

| 项目 | 内容 |
|------|------|
| 适用对象 | 安置残疾人员就业的企业 |
| 优惠内容 | 企业安置残疾人员的，在按照支付给残疾职工工资据实扣除的基础上，按照支付给残疾职工工资的100%加计扣除 |
| 政策依据 | 《中华人民共和国企业所得税法》第三十条第（二）项；《中华人民共和国企业所得税法实施条例》第九十六条 |
| 有效期限 | 长期有效 |
| 享受条件 | 1. 依法与残疾人签订一年以上劳动合同或服务协议；2. 为残疾人按月足额缴纳规定的社会保险；3. 通过银行等金融机构向残疾人实际支付了工资；4. 具备安置残疾人上岗工作的基本设施 |

## 四、减免税

### 4.1 高新技术企业优惠税率

| 项目 | 内容 |
|------|------|
| 适用对象 | 国家需要重点扶持的高新技术企业 |
| 优惠内容 | 减按15%的税率征收企业所得税 |
| 政策依据 | 《中华人民共和国企业所得税法》第二十八条第二款 |
| 有效期限 | 长期有效 |
| 享受条件 | 1. 拥有核心自主知识产权；2. 产品（服务）属于《国家重点支持的高新技术领域》范围；3. 研发费用占销售收入的比例符合规定（3%~5%）；4. 高新技术产品（服务）收入占企业总收入的比例不低于60%；5. 科技人员占企业职工总数的比例不低于10% |

### 4.2 技术先进型服务企业优惠税率

| 项目 | 内容 |
|------|------|
| 适用对象 | 经认定的技术先进型服务企业 |
| 优惠内容 | 减按15%的税率征收企业所得税 |
| 政策依据 | 《财政部 税务总局 商务部 科技部 国家发展改革委关于将技术先进型服务企业所得税政策推广至全国实施的通知》（财税〔2017〕79号） |
| 有效期限 | 长期有效 |

### 4.3 小型微利企业优惠

| 项目 | 内容 |
|------|------|
| 适用对象 | 从事国家非限制和禁止行业的小型微利企业 |
| 优惠内容 | 减按20%的税率征收企业所得税（实际税负根据最新政策可能进一步降低） |
| 政策依据 | 《中华人民共和国企业所得税法》第二十八条第一款；《财政部 税务总局关于进一步支持小微企业和个体工商户发展有关税费政策的公告》（财政部 税务总局公告2023年第12号）等后续政策 |
| 有效期限 | 以最新政策为准 |
| 享受条件 | 1. 年度应纳税所得额不超过300万元；2. 从业人数不超过300人；3. 资产总额不超过5000万元 |

### 4.4 西部大开发优惠税率

| 项目 | 内容 |
|------|------|
| 适用对象 | 设在西部地区的鼓励类产业企业 |
| 优惠内容 | 减按15%的税率征收企业所得税 |
| 政策依据 | 《财政部 税务总局 国家发展改革委关于延续西部大开发企业所得税政策的公告》（财政部 税务总局 国家发展改革委公告2020年第23号） |
| 有效期限 | 2021年1月1日至2030年12月31日 |
| 享受条件 | 1. 设在西部地区（包括12个省区市）；2. 以《西部地区鼓励类产业目录》中规定的产业项目为主营业务；3. 主营业务收入占企业收入总额60%以上 |

### 4.5 海南自由贸易港优惠税率

| 项目 | 内容 |
|------|------|
| 适用对象 | 注册在海南自由贸易港并实质性运营的鼓励类产业企业 |
| 优惠内容 | 减按15%的税率征收企业所得税 |
| 政策依据 | 《财政部 税务总局关于海南自由贸易港企业所得税优惠政策的通知》（财税〔2020〕31号） |
| 有效期限 | 2020年1月1日至2024年12月31日（后续可能延续） |
| 享受条件 | 1. 注册在海南自由贸易港；2. 实质性运营；3. 主营业务属于海南自由贸易港鼓励类产业目录 |

## 五、税额抵免

### 5.1 环境保护、节能节水、安全生产专用设备投资抵免

| 项目 | 内容 |
|------|------|
| 适用对象 | 购置并实际使用规定设备的企业 |
| 优惠内容 | 企业购置并实际使用《环境保护专用设备企业所得税优惠目录》《节能节水专用设备企业所得税优惠目录》和《安全生产专用设备企业所得税优惠目录》规定的环境保护、节能节水、安全生产等专用设备的，该专用设备的投资额的10%可以从企业当年的应纳税额中抵免；当年不足抵免的，可以在以后5个纳税年度结转抵免 |
| 政策依据 | 《中华人民共和国企业所得税法》第三十四条；《中华人民共和国企业所得税法实施条例》第一百条 |
| 有效期限 | 长期有效 |
| 享受条件 | 1. 设备必须在优惠目录范围内；2. 必须购置并实际使用；3. 5年内转让、出租的，应停止享受优惠并补缴已抵免的税款 |

## 六、加速折旧

### 6.1 固定资产加速折旧

| 项目 | 内容 |
|------|------|
| 适用对象 | 所有企业 |
| 优惠内容 | 1. 由于技术进步，产品更新换代较快的固定资产，可以缩短折旧年限或采取加速折旧方法；2. 常年处于强震动、高腐蚀状态的固定资产，可以缩短折旧年限或采取加速折旧方法；3. 单位价值不超过500万元的设备、器具，允许一次性计入当期成本费用在计算应纳税所得额时扣除 |
| 政策依据 | 《中华人民共和国企业所得税法》第三十二条；《财政部 税务总局关于设备 器具扣除有关企业所得税政策的公告》（财政部 税务总局公告2023年第37号） |
| 有效期限 | 一次性扣除政策以最新公告为准 |

### 6.2 集成电路和软件企业加速折旧

| 项目 | 内容 |
|------|------|
| 适用对象 | 集成电路生产企业和软件企业 |
| 优惠内容 | 生产设备的折旧年限可以适当缩短，最短可为3年 |
| 政策依据 | 《财政部 国家税务总局关于进一步鼓励软件产业和集成电路产业发展企业所得税政策的通知》（财税〔2012〕27号） |
| 有效期限 | 长期有效 |

## 七、特定行业优惠

### 7.1 集成电路生产企业优惠

| 项目 | 内容 |
|------|------|
| 适用对象 | 符合条件的集成电路生产企业 |
| 优惠内容 | 根据线宽不同，享受"两免三减半""五免五减半"或10年免征等不同优惠 |
| 政策依据 | 《财政部 税务总局 发展改革委 工业和信息化部关于促进集成电路产业和软件产业高质量发展企业所得税政策的公告》（财政部 税务总局 发展改革委 工业和信息化部公告2020年第45号） |
| 有效期限 | 长期有效 |

### 7.2 软件企业优惠

| 项目 | 内容 |
|------|------|
| 适用对象 | 符合条件的软件企业 |
| 优惠内容 | 自获利年度起，第一年至第二年免征企业所得税，第三年至第五年按照25%的法定税率减半征收企业所得税 |
| 政策依据 | 《财政部 国家税务总局关于进一步鼓励软件产业和集成电路产业发展企业所得税政策的通知》（财税〔2012〕27号）；《财政部 税务总局 发展改革委 工业和信息化部公告2020年第45号》 |
| 有效期限 | 长期有效 |

### 7.3 农林牧渔业项目减免

| 项目 | 内容 |
|------|------|
| 适用对象 | 从事农林牧渔业项目的企业 |
| 优惠内容 | 企业从事蔬菜、谷物、薯类、油料、豆类、棉花、麻类、糖料、水果、坚果的种植，农作物新品种的选育，中药材的种植，林木的培育和种植，牲畜、家禽的饲养，林产品的采集，灌溉、农产品初加工、兽医、农技推广、农机作业和维修等农、林、牧、渔服务业项目，远洋捕捞等项目的所得，免征企业所得税。企业从事花卉、茶以及其他饮料作物和香料作物的种植，海水养殖、内陆养殖等项目的所得，减半征收企业所得税 |
| 政策依据 | 《中华人民共和国企业所得税法》第二十七条第（一）项；《中华人民共和国企业所得税法实施条例》第八十六条 |
| 有效期限 | 长期有效 |

### 7.4 公共基础设施项目投资经营减免

| 项目 | 内容 |
|------|------|
| 适用对象 | 从事国家重点扶持的公共基础设施项目投资经营的所得 |
| 优惠内容 | 自项目取得第一笔生产经营收入所属纳税年度起，第一年至第三年免征企业所得税，第四年至第六年减半征收企业所得税（"三免三减半"） |
| 政策依据 | 《中华人民共和国企业所得税法》第二十七条第（二）项；《中华人民共和国企业所得税法实施条例》第八十七条 |
| 有效期限 | 长期有效 |
| 享受条件 | 项目必须属于《公共基础设施项目企业所得税优惠目录》规定的范围 |

### 7.5 环境保护、节能节水项目减免

| 项目 | 内容 |
|------|------|
| 适用对象 | 从事符合条件的环境保护、节能节水项目的所得 |
| 优惠内容 | 自项目取得第一笔生产经营收入所属纳税年度起，第一年至第三年免征企业所得税，第四年至第六年减半征收企业所得税（"三免三减半"） |
| 政策依据 | 《中华人民共和国企业所得税法》第二十七条第（三）项；《中华人民共和国企业所得税法实施条例》第八十八条 |
| 有效期限 | 长期有效 |

### 7.6 创业投资企业抵扣应纳税所得额

| 项目 | 内容 |
|------|------|
| 适用对象 | 创业投资企业 |
| 优惠内容 | 创业投资企业采取股权投资方式投资于未上市的中小高新技术企业2年以上的，可以按照其投资额的70%在股权持有满2年的当年抵扣该创业投资企业的应纳税所得额；当年不足抵扣的，可以在以后纳税年度结转抵扣 |
| 政策依据 | 《中华人民共和国企业所得税法》第三十一条；《中华人民共和国企业所得税法实施条例》第九十七条 |
| 有效期限 | 长期有效 |
