# 增值税税收优惠

## 一、小规模纳税人优惠

### 1.1 小规模纳税人免征增值税

| 项目 | 内容 |
|------|------|
| 适用对象 | 增值税小规模纳税人 |
| 优惠内容 | 月销售额10万元以下（含本数，以1个季度为1个纳税期的，季度销售额30万元以下）免征增值税 |
| 政策依据 | 《财政部 税务总局关于明确增值税小规模纳税人免征增值税政策的公告》（财政部 税务总局公告2021年第11号）；《财政部 税务总局关于延续实施应对疫情部分税费优惠政策的公告》（财政部 税务总局公告2022年第10号）延续执行 |
| 有效期限 | 长期执行（具体以最新公告为准） |
| 享受条件 | 1. 登记为增值税小规模纳税人；2. 月销售额不超过10万元（季度销售额不超过30万元） |

### 1.2 小规模纳税人减按1%征收率

| 项目 | 内容 |
|------|------|
| 适用对象 | 增值税小规模纳税人 |
| 优惠内容 | 适用3%征收率的应税销售收入，减按1%征收率征收增值税 |
| 政策依据 | 《财政部 税务总局关于延续实施应对疫情部分税费优惠政策的公告》（财政部 税务总局公告2022年第10号）；后续延续公告 |
| 有效期限 | 以最新公告为准 |
| 享受条件 | 小规模纳税人发生适用3%征收率的应税行为 |

## 三、出口退（免）税

### 3.1 出口货物退（免）税

| 项目 | 内容 |
|------|------|
| 适用对象 | 出口企业（包括外贸企业、生产企业等） |
| 优惠内容 | 出口货物实行退（免）税政策，具体包括免退税和免抵退税两种方式 |
| 政策依据 | 《财政部 国家税务总局关于出口货物劳务增值税和消费税政策的通知》（财税〔2012〕39号） |
| 有效期限 | 长期有效 |
| 享受条件 | 1. 必须是报关离境的出口货物；2. 必须是财务上已作销售处理的货物；3. 必须已收汇或符合视同收汇条件 |

### 3.2 跨境应税行为零税率或免税

| 项目 | 内容 |
|------|------|
| 适用对象 | 发生跨境应税行为的单位和个人 |
| 优惠内容 | 国际运输服务、航天运输服务适用零税率；部分跨境应税行为免征增值税 |
| 政策依据 | 《营业税改征增值税试点过渡政策的规定》（财税〔2016〕36号附件4） |
| 有效期限 | 长期有效 |

## 四、即征即退

### 4.1 软件产品增值税即征即退

| 项目 | 内容 |
|------|------|
| 适用对象 | 增值税一般纳税人销售其自行开发生产的软件产品 |
| 优惠内容 | 按13%税率征收增值税后，对其增值税实际税负超过3%的部分实行即征即退 |
| 政策依据 | 《财政部 国家税务总局关于软件产品增值税政策的通知》（财税〔2011〕100号） |
| 有效期限 | 长期有效 |
| 享受条件 | 1. 必须取得软件产业主管部门颁发的软件产品登记证书；2. 必须为自行开发生产的软件产品 |

### 4.2 安置残疾人就业增值税即征即退

| 项目 | 内容 |
|------|------|
| 适用对象 | 安置残疾人的单位和个体工商户 |
| 优惠内容 | 按安置残疾人的人数，限额即征即退增值税。每人每月退还限额为纳税人所在区县适用的经省级人民政府批准的月最低工资标准的4倍 |
| 政策依据 | 《财政部 国家税务总局关于促进残疾人就业增值税优惠政策的通知》（财税〔2016〕52号） |
| 有效期限 | 长期有效 |
| 享受条件 | 1. 实际安置的残疾人占在职职工人数的比例不低于25%；2. 安置的残疾人人数不少于10人；3. 依法与残疾人签订劳动合同；4. 为残疾人足额缴纳社会保险 |

### 4.3 资源综合利用产品增值税即征即退

| 项目 | 内容 |
|------|------|
| 适用对象 | 销售自产的资源综合利用产品和提供资源综合利用劳务的增值税一般纳税人 |
| 优惠内容 | 按不同产品类别实行30%~100%不等的即征即退比例 |
| 政策依据 | 《财政部 税务总局关于完善资源综合利用增值税政策的公告》（财政部 税务总局公告2021年第40号） |
| 有效期限 | 2022年3月1日起执行 |
| 享受条件 | 1. 产品原料中综合利用资源的比例符合规定；2. 纳税信用等级不为C级或D级；3. 符合环保相关要求 |

## 五、加计抵减

### 5.1 生产性服务业纳税人加计抵减

| 项目 | 内容 |
|------|------|
| 适用对象 | 提供邮政服务、电信服务、现代服务、生活服务取得的销售额占全部销售额比重超过50%的纳税人 |
| 优惠内容 | 按照当期可抵扣进项税额加计5%（或10%，以最新政策为准）抵减应纳税额 |
| 政策依据 | 《财政部 税务总局 海关总署关于深化增值税改革有关政策的公告》（财政部 税务总局 海关总署公告2019年第39号）及后续延续公告 |
| 有效期限 | 以最新延续公告为准 |

### 5.2 生活性服务业纳税人加计抵减

| 项目 | 内容 |
|------|------|
| 适用对象 | 提供生活服务取得的销售额占全部销售额比重超过50%的纳税人 |
| 优惠内容 | 按照当期可抵扣进项税额加计10%（或15%，以最新政策为准）抵减应纳税额 |
| 政策依据 | 《财政部 税务总局关于明确生活性服务业增值税加计抵减政策的公告》（财政部 税务总局公告2019年第87号）及后续延续公告 |
| 有效期限 | 以最新延续公告为准 |
