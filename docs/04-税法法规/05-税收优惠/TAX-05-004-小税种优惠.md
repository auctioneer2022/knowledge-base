# 小税种税收优惠

## 一、印花税优惠

### 1.1 小规模纳税人印花税减征

| 项目 | 内容 |
|------|------|
| 适用对象 | 增值税小规模纳税人、小型微利企业和个体工商户 |
| 优惠内容 | 可以在50%的税额幅度内减征印花税（不含证券交易印花税） |
| 政策依据 | 《财政部 税务总局关于进一步实施小微企业"六税两费"减免政策的公告》（财政部 税务总局公告2022年第10号）；《财政部 税务总局关于继续实施小微企业"六税两费"减免政策的公告》（财政部 税务总局公告2023年第12号） |
| 有效期限 | 2027年12月31日 |

### 1.2 个人销售或购买住房免征印花税

| 项目 | 内容 |
|------|------|
| 适用对象 | 个人 |
| 优惠内容 | 对个人销售或购买住房暂免征收印花税 |
| 政策依据 | 《财政部 国家税务总局关于调整房地产交易环节税收政策的通知》（财税〔2008〕137号） |
| 有效期限 | 长期有效（暂免） |

### 1.3 农产品购销合同免征印花税

| 项目 | 内容 |
|------|------|
| 适用对象 | 农民、家庭农场、农民专业合作社等 |
| 优惠内容 | 与农业生产者签订的农产品购销合同免征印花税 |
| 政策依据 | 《中华人民共和国印花税法》第十二条 |
| 有效期限 | 长期有效 |

### 1.4 无息、贴息贷款合同免征印花税

| 项目 | 内容 |
|------|------|
| 适用对象 | 签订无息、贴息贷款合同的各方 |
| 优惠内容 | 免征印花税 |
| 政策依据 | 《中华人民共和国印花税法》第十二条 |
| 有效期限 | 长期有效 |

### 1.5 财产所有权人将财产赠与政府、学校、社会福利机构、慈善组织书立的产权转移书据免征印花税

| 项目 | 内容 |
|------|------|
| 适用对象 | 财产所有权人 |
| 优惠内容 | 免征印花税 |
| 政策依据 | 《中华人民共和国印花税法》第十二条 |
| 有效期限 | 长期有效 |

## 六、城市维护建设税和教育费附加优惠

### 6.1 小规模纳税人"六税两费"减征

| 项目 | 内容 |
|------|------|
| 适用对象 | 增值税小规模纳税人、小型微利企业和个体工商户 |
| 优惠内容 | 城市维护建设税、教育费附加、地方教育附加可以在50%的税额幅度内减征 |
| 政策依据 | 《财政部 税务总局公告2022年第10号》；《财政部 税务总局公告2023年第12号》 |
| 有效期限 | 2027年12月31日 |

### 6.2 增值税、消费税减免时同步减免城建税和教育费附加

| 项目 | 内容 |
|------|------|
| 适用对象 | 享受增值税、消费税减免的纳税人 |
| 优惠内容 | 对增值税、消费税实行先征后返、先征后退、即征即退办法的，除另有规定外，对随增值税、消费税附征的城市维护建设税和教育费附加，一律不予退（返）还。但因减免增值税、消费税而退税的，可同时退还已征收的城市维护建设税和教育费附加 |
| 政策依据 | 《中华人民共和国城市维护建设税法》；《征收教育费附加的暂行规定》 |
| 有效期限 | 长期有效 |
