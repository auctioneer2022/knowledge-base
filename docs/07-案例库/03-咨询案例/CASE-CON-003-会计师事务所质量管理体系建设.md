# 会计师事务所质量管理体系建设案例

## 项目背景

某中型会计师事务所在中注协执业质量检查中被指出质量管理体系存在薄弱环节。随着《会计师事务所质量管理准则第5101号》的实施，事务所需要按照新准则的要求建立符合自身规模和质量风险特征的质量管理体系。事务所委托专业咨询机构协助完成体系建设。

## 服务过程

咨询团队首先对照质量管理准则的八个要素（事务所的风险评估程序、治理和领导层、职业道德、客户关系和具体业务的接受与保持、业务执行、资源、信息与沟通、监控和整改程序），对事务所现有质量管理状况进行了全面评估。随后识别出关键质量风险点，设计了针对性的质量管理政策和程序。

## 解决方案

主要措施包括：一是建立事务所层面的风险评估程序，定期识别和评估影响业务质量的风险；二是完善治理结构，明确质量管理责任人及其职责；三是加强职业道德管理，建立独立性声明和轮换制度；四是规范业务承接与保持程序，建立客户风险评估机制；五是完善业务执行的质量控制程序，包括指导、监督和复核；六是建立监控和整改程序，定期对质量管理体系进行内部检查。

## 成果总结

体系建设完成后，事务所顺利通过了中注协的执业质量检查，检查中未发现重大质量问题。事务所的业务质量明显提升，项目复核发现的问题数量减少约50%。该案例为中小会计师事务所建立质量管理体系提供了可借鉴的经验。
