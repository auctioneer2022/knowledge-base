# 企业并购财务尽职调查案例

## 项目背景

某上市公司拟收购一家同行业民营企业60%的股权，委托会计师事务所对目标公司进行财务尽职调查。目标公司成立10年，年营业收入约3亿元，但财务管理基础较为薄弱，存在会计政策与上市公司不一致、关联交易复杂等问题。

## 服务过程

尽职调查团队在两周内完成了对目标公司的全面财务调查。工作内容包括：审阅目标公司近三年财务报表及审计报告；分析收入确认政策和成本核算方法的合理性；核查主要客户和供应商的真实性及交易公允性；审查关联交易的完整性和定价公允性；评估资产质量和减值风险；分析税务合规性及潜在税务风险；识别表外负债和或有事项。

## 解决方案

调查发现的主要问题包括：一是目标公司收入确认时点与上市公司会计政策存在差异，需进行调整；二是存在未披露的关联方资金往来约2,000万元；三是部分应收账款账龄较长，坏账准备计提不足；四是存在未入账的环保整改义务。基于调查结果，咨询团队提出了交易价格调整建议和交割前整改要求。

## 成果总结

尽职调查报告为收购方的交易决策提供了重要依据。基于调查发现，收购方将交易价格下调约1,500万元，并在交易协议中设置了交割前整改条款和业绩承诺条款。该案例体现了财务尽职调查在并购交易中的关键价值，帮助收购方识别风险、合理定价。
