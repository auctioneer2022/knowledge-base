# 关联方交易未披露案例

## 案情概述

在某上市公司年度审计中，审计项目组发现该公司与多家企业存在大额资金往来和交易，但这些企业未被识别为关联方，相关交易也未在财务报表附注中披露。经深入调查发现，这些企业的实际控制人或关键管理人员与上市公司存在密切关系。

## 审计发现

审计项目组通过以下程序发现关联方交易未披露问题：一是检查大额资金往来的对手方信息，发现部分交易对方与上市公司董事、高管存在关联关系；二是通过企查查等工商信息平台查询交易对方的股权结构，发现存在共同控制人或重大影响关系；三是分析交易的商业实质，发现部分交易价格明显偏离市场价格，存在利益输送嫌疑；四是公司未建立完善的关联方识别和披露机制。

## 处理结果

审计项目组要求公司全面梳理关联方关系，补充识别并披露所有关联方及关联交易。调整涉及补充披露关联方关系12家，补充披露关联交易金额约8,000万元。同时对交易价格的公允性进行了评估，对不公允的部分提请公司进行调整。

## 要点分析

关联方交易审计是审计实务中的难点。审计师应运用多种手段识别关联方关系，包括查询工商信息、分析资金往来、关注异常交易等。对于识别出的关联方交易，应评估其商业实质和定价公允性。事务所应建立关联方审计的标准程序，确保关联方识别和披露的完整性。
