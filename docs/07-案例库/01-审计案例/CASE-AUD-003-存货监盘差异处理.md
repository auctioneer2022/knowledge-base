# 存货监盘差异处理案例

## 案情概述

在某制造业企业年度审计中，审计项目组在执行存货监盘程序时发现，企业账面存货数量与实际盘点数量存在重大差异。部分原材料和产成品的实际库存明显少于账面记录，差异金额约占存货总额的15%。

## 审计发现

审计项目组在监盘过程中发现以下问题：一是部分仓库的存货未纳入盘点范围，企业以"委托加工"为由未提供完整清单；二是部分产成品的规格型号与账面记录不符，存在以次充好的情况；三是存货存放地点分散，部分异地仓库的存货未实施监盘；四是企业存货管理制度不健全，出入库记录不完整，存在账实不符的系统性问题。

## 处理结果

审计项目组要求企业扩大盘点范围至所有存货存放地点，对差异原因进行逐项调查。经调查确认，差异主要由以下原因造成：一是部分存货已发出但未及时记账；二是部分存货因质量问题已报废但未进行账务处理；三是存在存货管理漏洞导致的盘亏。审计调整涉及冲减存货约2,000万元，相应调整营业成本和管理费用。

## 要点分析

存货监盘是验证存货存在性和完整性的重要审计程序。审计师应确保盘点范围覆盖所有存货存放地点，对异地存货应亲自监盘或委托其他审计师代为监盘。对于盘点差异，应深入调查原因并区分正常损耗和异常损失。存货管理薄弱的企业，审计师应考虑扩大实质性程序的范围。
