# 研发费用资本化与费用化调整案例

## 案情概述

在某高新技术企业年度审计中，审计项目组发现该公司研发费用的资本化与费用化处理存在不当之处。公司将部分尚处于研究阶段的开发支出资本化为无形资产，同时又将部分已进入开发阶段且满足资本化条件的支出费用化，导致资产和利润的计量不准确。

## 审计发现

审计项目组通过以下程序发现研发费用处理问题：一是检查研发项目的立项文件和技术可行性报告，发现部分资本化的研发项目尚未完成技术可行性论证，不满足资本化条件；二是分析研发支出明细，发现部分应费用化的日常技术改进支出被计入开发支出；三是检查无形资产的后续计量，发现部分已资本化的开发支出对应的技术已被淘汰，未计提减值准备；四是公司的研发费用归集和分摊方法不够规范。

## 处理结果

审计项目组要求公司按照《企业会计准则第6号——无形资产》的规定，重新划分研究阶段和开发阶段的支出。审计调整涉及将不符合资本化条件的开发支出约2,000万元调整为当期费用，同时对满足资本化条件但被费用化的支出约500万元调整为资本化。调整后相应影响当期利润和无形资产账面价值。

## 要点分析

研发费用的资本化与费用化划分是高新技术企业审计的重点。审计师应深入理解企业的研发流程，准确判断研究阶段和开发阶段的分界点，严格审查资本化的五项条件是否同时满足。对于研发支出金额较大的企业，应考虑利用专家工作评估技术可行性。
