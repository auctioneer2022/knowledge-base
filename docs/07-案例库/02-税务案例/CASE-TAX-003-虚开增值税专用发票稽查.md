# 虚开增值税专用发票稽查案例

## 案情概述

某贸易公司在经营过程中，为降低税负，通过支付手续费的方式从多家企业取得增值税专用发票用于抵扣进项税额。税务机关通过大数据分析和发票协查发现异常，对该企业立案稽查。

## 税务处理/争议焦点

税务机关查实的主要问题：一是企业取得的增值税专用发票对应的销售方与实际供货方不一致，存在"票货分离"情形；二是部分发票对应的交易缺乏真实的货物往来，资金流与发票流不匹配；三是企业通过个人账户向开票方支付手续费，资金回流痕迹明显；四是企业无法提供完整的购销合同、运输单据和入库记录等交易证据。

## 处理结果

税务机关认定企业取得虚开增值税专用发票，涉及税额约300万元。处理结果包括：追缴已抵扣的增值税税款300万元，加收滞纳金，并处少缴税款一倍的罚款。同时，将案件移送公安机关追究刑事责任。企业法定代表人和相关责任人被采取刑事强制措施。

## 要点分析

虚开增值税专用发票是严重的税收违法行为，不仅面临补税罚款，还可能承担刑事责任。企业应确保"三流一致"（资金流、发票流、货物流），建立完善的供应商审核和发票管理制度。税务机关利用大数据手段对发票异常的识别能力不断增强，企业切勿心存侥幸。
