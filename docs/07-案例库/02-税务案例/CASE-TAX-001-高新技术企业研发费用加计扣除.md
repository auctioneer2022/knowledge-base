# 高新技术企业研发费用加计扣除案例

## 案情概述

某高新技术企业在年度企业所得税汇算清缴中申报研发费用加计扣除，税务机关在后续核查中发现企业归集的研发费用中包含部分不符合加计扣除条件的支出，涉及金额约500万元，导致多享受加计扣除优惠。

## 税务处理/争议焦点

税务机关核查发现的主要问题：一是企业将部分日常生产技术改进支出归入研发费用，但该类活动不属于税法定义的研发活动；二是部分研发人员同时从事生产和研发工作，其工资薪金未按实际工时分摊，全额计入了研发费用；三是部分材料领用缺乏研发项目领料单，无法证明专门用于研发活动；四是委托外部研发的费用未按80%比例计算加计扣除基数。

## 处理结果

税务机关要求企业重新归集研发费用，剔除不符合条件的支出约500万元，补缴企业所得税及滞纳金约150万元。企业接受处理决定，并完善了研发费用辅助账和归集制度。

## 要点分析

研发费用加计扣除是企业重要的税收优惠政策，但享受优惠的前提是研发费用的归集必须规范、准确。企业应建立完善的研发费用辅助账，准确区分研发支出与生产经营支出，对共用人员工时应按合理方法分摊。税务机关对研发费用加计扣除的核查日益严格，企业应提前做好合规管理。
