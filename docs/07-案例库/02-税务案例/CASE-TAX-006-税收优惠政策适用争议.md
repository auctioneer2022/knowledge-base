# 税收优惠政策适用争议案例

## 案情概述

某软件企业在年度企业所得税汇算清缴中享受软件企业"两免三减半"优惠政策。税务机关核查后认为企业不符合软件企业的认定条件，取消其优惠资格并要求补缴税款。企业不服，申请税务行政复议。

## 税务处理/争议焦点

争议焦点主要围绕企业是否符合软件企业的认定条件：一是企业软件产品开发销售（营业）收入占企业收入总额的比例是否达到50%以上，税务机关认为企业将部分硬件销售收入混入软件收入；二是企业研发费用占企业销售（营业）收入的比例是否达到6%以上，税务机关对研发费用的归集范围提出异议；三是企业是否具有核心关键技术，税务机关认为企业的主要产品技术含量不足；四是企业职工总数中大专以上学历人员的比例是否达标。

## 处理结果

税务行政复议机关经审理认为，企业在软件收入占比和研发费用占比方面确实存在计算口径不当的问题，但核心关键技术的认定应综合考虑企业的实际情况。最终复议决定维持税务机关关于取消优惠资格的决定，但允许企业重新核算后再次申请认定。企业补缴税款约600万元。

## 要点分析

税收优惠政策的适用条件往往有明确的量化标准，企业在享受优惠前应仔细核对各项条件是否满足。对于认定标准存在争议的指标，应提前做好资料准备和论证工作。在税务机关提出异议时，企业应积极配合核查，必要时通过行政复议等法律途径维护合法权益。
