# 增值税进项税额抵扣争议案例

## 案情概述

某制造企业在增值税申报中抵扣了大量进项税额，税务机关在检查中认为部分进项税额不符合抵扣条件，要求企业作进项税额转出处理。企业对税务机关的处理决定不服，认为其抵扣行为符合税法规定，遂申请税务行政复议。

## 税务处理/争议焦点

争议的主要焦点包括：一是企业购进的部分原材料用于生产免税产品，对应的进项税额是否应转出，税务机关认为应按免税产品销售额占总销售额的比例计算转出；二是企业取得的增值税专用发票中，部分发票的货物名称与实际采购内容不完全一致，税务机关认为属于不合规发票不得抵扣；三是企业将部分购进货物用于职工福利，对应的进项税额是否已全额转出；四是企业不动产在建工程领用原材料的进项税额抵扣是否符合规定。

## 处理结果

税务行政复议机关经审理认为：用于免税产品的进项税额应按销售额比例计算转出，企业计算方法有误，需补转进项税额约80万元；发票货物名称不一致但能证明交易真实性的，可以抵扣；用于职工福利的进项税额企业已正确转出；不动产在建工程领用原材料的进项税额抵扣符合现行规定。最终企业补转进项税额80万元。

## 要点分析

增值税进项税额抵扣是税务实务中的常见问题。企业应严格区分可抵扣和不可抵扣的进项税额，特别是用于免税项目、简易计税项目、集体福利和个人消费的购进货物或服务，其进项税额不得抵扣。对于发票合规性问题，企业应确保取得的增值税专用发票信息与实际交易一致。
