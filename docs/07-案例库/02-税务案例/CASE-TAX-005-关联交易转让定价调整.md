# 关联交易转让定价调整案例

## 案情概述

某跨国公司在华子公司与其境外母公司之间存在大量关联交易，包括产品销售、技术服务费、商标使用费等。税务机关在转让定价调查中发现，该企业通过不合理的转让定价将利润转移至境外低税率地区，减少了在中国的应纳税所得额。

## 税务处理/争议焦点

税务机关调查的主要发现：一是企业向境外母公司支付的技术服务费费率明显高于市场水平，缺乏合理的定价依据；二是企业向关联方销售产品的价格低于向非关联方销售的价格，存在利润转移；三是企业使用的商标由境外母公司授权，支付的商标使用费占收入比例过高；四是企业未能提供完整的同期资料文档，无法证明关联交易符合独立交易原则。

## 处理结果

税务机关按照可比非受控价格法和交易净利润法对关联交易价格进行调整，调增企业应纳税所得额约5,000万元，补征企业所得税约1,250万元，并加收利息。企业接受调整后，重新制定了转让定价政策并完善了同期资料文档。

## 要点分析

转让定价是国际税收领域的重点监管方向。企业应确保关联交易符合独立交易原则，按照税法要求准备同期资料文档，包括主体文档、本地文档和国别报告。对于金额重大的关联交易，可以考虑申请预约定价安排，降低税务不确定性。
