# 企业重组特殊性税务处理案例

## 案情概述

某集团公司进行内部股权重组，母公司以其持有的子公司股权对另一子公司进行增资。企业选择适用特殊性税务处理，但在税务机关核查中被认定不符合特殊性税务处理的条件，需要调整为一般性税务处理并补缴税款。

## 税务处理/争议焦点

税务机关认定不符合特殊性税务处理条件的原因：一是重组交易中股权支付比例未达到85%的要求，部分对价以现金支付；二是重组后的连续12个月内，受让方改变了重组资产的原有实质性经营活动；三是企业未按规定在重组完成当年进行特殊性税务处理的备案；四是重组交易的商业目的说明不充分，难以证明具有合理的商业目的而非以减少税款为主要目的。

## 处理结果

税务机关要求企业调整为一般性税务处理，确认股权转让所得并补缴企业所得税约800万元及滞纳金。企业接受处理决定后，在后续重组交易中严格按照特殊性税务处理的条件进行操作，并提前做好备案工作。

## 要点分析

企业重组特殊性税务处理的条件较为严格，企业应重点关注股权支付比例、经营连续性、合理商业目的等核心要件。在重组方案设计阶段就应充分考虑税务影响，确保满足特殊性税务处理的全部条件，并按规定时限完成备案。建议在重大重组交易中提前与税务机关沟通。
