# 隐瞒收入偷逃企业所得税稽查案例

## 案情概述

某制造企业在日常经营中，通过个人账户收取部分销售收入不入账、以物易物不确认收入等方式隐瞒收入，偷逃企业所得税。税务机关通过银行流水分析和第三方信息比对发现异常，对企业进行税务稽查。

## 税务处理/争议焦点

税务机关查实的主要手段包括：一是企业通过法定代表人和财务人员的个人银行账户收取客户货款，未计入企业收入；二是企业以产品换取其他企业的服务或物资，未确认销售收入；三是企业将部分废料销售收入以现金方式收取，未入账核算；四是企业存在跨期调节收入的情形，将当年收入延迟至次年确认。

## 处理结果

税务机关查实企业隐瞒收入约2,000万元，涉及少缴企业所得税约500万元。处理结果包括：追缴企业所得税500万元，加收滞纳金，并处少缴税款50%的罚款，合计约800万元。企业接受处理决定并补缴全部税款和罚款。

## 要点分析

隐瞒收入是常见的税收违法行为，但随着金税四期的推进和部门间信息共享的加强，税务机关通过银行流水、第三方数据等手段发现隐瞒收入的能力大幅提升。企业应依法如实申报全部收入，建立健全的内部控制制度，确保所有经济业务均纳入会计核算。
