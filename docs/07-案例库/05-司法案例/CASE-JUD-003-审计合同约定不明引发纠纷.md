# 审计合同约定不明引发纠纷案例

## 案情概述

某企业与会计师事务所签订年度财务报表审计业务约定书，但约定书中对审计范围、审计费用调整机制、报告交付时间等关键条款约定不够明确。审计过程中，企业要求会计师事务所增加专项审计内容，双方就费用调整产生争议。会计师事务所完成审计后，企业以审计质量不达标为由拒绝支付审计费用。

## 争议焦点

本案的主要争议焦点：一是审计业务约定书中未明确的审计范围应如何界定；二是企业要求增加的专项审计是否属于原合同约定的范围；三是企业以审计质量不达标为由拒付费用的抗辩是否成立；四是审计费用的合理金额应如何确定。

## 裁判要旨

法院认为，审计业务约定书是确定双方权利义务的基础文件。对于约定不明的条款，应结合行业惯例和合同目的进行解释。企业要求增加的专项审计超出了原约定的年度财务报表审计范围，会计师事务所有权要求另行收费。关于审计质量，企业未能提供充分证据证明审计存在重大缺陷，拒付费用的抗辩不成立。最终判决企业支付审计费用及逾期付款利息。

## 要点分析

审计业务约定书的条款应当明确、具体，特别是审计范围、费用标准、报告交付时间、双方责任等关键条款。对于审计过程中可能出现的范围变更，应事先约定调整机制。本案例提醒会计师事务所在签订业务约定书时应充分沟通、明确约定，避免后续纠纷。
