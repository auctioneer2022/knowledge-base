# 注册会计师出具虚假验资报告民事责任案例

## 案情概述

某注册会计师事务所在为企业出具验资报告时，未对出资人的出资情况进行实质性审验，仅依据出资人提供的银行进账单即出具了验资报告。后经查明，该银行进账单系伪造，出资人实际未缴纳出资。企业债权人因企业无力偿还债务，起诉会计师事务所要求在虚假验资范围内承担赔偿责任。

## 争议焦点

本案的主要争议焦点：一是注册会计师在验资业务中应承担的注意义务标准；二是会计师事务所出具虚假验资报告与债权人损失之间的因果关系如何认定；三是会计师事务所的赔偿责任范围应如何确定。

## 裁判要旨

法院认为，注册会计师在执行验资业务时应当保持应有的职业谨慎，对出资证明材料的真实性进行合理审验。会计师事务所仅依据出资人单方提供的银行进账单即出具验资报告，未尽到基本的审验义务，存在过错。判决会计师事务所在虚假验资金额范围内对企业债务不能清偿的部分承担补充赔偿责任。

## 要点分析

验资业务虽然风险相对较低，但注册会计师仍需保持基本的职业谨慎。对于银行进账单等关键证据，应通过函证等方式进行验证，不能仅凭客户提供的材料即出具报告。本案提醒注册会计师在所有业务中都应严格执行必要的审验程序，不能因业务简单而降低执业标准。
