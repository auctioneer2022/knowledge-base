# 会计师事务所未尽审计义务导致投资者损失案例

## 案情概述

某上市公司连续多年财务造假，虚增收入和利润。为其提供审计服务的会计师事务所在多年审计中均出具了标准无保留意见的审计报告。财务造假被揭露后，公司股票价格大幅下跌，投资者遭受重大损失。投资者以证券虚假陈述为由，起诉上市公司、实际控制人及会计师事务所，要求赔偿投资损失。

## 争议焦点

本案的主要争议焦点：一是会计师事务所在多年审计中均未发现财务造假，是否构成未勤勉尽责；二是会计师事务所的过错程度如何认定，是否构成故意或重大过失；三是会计师事务所应承担的赔偿责任比例如何确定；四是投资者损失与虚假陈述之间的因果关系如何认定。

## 裁判要旨

法院认为，会计师事务所在连续多年的审计中均未发现公司系统性财务造假，说明其审计程序存在重大缺陷，未保持应有的职业怀疑，构成重大过失。但考虑到造假手段隐蔽、管理层串通舞弊等因素，会计师事务所的过错程度低于上市公司实际控制人。判决会计师事务所对投资者损失承担30%的连带赔偿责任。

## 要点分析

本案涉及证券虚假陈述中会计师事务所的民事责任认定。法院在确定赔偿比例时综合考虑了会计师事务所的过错程度、原因力大小以及造假行为的隐蔽性等因素。会计师事务所应特别关注管理层舞弊风险，对异常财务指标保持职业怀疑，严格执行审计准则规定的各项程序。
