# 会计师事务所审计侵权赔偿责任案例

## 案情概述

某上市公司因财务造假被证监会处罚，多名投资者以证券虚假陈述为由起诉上市公司及为其提供审计服务的会计师事务所，要求赔偿投资损失。会计师事务所辩称其已按照审计准则执行了审计程序，不应承担赔偿责任。

## 争议焦点

本案的主要争议焦点包括：一是会计师事务所在审计过程中是否存在过错，是否未尽到勤勉尽责义务；二是会计师事务所的审计行为与投资者损失之间是否存在因果关系；三是如果存在责任，会计师事务所应承担连带责任还是补充赔偿责任；四是投资者损失的计算方法和范围。

## 裁判要旨

法院经审理认为，会计师事务所在审计过程中存在未严格执行函证程序、对异常交易未保持职业怀疑等过错，出具的审计报告存在虚假记载，构成对投资者的侵权。但考虑到会计师事务所的过错程度和原因力大小，判决会计师事务所对投资者损失承担20%的补充赔偿责任，而非连带责任。

## 要点分析

本案确立了会计师事务所审计侵权责任的"比例责任"原则，即根据会计师事务所的过错程度确定赔偿比例，而非一律承担连带责任。这一裁判规则平衡了投资者保护和会计师事务所合理责任的关系。会计师事务所应严格执行审计准则，保持职业怀疑，完善审计工作底稿，以降低法律风险。
