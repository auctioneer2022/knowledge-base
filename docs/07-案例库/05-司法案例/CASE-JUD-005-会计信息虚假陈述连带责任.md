# 会计信息虚假陈述连带责任案例

## 案情概述

某上市公司在招股说明书和年度报告中存在严重的财务造假行为，虚增收入和利润数亿元。为其提供审计、保荐、法律等服务的多家中介机构均被投资者起诉。本案是近年来涉及中介机构连带责任的代表性案例，法院对各中介机构的责任进行了详细划分。

## 争议焦点

本案的主要争议焦点：一是各中介机构（会计师事务所、保荐机构、律师事务所）在虚假陈述中的责任如何区分；二是会计师事务所是否应对招股说明书和年度报告中的全部虚假内容承担连带责任；三是中介机构之间的责任是连带责任还是按份责任；四是投资者损失的计算基准日和计算方法。

## 裁判要旨

法院认为，各中介机构应根据其专业领域和职责范围承担相应的责任。会计师事务所作为财务信息的"看门人"，应对其审计的财务报表及相关信息的真实性承担主要责任。判决会计师事务所对投资者因财务信息虚假陈述造成的损失承担60%的连带赔偿责任，保荐机构承担40%的连带赔偿责任，律师事务所因其职责范围不涉及财务信息，不承担财务虚假陈述的赔偿责任。

## 要点分析

本案明确了证券虚假陈述中各中介机构的"看门人"责任边界。会计师事务所作为财务信息的专业把关者，对财务信息的真实性承担较高的注意义务和较重的责任。这一裁判规则进一步强化了中介机构的勤勉尽责义务，对审计行业具有深远的警示意义。
