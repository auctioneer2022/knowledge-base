## 说明

1. 审计准则应用指南与对应准则条文配套使用，提供具体的操作指引和示例。
2. 审计准则问题解答由中注协根据实务中遇到的问题发布，与审计准则及应用指南一并掌握和执行。
3. 职业道德守则问题解答针对职业道德守则实务应用中的问题提供指引。
4. 以上文件全文来源于中国注册会计师协会官方网站（cicpa.org.cn）。
