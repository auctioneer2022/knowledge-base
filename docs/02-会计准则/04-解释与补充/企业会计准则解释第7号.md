# 企业会计准则解释第7号

**发文字号：** 财会〔2015〕19号
**发布日期：** 2015年11月4日

财政部关于印发《企业会计准则解释第7号》的通知

各省、自治区、直辖市、计划单列市财政厅（局），新疆生产建设兵团财务局：

为了深入贯彻实施企业会计准则，解决执行中出现的问题，我部制定了《企业会计准则解释第7号》，现予印发，请遵照执行。

## 一、投资方因其他投资方对其子公司增资而导致本投资方持股比例下降，从而丧失控制权但能实施共同控制或施加重大影响的会计处理

### （一）相关准则

本问题涉及《企业会计准则第2号——长期股权投资》和《企业会计准则第33号——合并财务报表》。

### （二）会计处理

投资方应当区分个别财务报表和合并财务报表进行会计处理：

**个别财务报表：**

1. 应当对该项长期股权投资从成本法转为权益法核算。

2. 首先，按照新的持股比例确认本投资方应享有的原子公司因增资扩股而增加净资产的份额，与应结转持股比例下降部分所对应的长期股权投资原账面价值之间的差额计入当期损益。

3. 然后，按照新的持股比例视同自取得投资时即采用权益法核算进行调整。

**合并财务报表：**

应当按照《企业会计准则第33号——合并财务报表》的有关规定进行会计处理。

## 二、设定受益计划净负债或净资产的重新计量的会计处理

### （一）相关准则

本问题涉及《企业会计准则第9号——职工薪酬》。

### （二）会计处理

设定受益计划净负债或净资产的重新计量产生的变动，计入其他综合收益后，在后续会计期间不允许转回至损益。

在设定受益计划结算时，上述计入其他综合收益的金额应当在所有者权益内部结转，从其他综合收益转入未分配利润。

## 三、子公司发行优先股等金融工具时在合并利润表中的列报

### （一）相关准则

本问题涉及《企业会计准则第33号——合并财务报表》和《企业会计准则第37号——金融工具列报》。

### （二）会计处理

子公司发行的分类为权益工具的金融工具（如优先股等），在合并利润表中计算归属于母公司所有者的净利润时，应当按照以下规定扣除：

1. **累积型金融工具：** 无论当期是否宣告发放股利，均应当扣除归属于其他方持有人的当期应享有的股利，列示为少数股东损益。

2. **非累积型金融工具：** 仅在当期宣告发放股利时，扣除归属于其他方持有人的股利，列示为少数股东损益。

此前未按照上述规定列报的，应当对可比期间的数据进行追溯调整。

## 四、全资子公司改为分公司的会计处理

### （一）相关准则

本问题涉及《企业会计准则第2号——长期股权投资》和《企业会计准则第20号——企业合并》。

### （二）会计处理

母公司将其全资子公司改为分公司的，母公司应当按照以下规定进行会计处理：

1. 母公司应当以原取得该子公司投资时确定的各项资产、负债的公允价值（非同一控制下企业合并）或账面价值（同一控制下企业合并）为基础，持续计算该子公司各项资产、负债的价值。

2. 抵销母子公司之间的内部交易产生的未实现损益，并确认相应的递延所得税。

3. 原确认的商誉，在同一控制下的企业合并中，应当转入留存收益；在非同一控制下的企业合并中，应当作为分公司的资产列示。

4. 原确认的专项储备、其他综合收益等，应当转入母公司的相应科目。

5. 上述处理产生的差额，调整资本公积；资本公积不足冲减的，调整留存收益。

## 五、限制性股票激励计划的会计处理及每股收益的计算

### （一）相关准则

本问题涉及《企业会计准则第7号——股份支付》和《企业会计准则第34号——每股收益》。

### （二）会计处理

1. 企业收到职工认购限制性股票的款项时，应当确认股本和股本溢价。

2. 同时，就回购义务确认负债（作为收购库存股处理）。

3. 等待期内收到的现金股利，应当区分是否可以撤销分别处理：
   - **可撤销的：** 作为回购价格的减少处理。
   - **不可撤销的：** 按照利润分配处理。

4. **基本每股收益的计算：** 分子为归属于普通股股东的当期利润，分母为发行在外的普通股加权平均数。在等待期内，限制性股票不纳入基本每股收益的计算。

5. **稀释每股收益的计算：** 如果解锁条件包含业绩条件的，应当假设资产负债表日即为解锁日，据此评估稀释效应。
