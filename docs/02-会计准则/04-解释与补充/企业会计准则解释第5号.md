# 企业会计准则解释第5号

**发文字号：** 财会〔2012〕19号
**发布日期：** 2012年11月5日

财政部关于印发《企业会计准则解释第5号》的通知

各省、自治区、直辖市、计划单列市财政厅（局），新疆生产建设兵团财务局：

为了贯彻实施企业会计准则，根据企业会计准则执行情况和执行中出现的问题，我部制定了《企业会计准则解释第5号》，现予印发，请遵照执行。

## 一、非同一控制下企业合并中购买方取得的未入账无形资产的确认

### （一）相关准则

本问题涉及《企业会计准则第20号——企业合并》和《企业会计准则第6号——无形资产》。

### （二）会计处理

非同一控制下的企业合并中，购买方在对企业合并中取得的被购买方资产和负债进行确认时，对于被购买方拥有但在其个别财务报表中未确认的无形资产，应当进行充分识别。

满足以下条件之一的，应当确认为无形资产：

1. 源于合同性权利或其他法定权利。

2. 能够从被购买方中分离或者划分出来，并能单独或与相关合同、资产和负债一起，用于出售、转移、授予许可、租赁或交换。

购买方应当在附注中披露上述无形资产的估值方法和公允价值的确定依据。

## 二、信用风险缓释工具的会计处理

### （一）相关准则

本问题涉及《企业会计准则第22号——金融工具确认和计量》和《企业会计准则第25号——原保险合同》。

### （二）会计处理

信用风险缓释工具（如信用违约互换等）的交易双方，应当按照实质重于形式的原则，判断该工具是否构成财务担保合同。

1. **构成财务担保合同的：** 应当按照《企业会计准则第22号——金融工具确认和计量》中关于财务担保合同的有关规定进行会计处理。相关收入应当在合同期内按照系统合理的方法进行摊销。

2. **不构成财务担保合同的：** 应当作为衍生工具，以公允价值计量且其变动计入当期损益。

在财务报表列报和披露方面，应当按照相关金融工具和保险合同准则的规定执行。

## 三、附追索权的金融资产转让的终止确认

### （一）相关准则

本问题涉及《企业会计准则第23号——金融资产转移》。

### （二）会计处理

企业将持有的金融资产（如应收票据等）出售或背书转让，但附有追索权的，应当根据金融资产所有权上几乎所有的风险和报酬是否转移来判断是否终止确认：

1. **风险和报酬已转移的：** 应当终止确认该金融资产。

2. **风险和报酬未转移的：** 不应当终止确认该金融资产。

3. **既未转移也未保留几乎所有风险和报酬的：** 应当判断企业是否保留了对该金融资产的控制。未保留控制的，终止确认该金融资产；保留控制的，按照继续涉入程度继续确认该金融资产。

## 四、银行同业代付业务的会计处理

### （一）相关准则

本问题涉及《企业会计准则第22号——金融工具确认和计量》和《企业会计准则第37号——金融工具列报》。

### （二）会计处理

委托行和受托行应当根据同业代付业务的协议实质进行会计处理：

1. **委托行承担无条件还款义务的：** 委托行应当按照贷款进行会计处理；受托行应当按照拆出资金进行会计处理。

2. **申请人负责还款、委托行仅提供担保的：** 委托行应当将交易拆分为担保和代理两部分分别进行会计处理；受托行应当按照对申请人的贷款进行会计处理。

委托行和受托行不得将相关资产和负债相互抵销。

## 五、分步处置子公司股权至丧失控制权的会计处理

### （一）相关准则

本问题涉及《企业会计准则第33号——合并财务报表》和《企业会计准则第2号——长期股权投资》。

### （二）会计处理

企业通过多次交易分步处置对子公司股权投资直至丧失控制权的，如果上述交易属于"一揽子交易"，应当将各项交易作为一项处置子公司并丧失控制权的交易进行会计处理。

**"一揽子交易"的判断标准**包括但不限于：

1. 这些交易是同时或者在考虑了彼此影响的情况下订立的。

2. 这些交易整体才能达成一项完整的商业结果。

3. 一项交易的发生取决于其他至少一项交易的发生。

4. 一项交易单独看时是不经济的，但是和其他交易一并考虑时是经济的。

在丧失控制权之前，每次处置价款与处置投资对应的享有该子公司净资产份额的差额，在合并财务报表中应当确认为其他综合收益，在丧失控制权时一并转入丧失控制权当期的损益。

## 六、接受非控股股东代为偿债或捐赠的会计处理

### （一）相关准则

本问题涉及《企业会计准则——基本准则》和《企业会计准则第16号——政府补助》。

### （二）会计处理

企业接受非控股股东（或非控股股东的子公司）直接或间接代为偿债、债务豁免或捐赠的，通常应当计入当期损益。

但是，如果有确凿证据表明上述交易的经济实质属于非控股股东对企业的资本性投入，应当将相关利得计入资本公积（资本溢价）。

在企业破产重整过程中，非控股股东通过让渡其持有的企业股份来偿还企业债务的，企业应当按照让渡日该股份的公允价值增加资本公积（资本溢价），同时减少相关债务的账面价值，差额计入当期损益。

本解释自2013年1月1日起施行，不要求进行追溯调整。
