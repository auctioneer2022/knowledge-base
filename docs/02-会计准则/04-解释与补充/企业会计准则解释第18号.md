# 企业会计准则解释第18号

**发文字号：** 财会〔2024〕x号
**发布日期：** 2024年

财政部关于印发《企业会计准则解释第18号》的通知

各省、自治区、直辖市、计划单列市财政厅（局），新疆生产建设兵团财务局：

为了深入贯彻实施企业会计准则，解决执行中出现的问题，我部制定了《企业会计准则解释第18号》，现予印发，请遵照执行。

---

> **注：** 本解释全文待补充。

## 概述

企业会计准则解释第18号由财政部于2024年发布，旨在对企业会计准则执行中出现的具体问题进行解释和补充规定。

根据上市公司会计政策变更公告的披露，企业会计准则解释第18号涉及以下方面的会计处理规定（具体以财政部正式发布的文件为准）：

1. 相关会计处理问题的补充规定。
2. 对企业会计准则执行中常见问题的进一步明确。
