# 政府会计准则制度概述

## 一、政府会计标准体系的构成

我国政府会计标准体系由以下四个层次构成：

### （一）基本准则

基本准则是政府会计准则体系的"概念框架"，由财政部以部长令形式发布（财政部令第78号，2015年发布，2023年修改）。基本准则对政府会计目标、会计主体、会计信息质量要求、会计要素的定义和确认计量原则、决算报告和财务报告等作出总体规定，在政府会计准则体系中起统驭作用。

### （二）具体准则

具体准则依据基本准则制定，对政府会计主体发生的各项经济业务和事项的确认、计量、记录和报告作出具体规定。截至2024年底，财政部已陆续发布以下具体准则：

| 准则编号 | 准则名称 | 发文字号 | 发布时间 |
|----------|----------|----------|----------|
| 第1号 | 存货 | 财会〔2016〕12号 | 2016年7月 |
| 第2号 | 投资 | 财会〔2016〕12号 | 2016年7月 |
| 第3号 | 固定资产 | 财会〔2016〕12号 | 2016年7月 |
| 第4号 | 无形资产 | 财会〔2016〕12号 | 2016年7月 |
| 第5号 | 公共基础设施 | 财会〔2017〕11号 | 2017年7月 |
| 第6号 | 政府储备物资 | 财会〔2017〕11号 | 2017年7月 |
| 第7号 | 会计调整 | 财会〔2018〕28号 | 2018年10月 |
| 第8号 | 负债 | 财会〔2018〕32号 | 2018年11月 |
| 第9号 | 财务报表编制和列报 | 财会〔2019〕6号 | 2019年5月 |
| 第10号 | 政府和社会资本合作项目合同 | 财会〔2019〕23号 | 2019年12月 |

此外，财政部还发布了职工薪酬等相关准则，对政府会计主体应付职工薪酬的确认、计量和披露进行了规范。

### （三）应用指南

应用指南是对具体准则的操作性指引，为具体准则的实施提供详细指导，包括具体准则涉及的主要账务处理、报表列示等内容。应用指南有助于政府会计主体准确理解和执行具体准则。

### （四）解释

政府会计准则解释是对具体准则及应用指南实施过程中遇到的具体问题进行进一步明确和补充的规范性文件，与具体准则及应用指南具有同等效力。财政部已发布多项政府会计准则解释，涵盖固定资产、无形资产、公共基础设施等领域的实务问题。

## 二、"双功能、双基础、双报告"模式

政府会计准则制度体系的核心特征是构建了"双功能、双基础、双报告"的会计核算模式，这是区别于企业会计制度的重要制度创新。

### （一）双功能

政府会计同时具备预算会计和财务会计两项功能：

- **预算会计**：反映和监督预算收支执行情况，服务于预算管理需要。通过预算收入、预算支出和预算结余三个要素进行核算。
- **财务会计**：反映和监督政府会计主体的财务状况、运行情况和现金流量等。通过资产、负债、净资产、收入和费用五个要素进行核算。

### （二）双基础

- **预算会计实行收付实现制**（国务院另有规定的依照其规定）：以现金的实际收付为标志确认本期收入和支出。
- **财务会计实行权责发生制**：以取得收取款项的权利或支付款项的义务为标志确认本期收入和费用。

### （三）双报告

- **决算报告**：基于预算会计核算数据编制，综合反映政府会计主体年度预算收支执行结果。
- **财务报告**：基于财务会计核算数据编制，反映政府会计主体特定日期的财务状况和某一会计期间的运行情况、现金流量等。

## 三、政府会计制度

2017年10月，财政部印发《政府会计制度——行政事业单位会计科目和报表》（财会〔2017〕25号），自2019年1月1日起施行。该制度是政府会计准则体系的重要组成部分，统一了行政单位和事业单位的会计核算制度，规定了会计科目名称、核算内容、账务处理及财务报表格式等，实现了"一套制度"的目标。

该制度的主要内容包括：

1. **会计科目设置**：按照财务会计和预算会计"平行记账"的原则，设置了资产类、负债类、净资产类、收入类和费用类科目（财务会计），以及预算收入类、预算支出类和预算结余类科目（预算会计）。
2. **平行记账规则**：对于纳入部门预算管理的现金收支业务，在进行财务会计核算的同时，应当同时进行预算会计核算；对于其他业务，仅需要进行财务会计核算。
3. **财务报表体系**：包括资产负债表、收入费用表、净资产变动表、现金流量表及附注，同时要求编制预算收入支出表、预算结转结余变动表等预算报表。

## 四、适用范围

政府会计准则制度适用于各级政府部门、各单位（即行政事业单位），包括：

- 各级政府财政部门
- 各级政府组成部门、直属机构
- 事业单位（教育、科技、文化、卫生等领域）
- 其他纳入政府会计主体范围的单位

军队、已纳入企业财务管理体系的单位不执行政府会计准则制度。

## 五、改革意义

政府会计准则制度的建立和实施，标志着我国政府会计改革取得重大突破，具有以下重要意义：

1. **提升财政透明度**：通过权责发生制下的财务报告，全面反映政府的资产、负债和运行成本，增强政府财务信息的透明度。
2. **加强政府资产管理**：有助于摸清政府资产"家底"，防范财政风险，为政府信用评级和政府债券发行提供数据支撑。
3. **完善预算管理制度**：双功能、双基础的核算模式使预算执行信息与财务信息相互补充，有利于建立全面规范透明、标准科学、约束有力的预算制度。
4. **促进国家治理现代化**：为开展政府绩效考核、加强公共财政管理提供信息基础，推进国家治理体系和治理能力现代化。
