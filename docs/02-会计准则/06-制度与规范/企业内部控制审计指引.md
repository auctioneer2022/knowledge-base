# 企业内部控制审计指引

## 通知正文

财会〔2010〕11号

（通知正文同《企业内部控制配套指引》通知）

---

**第一条** 为了规范注册会计师执行企业内部控制审计业务，提高内部控制审计质量，根据《中华人民共和国注册会计师法》和《企业内部控制基本规范》，制定本指引。

**第二条** 本指引所称内部控制审计，是指会计师事务所接受委托，对特定基准日内部控制设计与运行的有效性进行审计。

**第三条** 注册会计师应当根据《企业内部控制基本规范》、应用指引、本指引及中国注册会计师相关执业准则的规定，计划和实施审计工作，对内部控制的有效性发表审计意见。

**第四条** 建立健全内部控制并评价其有效性是企业董事会的责任。注册会计师的责任是在实施审计工作的基础上对内部控制的有效性发表审计意见。

**第五条** 注册会计师应当保持职业怀疑态度，合理保证内部控制不存在重大缺陷。由于内部控制存在固有局限性，注册会计师不能对内部控制的有效性提供绝对保证。

**第六条** 注册会计师应当了解企业及其环境，包括内部控制环境，以识别可能存在的重大错报风险，并据此设计和实施进一步审计程序。

**第七条** 注册会计师应当评估企业层面控制的有效性。企业层面控制包括：

（一）与控制环境相关的控制。

（二）针对管理层凌驾于控制之上的控制。

（三）企业的风险评估过程。

（四）对内部信息传递和期末财务报告流程的控制。

（五）集中化的处理和控制。

（六）监督经营成果的控制。

**第八条** 注册会计师应当了解、测试和评价与重要账户和列报相关的业务流程层面控制的设计和运行有效性。

**第九条** 注册会计师在确定拟测试的控制时，应当考虑控制对其他控制有效性的依赖关系，以及控制的性质、控制执行的频率和控制的重要性。

**第十条** 注册会计师应当测试控制设计的有效性。如果控制设计无效，注册会计师无需测试控制运行的有效性。

**第十一条** 注册会计师应当测试控制运行的有效性，即控制是否一贯地被执行，以及执行控制的人员是否具备必要的授权和能力。

**第十二条** 注册会计师在测试控制运行的有效性时，应当综合运用询问、观察、检查和重新执行等审计程序，获取充分、适当的审计证据。

**第十三条** 注册会计师应当评估内部控制缺陷的严重程度。内部控制缺陷的严重程度取决于：

（一）控制缺陷导致账户余额或列报错报的可能性。

（二）控制缺陷导致一项或多项控制缺陷组合导致重大错报的可能性。

**第十四条** 内部控制缺陷按照其严重程度分为重大缺陷、重要缺陷和一般缺陷。

重大缺陷，是指一个或多个控制缺陷的组合，合理可能导致企业不能及时防止或发现并纠正财务报告的重大错报。

重要缺陷，是指一个或多个控制缺陷的组合，其严重程度低于重大缺陷，但足以引起负责监督企业财务报告的相关人员的关注。

一般缺陷，是指除重大缺陷和重要缺陷之外的其他缺陷。

**第十五条** 注册会计师应当根据获取的审计证据，确定内部控制是否存在重大缺陷。

如果存在下列情形之一，注册会计师应当认为内部控制存在重大缺陷：

（一）注册会计师发现企业董事、监事和高级管理人员的任何级别的舞弊行为。

（二）企业更正了已经公布的财务报表，以更正由于舞弊或错误导致的重大错报。

（三）注册会计师发现当期财务报表存在重大错报，而内部控制在运行过程中未能发现该错报。

（四）审计委员会和内部审计机构对内部控制的监督无效。

**第十六条** 注册会计师应当形成审计意见。审计意见的类型包括：

（一）无保留意见。如果内部控制不存在重大缺陷，注册会计师应当出具无保留意见的审计报告。

（二）否定意见。如果内部控制存在一项或多项重大缺陷，注册会计师应当出具否定意见的审计报告，除非审计范围受到限制。

（三）无法表示意见。如果审计范围受到限制，导致注册会计师无法获取充分、适当的审计证据以支持对内部控制有效性的意见，注册会计师应当出具无法表示意见的审计报告。

**第十七条** 注册会计师应当在审计报告中说明下列内容：

（一）企业的董事会对内部控制有效性的责任。

（二）注册会计师的责任。

（三）内部控制的固有局限性。

（四）内部控制评价所采用的标准。

（五）审计意见。

（六）内部控制重大缺陷的定义和特征。

（七）如果存在重大缺陷，说明重大缺陷的性质及其对内部控制有效性的影响。

**第十八条** 注册会计师应当在审计报告中说明，其审计意见是对特定基准日内部控制的有效性发表的意见。

**第十九条** 如果存在重大缺陷，注册会计师应当在审计报告中说明重大缺陷的性质和对实现控制目标的影响程度，以及重大缺陷在基准日之前已存在的事实。

**第二十条** 注册会计师应当关注基准日至审计报告日之间发生的期后事项。如果期后事项对内部控制有效性产生重大影响，注册会计师应当在审计报告中增加强调事项段予以说明。

**第二十一条** 注册会计师应当就下列事项与企业董事会和管理层进行沟通：

（一）注册会计师识别的重大缺陷和重要缺陷。

（二）审计过程中发现的、已与管理层沟通但尚未解决的其他内部控制缺陷。

（三）其他需要沟通的事项。

**第二十二条** 注册会计师应当编制审计工作底稿，详细记录审计计划、审计程序、审计发现和审计结论。审计工作底稿应当足以使未曾接触该项审计工作的注册会计师了解审计工作的全过程。

**第二十三条** 注册会计师应当在审计工作完成后，及时出具审计报告。审计报告应当由注册会计师签名并盖章，加盖会计师事务所公章。

**第二十四条** 注册会计师应当遵守职业道德守则，保持独立性和专业胜任能力。

**第二十五条** 本指引自2011年1月1日起施行。
