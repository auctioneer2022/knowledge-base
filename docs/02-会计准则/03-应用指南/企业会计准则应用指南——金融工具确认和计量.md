# 企业会计准则应用指南——金融工具确认和计量

> 对应准则：《企业会计准则第22号——金融工具确认和计量》（CAS 22，2017年修订）
> 应用指南版本：2018年

## 一、总体概述

本指南规范了金融工具确认和计量的操作指引，核心变化包括：
1. 金融资产分类由"四分类"改为"三分类"（基于业务模式和合同现金流量特征）；
2. 金融资产减值由"已发生损失法"改为"预期信用损失法"；
3. 嵌入衍生工具不再需要从混合合同中分拆（金融资产的场合）。

## 二、相关定义

### （一）金融资产

金融资产是指企业持有的现金、其他方的权益工具以及符合下列条件之一的资产：
1. 从其他方收取现金或其他金融资产的合同权利；
2. 在潜在有利条件下，与其他方交换金融资产或金融负债的合同权利。

### （二）金融负债

金融负债是指企业符合下列条件之一的负债：
1. 向其他方交付现金或其他金融资产的合同义务；
2. 在潜在不利条件下，与其他方交换金融资产或金融负债的合同义务。

### （三）衍生工具

衍生工具是指同时具有下列特征的金融工具或其他合同：
1. 其价值随特定变量的变动而变动；
2. 不要求初始净投资，或初始净投资很小；
3. 在未来某一日期结算。

## 三、金融资产的分类

### （一）分类标准

金融资产的分类取决于两个标准：
1. **企业管理金融资产的业务模式**；
2. **金融资产的合同现金流量特征**（即是否仅为对本金和以未偿付本金金额为基础的利息的支付，简称"SPPI测试"）。

### （二）三类金融资产

| 分类 | 业务模式 | 合同现金流量特征 | 计量方式 |
|------|----------|-----------------|----------|
| 以摊余成本计量的金融资产（AMC） | 收取合同现金流量 | 通过SPPI测试 | 摊余成本 |
| 以公允价值计量且其变动计入其他综合收益的金融资产（FVOCI） | 收取合同现金流量+出售 | 通过SPPI测试 | 公允价值，变动计入OCI |
| 以公允价值计量且其变动计入当期损益的金融资产（FVTPL） | 其他业务模式 | 未通过SPPI测试或指定 | 公允价值，变动计入损益 |

### （三）指定权

- **不可撤销指定为FVTPL**：如果在初始确认时能够消除或显著减少会计错配，可以将金融资产不可撤销地指定为以公允价值计量且其变动计入当期损益。
- **不可撤销指定为FVOCI（权益工具）**：对于非交易性权益工具投资，可以在初始确认时不可撤销地指定为以公允价值计量且其变动计入其他综合收益。

## 四、会计科目设置和主要账务处理

### （一）资产类科目

#### 1101 交易性金融资产

- **核算内容**：核算企业以公允价值计量且其变动计入当期损益的金融资产。包括分类为FVTPL的金融资产以及直接指定为FVTPL的金融资产。
- **明细设置**：
  - "成本"：记录初始确认时的公允价值；
  - "公允价值变动"：记录后续公允价值变动。
- **主要账务处理**：

  **取得时**：
  ```
  借：交易性金融资产——成本（公允价值）
      投资收益（交易费用）
      应收股利/应收利息（已宣告但尚未发放的股利或利息）
    贷：银行存款/其他货币资金等
  ```

  **资产负债表日公允价值变动**：
  ```
  # 公允价值上升
  借：交易性金融资产——公允价值变动
    贷：公允价值变动损益

  # 公允价值下降
  借：公允价值变动损益
    贷：交易性金融资产——公允价值变动
  ```

  **处置时**：
  ```
  借：银行存款（实际收到金额）
    贷：交易性金融资产——成本
        交易性金融资产——公允价值变动（也可能在借方）
        投资收益（差额）
  ```

#### 1501 债权投资

- **核算内容**：核算企业以摊余成本计量的债权投资（如持有至到期的债券投资等）。
- **明细设置**：
  - "面值"；
  - "利息调整"（溢折价及交易费用的摊销）；
  - "应计利息"。
- **主要账务处理**：

  **取得时**：
  ```
  借：债权投资——面值
      应收利息（已到付息期但尚未领取的利息）
    贷：银行存款（实际支付金额）
        债权投资——利息调整（差额，也可能在借方）
  ```

  **资产负债表日确认利息收入**：
  ```
  借：应收利息（票面利息）
      债权投资——利息调整（差额，也可能在贷方）
    贷：投资收益（按实际利率计算的利息收入）
  ```

  **计提减值准备**：
  ```
  借：信用减值损失
    贷：债权投资减值准备
  ```

#### 1502 债权投资减值准备

- **核算内容**：核算以摊余成本计量的债权投资计提的减值准备。

#### 1503 其他债权投资

- **核算内容**：核算企业以公允价值计量且其变动计入其他综合收益的债权投资。
- **明细设置**：
  - "成本"；
  - "利息调整"；
  - "应计利息"；
  - "公允价值变动"。
- **主要账务处理**：

  **取得时**：
  ```
  借：其他债权投资——成本（面值）
      应收利息
    贷：银行存款
        其他债权投资——利息调整（差额，也可能在借方）
  ```

  **资产负债表日**：
  ```
  # 确认利息收入（同债权投资）
  借：应收利息
      其他债权投资——利息调整（差额）
    贷：投资收益

  # 公允价值变动
  借：其他债权投资——公允价值变动
    贷：其他综合收益——其他债权投资公允价值变动
  # 或相反分录

  # 计提减值准备
  借：信用减值损失
    贷：其他综合收益——信用减值准备
  ```

  **处置时**：
  ```
  借：银行存款
    贷：其他债权投资——成本
        其他债权投资——利息调整（也可能在借方）
        其他债权投资——公允价值变动（也可能在借方）
        投资收益（差额）

  # 同时结转其他综合收益
  借：其他综合收益——其他债权投资公允价值变动
    贷：投资收益
  # 或相反分录
  ```

#### 1504 其他权益工具投资

- **核算内容**：核算企业指定为以公允价值计量且其变动计入其他综合收益的非交易性权益工具投资。
- **明细设置**：
  - "成本"；
  - "公允价值变动"。
- **主要账务处理**：

  **取得时**：
  ```
  借：其他权益工具投资——成本（公允价值+交易费用）
      应收股利
    贷：银行存款
  ```

  **资产负债表日公允价值变动**：
  ```
  借：其他权益工具投资——公允价值变动
    贷：其他综合收益——其他权益工具投资公允价值变动
  # 或相反分录
  ```

  **持有期间宣告发放股利**：
  ```
  借：应收股利
    贷：投资收益
  ```

  **处置时**：
  ```
  借：银行存款
    贷：其他权益工具投资——成本
        其他权益工具投资——公允价值变动（也可能在借方）
        盈余公积/利润分配——未分配利润（差额）

  # 同时结转其他综合收益（不转入损益，转入留存收益）
  借：其他综合收益——其他权益工具投资公允价值变动
    贷：盈余公积
        利润分配——未分配利润
  # 或相反分录
  ```

  **特别注意**：处置其他权益工具投资时，累计计入其他综合收益的金额不得转入当期损益，而是转入留存收益。

### （二）负债类科目

#### 2201 交易性金融负债

- **核算内容**：核算企业以公允价值计量且其变动计入当期损益的金融负债。
- **明细设置**：
  - "本金"；
  - "公允价值变动"。

### （三）损益类科目

#### 6111 投资收益

- **核算内容**：核算企业确认的投资收益或投资损失。
- **明细设置**：按投资项目设置。

#### 6114（新增）信用减值损失

- **核算内容**：核算企业按照预期信用损失法计提的各项金融工具减值准备所形成的损失。
- **包括**：应收账款减值损失、其他应收款减值损失、债权投资减值损失、贷款减值损失、合同资产减值损失等。

#### 6061 公允价值变动损益

- **核算内容**：核算企业交易性金融资产、交易性金融负债等公允价值变动形成的应计入当期损益的利得或损失。

## 五、金融资产的减值——预期信用损失法

### （一）适用范围

以下金融工具需按预期信用损失法计提减值准备：
1. 以摊余成本计量的金融资产（如应收账款、债权投资、贷款等）；
2. 以公允价值计量且其变动计入其他综合收益的债权投资（其他债权投资）；
3. 租赁应收款；
4. 合同资产；
5. 贷款承诺和财务担保合同。

### （二）三阶段模型

| 阶段 | 信用风险变化 | 损失准备计量基础 | 利息收入计算基础 |
|------|-------------|-----------------|-----------------|
| 第一阶段 | 自初始确认后信用风险未显著增加 | 未来12个月的预期信用损失 | 账面总额（摊余成本） |
| 第二阶段 | 自初始确认后信用风险显著增加 | 整个存续期的预期信用损失 | 账面总额（摊余成本） |
| 第三阶段 | 已发生信用减值 | 整个存续期的预期信用损失 | 摊余成本（扣除减值准备后的净额） |

### （三）应收账款的简化处理

对于不含重大融资成分的应收账款和合同资产，企业应当始终按照相当于整个存续期内预期信用损失的金额计量损失准备（即始终处于第二阶段），无需评估信用风险是否显著增加。

### （四）预期信用损失的计量

预期信用损失 = 违约概率 × 违约损失率 × 违约风险敞口

企业应考虑：
1. 无偏概率加权金额（通过评估一系列可能的结果确定）；
2. 货币时间价值；
3. 无须付出不必要的额外成本或努力即可获得的有关过去事项、当前状况及未来经济状况预测的合理且有依据的信息。

## 六、金融资产的重分类

当且仅当企业改变管理金融资产的业务模式时，才应对受影响的金融资产进行重分类。重分类日为企业改变业务模式后的首个报告期间的第一天。

### 重分类的会计处理

| 从 → 到 | 会计处理 |
|---------|---------|
| AMC → FVTPL | 按公允价值重新计量，差额计入当期损益 |
| AMC → FVOCI | 按公允价值重新计量，差额计入其他综合收益 |
| FVTPL → AMC | 以重分类日公允价值为新的摊余成本 |
| FVTPL → FVOCI | 以重分类日公允价值为新的账面余额 |
| FVOCI → AMC | 将累计计入OCI的利得或损失调整资产账面价值 |
| FVOCI → FVTPL | 将累计计入OCI的利得或损失转入当期损益 |

## 七、金融负债的分类和计量

### （一）分类

1. **以摊余成本计量的金融负债**（默认分类）；
2. **以公允价值计量且其变动计入当期损益的金融负债**：
   - 分类为FVTPL的金融负债（如交易性金融负债）；
   - 指定为FVTPL的金融负债（消除会计错配或包含嵌入衍生工具）。

### （二）指定为FVTPL的金融负债——自身信用风险变动

对于指定为以公允价值计量且其变动计入当期损益的金融负债，由企业自身信用风险变动引起的公允价值变动金额应计入其他综合收益，其余变动计入当期损益。

## 八、报表列示

### （一）资产负债表

- 交易性金融资产/负债：单独列示于流动资产/流动负债。
- 债权投资：列示于非流动资产（扣除减值准备后的净额）。
- 其他债权投资：按流动性分别列示于流动资产或非流动资产。
- 其他权益工具投资：按流动性分别列示于流动资产或非流动资产。

### （二）利润表

- 投资收益：反映处置损益、股利收入、利息收入等。
- 公允价值变动损益：反映交易性金融资产/负债的公允价值变动。
- 信用减值损失：反映按预期信用损失法计提的减值损失。

### （三）附注披露

企业应披露：
1. 金融资产和金融负债的分类基础、计量方法和利得损失；
2. 套期会计的相关信息；
3. 信用风险管理及预期信用损失的信息；
4. 公允价值层级信息。
