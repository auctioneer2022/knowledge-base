# 企业会计准则应用指南——套期会计

> 对应准则：《企业会计准则第24号——套期会计》（CAS 24）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了套期关系的指定、套期工具的选择、套期有效性的评估以及套期会计的处理方法。套期会计的目标是在财务报表中反映企业运用金融工具管理风险活动的实质。

**套期关系的三要素**：
1. **套期工具**：企业指定的用于套期的金融工具（如衍生工具）。
2. **被套期项目**：企业指定的面临风险敞口的项目（如已确认资产/负债、未确认确定承诺、很可能发生的预期交易、境外经营净投资等）。
3. **套期关系**：企业指定的套期工具与被套期项目之间的对冲关系。

**套期会计的适用条件**：
1. 套期关系仅由符合条件的套期工具和被套期项目组成；
2. 在套期开始时，企业正式指定了套期工具和被套期项目，并准备了套期关系的书面文件；
3. 该套期关系符合套期有效性要求。

## 二、会计科目设置和主要账务处理

### （一）被套期项目

- **核算内容**：核算企业开展套期保值业务指定的被套期项目。被套期项目可以在相关资产或负债科目下设置明细科目进行核算。
- **明细设置**：如"库存商品——被套期商品""短期借款——被套期借款"等。

### （二）套期工具

- **核算内容**：核算企业开展套期保值业务指定的套期工具。套期工具为衍生工具的，通过"衍生工具"科目核算，并设置"套期工具"明细科目。
- **主要账务处理**：
  ```
  # 指定套期工具
  借：衍生工具——套期工具
    贷：银行存款等（或相反分录）

  # 套期工具公允价值变动
  借：衍生工具——套期工具
    贷：套期损益（公允价值套期）
        其他综合收益——套期储备（现金流量套期）
  ```

### （三）公允价值套期的会计处理

公允价值套期是指对已确认资产或负债、尚未确认的确定承诺，或上述项目组成部分的公允价值变动风险敞口进行的套期。

**主要账务处理**：
```
# 套期工具产生利得
借：衍生工具——套期工具
  贷：套期损益

# 被套期项目因被套期风险形成的损失
借：套期损益
  贷：被套期项目（或相反分录）

# 被套期项目为非金融资产/负债的，调整其账面价值
借：套期损益
  贷：库存商品——被套期商品（或相反分录）

# 套期关系终止后，被套期项目不再调整
```

**特殊处理**：
- 被套期项目为以摊余成本计量的金融工具的，对被套期项目账面价值所作的调整应当按照实际利率法在调整期间内摊销，计入当期损益。
- 被套期项目为以公允价值计量且其变动计入其他综合收益的金融资产的，对被套期项目账面价值所作的调整计入当期损益。

### （四）现金流量套期的会计处理

现金流量套期是指对现金流量变动风险敞口进行的套期。

**主要账务处理**：
```
# 套期工具产生利得（有效部分）
借：衍生工具——套期工具
  贷：其他综合收益——套期储备

# 套期工具产生利得（无效部分）
借：衍生工具——套期工具
  贷：套期损益

# 被套期项目影响损益时，将套期储备转出
借：其他综合收益——套期储备
  贷：主营业务收入/主营业务成本等（或相反分录）

# 被套期项目为预期交易，后续形成非金融资产/负债
# 将套期储备转入资产/负债初始成本（基础调整）
借：其他综合收益——套期储备
  贷：库存商品/固定资产等（或相反分录）
```

### （五）境外经营净投资套期的会计处理

境外经营净投资套期是指对境外经营净投资外汇风险敞口进行的套期。其会计处理与现金流量套期类似。

```
# 套期工具产生利得（有效部分）
借：衍生工具——套期工具
  贷：其他综合收益——套期储备

# 处置境外经营时，将套期储备转入当期损益
借：其他综合收益——套期储备
  贷：投资收益（或相反分录）
```

## 三、套期有效性评估与套期关系再平衡

### （一）套期有效性评估

套期有效性是指套期工具的公允价值或现金流量变动能够抵销被套期项目公允价值或现金流量变动的程度。

**评估方法**：
1. **主要条款比较法**：比较套期工具和被套期项目的主要条款，如果主要条款匹配，则套期有效。
2. **比率分析法**：计算套期工具与被套期项目的价值变动比率，在80%至125%的范围内认为套期有效。

### （二）套期关系再平衡

当套期比率不再反映套期工具与被套期项目之间实际存在的经济关系时，企业应当调整套期关系，使其重新满足套期有效性要求。

### （三）套期关系的终止

企业应当在下列情况下终止套期会计：
1. 套期关系不再满足套期有效性要求；
2. 套期工具到期、被出售、被终止或被执行；
3. 被套期项目不再存在；
4. 企业撤销了对套期关系的指定。

## 四、财务报表列示要求

### （一）资产负债表

- **套期工具**：根据"衍生工具——套期工具"科目期末余额填列。
- **被套期项目**：在相关资产或负债项目中列示，但应单独披露。

### （二）附注披露

企业应当在附注中披露：
1. 套期关系的描述，包括套期工具和被套期项目的性质；
2. 套期策略的描述；
3. 套期工具和被套期项目的公允价值；
4. 套期有效性评估的结果；
5. 套期会计对财务报表的影响；
6. 现金流量套期储备的变动情况；
7. 境外经营净投资套期的相关信息。
