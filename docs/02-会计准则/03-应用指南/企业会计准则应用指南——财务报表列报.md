# 企业会计准则应用指南——财务报表列报

> 对应准则：《企业会计准则第30号——财务报表列报》（CAS 30，2014年修订）
> 应用指南版本：2014年

## 一、总体概述

本指南规范了财务报表的列报要求。一套完整的财务报表至少应当包括下列组成部分（"四表一注"）：
1. 资产负债表；
2. 利润表；
3. 现金流量表；
4. 所有者权益（或股东权益）变动表；
5. 附注。

现金流量表的编制还应遵循《企业会计准则第31号——现金流量表》。

## 二、财务报表列报的基本要求

### （一）编制基础

1. **持续经营**：企业应当以持续经营为基础编制财务报表。如果管理层判断企业持续经营能力存在重大不确定性，应当充分披露。如果企业不处于持续经营状态，则不得以持续经营为基础编制。
2. **权责发生制**：除现金流量表按照收付实现制编制外，企业应当按照权责发生制编制财务报表。
3. **列报的一致性**：财务报表项目的列报应当在各个会计期间保持一致，不得随意变更。

### （二）重要性

性质或功能不同的项目，应当在财务报表中单独列报，但不具有重要性的项目除外。重要性应当根据企业所处的具体环境，从项目的性质和金额大小两方面予以判断。

### （三）抵销

财务报表中的资产项目和负债项目的金额、收入项目和费用项目的金额不得相互抵销，但其他会计准则另有规定的除外。

### （四）比较信息

企业在列报当期财务报表时，至少应当提供所有列报项目上一个可比会计期间的比较数据。

## 五、所有者权益变动表

### （一）格式说明

所有者权益变动表应当反映构成所有者权益的各组成部分当期的增减变动情况。

矩阵结构：
- **纵列**：实收资本（股本）、其他权益工具、资本公积、库存股、其他综合收益、专项储备、盈余公积、未分配利润、所有者权益合计、少数股东权益。
- **横行**：综合收益总额、所有者投入和减少资本、利润分配、所有者权益内部结转等。

### （二）主要变动事项

1. **综合收益总额**：当期净利润和其他综合收益的税后净额。
2. **所有者投入和减少资本**：
   - 所有者投入的普通股；
   - 其他权益工具持有者投入资本；
   - 股份支付计入所有者权益的金额；
   - 其他。
3. **利润分配**：
   - 提取盈余公积；
   - 对所有者（或股东）的分配。
4. **所有者权益内部结转**：
   - 资本公积转增资本（或股本）；
   - 盈余公积转增资本（或股本）；
   - 盈余公积弥补亏损；
   - 其他综合收益结转留存收益。

## 六、附注

### （一）附注的披露顺序和内容

附注一般应当按照下列顺序披露：

1. **企业的基本情况**：
   - 企业注册地、组织形式和总部地址；
   - 企业的业务性质和主要经营活动；
   - 母公司以及集团最终母公司的名称；
   - 财务报告的批准报出者和批准报出日。

2. **财务报表的编制基础**：持续经营假设或非持续经营假设。

3. **遵循企业会计准则的声明**。

4. **重要会计政策和会计估计**：
   - 会计期间；
   - 营业周期；
   - 记账本位币；
   - 计量属性；
   - 各项重要会计政策（如收入确认、金融工具分类、固定资产折旧方法、无形资产摊销方法等）；
   - 重要会计估计和判断。

5. **会计政策和会计估计变更以及差错更正的说明**。

6. **报表重要项目的说明**：对资产负债表、利润表、现金流量表、所有者权益变动表中列示的重要项目作进一步说明。

7. **或有事项和承诺事项**。

8. **资产负债表日后事项**。

9. **关联方关系及其交易**。

10. **其他需要说明的重要事项**。
