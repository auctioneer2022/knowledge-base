# 企业会计准则应用指南——再保险合同

> 对应准则：《企业会计准则第26号——再保险合同》（CAS 26）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了再保险合同的会计处理，包括分出业务和分入业务的确认、计量和列报要求。

**再保险合同的定义**：再保险合同是指再保险分出人（原保险人）将其承担的保险风险，部分转移给再保险接受人（再保险人）的协议。

**再保险合同的分类**：
1. **分出业务**：原保险人将部分保险风险转移给再保险人的业务。
2. **分入业务**：再保险人接受原保险人转移的保险风险的业务。

**再保险合同的类型**：
1. **比例再保险**：按照保费或保额的一定比例分配风险和保费。
2. **非比例再保险**：按照赔款金额的一定额度分配风险。

## 二、会计科目设置和主要账务处理

### （一）分出业务

**1. 2621 分出保费**
- **核算内容**：核算再保险分出人向再保险接受人分出的保费。
- **明细设置**：按再保险合同或再保险接受人设置明细科目。
- **主要账务处理**：
  ```
  # 确认分出保费
  借：分出保费
    贷：应付分保账款

  # 支付分出保费
  借：应付分保账款
    贷：银行存款
  ```

**2. 2622 摊回赔付支出**
- **核算内容**：核算再保险分出人从再保险接受人摊回的赔付支出。
- **明细设置**：按再保险合同或再保险接受人设置明细科目。
- **主要账务处理**：
  ```
  # 确认摊回赔付支出
  借：应收分保账款
    贷：摊回赔付支出

  # 收到摊回款项
  借：银行存款
    贷：应收分保账款
  ```

**3. 摊回分保费用**
- **核算内容**：核算再保险分出人从再保险接受人摊回的分保费用。
- **主要账务处理**：
  ```
  # 确认摊回分保费用
  借：应收分保账款
    贷：摊回分保费用
  ```

**4. 摊回未到期责任准备金/未决赔款准备金**
- **核算内容**：核算再保险分出人从再保险接受人摊回的准备金。
- **主要账务处理**：
  ```
  # 确认摊回准备金
  借：应收分保账款
    贷：摊回未到期责任准备金/摊回未决赔款准备金

  # 冲回摊回准备金
  借：摊回未到期责任准备金/摊回未决赔款准备金
    贷：应收分保账款
  ```

**5. 应收分保账款**
- **核算内容**：核算再保险分出人应收再保险接受人的款项。
- **明细设置**：按再保险接受人设置明细科目。

### （二）分入业务

**1. 分保费收入**
- **核算内容**：核算再保险接受人接受的分保费收入。
- **明细设置**：按再保险合同或再保险分出人设置明细科目。
- **主要账务处理**：
  ```
  # 确认分保费收入
  借：应收分保账款
    贷：分保费收入
  ```

**2. 分保赔付支出**
- **核算内容**：核算再保险接受人承担的分保赔付支出。
- **主要账务处理**：
  ```
  # 确认分保赔付支出
  借：分保赔付支出
    贷：应付分保账款
  ```

**3. 提取分保未到期责任准备金/未决赔款准备金**
- **核算内容**：核算再保险接受人为履行分保合同义务而提取的准备金。
- **主要账务处理**：
  ```
  # 提取分保准备金
  借：提取分保未到期责任准备金/提取分保未决赔款准备金
    贷：分保未到期责任准备金/分保未决赔款准备金
  ```

### （三）2631 保户质押贷款

- **核算内容**：核算保险人向投保人提供的质押贷款，以保单现金价值为质押。
- **主要账务处理**：
  ```
  # 发放质押贷款
  借：保户质押贷款
    贷：银行存款/现金

  # 收回质押贷款
  借：银行存款/现金
    贷：保户质押贷款
        利息收入

  # 质押贷款逾期，以保单现金价值抵偿
  借：寿险责任准备金/长期健康险责任准备金
    贷：保户质押贷款
        营业外收入（差额）
  ```

## 三、再保险合同资产与负债的列报

### （一）资产负债表

**分出人**：
- **应收分保账款**：根据"应收分保账款"科目期末余额填列，列示于资产项目。
- **应付分保账款**：根据"应付分保账款"科目期末余额填列，列示于负债项目。
- **分出保费**：作为保费收入的抵减项目，在利润表中单独列示。
- **摊回赔付支出**：作为赔付支出的抵减项目，在利润表中单独列示。

**分入人**：
- **应收分保账款**：根据"应收分保账款"科目期末余额填列。
- **应付分保账款**：根据"应付分保账款"科目期末余额填列。
- **分保费收入**：在利润表中单独列示。
- **分保赔付支出**：在利润表中单独列示。

### （二）利润表

**分出人利润表项目**：
1. 保费收入（净额）
2. 减：分出保费
3. 摊回赔付支出
4. 摊回分保费用
5. 提取未到期责任准备金（净额）
6. 提取未决赔款准备金（净额）

**分入人利润表项目**：
1. 分保费收入
2. 分保赔付支出
3. 提取分保未到期责任准备金
4. 提取分保未决赔款准备金

### （三）附注披露

企业应当在附注中披露：
1. 再保险安排的性质和范围；
2. 分出保费和分保费收入的金额；
3. 应收分保账款和应付分保账款的账龄分析；
4. 再保险合同的信用风险集中度；
5. 再保险合同的重大假设和不确定性；
6. 再保险合同对财务报表的影响。
