# 企业会计准则应用指南——长期股权投资

> 对应准则：《企业会计准则第2号——长期股权投资》（CAS 2，2014年修订）
> 应用指南版本：2014年，2024年更新

## 一、总体概述

本指南规范了长期股权投资的确认、计量和列报要求。长期股权投资包括：
1. 投资方能够对被投资单位实施**控制**的权益性投资（对子公司投资）；
2. 投资方与其他参与方一起对被投资单位实施**共同控制**的权益性投资（对合营企业投资）；
3. 投资方对被投资单位具有**重大影响**的权益性投资（对联营企业投资）。

## 二、会计科目设置和主要账务处理

### （一）1511 长期股权投资

- **核算内容**：核算企业持有的长期股权投资。
- **明细设置**：
  - 按被投资单位设置明细科目；
  - 权益法下还需设置以下明细科目：
    - "投资成本"
    - "损益调整"
    - "其他综合收益"
    - "其他权益变动"

### （二）1512 长期股权投资减值准备

- **核算内容**：核算企业长期股权投资发生减值时计提的减值准备。
- **重要规则**：长期股权投资减值损失一经确认，在以后会计期间不得转回。

### （三）1531 应收股利

- **核算内容**：核算企业应收取的现金股利和应收取其他单位分配的利润。

### （四）损益类科目

- **投资收益**：核算企业确认的投资收益或投资损失。
- **其他综合收益**：核算权益法下应享有或应分担的被投资单位其他综合收益的份额。

## 三、初始计量

### （一）企业合并形成的长期股权投资

#### 1. 同一控制下的企业合并

- 合并方以支付现金、转让非现金资产或承担债务方式作为合并对价的：
  ```
  借：长期股权投资（被合并方所有者权益在最终控制方合并报表中的账面价值份额）
    贷：银行存款/相关资产/负债（账面价值）
        资本公积——资本溢价/股本溢价（差额，也可能在借方）
  ```
  资本公积不足冲减的，依次冲减盈余公积和未分配利润。

- 合并方以发行权益性证券作为合并对价的：
  ```
  借：长期股权投资（被合并方所有者权益在最终控制方合并报表中的账面价值份额）
    贷：股本（面值）
        资本公积——股本溢价（差额）
  ```

#### 2. 非同一控制下的企业合并

- 购买日长期股权投资成本 = 合并成本（支付对价的公允价值）
  ```
  借：长期股权投资（合并成本）
    贷：银行存款/相关资产（公允价值）
        主营业务收入/其他业务收入（非现金资产公允价值）
        应交税费——应交增值税（销项税额）

  # 同时结转非现金资产成本
  借：主营业务成本/其他业务成本
    贷：库存商品/原材料等
  ```

- 合并成本与取得的被购买方可辨认净资产公允价值份额之间的差额：
  - 合并成本 > 份额 → 确认为**商誉**（在合并报表中体现）；
  - 合并成本 < 份额 → 差额计入当期**营业外收入**（在合并报表中体现）。

- 为企业合并发生的审计、法律服务、评估咨询等中介费用：计入当期**管理费用**。
- 发行权益性证券或债务性证券的交易费用：计入权益性证券或债务性证券的初始确认金额。

### （二）非企业合并形成的长期股权投资

- 以支付现金取得的：按实际支付的购买价款（含直接相关费用、税金等）作为初始投资成本。
  ```
  借：长期股权投资
    贷：银行存款
  ```

- 以发行权益性证券取得的：按权益性证券的公允价值作为初始投资成本。

- 通过非货币性资产交换、债务重组等方式取得的：分别按照相关准则确定。

## 四、后续计量

### （一）成本法

**适用范围**：投资方能够对被投资单位实施控制的长期股权投资（对子公司投资）。

**核算原则**：
- 长期股权投资按初始投资成本计量；
- 被投资单位宣告分派的现金股利或利润，确认为当期投资收益：
  ```
  借：应收股利
    贷：投资收益
  ```
- 追加或收回投资时调整长期股权投资的成本。

### （二）权益法

**适用范围**：对合营企业投资和对联营企业投资。

#### 1. 初始投资成本的调整

- 初始投资成本 > 投资时应享有被投资单位可辨认净资产公允价值份额：不调整长期股权投资初始投资成本（差额实质为商誉）。
- 初始投资成本 < 投资时应享有被投资单位可辨认净资产公允价值份额：差额计入当期损益，同时调整长期股权投资成本：
  ```
  借：长期股权投资——投资成本
    贷：营业外收入
  ```

#### 2. 投资损益的确认

- 被投资单位实现净利润时：
  ```
  借：长期股权投资——损益调整
    贷：投资收益
  ```

- 被投资单位发生净亏损时：
  ```
  借：投资收益
    贷：长期股权投资——损益调整
  ```
  以长期股权投资账面价值减记至零为限（但投资方负有额外损失义务的除外）。

**确认投资损益时的调整**：
1. 以取得投资时被投资单位可辨认资产、负债的公允价值为基础，对被投资单位的净利润进行调整后确认；
2. 抵销投资方与被投资单位之间内部交易未实现损益（按持股比例）。

#### 3. 被投资单位其他综合收益变动

```
# 被投资单位其他综合收益增加
借：长期股权投资——其他综合收益
  贷：其他综合收益

# 被投资单位其他综合收益减少
借：其他综合收益
  贷：长期股权投资——其他综合收益
```

#### 4. 被投资单位除净损益、其他综合收益和利润分配以外的所有者权益变动

```
# 被投资单位其他权益变动
借：长期股权投资——其他权益变动
  贷：资本公积——其他资本公积

# 或相反分录
```

#### 5. 被投资单位宣告分派现金股利

```
借：应收股利
  贷：长期股权投资——损益调整
```

### （三）超额亏损的处理顺序

投资方确认被投资单位发生的净亏损，应当按以下顺序处理：
1. 冲减长期股权投资的账面价值；
2. 长期股权投资的账面价值不足以冲减的，以其他实质上构成对被投资单位净投资的长期权益（如长期应收款）的账面价值为限继续确认投资损失；
3. 经过上述处理后，按投资合同或协议约定投资方仍承担额外义务的，按预计承担的义务确认预计负债。

## 五、核算方法的转换

### （一）成本法转权益法

因减少投资等原因导致不再具有控制但仍具有共同控制或重大影响的：
- 剩余股权按公允价值重新计量；
- 公允价值与账面价值的差额计入当期损益；
- 然后按权益法进行后续核算（视同自取得投资时即采用权益法核算，进行追溯调整）。

### （二）权益法转成本法

因追加投资等原因能够对被投资单位实施控制的：
- 原持有的股权投资按账面价值加上新增投资成本之和作为成本法下的初始投资成本；
- 原权益法下确认的其他综合收益暂不处理，待处置时再按规定处理。

### （三）成本法或权益法转为金融资产

因减少投资等原因导致不再具有控制、共同控制或重大影响，且无活跃市场报价的：
- 剩余股权按公允价值重新计量；
- 公允价值与账面价值的差额计入当期损益。
- 后续按金融工具准则核算。

### （四）公允价值计量转为权益法

因追加投资等原因能够对被投资单位施加重大影响或实施共同控制的：
- 原持有的股权投资公允价值加上新增投资成本之和作为权益法下的初始投资成本；
- 原计入其他综合收益的累计公允价值变动转入改按权益法核算的当期损益。

## 六、长期股权投资的处置

### （一）全部处置

```
借：银行存款（实际取得价款）
    长期股权投资减值准备
  贷：长期股权投资（账面余额）
      投资收益（差额，也可能在借方）
```

同时，结转原计入其他综合收益和资本公积的金额：
```
# 权益法下原计入其他综合收益的部分
借：其他综合收益
  贷：投资收益（或相反分录）

# 权益法下原计入资本公积的部分
借：资本公积——其他资本公积
  贷：投资收益（或相反分录）
```

### （二）部分处置

部分处置后仍采用权益法的：
- 按处置比例结转长期股权投资账面价值；
- 原计入其他综合收益和资本公积的部分按处置比例结转。

部分处置后改按金融资产核算的：
- 剩余股权按公允价值重新计量；
- 差额计入当期损益。

## 七、重大影响的判断

重大影响是指投资方对被投资单位的财务和经营政策有参与决策的权力，但并不能够控制或者与其他方一起共同控制这些政策的制定。

**判断重大影响的迹象**（包括但不限于）：
1. 在被投资单位的董事会或类似权力机构中派有代表；
2. 参与被投资单位的财务和经营政策制定过程；
3. 与被投资单位之间发生重要交易；
4. 向被投资单位派出管理人员；
5. 向被投资单位提供关键技术资料。

**持股比例参考**：通常当投资方直接或通过子公司间接持有被投资单位20%或以上但低于50%的表决权时，认为具有重大影响（但有确凿证据表明不具有重大影响的除外）。

## 八、报表列示

### （一）资产负债表

- **长期股权投资**：根据"长期股权投资"科目期末余额减去"长期股权投资减值准备"科目期末余额后的净额填列。

### （二）利润表

- **投资收益**：包括成本法下确认的股利收入、权益法下确认的投资损益、处置长期股权投资产生的损益等。

### （三）附注披露

企业应当在附注中披露：
1. 子公司、合营企业和联营企业的清单；
2. 重要合营企业和联营企业的主要财务信息；
3. 长期股权投资的减值情况；
4. 投资方与合营企业或联营企业之间发生的内部交易；
5. 对合营企业投资存在的重大限制。
