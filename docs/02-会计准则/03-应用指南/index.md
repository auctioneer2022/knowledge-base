# 应用指南

收录企业会计准则应用指南，为各具体准则提供操作层面的指导。

应用指南主要包含两大部分：**会计科目和主要账务处理**（为各准则涉及的科目设置及分录提供指引）和**财务报表格式**（资产负债表、利润表、现金流量表及所有者权益变动表的标准格式）。

## 说明

1. 应用指南与各项具体准则配套使用，为准则的贯彻实施提供操作性指引。
2. 应用指南的核心内容通常包括：
   - **会计科目和主要账务处理**：明确各准则涉及的会计科目设置、核算内容及主要会计分录。
   - **财务报表格式**：提供资产负债表、利润表、现金流量表和所有者权益变动表的标准格式及编制说明。
3. CAS-Guide-030（建造合同）原对应《企业会计准则第15号——建造合同》，该准则已于2017年修订后并入《企业会计准则第14号——收入》（CAS 14），不再单独保留。
4. 本知识库收录的应用指南以财政部发布的正式文本为准。
