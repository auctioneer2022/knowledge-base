# 企业会计准则应用指南——企业年金基金

> 对应准则：《企业会计准则第10号——企业年金基金》（CAS 10）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了企业年金基金的会计处理和财务报表列报要求。企业年金基金是指根据依法制定的企业年金计划筹集的资金及其投资运营收益形成的补充养老保险基金。

**重要特点**：企业年金基金是独立的报告主体，不属于企业资产，也不属于受托人、账户管理人、托管人或投资管理人的资产。企业年金基金会计以收付实现制为基础，采用权责发生制进行后续计量。

**参与主体**：
1. **委托人**：建立企业年金计划的企业及其职工。
2. **受托人**：受托管理企业年金基金的企业年金理事会或法人受托机构。
3. **账户管理人**：管理企业年金基金账户的专业机构。
4. **托管人**：保管企业年金基金财产的商业银行。
5. **投资管理人**：管理企业年金基金投资的专业机构。

## 二、会计科目设置和主要账务处理

### （一）企业年金基金资产类科目

**1. 现金及现金等价物**
- **核算内容**：核算企业年金基金持有的现金及现金等价物。
- **主要账务处理**：
  ```
  # 收到缴费
  借：现金及现金等价物
    贷：企业年金缴费收入

  # 支付管理费用
  借：管理费用——受托人管理费/账户管理费等
    贷：现金及现金等价物
  ```

**2. 投资类科目**
- **核算内容**：核算企业年金基金的投资资产，包括股票、基金、债券等。
- **明细设置**：按投资品种设置明细科目。
- **主要账务处理**：
  ```
  # 购入投资产品
  借：交易性金融资产——股票/基金/债券等
    贷：现金及现金等价物

  # 公允价值变动
  借：交易性金融资产——公允价值变动
    贷：公允价值变动损益（或相反分录）

  # 出售投资产品
  借：现金及现金等价物
    贷：交易性金融资产——成本
        交易性金融资产——公允价值变动（或借方）
        投资收益（差额，或借方）
  ```

**3. 应收类科目**
- **核算内容**：核算企业年金基金的应收款项，包括应收利息、应收股利、应收申购款等。
- **主要账务处理**：
  ```
  # 确认应收利息
  借：应收利息
    贷：利息收入

  # 收到利息
  借：现金及现金等价物
    贷：应收利息
  ```

### （二）企业年金基金负债类科目

- **核算内容**：核算企业年金基金的负债，包括应付受益人待遇、应付受托人管理费、应付托管费、应付投资管理人管理费等。
- **主要账务处理**：
  ```
  # 计提管理费用
  借：管理费用——受托人管理费/托管费/投资管理费
    贷：应付受托人管理费/应付托管费/应付投资管理费

  # 支付管理费用
  借：应付受托人管理费/应付托管费/应付投资管理费
    贷：现金及现金等价物

  # 确认应付受益人待遇
  借：受益人待遇支出
    贷：应付受益人待遇

  # 支付受益人待遇
  借：应付受益人待遇
    贷：现金及现金等价物
  ```

### （三）企业年金基金净资产类科目

**1. 企业年金缴费收入**
- **核算内容**：核算企业年金基金收到的企业和职工缴费。
- **主要账务处理**：
  ```
  # 收到企业缴费
  借：现金及现金等价物
    贷：企业年金缴费收入——企业缴费

  # 收到职工缴费
  借：现金及现金等价物
    贷：企业年金缴费收入——职工缴费
  ```

**2. 投资收益类科目**
- **核算内容**：核算企业年金基金投资运营产生的收益，包括利息收入、投资收益、公允价值变动损益等。

**3. 受益人待遇支出**
- **核算内容**：核算企业年金基金向受益人支付的待遇。
- **主要账务处理**：
  ```
  # 确认待遇支付
  借：受益人待遇支出
    贷：现金及现金等价物
  ```

**4. 管理费用**
- **核算内容**：核算企业年金基金运营过程中发生的各项管理费用。

## 三、财务报表列示要求

### （一）企业年金基金财务报表

企业年金基金应当编报下列财务报表：
1. **资产负债表**：反映企业年金基金在某一特定日期的财务状况。
2. **净资产变动表**：反映企业年金基金在一定期间内净资产的变动情况。

### （二）资产负债表格式

企业年金基金资产负债表应当包括下列项目：
- **资产**：现金及现金等价物、交易性金融资产、应收款项、其他资产。
- **负债**：应付受益人待遇、应付管理费用、其他负债。
- **净资产**：企业年金基金净资产。

### （三）净资产变动表格式

净资产变动表应当反映下列项目：
1. 期初净资产；
2. 本期企业缴费；
3. 本期职工缴费；
4. 本期投资收益；
5. 本期受益人待遇支出；
6. 本期管理费用；
7. 期末净资产。

### （四）附注披露

企业年金基金应当在附注中披露：
1. 企业年金基金的基本情况；
2. 企业年金基金的投资组合及收益情况；
3. 企业年金基金的管理费用情况；
4. 企业年金基金的受益人待遇支付情况；
5. 企业年金基金的重大风险及不确定性。
