# 企业会计准则应用指南——建造合同

> **状态说明**：本指南原对应《企业会计准则第15号——建造合同》（CAS 15），该准则已于2017年修订后并入《企业会计准则第14号——收入》（CAS 14），不再单独保留。相关内容请参见[企业会计准则应用指南——收入](企业会计准则应用指南——收入.md)。

## 说明

原建造合同应用指南的核心内容（包括完工百分比法、合同收入和合同成本的确认等）已整合入收入准则应用指南。在实务操作中，涉及建造合同的会计处理应按照修订后的CAS 14执行。
