# 企业会计准则应用指南——或有事项

> 对应准则：《企业会计准则第13号——或有事项》（CAS 13）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了或有事项的确认、计量和相关信息的披露要求。或有事项是指过去的交易或者事项形成的，其结果须由某些未来事项的发生或不发生才能决定的不确定事项。

或有事项的特征：
1. 由过去的交易或事项形成；
2. 结果具有不确定性；
3. 结果须由未来事项决定。

常见的或有事项包括：未决诉讼或仲裁、产品质量保证、环境污染整治、承诺、亏损合同、重组义务等。

## 二、会计科目设置和主要账务处理

### （一）2211 预计负债

- **核算内容**：核算企业根据或有事项准则确认的各项预计负债，包括对外提供担保、未决诉讼、产品质量保证、重组义务、亏损合同等产生的预计负债。
- **明细设置**：按形成预计负债的交易或事项设置明细科目，如"预计负债——未决诉讼""预计负债——产品质量保证""预计负债——亏损合同"等。
- **主要账务处理**：

  **未决诉讼**：
  ```
  # 确认预计负债
  借：营业外支出——诉讼赔偿
    贷：预计负债——未决诉讼

  # 实际支付赔偿款
  借：预计负债——未决诉讼
    贷：银行存款
  ```

  **产品质量保证**：
  ```
  # 确认预计负债
  借：销售费用——产品质量保证
    贷：预计负债——产品质量保证

  # 实际发生保修费用
  借：预计负债——产品质量保证
    贷：原材料/应付职工薪酬/银行存款等

  # 保修期满，冲销余额
  借：预计负债——产品质量保证
    贷：销售费用——产品质量保证
  ```

  **亏损合同**：
  ```
  # 确认预计负债
  借：营业外支出——亏损合同损失
    贷：预计负债——亏损合同

  # 实际履行合同发生损失
  借：预计负债——亏损合同
    贷：库存商品/银行存款等
  ```

  **重组义务**：
  ```
  # 确认预计负债
  借：管理费用/营业外支出
    贷：预计负债——重组义务

  # 实际支付重组费用
  借：预计负债——重组义务
    贷：银行存款/应付职工薪酬等
  ```

### （二）预计负债的确认条件

与或有事项相关的义务同时满足下列条件的，应当确认为预计负债：
1. 该义务是企业承担的现时义务；
2. 履行该义务很可能导致经济利益流出企业；
3. 该义务的金额能够可靠地计量。

**"很可能"的判断标准**：
- 基本确定：大于95%但小于100%
- 很可能：大于50%但小于或等于95%
- 可能：大于5%但小于或等于50%
- 极小可能：大于0但小于或等于5%

## 三、预计负债的计量

### （一）最佳估计数

预计负债应当按照履行相关现时义务所需支出的最佳估计数进行初始计量。

**最佳估计数的确定**：
1. **所需支出存在一个连续范围**：最佳估计数按照该范围内的中间值确定。
2. **所需支出不存在一个连续范围**：
   - 涉及单个项目：按照最可能发生金额确定。
   - 涉及多个项目：按照各种可能结果及相关概率计算确定（期望值法）。

### （二）考虑因素

1. **风险和不确定性**：在确定最佳估计数时应当考虑风险和不确定性。
2. **货币时间价值**：如果货币时间价值影响重大，预计负债应当按照相关支出的现值确定。
3. **未来事项**：有确凿证据表明未来事项将影响履行义务所需支出的，应当考虑未来事项的影响。

### （三）预期补偿

企业清偿预计负债所需支出全部或部分预期由第三方补偿的，补偿金额只有在基本确定能够收到时才能作为资产单独确认，且确认的补偿金额不应超过预计负债的账面价值。

```
# 确认预期补偿
借：其他应收款
  贷：营业外支出（冲减相关费用）
```

## 四、财务报表列示要求

### （一）资产负债表

- **预计负债**：根据"预计负债"科目期末余额填列，列示于非流动负债项目。

### （二）附注披露

企业应当在附注中披露：

**预计负债**：
1. 预计负债的种类、形成原因；
2. 各类预计负债的期初余额、本期增加额、本期减少额和期末余额；
3. 经济利益流出的不确定性说明；
4. 预期补偿的金额及不确定性说明。

**或有负债**（不确认但需披露）：
1. 或有负债的种类及其形成原因；
2. 经济利益流出不确定性的说明；
3. 预计产生的财务影响（如无法作出，应说明原因）。

**或有资产**（不确认但可能需披露）：
1. 或有资产的种类及其形成原因；
2. 预期产生的财务影响（如无法作出，应说明原因）。
