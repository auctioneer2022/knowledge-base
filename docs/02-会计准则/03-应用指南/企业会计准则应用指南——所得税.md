# 企业会计准则应用指南——所得税

> 对应准则：《企业会计准则第18号——所得税》（CAS 18）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了所得税的会计处理方法。企业应当采用**资产负债表债务法**核算所得税。

资产负债表债务法的核心逻辑：
1. 比较资产和负债的**账面价值**与其**计税基础**，确定**暂时性差异**；
2. 根据暂时性差异确认**递延所得税资产**或**递延所得税负债**；
3. 确认当期**所得税费用**（包括当期所得税和递延所得税）。

## 二、核心概念

### （一）账面价值

资产或负债的账面价值是指按照企业会计准则规定确定的金额，即资产负债表中列示的金额。

### （二）计税基础

#### 资产的计税基础

资产的计税基础是指企业收回资产账面价值过程中，按照税法规定可以税前扣除的金额。

- **公式**：资产的计税基础 = 未来可税前扣除的金额
- **举例**：
  - 固定资产的计税基础 = 原值 - 按税法规定已计提的累计折旧
  - 应收账款的计税基础 = 账面余额（税法不认可坏账准备，因此未来收回时可全额扣除的金额等于账面余额）

#### 负债的计税基础

负债的计税基础是指负债的账面价值减去未来期间按照税法规定可予税前扣除的金额后的余额。

- **公式**：负债的计税基础 = 账面价值 - 未来可税前扣除的金额
- **举例**：
  - 预计负债（产品质量保证）的计税基础 = 账面价值 - 未来实际发生时可税前扣除的金额 = 0
  - 预收账款的计税基础 = 账面价值 - 未来确认收入时不再纳税的金额 = 0

### （三）暂时性差异

暂时性差异是指资产或负债的账面价值与其计税基础之间的差额。

| 类型 | 定义 | 结果 |
|------|------|------|
| **应纳税暂时性差异** | 资产账面价值 > 计税基础，或负债账面价值 < 计税基础 | 确认递延所得税负债 |
| **可抵扣暂时性差异** | 资产账面价值 < 计税基础，或负债账面价值 > 计税基础 | 确认递延所得税资产 |

### （四）特殊项目的暂时性差异

#### 1. 未作为资产和负债确认的项目

某些交易或事项虽然按照会计准则规定未确认为资产或负债，但按照税法规定可以确定其计税基础的，该计税基础与其账面价值（零）之间的差额也构成暂时性差异。例如：
- **开办费**：会计上在发生时已费用化（账面价值为零），但税法允许在一定期间内摊销扣除。
- **研发费用加计扣除**：形成的无形资产按150%摊销，产生可抵扣暂时性差异。

#### 2. 可抵扣亏损

按照税法规定可以结转以后年度的未弥补亏损和税款抵减，视同可抵扣暂时性差异处理。

## 三、会计科目设置和主要账务处理

### （一）1801 递延所得税资产

- **核算内容**：核算企业根据所得税准则确认的可抵扣暂时性差异产生的所得税资产，以及按照税法规定可用以后年度抵减的未弥补亏损和税款抵减产生的所得税资产。
- **明细设置**：按暂时性差异的项目设置明细科目。
- **确认条件**：
  - 以**很可能取得**用来抵扣可抵扣暂时性差异的应纳税所得额为限。
  - 对于能够结转以后年度的可抵扣亏损和税款抵减，以很可能获得用来抵扣的可抵扣亏损和税款抵减的未来应纳税所得额为限确认递延所得税资产。

- **主要账务处理**：

  **确认递延所得税资产**：
  ```
  借：递延所得税资产
    贷：所得税费用——递延所得税费用
        其他综合收益（与其他综合收益相关的项目）
        资本公积——其他资本公积（与直接计入所有者权益的交易相关）
        商誉（与企业合并相关）
  ```

  **减记递延所得税资产**（当预期未来无法获得足够的应纳税所得额时）：
  ```
  借：所得税费用——递延所得税费用
      其他综合收益
      资本公积——其他资本公积
    贷：递延所得税资产
  ```

  **转回减记金额**（当预期未来能够获得足够的应纳税所得额时）：
  ```
  借：递延所得税资产
    贷：所得税费用——递延所得税费用
        其他综合收益
        资本公积——其他资本公积
  ```

### （二）1811 递延所得税负债

- **核算内容**：核算企业根据所得税准则确认的应纳税暂时性差异产生的所得税负债。
- **明细设置**：按暂时性差异的项目设置明细科目。

- **例外情况**（不确认递延所得税负债）：
  1. 商誉的初始确认；
  2. 既不影响会计利润也不影响应纳税所得额（或可抵扣亏损）的交易中产生的资产或负债的初始确认（但该交易不是企业合并）；
  3. 对于与子公司、联营企业及合营企业投资相关的应纳税暂时性差异，如果投资方能够控制暂时性差异转回的时间且该暂时性差异在可预见的未来很可能不会转回。

- **主要账务处理**：

  **确认递延所得税负债**：
  ```
  借：所得税费用——递延所得税费用
      其他综合收益（与其他综合收益相关的项目）
      资本公积——其他资本公积（与直接计入所有者权益的交易相关）
      商誉（与企业合并相关）
    贷：递延所得税负债
  ```

  **转回递延所得税负债**：
  ```
  借：递延所得税负债
    贷：所得税费用——递延所得税费用
        其他综合收益
        资本公积——其他资本公积
  ```

### （三）5701 所得税费用

- **核算内容**：核算企业确认的应从当期利润总额中扣除的所得税费用。
- **明细设置**：
  - "当期所得税费用"：核算当期应交所得税；
  - "递延所得税费用"：核算递延所得税的影响。

- **主要账务处理**：

  **确认当期所得税**：
  ```
  借：所得税费用——当期所得税费用
    贷：应交税费——应交所得税
  ```

  **确认递延所得税**：
  ```
  # 递延所得税费用增加
  借：所得税费用——递延所得税费用
    贷：递延所得税负债

  # 或
  借：递延所得税资产
    贷：所得税费用——递延所得税费用
  ```

  **期末结转**：
  ```
  借：本年利润
    贷：所得税费用
  ```

## 四、常见暂时性差异项目

### （一）资产类

| 项目 | 账面价值 | 计税基础 | 暂时性差异类型 |
|------|---------|---------|--------------|
| 固定资产（会计加速折旧 vs 税法直线法） | 较小 | 较大 | 应纳税 |
| 固定资产（会计直线法 vs 税法加速折旧） | 较大 | 较小 | 可抵扣 |
| 交易性金融资产（公允价值上升） | 较大 | 成本 | 应纳税 |
| 交易性金融资产（公允价值下降） | 较小 | 成本 | 可抵扣 |
| 应收账款（计提坏账准备） | 较小（净值） | 余额 | 可抵扣 |
| 存货（计提跌价准备） | 较小（净值） | 成本 | 可抵扣 |
| 其他权益工具投资（公允价值上升） | 较大 | 成本 | 应纳税 |
| 其他债权投资（公允价值上升） | 较大 | 成本 | 应纳税 |

### （二）负债类

| 项目 | 账面价值 | 计税基础 | 暂时性差异类型 |
|------|---------|---------|--------------|
| 预计负债（产品质量保证） | 有余额 | 0 | 可抵扣 |
| 预计负债（未决诉讼） | 有余额 | 0（实际支付时可扣除） | 可抵扣 |
| 合同负债/预收账款（税法收到时即纳税） | 有余额 | 0 | 可抵扣 |
| 应付职工薪酬（税法实际支付时扣除） | 有余额 | 0 | 可抵扣 |

## 五、所得税的确认和计量

### （一）当期所得税

当期所得税是指企业按照税法规定计算确定的针对当期发生的交易和事项应交纳的所得税金额。

```
当期应交所得税 = 应纳税所得额 × 适用税率
```

**应纳税所得额** = 利润总额 ± 纳税调整项目

常见的纳税调整项目包括：
- 国债利息收入（免税）：调减
- 超过标准的业务招待费、广告费等：调增
- 资产减值损失（税法不认可）：调增
- 公允价值变动损益（税法不认可）：调增或调减
- 研发费用加计扣除：调减

### （二）递延所得税

递延所得税按照暂时性差异和适用税率计算：

```
递延所得税资产 = 可抵扣暂时性差异 × 适用税率
递延所得税负债 = 应纳税暂时性差异 × 适用税率
```

**递延所得税费用** = 当期递延所得税负债增加额 - 当期递延所得税资产增加额

### （三）税率选择

递延所得税资产和递延所得税负债应当按照预期收回该资产或清偿该负债期间的适用税率计量。

### （四）不折现

递延所得税资产和递延所得税负债不应当折现。

## 六、特殊交易的所得税处理

### （一）企业合并

- 非同一控制下企业合并中取得的资产和负债按公允价值入账，与计税基础之间的差额确认为递延所得税资产或递延所得税负债，相应调整商誉。
- 合并中产生的可抵扣暂时性差异不符合递延所得税资产确认条件的，不确认递延所得税资产。购买日后12个月内取得新证据表明购买日已存在的情况满足确认条件的，追溯调整商誉。

### （二）直接计入所有者权益的交易

某些交易或事项产生的递延所得税影响直接计入所有者权益（其他综合收益或资本公积），而不影响当期所得税费用。例如：
- 其他权益工具投资公允价值变动；
- 其他债权投资公允价值变动；
- 自用房地产转为公允价值模式计量的投资性房地产时公允价值大于账面价值的差额。

### （三）股份支付

- 与股份支付相关的支出按照税法规定可在税前扣除的，如果账面价值与计税基础之间存在暂时性差异，应当确认递延所得税。

### （四）可抵扣亏损

- 对于能够结转以后年度的可抵扣亏损，以很可能获得用来抵扣的未来应纳税所得额为限，确认递延所得税资产。

## 七、所得税的列报

### （一）资产负债表

- **递延所得税资产**：在非流动资产中单独列示。
- **递延所得税负债**：在非流动负债中单独列示。
- **抵销条件**：同时满足以下条件时，递延所得税资产和递延所得税负债以抵销后的净额列示：
  1. 企业拥有以净额结算当期所得税资产和当期所得税负债的法定权利；
  2. 递延所得税资产和递延所得税负债是与同一税收征管部门对同一纳税主体征收的所得税相关，或者是对不同的纳税主体相关但在未来每一具有重要性的递延所得税资产和负债转回的期间内，涉及的纳税主体意图以净额结算当期所得税资产和负债或是同时取得资产、清偿负债。

### （二）利润表

- **所得税费用**：在利润表中单独列示。
- 应当在附注中披露所得税费用（收益）的主要组成部分。

### （三）附注披露

企业应当在附注中披露：
1. 所得税费用（收益）的主要组成部分；
2. 会计利润与所得税费用的关系说明；
3. 未确认递延所得税资产的可抵扣暂时性差异、可抵扣亏损的金额；
4. 递延所得税资产和递延所得税负债的明细；
5. 与直接计入所有者权益的交易相关的当期所得税和递延所得税。
