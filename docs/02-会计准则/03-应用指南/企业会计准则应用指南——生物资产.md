# 企业会计准则应用指南——生物资产

> 对应准则：《企业会计准则第5号——生物资产》（CAS 5）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了生物资产的确认、计量和相关信息的披露要求。生物资产是指有生命的动物和植物。生物资产分为三类：

1. **消耗性生物资产**：是指为出售而持有的、或在将来收获为农产品的生物资产，如生长中的大田作物、蔬菜、用材林以及存栏待售的牲畜等。
2. **生产性生物资产**：是指为产出农产品、提供劳务或出租等目的而持有的生物资产，如经济林、薪炭林、产畜和役畜等。
3. **公益性生物资产**：是指以防护、环境保护为主要目的的生物资产，如防风固沙林、水土保持林和水源涵养林等。

## 二、会计科目设置和主要账务处理

### （一）1601 消耗性生物资产

- **核算内容**：核算企业持有的消耗性生物资产成本。
- **明细设置**：按消耗性生物资产的种类、群别设置明细科目。
- **主要账务处理**：

  **外购的消耗性生物资产**：
  ```
  借：消耗性生物资产（购买价款+相关税费+运输费+保险费等可直接归属于购买该资产的支出）
    贷：银行存款/应付账款等
  ```

  **自行营造的消耗性生物资产**：
  ```
  # 发生造林费、抚育管护费等
  借：消耗性生物资产
    贷：原材料/应付职工薪酬/银行存款等

  # 郁闭成林后
  借：农产品
    贷：消耗性生物资产
  ```

### （二）1621 生产性生物资产

- **核算内容**：核算企业持有的生产性生物资产原价。
- **明细设置**：按生产性生物资产的种类设置明细科目。
- **主要账务处理**：

  **外购的生产性生物资产**：
  ```
  借：生产性生物资产（购买价款+相关税费+可直接归属于购买该资产的其他支出）
    贷：银行存款/应付账款等
  ```

  **自行营造或繁殖的生产性生物资产**：
  ```
  # 达到预定生产经营目的前发生的造林费、抚育管护费等
  借：生产性生物资产——未成熟生产性生物资产
    贷：原材料/应付职工薪酬/银行存款等

  # 达到预定生产经营目的时
  借：生产性生物资产——成熟生产性生物资产
    贷：生产性生物资产——未成熟生产性生物资产
  ```

### （三）1622 生产性生物资产累计折旧

- **核算内容**：核算企业生产性生物资产计提的累计折旧。
- **明细设置**：按生产性生物资产种类设置。
- **主要账务处理**：

  **按月计提折旧**：
  ```
  借：农业生产成本（产出农产品）
      管理费用（出租的生产性生物资产）
    贷：生产性生物资产累计折旧
  ```

  **折旧方法**：年限平均法、工作量法、产量法等。
  **折旧起始**：当月增加的生产性生物资产，当月不计提折旧，从下月起计提折旧；当月减少的，当月仍计提折旧，从下月起不计提折旧。

### （四）公益性生物资产

- **核算内容**：核算企业持有的公益性生物资产成本。
- **特点**：公益性生物资产不计提折旧，也不计提减值准备。
- **主要账务处理**：
  ```
  借：公益性生物资产
    贷：银行存款/应付职工薪酬等
  ```

### （五）生物资产减值准备

- **核算内容**：核算消耗性生物资产和生产性生物资产发生减值时计提的减值准备。
- **重要规则**：消耗性生物资产减值的影响因素已经消失的，减记金额应当予以恢复，并在原已计提的跌价准备金额内转回。生产性生物资产减值损失一经确认，不得转回。
- **主要账务处理**：
  ```
  借：资产减值损失
    贷：消耗性生物资产跌价准备/生产性生物资产减值准备
  ```

## 三、财务报表列示要求

### （一）资产负债表

- **消耗性生物资产**：根据"消耗性生物资产"科目期末余额减去"消耗性生物资产跌价准备"科目期末余额后的净额填列，列示于存货项目。
- **生产性生物资产**：根据"生产性生物资产"科目期末余额减去"生产性生物资产累计折旧"和"生产性生物资产减值准备"科目期末余额后的净额填列。

### （二）附注披露

企业应当在附注中披露：
1. 生物资产的分类及各类生物资产的账面价值；
2. 各类生物资产的增减变动情况；
3. 生产性生物资产的折旧方法、使用寿命和预计净残值；
4. 生物资产的减值情况；
5. 所有权或使用权受到限制的生物资产；
6. 与生物资产相关的风险及其管理政策。
