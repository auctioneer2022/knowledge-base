# 企业会计准则应用指南——合并财务报表

> 对应准则：《企业会计准则第33号——合并财务报表》（CAS 33，2014年修订）
> 应用指南版本：2014年

## 一、总体概述

本指南规范了合并财务报表的编制和列报要求。核心原则是：母公司应当将其全部子公司（包括母公司所控制的被投资单位可分割部分、结构化主体等）纳入合并范围，以母公司和子公司的财务报表为基础，根据其他有关资料，编制合并财务报表。

合并财务报表旨在反映企业集团整体的财务状况、经营成果和现金流量。

## 二、合并范围的确定

### （一）控制的定义

控制是指投资方拥有对被投资方的权力，通过参与被投资方的相关活动而享有可变回报，并且有能力运用对被投资方的权力影响其回报金额。

控制的三个基本要素：
1. **权力**：投资方拥有对被投资方的权力（主导相关活动的能力）；
2. **可变回报**：投资方因参与被投资方的活动而享有可变回报；
3. **权力与回报的联系**：投资方有能力运用权力影响其回报金额。

### （二）纳入合并范围的主体

- 母公司应当将其全部子公司纳入合并范围；
- 子公司是指被母公司控制的主体（包括企业、被投资单位中可分割的部分以及结构化主体）。

### （三）不纳入合并范围的特殊情况

- **投资性主体**：如果母公司是投资性主体，则应仅将为其投资活动提供相关服务的子公司纳入合并范围，其他子公司按公允价值计量且其变动计入当期损益。

### （四）实质性权利与保护性权利

- **实质性权利**：持有人有实际能力行使的权利（在判断控制时应考虑）；
- **保护性权利**：仅为了保护权利持有人利益但未赋予其主导相关活动能力的权利（不构成控制）。

## 三、合并财务报表的编制程序

### （一）编制前的准备工作

1. **统一会计政策和会计期间**：母公司应统一子公司的会计政策和会计期间。子公司采用的会计政策与母公司不一致的，应按母公司的会计政策对子公司财务报表进行调整。
2. **外币财务报表折算**：子公司记账本位币与母公司不同的，应按外币折算准则的规定折算为母公司的记账本位币。
3. **对子公司长期股权投资按权益法调整**：在合并工作底稿中，将母公司对子公司的长期股权投资由成本法调整为权益法。

### （二）合并资产负债表

合并资产负债表以母公司和子公司的个别资产负债表为基础，在抵销内部交易的影响后编制。

#### 需要抵销的项目

**1. 母公司对子公司长期股权投资与子公司所有者权益的抵销**

```
借：股本/实收资本（子公司）
    资本公积（子公司）
    其他综合收益（子公司）
    盈余公积（子公司）
    未分配利润——年末（子公司）
    商誉（借方差额）
  贷：长期股权投资（母公司对子公司）
      少数股东权益
      商誉（贷方差额，如为负商誉则贷记"营业外收入"）
```

**2. 内部债权债务的抵销**

```
借：应付账款
  贷：应收账款

借：应付票据
  贷：应收票据

借：应付债券
  贷：债权投资/其他债权投资
```

同时抵销相应的坏账准备：
```
借：应收账款——坏账准备
  贷：信用减值损失
```

**3. 内部商品交易的抵销**

```
# 抵销内部销售收入和成本
借：营业收入（内部销售收入金额）
  贷：营业成本（内部销售成本金额）

# 抵销期末存货中包含的未实现内部销售利润
借：营业成本
  贷：存货
```

**4. 内部固定资产交易的抵销**

```
# 抵销内部交易固定资产的未实现利润
借：营业收入（内部销售收入）
  贷：营业成本（内部销售成本）
      固定资产——原价（未实现利润）

# 抵销多提的折旧
借：固定资产——累计折旧
  贷：管理费用等
```

**5. 内部投资收益（利息收入与利息支出）的抵销**

```
借：投资收益
  贷：财务费用
```

**6. 内部现金股利的抵销**

```
借：投资收益（母公司确认的投资收益）
  贷：利润分配——应付现金股利
```

### （三）合并利润表

合并利润表以母公司和子公司的个别利润表为基础，在抵销内部交易的影响后编制。

**需要抵销的项目**：
1. 内部营业收入与营业成本；
2. 内部投资收益与利息收入/支出；
3. 内部资产减值损失；
4. 内部交易产生的未实现损益。

**少数股东损益**：子公司当期净损益中属于少数股东的份额，在合并利润表中"净利润"项目下单独列示为"少数股东损益"。

### （四）合并现金流量表

合并现金流量表以母公司和子公司的个别现金流量表为基础，在抵销内部交易的影响后编制。

**需要抵销的项目**：
1. 母公司与子公司、子公司相互之间当期以现金投资或收购股权所支付的现金与收到的现金；
2. 母公司取得子公司收到的现金净额（或支付的现金净额）；
3. 母公司与子公司、子公司相互之间当期取得投资收益收到的现金与分配股利、利润或偿付利息支付的现金；
4. 母公司与子公司、子公司相互之间以现金结算债权与债务所产生的现金；
5. 母公司与子公司、子公司相互之间当期销售商品所产生的现金；
6. 母公司与子公司、子公司相互之间处置固定资产、无形资产和其他长期资产所产生的现金。

### （五）合并所有者权益变动表

合并所有者权益变动表以母公司和子公司的个别所有者权益变动表为基础，在抵销内部交易的影响后编制。

**需要抵销的项目**：
1. 母公司对子公司的投资与子公司所有者权益中归属于母公司的份额；
2. 母公司对子公司的投资收益与子公司的利润分配；
3. 母公司与子公司、子公司相互之间发生的其他内部交易对所有者权益的影响。

**少数股东权益**：在合并所有者权益变动表中单独列示"少数股东权益"栏。

## 四、特殊交易的合并处理

### （一）追加投资取得子公司

- **非同一控制下**：购买日之前持有的股权按购买日公允价值重新计量，差额计入当期损益；原计入其他综合收益的金额转入当期损益。
- **同一控制下**：视同合并后形成的报告主体自最终控制方开始实施控制时一直存在。

### （二）处置子公司股权

**不丧失控制权**：作为权益性交易处理，处置价款与处置投资对应的享有子公司自购买日持续计算的净资产份额之间的差额调整资本公积（资本溢价），不足冲减的调整留存收益。

**丧失控制权**：
1. 终止确认子公司的资产、负债和少数股东权益；
2. 剩余股权按公允价值重新计量；
3. 处置价款与剩余股权公允价值之和，减去按原持股比例计算应享有原有子公司自购买日开始持续计算的净资产的份额与商誉之和的差额，计入丧失控制权当期的投资收益；
4. 与原子公司股权投资相关的其他综合收益，在丧失控制权时转入当期损益。

### （三）分步实现企业合并

通过多次交易分步实现非同一控制下企业合并的：
- 购买日之前持有的股权按公允价值重新计量；
- 公允价值与账面价值的差额计入当期损益；
- 原计入其他综合收益的金额转入当期损益。

### （四）"一揽子交易"的判断

多项交易的条款、条件以及经济影响符合以下一种或多种情况的，通常表明应将多次交易作为"一揽子交易"进行会计处理：
1. 这些交易是同时或者在考虑了彼此影响的情况下订立的；
2. 这些交易整体才能达成一项完整的商业结果；
3. 一项交易的发生取决于其他至少一项交易的发生；
4. 一项交易单独看是不经济的，但是和其他交易一并考虑时是经济的。

## 五、合并财务报表的格式

### （一）合并资产负债表

合并资产负债表的格式与母公司个别资产负债表基本一致，但增加以下项目：
- **商誉**：非同一控制下企业合并形成的商誉。
- **少数股东权益**：在所有者权益项目中单独列示。

### （二）合并利润表

合并利润表的格式与母公司个别利润表基本一致，但增加以下项目：
- **少数股东损益**：在"净利润"项目下单独列示。

### （三）合并现金流量表

格式与个别现金流量表一致。

### （四）合并所有者权益变动表

在个别所有者权益变动表的基础上增加"少数股东权益"栏。

## 六、附注披露要求

合并财务报表附注应披露：
1. 子公司的清单及基本情况；
2. 合并范围的变化（新增或减少子公司）；
3. 为纳入合并范围的结构化主体提供的财务支持；
4. 投资性主体的判断依据及未纳入合并范围的子公司信息；
5. 对控制判断有重大影响的判断和假设；
6. 与纳入合并范围的非全资子公司相关的信息（包括少数股东权益、少数股东损益等）。
