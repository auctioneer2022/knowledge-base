# 企业会计准则应用指南——保险合同

> 对应准则：《企业会计准则第25号——保险合同》（CAS 25）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了保险合同的分类、保费收入确认、保险合同准备金计量、理赔与给付以及分出业务与分入业务的会计处理。

**保险合同的定义**：保险合同是指保险人与投保人约定保险权利义务关系的协议。保险合同的核心特征是保险风险转移，即投保人将保险风险转移给保险人。

**保险风险显著性测试**：
保险合同与非保险合同的区分取决于保险风险的显著性。如果保险人在合同中承担了显著的保险风险，则该合同为保险合同；否则为投资合同或其他合同。

## 二、会计科目设置和主要账务处理

### （一）2601 保险合同准备金

- **核算内容**：核算保险人为履行保险合同义务而提取的各项准备金。
- **明细设置**：
  1. **未到期责任准备金**：核算非寿险保险人为尚未到期的保险责任提取的准备金。
  2. **未决赔款准备金**：核算保险人为已发生但尚未结案的赔案提取的准备金。
  3. **寿险责任准备金**：核算寿险保险人为尚未终止的寿险责任提取的准备金。
  4. **长期健康险责任准备金**：核算保险人为尚未终止的长期健康险责任提取的准备金。

- **主要账务处理**：

  **非寿险业务**：
  ```
  # 提取未到期责任准备金
  借：提取未到期责任准备金
    贷：未到期责任准备金

  # 提取未决赔款准备金
  借：提取未决赔款准备金
    贷：未决赔款准备金

  # 冲回多提的准备金
  借：未到期责任准备金/未决赔款准备金
    贷：提取未到期责任准备金/提取未决赔款准备金
  ```

  **寿险业务**：
  ```
  # 提取寿险责任准备金
  借：提取保险责任准备金
    贷：寿险责任准备金

  # 提取长期健康险责任准备金
  借：提取保险责任准备金
    贷：长期健康险责任准备金

  # 冲回多提的准备金
  借：寿险责任准备金/长期健康险责任准备金
    贷：提取保险责任准备金
  ```

### （二）保费收入

- **核算内容**：核算保险人确认的保费收入。
- **明细设置**：按险种设置明细科目。
- **主要账务处理**：
  ```
  # 确认保费收入
  借：银行存款/应收账款/应收保费等
    贷：保费收入

  # 发生退保
  借：保费收入
    贷：银行存款/应付账款等
  ```

### （三）赔付支出

- **核算内容**：核算保险人实际支付的赔款及给付。
- **明细设置**：按险种设置明细科目。
- **主要账务处理**：
  ```
  # 支付赔款
  借：赔付支出
    贷：银行存款/现金等

  # 冲回多提的未决赔款准备金
  借：未决赔款准备金
    贷：赔付支出
  ```

### （四）提取保险责任准备金

- **核算内容**：核算保险人为履行保险合同义务而提取的各项准备金所形成的费用。

### （五）手续费及佣金收入

- **核算内容**：核算保险人向投保人收取的手续费及佣金。

## 三、保险合同准备金计量

### （一）充足性测试

保险人应当在资产负债表日对保险合同准备金进行充足性测试。如果保险合同准备金的账面价值不足以满足未来现金流出的需要，应当补提准备金，计入当期损益。

```
# 补提准备金
借：资产减值损失/提取未到期责任准备金等
  贷：未到期责任准备金/未决赔款准备金等
```

### （二）计量方法

**未到期责任准备金**：
- 比例法：按照保费收入的一定比例提取。
- 风险分布法：按照风险分布情况提取。

**未决赔款准备金**：
- 逐案估计法：对每个赔案逐一估计。
- 案均赔款法：按照历史案均赔款水平估计。
- 准备金进展法：按照准备金进展规律估计。

**寿险责任准备金**：
- 未来法：按照未来现金流出的现值计量。
- 过去法：按照已收保费的积累值计量。

## 四、财务报表列示要求

### （一）资产负债表

- **保险合同准备金**：根据"保险合同准备金"科目期末余额填列，分为流动和非流动部分。
- **应收保费**：根据"应收账款——应收保费"科目期末余额填列。

### （二）利润表

- **保费收入**：根据"保费收入"科目本期发生额填列。
- **赔付支出**：根据"赔付支出"科目本期发生额填列。
- **提取保险责任准备金**：根据"提取保险责任准备金"科目本期发生额填列。

### （三）附注披露

企业应当在附注中披露：
1. 保险合同准备金的种类、金额及增减变动情况；
2. 保费收入的种类及金额；
3. 赔付支出的种类及金额；
4. 保险合同准备金充足性测试的结果；
5. 分出业务和分入业务的情况；
6. 与保险合同相关的重大假设和不确定性。
