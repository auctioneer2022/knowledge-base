# 企业会计准则应用指南——每股收益

> 对应准则：《企业会计准则第34号——每股收益》（CAS 34）
> 应用指南版本：2006年发布

## 一、总体概述

本指南规范了每股收益的计算和列报要求，适用于普通股或潜在普通股已公开交易的企业以及正处于公开市场发行过程中的企业。

每股收益是衡量企业盈利能力的重要指标，分为**基本每股收益**和**稀释每股收益**。企业应当在利润表中单独列示基本每股收益和稀释每股收益。

## 二、基本每股收益

### （一）计算公式

**基本每股收益 = 归属于普通股股东的当期净利润 ÷ 发行在外普通股的加权平均数**

### （二）分子的确定

- 归属于普通股股东的当期净利润 = 当期净利润 - 优先股股利
- 优先股股利包括当期宣告发放的累积优先股股利和当期宣告发放的非累积优先股股利；
- 在合并财务报表中，分子为归属于母公司普通股股东的合并净利润（扣除子公司少数股东损益和优先股股利后）。

### （三）分母的确定

**发行在外普通股加权平均数** = 期初发行在外普通股股数 + 当期新发行普通股股数 ×（已发行时间 ÷ 报告期时间）- 当期回购普通股股数 ×（已回购时间 ÷ 报告期时间）

- 新发行普通股包括：公开增发、配股、以现金发行新股等；
- 时间权重：通常按天数计算，也可按月份数合理近似。

## 三、稀释每股收益

### （一）基本原则

稀释每股收益是在基本每股收益的基础上，考虑所有**稀释性潜在普通股**的影响后计算的每股收益。潜在普通股包括可转换公司债券、认股权证、股份期权等。

**稀释每股收益 = 调整后的归属于普通股股东的净利润 ÷ 调整后的发行在外普通股加权平均数**

### （二）可转换公司债券的稀释效应

- **分子调整**：将可转换公司债券当期已确认为费用的利息（税后）加回归属于普通股股东的净利润；
- **分母调整**：假设可转换公司债券在期初（或发行日）全部转换为普通股，增加相应的普通股股数。

### （三）认股权证和股份期权的稀释效应

认股权证和股份期权是否具有稀释性，取决于其行权价格与当期普通股平均市场价格的比较：

- **行权价格 < 平均市场价格**：具有稀释性；
- **行权价格 ≥ 平均市场价格**：不具有稀释性。

具有稀释性时，采用**库存股法**计算增加的普通股股数：

**增加的普通股股数 = 拟行权时转换的普通股股数 - 按平均市场价格回购的普通股股数**

即：增加的股数 = 行权股数 ×（平均市场价格 - 行权价格）÷ 平均市场价格

### （四）反稀释原则

如果潜在普通股具有**反稀释效应**（即计算出的每股收益高于基本每股收益），则不应纳入稀释每股收益的计算。企业应当逐一测试各潜在普通股的稀释性，仅纳入具有稀释性的潜在普通股。

### （五）多项潜在普通股同时存在

当企业同时存在多种稀释性潜在普通股时，应当按照其稀释程度从大到小的顺序依次计入，直至稀释每股收益达到最小值。

## 四、重新计算

### （一）股票股利、拆细和并股

当企业发生下列事项时，应当按调整后的股数重新计算各列报期间的每股收益：
1. **派发股票股利**（送红股）；
2. **资本公积转增资本**；
3. **股票拆细**（如1股拆为2股）；
4. **并股**（如2股合并为1股）；
5. **配股**（含送股因素的配股需调整）。

上述事项改变了发行在外的普通股数量，但不影响企业的所有者权益总额，因此应追溯调整各列报期间的每股收益。

### （二）配股中的送股因素

配股价格低于市价时，配股中包含送股因素。计算每股收益时，应调整配股前的股数：

**调整系数 = 行权前一日公允价值 ÷ 理论除权价格**

理论除权价格 =（行权前一日公允价值 × 行权前股数 + 配股价格 × 配股数）÷（行权前股数 + 配股数）

## 五、列报与披露要求

### （一）利润表列示

- 企业应当在利润表中单独列示**基本每股收益**和**稀释每股收益**；
- 即使每股收益为负数（亏损），也应当列示。

### （二）附注披露

企业应当在附注中披露：
1. 基本每股收益和稀释每股收益的计算过程，包括分子和分母的确定；
2. 各期列报的基本每股收益和稀释每股收益的金额；
3. 资产负债表日至财务报告批准报出日之间发生的、导致发行在外普通股股数发生重大变化的事项；
4. 潜在普通股的种类、数量及其对每股收益的影响；
5. 当期发行或转换、到期或取消的潜在普通股情况。
