# 企业会计准则应用指南——石油天然气开采

> 对应准则：《企业会计准则第27号——石油天然气开采》（CAS 27）
> 应用指南版本：2006年发布

## 一、总体概述

本指南规范了石油天然气开采活动的会计处理，适用于石油天然气开采企业（以下简称"油气企业"）。石油天然气开采活动包括矿区权益的取得、油气勘探、油气开发和油气生产等阶段。

本准则涉及的会计处理方法主要包括**成果法**和**完全成本法**两种。成果法下，只有发现探明经济可采储量的勘探支出才予以资本化；完全成本法下，所有在矿区权益取得、勘探和开发过程中发生的支出均予以资本化。我国准则以成果法为基本方法。

## 二、会计科目设置和主要账务处理

### （一）1631 油气资产

- **核算内容**：核算油气企业持有的矿区权益和油气生产设施等油气资产的原价。
- **明细设置**：按矿区权益、油气生产设施等类别设置明细科目。
- **主要账务处理**：

  **取得矿区权益**：
  ```
  借：油气资产——矿区权益
    贷：银行存款/应付账款等
  ```

  **油气开发完成形成油气资产**：
  ```
  借：油气资产——油气设施
    贷：在建工程——油气开发支出
  ```

### （二）1632 累计折耗

- **核算内容**：核算油气资产累计计提的折耗。油气资产折耗类似于一般企业的固定资产折旧，但采用**产量法**（单位产量法）计提。
- **明细设置**：按油气资产类别设置。
- **折耗计算**：
  - 单位产量折耗额 = 油气资产账面价值 ×（当期产量 / 期末探明已开发经济可采储量）
- **主要账务处理**：
  ```
  借：生产成本
    贷：累计折耗
  ```

### （三）油气勘探支出（在"在建工程"下核算）

- **核算内容**：核算油气勘探过程中发生的各项支出，包括地质调查、地球物理勘探、钻探探井等支出。
- **明细设置**：按勘探项目设置明细科目。
- **主要账务处理**：

  **发生勘探支出时**：
  ```
  借：在建工程——油气勘探支出
    贷：银行存款/原材料/应付职工薪酬等
  ```

  **勘探结果处理**：
  ```
  # 发现探明经济可采储量（资本化）
  借：油气资产——矿区权益
    贷：在建工程——油气勘探支出

  # 未发现探明经济可采储量（费用化）
  借：勘探费用
    贷：在建工程——油气勘探支出
  ```

  **成果法下的特殊规定**：勘探井完钻后，如果未能发现探明经济可采储量，但满足以下条件之一的，可暂时作为资产保留（不超过一年）：（1）已发现足够数量的储量，需进一步钻探评价井；（2）正在钻探评价井且预计能发现经济可采储量。超过保留期限仍未证实的，应费用化处理。

### （四）油气开发支出（在"在建工程"下核算）

- **核算内容**：核算油气开发过程中发生的各项支出，包括钻探开发井、建造生产设施等支出。
- **明细设置**：按开发项目设置明细科目。
- **主要账务处理**：

  **发生开发支出时**：
  ```
  借：在建工程——油气开发支出
    贷：银行存款/原材料/应付职工薪酬等
  ```

  **开发完成转入油气资产**：
  ```
  借：油气资产
    贷：在建工程——油气开发支出
  ```

  油气开发支出均予以资本化，最终形成油气资产。

### （五）资产减值

油气资产发生减值的，按照资产减值准则处理：
```
借：资产减值损失
  贷：油气资产减值准备
```
油气资产减值损失一经确认，在以后会计期间不得转回。减值测试以矿区（或矿区组）为资产组进行。

### （六）弃置义务

油气企业在矿区开发过程中承担的环境恢复和设施拆除等弃置义务，满足预计负债确认条件的，应当确认预计负债并计入油气资产成本：
```
# 初始确认
借：油气资产
  贷：预计负债——弃置费用

# 后续期间（利息费用）
借：财务费用
  贷：预计负债——弃置费用
```

## 三、财务报表列示要求

### （一）资产负债表

- **油气资产**：根据"油气资产"科目期末余额减去"累计折耗"和"油气资产减值准备"科目期末余额后的净额填列。
- **在建工程**中包含的油气勘探支出和油气开发支出，按扣除减值准备后的净额列示。

### （二）附注披露

油气企业应当在附注中披露：
1. 矿区权益的取得方式、账面价值及变动情况；
2. 采用的勘探支出会计处理方法（成果法或完全成本法）；
3. 油气资产的类别、折耗方法和折耗率；
4. 探明经济可采储量的估计情况；
5. 油气资产的减值情况及减值测试方法；
6. 弃置义务的确认和计量情况。
