# 企业会计准则应用指南——外币折算

> 对应准则：《企业会计准则第19号——外币折算》（CAS 19）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了外币交易的会计处理、外币财务报表的折算以及汇兑差额的列报。

**记账本位币**：是指企业经营所处的主要经济环境中的货币。企业通常应选择人民币作为记账本位币。业务收支以人民币以外的货币为主的企业，可以按规定选定其中一种货币作为记账本位币，但编报的财务报表应当折算为人民币。

**确定记账本位币应考虑的因素**：
1. 该货币主要影响商品和劳务的销售价格；
2. 该货币主要影响商品和劳务所需人工、材料和其他费用；
3. 融资活动获得的货币以及保存从经营活动中收取款项所使用的货币。

## 二、会计科目设置和主要账务处理

### （一）外币交易的会计处理

企业发生外币交易时，应当将外币金额折算为记账本位币金额。

**折算汇率**：
1. 交易日即期汇率（通常采用）
2. 按照系统合理的方法确定的、与交易日即期汇率近似的汇率（如当期平均汇率或加权平均汇率）

**外币兑换业务**：
```
# 企业将美元兑换为人民币
借：银行存款——人民币户（银行买入价×美元金额）
    财务费用——汇兑差额（差额）
  贷：银行存款——美元户（即期汇率×美元金额）

# 企业用人民币购买美元
借：银行存款——美元户（即期汇率×美元金额）
    财务费用——汇兑差额（差额）
  贷：银行存款——人民币户（银行卖出价×美元金额）
```

**外币购销业务**：
```
# 进口商品（以美元结算）
借：库存商品/原材料（即期汇率×美元金额）
  贷：应付账款——美元（即期汇率×美元金额）

# 出口商品（以美元结算）
借：应收账款——美元（即期汇率×美元金额）
  贷：主营业务收入（即期汇率×美元金额）
```

**外币借款业务**：
```
# 取得外币借款
借：银行存款——美元户（即期汇率×美元金额）
  贷：短期借款/长期借款——美元（即期汇率×美元金额）
```

### （二）资产负债表日外币项目的折算

**货币性项目**：
- 包括：库存现金、银行存款、应收账款、其他应收款、长期应收款、短期借款、应付账款、其他应付款、长期借款、应付债券、长期应付款等。
- 折算方法：采用资产负债表日即期汇率折算。
- 汇兑差额处理：
  ```
  # 外币货币性资产产生汇兑收益
  借：应收账款/银行存款等
    贷：财务费用——汇兑差额

  # 外币货币性资产产生汇兑损失
  借：财务费用——汇兑差额
    贷：应收账款/银行存款等
  ```

**非货币性项目**：
1. **以历史成本计量的外币非货币性项目**：仍采用交易发生日的即期汇率折算，不改变其记账本位币金额。如存货、固定资产、无形资产等。
2. **以公允价值计量的外币非货币性项目**：采用公允价值确定日的即期汇率折算，折算后的记账本位币金额与原记账本位币金额的差额，作为公允价值变动（含汇率变动）处理。
   ```
   # 交易性金融资产（外币股票）
   借：交易性金融资产——公允价值变动
     贷：公允价值变动损益（含汇率变动影响）
   ```

### （三）外币专门借款的汇兑差额

在借款费用资本化期间内，外币专门借款本金及利息的汇兑差额，应当予以资本化，计入符合资本化条件的资产的成本。

```
# 资本化期间
借：在建工程
  贷：长期借款——美元（汇兑差额）

# 费用化期间
借：财务费用
  贷：长期借款——美元（汇兑差额）
```

## 三、境外经营财务报表折算

### （一）折算方法

境外经营的财务报表折算为人民币财务报表时，应当按照下列方法进行：

1. **资产负债表中的资产和负债项目**：采用资产负债表日的即期汇率折算。
2. **所有者权益项目**：除"未分配利润"项目外，其他项目采用发生时的即期汇率折算。
3. **利润表中的收入和费用项目**：采用交易发生日的即期汇率折算；也可以采用按照系统合理的方法确定的、与交易日即期汇率近似的汇率折算。

**折算差额**：上述折算产生的外币财务报表折算差额，在资产负债表中所有者权益项目下单独列示，计入其他综合收益。

### （二）处置境外经营

企业处置境外经营时，应当将资产负债表中所有者权益项目下列示的、与该境外经营相关的外币财务报表折算差额，自所有者权益项目转入处置当期损益。

```
# 处置境外经营时
借：其他综合收益——外币财务报表折算差额
  贷：投资收益（或相反分录）
```

## 四、财务报表列示要求

### （一）资产负债表

- 外币货币性项目按资产负债表日即期汇率折算后的金额列示。
- 外币财务报表折算差额在所有者权益中单独列示。

### （二）附注披露

企业应当在附注中披露：
1. 记账本位币及其选择依据；
2. 记账本位币发生变动的，应当说明理由；
3. 计入当期损益的汇兑差额；
4. 处置境外经营对外币财务报表折算差额的影响。
