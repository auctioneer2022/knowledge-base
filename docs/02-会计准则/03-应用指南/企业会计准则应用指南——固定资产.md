# 企业会计准则应用指南——固定资产

> 对应准则：《企业会计准则第4号——固定资产》（CAS 4）
> 应用指南版本：2006年发布，2024年更新

## 一、总体概述

本指南规范了固定资产的确认、计量和列报要求。固定资产是指同时具有下列特征的有形资产：
1. 为生产商品、提供劳务、出租或经营管理而持有的；
2. 使用寿命超过一个会计年度。

## 二、会计科目设置和主要账务处理

### （一）1601 固定资产

- **核算内容**：核算企业持有的固定资产原价。
- **明细设置**：按固定资产类别设置明细科目。
- **主要账务处理**：

  **外购固定资产**：
  ```
  借：固定资产（购买价款+相关税费+使固定资产达到预定可使用状态前所发生的可归属于该项资产的运输费、装卸费、安装费和专业人员服务费等）
    贷：银行存款/应付账款等
  ```

  **自行建造固定资产**：
  ```
  # 建设期间
  借：在建工程
    贷：工程物资/原材料/应付职工薪酬/银行存款等

  # 达到预定可使用状态时
  借：固定资产
    贷：在建工程
  ```

  **投资者投入固定资产**：
  ```
  借：固定资产（投资合同或协议约定的公允价值）
    贷：实收资本/股本
        资本公积——资本溢价/股本溢价（差额）
  ```

  **存在弃置义务的固定资产**：
  ```
  借：固定资产（弃置费用现值）
    贷：预计负债
  ```

### （二）1602 累计折旧

- **核算内容**：核算企业固定资产计提的累计折旧。
- **明细设置**：按固定资产类别设置。
- **主要账务处理**：

  **按月计提折旧**：
  ```
  借：制造费用（生产车间用）
      管理费用（管理部门用）
      销售费用（销售部门用）
      其他业务成本（出租的固定资产）
      在建工程（自建工程用）
      研发支出（研发用）
    贷：累计折旧
  ```

### （三）1603 固定资产减值准备

- **核算内容**：核算企业固定资产发生减值时计提的减值准备。
- **重要规则**：固定资产减值损失一经确认，在以后会计期间不得转回。
- **主要账务处理**：
  ```
  借：资产减值损失
    贷：固定资产减值准备
  ```

### （四）1604 在建工程

- **核算内容**：核算企业基建、更新改造等在建工程发生的支出。
- **明细设置**：按工程项目设置明细科目。
- **主要账务处理**：

  **发包的在建工程**：
  ```
  # 按工程进度结算工程价款
  借：在建工程
    贷：银行存款/应付账款等
  ```

  **自营的在建工程**：
  ```
  # 领用工程物资
  借：在建工程
    贷：工程物资

  # 领用原材料
  借：在建工程
    贷：原材料

  # 确认职工薪酬
  借：在建工程
    贷：应付职工薪酬

  # 工程达到预定可使用状态
  借：固定资产
    贷：在建工程
  ```

  **试运行销售**：
  试运行期间产出的产品或副产品对外销售的，应当按照收入准则和存货准则的规定，分别确认收入和成本，计入当期损益，不得将相关销售收入冲减在建工程成本。

### （五）1605 工程物资

- **核算内容**：核算企业为在建工程准备的各种物资的成本。
- **明细设置**：按物资品种设置。

### （六）1606 固定资产清理

- **核算内容**：核算企业因出售、报废、毁损、对外投资、非货币性资产交换、债务重组等原因转出的固定资产价值以及在清理过程中发生的费用等。
- **主要账务处理**：

  **固定资产转入清理**：
  ```
  借：固定资产清理
      累计折旧
      固定资产减值准备
    贷：固定资产
  ```

  **发生清理费用**：
  ```
  借：固定资产清理
    贷：银行存款
        应交税费——应交增值税（销项税额）
  ```

  **收回残料或变价收入**：
  ```
  借：银行存款/原材料等
    贷：固定资产清理
        应交税费——应交增值税（销项税额）
  ```

  **保险赔偿**：
  ```
  借：其他应收款/银行存款
    贷：固定资产清理
  ```

  **清理完毕结转净损益**：
  ```
  # 净损失（正常出售转让）
  借：资产处置损益
    贷：固定资产清理

  # 净收益（正常出售转让）
  借：固定资产清理
    贷：资产处置损益

  # 报废毁损损失
  借：营业外支出
    贷：固定资产清理

  # 报废毁损收益
  借：固定资产清理
    贷：营业外收入
  ```

## 三、固定资产的确认

### （一）确认条件

固定资产在同时满足下列条件时才能予以确认：
1. 与该固定资产有关的经济利益很可能流入企业；
2. 该固定资产的成本能够可靠地计量。

### （二）固定资产的各组成部分

固定资产的各组成部分具有不同使用寿命或者以不同方式为企业提供经济利益，适用不同折旧率或折旧方法的，应当分别将各组成部分确认为单项固定资产。

### （三）备品备件和维修设备

备品备件和维修设备通常确认为存货，但某些备品备件和维修设备需要与相关固定资产组合发挥效用，且金额较大、使用期限较长，则应当确认为固定资产。

## 四、固定资产的初始计量

### （一）外购固定资产

成本包括购买价款、相关税费、使固定资产达到预定可使用状态前所发生的可归属于该项资产的运输费、装卸费、安装费和专业人员服务费等。

- **延期付款**：购买固定资产的价款超过正常信用条件延期支付，实质上具有融资性质的，固定资产的成本以购买价款的现值为基础确定。实际支付的价款与购买价款的现值之间的差额，除按照借款费用准则应予资本化的以外，应当在信用期间内计入当期损益。

### （二）自行建造固定资产

成本由建造该项资产达到预定可使用状态前所发生的必要支出构成。

### （三）存在弃置义务的固定资产

对于特殊行业的特定固定资产（如核电站、石油开采设施），确定其初始成本时应当考虑预计弃置费用。弃置费用的现值计入固定资产成本，同时确认相应的预计负债。

## 五、固定资产的后续计量

### （一）折旧

#### 应计折旧额

应计折旧额 = 固定资产原价 - 预计净残值 - 已计提的减值准备累计金额

#### 折旧方法

企业可选用的折旧方法包括：
1. **年限平均法（直线法）**：年折旧额 = (原价 - 预计净残值) / 预计使用寿命
2. **工作量法**：单位工作量折旧额 = (原价 - 预计净残值) / 预计总工作量
3. **双倍余额递减法**：年折旧额 = 期初账面净值 × 2/预计使用寿命
4. **年数总和法**：年折旧额 = (原价 - 预计净残值) × 尚可使用年限/年数总和

#### 折旧的起始和终止

- **起始**：固定资产应当按月计提折旧。当月增加的固定资产，当月不计提折旧，从下月起计提折旧。
- **终止**：当月减少的固定资产，当月仍计提折旧，从下月起不计提折旧。
- **提前处置**：固定资产提足折旧后，不论能否继续使用，均不再计提折旧；提前报废的固定资产，也不再补提折旧。

#### 复核

企业至少应当于每年年度终了，对固定资产的使用寿命、预计净残值和折旧方法进行复核。如有差异，应当作为会计估计变更处理。

### （二）后续支出

- **资本化**：与固定资产有关的后续支出，满足固定资产确认条件的（即能够提升资产效能的），应当计入固定资产成本。如有被替换的部分，应扣除其账面价值。
- **费用化**：不满足固定资产确认条件的后续支出（如日常修理费），应当在发生时计入当期损益。

## 六、固定资产的处置

### （一）终止确认条件

固定资产满足下列条件之一的，应当予以终止确认：
1. 该固定资产处于处置状态；
2. 该固定资产预期通过使用或处置不能产生经济利益。

### （二）盘亏和盘盈

**盘亏**：
```
# 发现盘亏时
借：待处理财产损溢
    累计折旧
    固定资产减值准备
  贷：固定资产

# 经批准后
借：营业外支出——盘亏损失
  贷：待处理财产损溢
```

**盘盈**：作为前期差错处理，通过"以前年度损益调整"科目核算。
```
借：固定资产
  贷：以前年度损益调整
```

## 七、报表列示

### （一）资产负债表

- **固定资产**：根据"固定资产"科目期末余额减去"累计折旧"和"固定资产减值准备"科目期末余额后的净额填列。
- **在建工程**：根据"在建工程"科目期末余额减去"在建工程减值准备"科目期末余额后的净额填列。
- **固定资产清理**：根据"固定资产清理"科目期末余额填列。

### （二）附注披露

企业应当在附注中披露：
1. 固定资产的确认条件、分类、折旧方法、使用寿命和预计净残值；
2. 各类固定资产的期初和期末原价、累计折旧、减值准备及账面价值；
3. 当期计提的折旧额；
4. 当期处置的固定资产及处置损益；
5. 所有权或使用权受到限制的固定资产；
6. 暂时闲置的固定资产；
7. 持有待售的固定资产；
8. 未办妥产权证书的固定资产情况。
