# 企业会计准则应用指南——合营安排

> 对应准则：《企业会计准则第40号——合营安排》（CAS 40）
> 应用指南版本：2014年发布

## 一、总体概述

本指南规范了合营安排的分类、判断和会计处理。合营安排是指一项由两个或两个以上的参与方共同控制的安排。共同控制是指按照相关约定对某项安排所共有的控制，并且该安排的相关活动必须经过分享控制权的参与方一致同意后才能决策。

合营安排分为两类：**共同经营**和**合营企业**。两者的核心区别在于合营方是否对安排的净资产享有权利（合营企业），还是对安排的资产和负债分别享有权利和承担义务（共同经营）。

## 二、合营安排的分类

### （一）共同经营

共同经营是指合营方享有该安排相关资产且承担该安排相关负债的合营安排。在共同经营中，各合营方按照约定的比例分别享有资产、承担负债、分享收入、分担费用。

### （二）合营企业

合营企业是指合营方仅对该安排的净资产享有权利的合营安排。合营方通过持有合营企业的权益份额参与其利润分配，但不直接享有资产和承担负债。

## 三、合营安排的判断标准

### （一）单独主体

判断合营安排分类时，首先应当判断该安排是否通过**单独主体**（即具有单独可辨认财务架构的主体，包括法人主体和非法人主体）达成：
- 未通过单独主体达成的合营安排 → 共同经营；
- 通过单独主体达成的合营安排 → 需要进一步分析。

### （二）通过单独主体达成的安排

通过单独主体达成的合营安排，应当综合考虑以下因素进行分类判断：

1. **法律形式**：单独主体的法律形式是否赋予合营方对资产和负债的权利和义务。如果法律形式使合营方直接享有资产权利和承担负债义务，则为共同经营。
2. **合同条款**：合同安排中是否约定合营方对安排的资产和负债分别享有权利和承担义务。
3. **其他事实和情况**：
   - 合营方是否实质上承担了安排的全部或几乎全部产出义务；
   - 安排对合营方提供资金的依赖程度；
   - 安排的负债是否主要由合营方担保或提供资金支持。

如果综合以上因素，合营方实质上对安排的资产和负债分别享有权利和承担义务，则分类为共同经营；否则分类为合营企业。

## 四、共同经营的会计处理

### （一）基本原则

合营方在共同经营中应当确认与其利益份额相关的下列项目：
1. 单独所持有的资产，以及按份额共同持有的资产；
2. 单独所承担的负债，以及按份额共同承担的负债；
3. 出售其享有的共同经营产出份额所产生的收入；
4. 按份额确认共同经营因出售产出所产生的收入；
5. 单独所发生的费用，以及按份额共同发生的费用。

### （二）主要账务处理

共同经营的会计处理通过各合营方自身的会计科目进行，不通过"长期股权投资"核算：

**确认共同经营中的资产份额**：
```
借：固定资产/无形资产/存货等（按份额）
  贷：银行存款/应付账款等
```

**确认共同经营中的收入和费用份额**：
```
# 确认收入份额
借：应收账款/银行存款等
  贷：主营业务收入（按份额）

# 确认费用份额
借：生产成本/管理费用等（按份额）
  贷：应付账款/银行存款等
```

### （三）与共同经营之间的交易

- **合营方向共同经营投出或出售资产**：在共同经营将该资产出售给第三方之前，合营方仅确认归属于其他合营方的损益部分；
- **合营方自共同经营购买资产**：在将该资产出售给第三方之前，合营方不确认共同经营因该交易产生的损益中归属于自身的份额。

## 五、合营企业的会计处理

### （一）权益法核算

合营方对合营企业的投资应当按照**长期股权投资——权益法**进行核算：

**初始确认**：
```
借：长期股权投资——合营企业（投资成本）
  贷：银行存款等
```

**确认投资收益**：
```
# 合营企业实现净利润
借：长期股权投资——合营企业（损益调整）
  贷：投资收益

# 合营企业发生净亏损
借：投资收益
  贷：长期股权投资——合营企业（损益调整）
```

**宣告分派现金股利**：
```
借：应收股利
  贷：长期股权投资——合营企业（损益调整）
```

### （二）不纳入合并范围

合营企业对合营企业的投资采用权益法核算，不纳入合营方的合并财务报表范围（即不进行逐行合并）。

### （三）分类变更

当合营安排的分类发生变化时（如从共同经营变更为合营企业，或从合营企业变更为共同经营），合营方应当自分类变更之日起采用新的会计处理方法。变更前后的会计处理差异作为会计政策变更处理。

## 六、财务报表列示要求

### （一）资产负债表

- **共同经营**：合营方按份额确认的资产和负债，分别并入合营方自身资产负债表的相应项目列示；
- **合营企业**：合营方对合营企业的投资在"长期股权投资"项目中列示。

### （二）附注披露

企业应当在附注中披露：
1. 合营安排的名称、所在地、主要活动；
2. 企业在合营安排中的利益性质和持有份额；
3. 合营安排的分类（共同经营或合营企业）及分类依据；
4. 共同经营中确认的资产、负债、收入和费用的汇总信息；
5. 合营企业中按权益法确认的投资收益、长期股权投资账面价值等。
