# 企业会计准则应用指南——租赁

> 对应准则：《企业会计准则第21号——租赁》（CAS 21，2018年修订）
> 应用指南版本：2019年

## 一、总体概述

本指南规范了租赁的确认、计量和列报要求。新租赁准则的核心变化是取消了承租人关于融资租赁与经营租赁的分类，要求承租人对所有租赁（短期租赁和低价值资产租赁除外）确认使用权资产和租赁负债。

## 二、租赁的识别

### （一）租赁的定义

租赁是指在一定期间内，出租人将资产的使用权让与承租人以获取对价的合同。

**判断合同是否包含租赁的关键**：
1. 是否存在已识别资产（明确指定或隐性指定，且供应方无实质性替换权）；
2. 客户是否获得在使用期间内因使用已识别资产所产生的几乎全部经济利益；
3. 客户是否在使用期间内有权主导已识别资产的使用。

### （二）租赁的分拆与合并

- **合同分拆**：当合同同时包含租赁和非租赁组成部分时，承租人和出租人应将合同中的租赁组成部分与非租赁组成部分进行分拆。
- **实务简化**：承租人可以按照租赁资产的类别选择是否将租赁和非租赁组成部分合并处理。
- **合同合并**：满足特定条件时，两份或多份合同应合并为一份合同处理。

### （三）租赁期

租赁期是指承租人有权使用租赁资产且不可撤销的期间，包括合理确定承租人将行使续租选择权的期间以及不行使终止选择权的期间。

## 三、承租人的会计处理

### （一）会计科目设置

#### 1521（新增）使用权资产

- **核算内容**：核算承租人持有的使用权资产的原值。
- **明细设置**：按租赁资产类别设置明细科目。
- **初始计量**（入账价值包括）：
  1. 租赁负债的初始计量金额；
  2. 在租赁期开始日或之前支付的租赁付款额（扣除已享受的租赁激励）；
  3. 承租人发生的初始直接费用；
  4. 承租人为拆卸及移除租赁资产、复原租赁资产所在场地或将租赁资产恢复至租赁条款约定状态预计将发生的成本现值。

- **后续计量**：
  - 按成本模式计量，即：原值 - 累计折旧 - 减值准备 + 租赁负债重新计量调整。
  - **折旧**：在租赁期与使用权资产使用寿命两者孰短的期间内计提折旧。如果能合理确定租赁期届满时取得租赁资产所有权，则在使用寿命内计提。
  - 折旧方法参照固定资产准则（通常为直线法）。

#### 1522（新增）使用权资产累计折旧

- **核算内容**：核算使用权资产计提的累计折旧。
- **明细设置**：按租赁资产类别设置。

#### 1523（新增）使用权资产减值准备

- **核算内容**：核算使用权资产发生减值时计提的减值准备。
- **减值测试**：按资产减值准则（CAS 8）执行。

#### 2501（新增）租赁负债

- **核算内容**：核算承租人尚未支付的租赁付款额的现值。
- **明细设置**：
  - "租赁付款额"：记录尚未支付的固定付款额及实质固定付款额（扣除租赁激励）、可变租赁付款额（取决于指数或比率的）、购买选择权的行权价格（合理确定将行使时）、终止租赁的罚款（租赁期反映将行使终止权时）、承租人担保的余值。
  - "未确认融资费用"：记录租赁付款额与其现值之间的差额。

- **初始计量**：
  - 租赁付款额按租赁内含利率折现（无法确定时采用承租人增量借款利率）。

- **后续计量**：
  - 按固定的周期性利率（折现率）计算各期利息费用：借记"财务费用"科目，贷记"租赁负债——未确认融资费用"科目。
  - 支付租赁付款额时：借记"租赁负债——租赁付款额"科目，贷记"银行存款"等科目。
  - 租赁变更、续租选择权或购买选择权评估变化等导致租赁负债重新计量的，相应调整使用权资产。

### （二）主要账务处理

#### 1. 租赁期开始日

```
借：使用权资产                                    XXX
    租赁负债——未确认融资费用                       XXX
  贷：租赁负债——租赁付款额                          XXX
      银行存款（预付租金、初始直接费用等）            XXX
      预计负债（拆卸移除复原义务）                    XXX
```

#### 2. 租赁期内各期

```
# 确认利息费用
借：财务费用                                      XXX
  贷：租赁负债——未确认融资费用                       XXX

# 支付租金
借：租赁负债——租赁付款额                            XXX
  贷：银行存款                                     XXX

# 计提折旧
借：管理费用/销售费用/制造费用等                     XXX
  贷：使用权资产累计折旧                              XXX
```

#### 3. 租赁变更

租赁变更导致租赁范围缩小或租赁期缩短的：
```
借：租赁负债——租赁付款额
  贷：使用权资产
      租赁负债——未确认融资费用（差额）
```
部分终止或完全终止租赁的利得或损失计入当期损益。

#### 4. 租赁负债重新计量

因指数或比率变动、续租/购买选择权评估变化等导致租赁付款额变动时：
```
# 租赁付款额增加
借：使用权资产
  贷：租赁负债——租赁付款额

# 租赁付款额减少
借：租赁负债——租赁付款额
  贷：使用权资产
```

### （三）简化处理

#### 短期租赁

租赁期不超过12个月且不包含购买选择权的租赁，可以选择简化处理，不确认使用权资产和租赁负债，将租赁付款额在租赁期内按直线法（或其他系统合理方法）计入当期损益。

#### 低价值资产租赁

单项租赁资产为全新资产时价值较低的租赁（通常不超过人民币4万元），可以选择简化处理。

### （四）可变租赁付款额

- **取决于指数或比率的**（如与消费者价格指数挂钩的租金）：纳入租赁负债的初始计量。
- **其他可变租赁付款额**（如与销售额挂钩的租金）：在实际发生时计入当期损益。

## 四、出租人的会计处理

### （一）租赁分类

出租人应当在租赁开始日将租赁分为**融资租赁**和**经营租赁**。

**融资租赁的判断标准**（实质上转移了与租赁资产所有权有关的几乎全部风险和报酬）：
1. 租赁期届满时，租赁资产的所有权转移给承租人；
2. 承租人有购买租赁资产的选择权，且行权价格预计将远低于行使选择权时租赁资产的公允价值；
3. 资产的所有权虽然不转移，但租赁期占租赁资产使用寿命的大部分（通常≥75%）；
4. 租赁付款额的现值几乎相当于租赁开始日租赁资产公允价值（通常≥90%）；
5. 租赁资产性质特殊，如果不作较大改造，只有承租人才能使用。

### （二）融资租赁的会计处理

#### 5301（新增）应收融资租赁款

- **核算内容**：核算出租人融资租赁产生的租赁投资净额。
- **明细设置**：
  - "租赁收款额"：记录尚未收到的租赁付款额；
  - "未实现融资收益"：记录租赁收款额与租赁投资净额之间的差额。

#### 主要账务处理

**租赁期开始日**：
```
借：应收融资租赁款——租赁收款额                      XXX
  贷：融资租赁资产（账面价值）                        XXX
      应收融资租赁款——未实现融资收益                  XXX
      银行存款（初始直接费用）                        XXX
      资产处置损益（差额，也可能在借方）               XXX
```

**各期确认融资收益**：
```
借：应收融资租赁款——未实现融资收益                   XXX
  贷：租赁收入/其他业务收入                           XXX
```

**收到租赁付款额**：
```
借：银行存款
  贷：应收融资租赁款——租赁收款额
```

### （三）经营租赁的会计处理

经营租赁的租赁收款额在租赁期内按直线法或其他系统合理方法确认为租金收入。

```
# 确认租金收入
借：应收账款/银行存款
  贷：租赁收入/其他业务收入

# 计提折旧
借：其他业务成本
  贷：累计折旧
```

## 五、转租赁

转租情况下，中间出租人（原承租人）应：
- 对原租赁（转租赁的标的）：继续按承租人准则处理；
- 对转租赁：分类为融资租赁或经营租赁（参照原租赁的使用权资产而非标的资产进行分类）。

## 六、售后租回交易

### （一）判断标准

售后租回交易中的资产转让属于销售的：
- 卖方（承租人）：按原资产账面价值中与租回获得的使用权有关的部分计量使用权资产，仅就转让至买方的权利确认相关利得或损失。
- 买方（出租人）：按购买资产进行会计处理，按租赁进行会计处理。

售后租回交易中的资产转让不属于销售的：
- 卖方（承租人）：继续确认被转让资产，同时确认一项与转让收入等额的金融负债。
- 买方（出租人）：不确认被转让资产，但确认一项与转让收入等额的金融资产。

## 七、报表列示

### （一）承租人

**资产负债表**：
- "使用权资产"：单独列示于非流动资产或流动资产项目下。
- "租赁负债"：分别列示于非流动负债和流动负债（一年内到期的部分）。

**利润表**：
- 使用权资产折旧费：按功能归集于营业成本或期间费用。
- 租赁负债利息费用：计入"财务费用"。
- 短期租赁和低价值资产租赁费用：计入相关费用。
- 可变租赁付款额（未纳入租赁负债计量的）：计入相关费用。

**现金流量表**：
- 偿还租赁负债本金：筹资活动现金流出。
- 支付租赁负债利息：筹资活动或经营活动现金流出。
- 短期租赁、低价值资产租赁和可变租赁付款额：经营活动现金流出。

### （二）出租人

**资产负债表**：
- 应收融资租赁款：按摊余成本列示。
- 经营租赁资产：在固定资产等项目中列示。

**利润表**：
- 融资租赁收入：计入"租赁收入"或"其他业务收入"。
- 经营租赁收入：计入"租赁收入"或"其他业务收入"。

## 八、附注披露要求

### （一）承租人应披露

1. 各类使用权资产的增加额、折旧、减值和期末余额；
2. 租赁负债的利息费用；
3. 短期租赁和低价值资产租赁费用；
4. 可变租赁付款额；
5. 未来各期的租赁付款额到期分析；
6. 重大判断和估计的说明。

### （二）出租人应披露

1. 融资租赁的投资净额及未实现融资收益；
2. 经营租赁资产的账面价值；
3. 未来各期的租赁收款额到期分析。
